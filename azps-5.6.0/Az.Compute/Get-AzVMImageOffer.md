# Get-AzVMImageOffer

## SYNOPSIS
Возвращает типы предложений VMImage.

## СИНТАКСИС

```
Get-AzVMImageOffer -Location <String> -PublisherName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для **получения типов предложений VMImage возвращается cmdlet Get-AzVMImageOffer.**

## ПРИМЕРЫ

### Пример 1. Типы предложений для издателя
```
PS C:\> Get-AzVMImageOffer -Location "Central US" -PublisherName "Fabrikam"
```

Эта команда получает типы предложений для указанного издателя в центральной части США.

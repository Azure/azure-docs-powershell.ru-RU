# Get-AzRemoteRenderingAccount

## КРАТКИй обзор
Получить учетную запись удаленной отрисовки

## Максимальное

### ListParameterSet (по умолчанию)
```
Get-AzRemoteRenderingAccount [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Параметр "параметризованный" (по умолчанию)
```
Get-AzRemoteRenderingAccount -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzRemoteRenderingAccount -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Получение или Просмотр списка учетных записей удаленной отрисовки в определенной подписке и группе ресурсов.

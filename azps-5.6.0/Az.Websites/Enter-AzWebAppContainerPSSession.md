# Enter-AzWebAppContainerPSSession

## SYNOPSIS
Открытие удаленного сеанса PowerShell в контейнере Windows, указанном на определенном сайте или в слоте и заданной группе ресурсов

## СИНТАКСИС

### S1 (по умолчанию)
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [[-SlotName] <String>] [-ResourceGroupName] <String>
 [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Открывает удаленный сеанс PowerShell в контейнере Windows, указанном на определенном сайте или в слоте и заданной группе ресурсов

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Enter-AzWebAppContainerPSSession -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Эта команда открывает удаленный сеанс PowerShell в приложении contosoASP контейнера Windows

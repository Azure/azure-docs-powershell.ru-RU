# Remove-AzAutomationDscConfiguration

## SYNOPSIS
Удаляет конфигурации DSC из автоматизации.

## СИНТАКСИС

```
Remove-AzAutomationDscConfiguration [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
При этом из автоматизации Azure удаляются конфигурации **APS.AzAutomationDscConfiguration.**

## ПРИМЕРЫ

### Пример 1

Удаляет конфигурации DSC из автоматизации. (autogenerated)

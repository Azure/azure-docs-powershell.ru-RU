# Remove-AzDataLakeStoreAccount

## SYNOPSIS
Удаляет учетную запись Data Lake Store навсегда.

## СИНТАКСИС

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его учетной** записью Будет окончательно удалена учетная запись магазина Data Lake Store.

## ПРИМЕРЫ

### Пример 1. Удаление учетной записи Data Lake Store
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Эта команда удаляет учетную запись ContosoADL из магазина Data Lake Store.

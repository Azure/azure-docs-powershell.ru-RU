# Get-AzureVirtualNetworkGateway

## КРАТКИй обзор
Возвращает шлюз виртуальной сети.

## Максимальное

```
Get-AzureVirtualNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureVirtualNetworkGateway** получает шлюз виртуальной сети Azure.

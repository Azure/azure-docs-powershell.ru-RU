# Get-AzureRmRedisCachePatchSchedule

## КРАТКИй обзор
Возвращает расписание исправлений.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmRedisCachePatchSchedule** получает расписание исправлений кэша в Azure Redis Cache.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение расписания исправлений
```
PS C:\>Get-AzureRmRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Эта команда получает расписание исправлений из кэша с именем RedisCache06.

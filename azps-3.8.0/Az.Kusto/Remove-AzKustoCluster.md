# Remove-AzKustoCluster

## КРАТКИй обзор
Удаляет существующий кластер Kusto.

## Максимальное

### ByNameAndResourceGroup (по умолчанию)
```
Remove-AzKustoCluster [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzKustoCluster [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKustoCluster [-InputObject] <PSKustoCluster> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Удаляет существующий кластер Kusto.

## ИЛЛЮСТРИРУЮТ

### Пример 1. Удалите существующий кластер Kusto по имени

```
PS C:\> Remove-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster
```

Приведенная выше команда удаляет кластер Kusto с именем "mykustocluster" в группе ресурсов "testrg".

### Пример 2. Удаление существующего кластера Kusto с помощью трубопроводов

```
PS C:\> Get-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster | Remove-AzKustoCluster
```

В приведенной выше команде получается кластер Kusto с именем "mykustocluster" в группе ресурсов "testrg" с помощью `Get-AzKustoCluster` командлета, а затем `Remove-AzKustoCluster` продается результат, чтобы удалить его.

# Set-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Обновляет пул конечных адресов для шлюза приложения.

## СИНТАКСИС

```
Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-BackendIPAddresses <String[]>] [-BackendFqdns <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Set-AzApplicationGatewayBackendAddressPool** обновляет пул конечных адресов для шлюза приложения Azure.
Конечные адреса можно укандовать как IP-адреса, полное доменное имя (FQDN) или как ID конфигурации IP-адресов.

## ПРИМЕРЫ

### Пример 1. Настройка пула конечных адресов с помощью FQDNs
```
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "Pool02" -BackendFqdns "contoso1.com", "contoso2.com"
```

Первая команда получает шлюз приложения ApplicationGateway01 в группе ресурсов ResourceGroup01 и сохраняет его в $AppGw переменной.
Вторая команда обновляет пул конечных адресов шлюза приложения в $AppGw с помощью FQDNs.

### Пример 2. Настройка пула серверных адресов с помощью IP-адресов сервера серверов
```
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "Pool02" -BackendIPAddresses "10.10.10.10", "10.10.10.11"
```

Первая команда получает шлюз приложения ApplicationGateway01 в группе ресурсов ResourceGroup01 и сохраняет его в $AppGw переменной.
Вторая команда обновляет пул конечных адресов шлюза приложения в $AppGw с помощью IP-адресов.

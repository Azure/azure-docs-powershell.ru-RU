# Get-AzSqlServerDisasterRecoveryConfigurationActivity

## SYNOPSIS
Возвращает действие для конфигурации восстановления сервера базы данных.

## СИНТАКСИС

```
Get-AzSqlServerDisasterRecoveryConfigurationActivity [-ServerName] <String>
 -ServerDisasterRecoveryConfigurationName <String> [-OperationId <Guid>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для конфигурации восстановления сервера базы данных на сервере базы данных **Get-AzSqlServerDisasterRecoveryConfigurationActivity** возвращается действие для SQL сервера базы данных.

# Remove-AzureBatchComputeNode

## КРАТКИй обзор
Удаляет из пула вычислительные узлы.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### ID (по умолчанию)
```
Remove-AzureBatchComputeNode [-PoolId] <String> [-Ids] <String[]>
 [-DeallocationOption <ComputeNodeDeallocationOption>] [-ResizeTimeout <TimeSpan>] [-Force]
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### InputObject
```
Remove-AzureBatchComputeNode [[-ComputeNode] <PSComputeNode>]
 [-DeallocationOption <ComputeNodeDeallocationOption>] [-ResizeTimeout <TimeSpan>] [-Force]
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureBatchComputeNode** удаляет пакетные вычисляемые узлы Azure из пула.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление вычислительного узла
```
PS C:\>Remove-AzureBatchComputeNode -PoolId "Pool07" -Ids "tvm-2316545714_1-20150725t213220z" -DeallocationOption Terminate -ResizeTimeout ([TimeSpan]::FromMinutes(10)) -BatchContext $Context
```

Эта команда удаляет узел COMPUTE node с указанным ИДЕНТИФИКАТОРом из пула с ИД Pool07.
Команда задает параметр завершения освобождения.
Время ожидания изменения размера составляет 10 минут.

### Пример 2: удаление вычислительного узла с помощью конвейера
```
PS C:\>Get-AzureBatchComputeNode -PoolId "Pool07" -Id "tvm-2316545714_1-20150725t213220z" -BatchContext $Context | Remove-AzureBatchComputeNode -Force -BatchContext $Context
```

Эта команда получает вычислительный узел с указанным ИДЕНТИФИКАТОРом из пула с ИД Pool07 с помощью командлета Get-AzureBatchComputeNode.
Команда передает этот узел текущему командлету с помощью конвейера.
Текущий командлет удаляет вычисляемый узел.
Команда задает параметр *Force* .
Таким образом, команда не запрашивает подтверждение.

### Пример 3: удаление нескольких узлов
```
PS C:\>Remove-AzureBatchComputeNode -PoolId "Pool07" @("tvm-1783593343_28-20151117t214257z","tvm-1783593343_29-20151117t214257z") -Force -BatchContext $Context
```

Эта команда удаляет два вычислительных узла из пула с ИД Pool07.
Команда не запрашивает подтверждение.

### -BatchContext
Указывает экземпляр **BatchAccountContext** , используемый этим командлетом для взаимодействия со службой пакетной обработки.
Если для получения BatchAccountContext используется командлет Get-AzureRmBatchAccount, проверка подлинности Azure Active Directory будет использоваться при взаимодействии с пакетной службой. Чтобы использовать проверку подлинности с использованием общего ключа, используйте командлет Get-AzureRmBatchAccountKeys, чтобы получить объект BatchAccountContext с заполненными ключами доступа. При использовании проверки подлинности с использованием общего ключа по умолчанию используется первичный ключ доступа. Чтобы изменить ключ на использование, задайте свойство BatchAccountContext.

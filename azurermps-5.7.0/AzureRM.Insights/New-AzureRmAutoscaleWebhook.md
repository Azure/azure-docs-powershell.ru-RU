# New-AzureRmAutoscaleWebhook

## КРАТКИй обзор
Создание веб-перехватчика автомасштабирования.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmAutoscaleWebhook** создает интерфейс автомасштабирования.

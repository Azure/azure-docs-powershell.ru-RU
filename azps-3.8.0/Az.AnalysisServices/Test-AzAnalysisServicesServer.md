# Test-AzAnalysisServicesServer

## КРАТКИй обзор
Проверка существования экземпляра сервера служб Analysis Services

## Максимальное

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Test-AzAnalysisServicesServer проверяет наличие экземпляра сервера служб Analysis Services

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Эта команда пройдет проверку на наличие сервера с именем TestServer в группе resourcegroup testgroup

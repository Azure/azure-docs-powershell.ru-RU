# Remove-AzRouteTable

## КРАТКИй обзор
Удаляет таблицу маршрутов.

## Максимальное

```
Remove-AzRouteTable -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzRouteTable** удаляет таблицу маршрута Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление таблицы маршрутов
```
PS C:\>Remove-AzRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Эта команда удаляет таблицу маршрутов с именем RouteTable01 в группе ресурсов с именем ResourceGroup11.
Перед удалением таблицы командлет запрашивает подтверждение.

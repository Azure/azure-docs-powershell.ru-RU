# Get-AzApplicationGatewayHttpListenerCustomError

## SYNOPSIS
Возвращает пользовательские ошибки, выполненные http-прослушиванием шлюза приложения.

## СИНТАКСИС

```
Get-AzApplicationGatewayHttpListenerCustomError [-StatusCode <String>]
 -HttpListener <PSApplicationGatewayHttpListener> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzApplicationGateWayCustomError** пользовательская ошибка возвращается от http-слушателя шлюза приложения.

## ПРИМЕРЫ

### Пример 1. Возвращает настраиваемую ошибку в http-прослушивание
```powershell
PS C:\> $ce = Get-AzApplicationGatewayCustomError -HttpListener $listener01 -StatusCode HttpStatus502
```

Эта команда возвращает пользовательскую ошибку http-кода состояния 502 от программы http listener $listener 01.

### Пример 2. Список всех пользовательских ошибок http-прослушивание
```powershell
PS C:\> $ces = Get-AzApplicationGatewayCustomError -HttpListener $listener01
```

Эта команда возвращает список всех пользовательских ошибок http-$listener 01.

# Remove-AzDataFactoryPipeline

## КРАТКИй обзор
Удаляет конвейер из фабрики данных Azure.

## Максимальное

### ByFactoryName (по умолчанию)
```
Remove-AzDataFactoryPipeline [-Force] [-Name] <String> [-DataFactoryName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryPipeline [-Force] [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzDataFactoryPipeline** удаляет конвейер из фабрики данных Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление конвейера
```
PS C:\>Remove-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to remove pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Этот командлет удаляет конвейер с именем DPWikisample из фабрики данных с именем WikiADF.
Команда возвращает значение $True.

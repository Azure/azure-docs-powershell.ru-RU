# Get-AzureRmSqlServerUpgrade

## КРАТКИй обзор
Получает состояние обновления сервера базы данных Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlServerUpgrade -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSqlServerUpgrade** получает состояние обновления сервера базы данных Azure SQL.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение состояния обновления
```
PS C:\>Get-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Format-List
ResourceGroupName               : resourcegroup01
ServerName                      : server01
Status                          : Queued
```

Эта команда получает состояние обновления с сервера с именем Server01 в группе ресурсов с именем ResourceGroup01.

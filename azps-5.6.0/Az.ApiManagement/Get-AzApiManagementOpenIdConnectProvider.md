# Get-AzApiManagementOpenIdConnectProvider

## SYNOPSIS
Получает поставщики OpenID Connect.

## СИНТАКСИС

### GetAllOpenIdConnectProviders (по умолчанию)
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByOpenIdConnectProviderId
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-OpenIdConnectProviderId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByName
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для управления API Azure поставщики OpenID Connect получаются с использованием cmdlet **Get-AzApiManagementOpenIdProvider.**
ClientSecret не включается в сведения о результатах. Чтобы получить секрет клиента, используйте **Get-AzApiManagementOpenIdConnectProviderClientSecret.**

## ПРИМЕРЫ

### Пример 1. Получить всех поставщиков
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext
```

Эта команда получает все поставщики OpenID Connect для заданного контекста.

### Пример 2. Как получить поставщика с помощью ИД
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvider01"
```

Эта команда возвращает поставщика ID OICProvider01.

### Пример 3. Как получить поставщика по имени
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -Name "Contoso OpenID Connect Provider"
```

Эта команда возвращает поставщика Contoso OpenID Connect Provider.

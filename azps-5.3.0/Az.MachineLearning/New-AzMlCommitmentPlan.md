# New-AzMlCommitmentPlan

## SYNOPSIS
Создается новый план обязательств.

## СИНТАКСИС

```
New-AzMlCommitmentPlan -ResourceGroupName <String> -Location <String> -Name <String> -SkuName <String>
 -SkuTier <String> [-SkuCapacity <Int32>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Создает план обязательств по машинным обучению Azure в существующей группе ресурсов.
Если в группе ресурсов существует план обязательств с таким же именем, вызов будет выступать в качестве операции обновления, а существующий план обязательств будет перезаписан.

## ПРИМЕРЫ

### Пример 1. Создание нового плана обязательств
```
New-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName" -Location "South Central US" -SkuName DevTest -SkuTier Standard -SkuCapacity 1
```

Создание нового плана обязательств по машинным обучению Azure с именем MyCommitmentPlanName в группе MyResourceGroup и регионом Южный Центр США. В этом примере используется SKU DevTest/Standard, то есть ресурсы, предоставляемые планом обязательств, определяются ограничениями devTest/Standard. В этом примере SkuCapacity имеет значение 1, то есть стоимость плана будет на 1x стоить devTest, а ресурсы, которые содержит план, будут в 1x больше затрат, чем devTest.

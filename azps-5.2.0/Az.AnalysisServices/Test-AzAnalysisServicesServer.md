# Test-AzAnalysisServicesServer

## SYNOPSIS
Проверяет наличие экземпляра сервера Analysis Services.

## СИНТАКСИС

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Test-AzAnalysisServicesServer проверяет наличие экземпляра сервера Analysis Services.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Эта команда проверяет, есть ли сервер с именем testerver в тестовой группе ресурсов

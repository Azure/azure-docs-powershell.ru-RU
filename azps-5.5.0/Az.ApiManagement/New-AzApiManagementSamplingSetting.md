# New-AzApiManagementSamplingSetting

## SYNOPSIS
Создание нового параметра выборки для диагностики

## СИНТАКСИС

```
New-AzApiManagementSamplingSetting [-SamplingType <String>] [-SamplingPercentage <Double>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С помощью cmdlet **New-AzApiManagementSamplingSetting** создается новая настройка выборки для диагностики.

## ПРИМЕРЫ

### Пример 1. Создание простого параметра выборки
```powershell
PS C:\> New-AzApiManagementSamplingSetting -SamplingType fixed -Percentage 100

SamplingType Percentage
------------ ----------
fixed               100
```

Создание параметра выборки типа с ведением журнала `Fixed` для 100 % запросов и ответов.

### Пример 2

Создание нового параметра выборки для диагностики. (autogenerated)

```powershell
<!-- Aladdin Generated Example --> 
New-AzApiManagementSamplingSetting -SamplingPercentage 100 -SamplingType fixed
```

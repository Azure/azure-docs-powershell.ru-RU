# New-AzureRmServiceBusNamespaceAuthorizationRule

## КРАТКИй обзор
Создает новое правило авторизации для указанного пространства имен служебной шины.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmServiceBusNamespaceAuthorizationRule [-ResourceGroup] <String> -Namespace <String> -Name <String>
 [-Rights] <String[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmServiceBusNamespaceAuthorizationRule** создает новое правило авторизации для указанного пространства имен служебной шины.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> New-AzureRmServiceBusNamespaceAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -AuthorizationRuleName AuthoRule1 -Rights @("Listen","Send")
```

Создание `AuthoRule1` с правами на **прослушивание** и **отправку** для пространства имен `SB-Example1` .

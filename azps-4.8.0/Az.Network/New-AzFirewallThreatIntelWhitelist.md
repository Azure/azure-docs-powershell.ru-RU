# New-AzFirewallThreatIntelWhitelist

## КРАТКИй обзор
Создание нового добавлениеного каталога для угроз для брандмауэра Azure

## Максимальное

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzFirewallThreatIntelWhitelist** создает объект Threat Intel Добавление, который можно использовать при создании или настройке брандмауэра Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

В этом примере создается угроза Intel Добавление, содержащая полное доменное имя Добавление из двух записей и IP-адреса добавление с двумя записями.

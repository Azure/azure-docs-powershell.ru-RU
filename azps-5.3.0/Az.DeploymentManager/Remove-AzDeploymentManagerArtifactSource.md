# Remove-AzDeploymentManagerArtifactSource

## SYNOPSIS
Удаляет указанный источник артефакта.

## СИНТАКСИС

### Интерактивный (по умолчанию)
```
Remove-AzDeploymentManagerArtifactSource [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerArtifactSource [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerArtifactSource [-InputObject] <PSArtifactSource> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Remove-AzDeploymentManagerArtifactSource** можно удалить источник артефакта, указав его имя и имя группы ресурсов. Кроме того, вы можете предоставить объект ArtifactSource или ResourceId.

## ПРИМЕРЫ

### Пример 1. Удаление источника артефакта
### Пример 1
```powershell
PS C:\> Remove-AzDeploymentManagerArtifactSource -ResourceGroupName "ContosoResourceGroup" -Name "ContosoArtifactSource"
```

Эта команда удаляет источник артефакта ContosoArtifactSource в ContosoResourceGroup.

### Пример 2. Удаление источника артефакта с помощью идентификатора ресурса
```powershell
PS C:\> Remove-AzDeploymentManagerArtifactSource -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/artifactSources/ContosoArtifactSource"
```

Эта команда удаляет источник артефакта ContosoArtifactSource в ContosoResourceGroup.

### Пример 3. Удаление источника артефакта с помощью объекта
```powershell
PS C:\> Remove-AzDeploymentManagerArtifactSource -InputObject $artifactSourceObject
```

Эта команда удаляет источник артефакта, имя и группа ресурсов которого соответствуют свойствам Name и ResourceGroupName $artifactSourceObject соответственно.

# Get-AzureRmRelayHybridConnection

## КРАТКИй обзор
Возвращает описание указанного HybridConnection в пространстве имен ретрансляции.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmRelayHybridConnection** Возвращает описание указанной HybridConnection в пространстве имен ретрансляции.

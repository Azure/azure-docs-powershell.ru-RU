# Get-AzIntegrationAccountCertificate

## SYNOPSIS
Получает сертификаты учетной записи интеграции из группы ресурсов.

## СИНТАКСИС

```
Get-AzIntegrationAccountCertificate [-ResourceGroupName <String>] [-Name <String>] [-CertificateName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Get-AzIntegrationAccountCertificate** получает сертификаты учетных записей интеграции из группы ресурсов.
Укажите имя учетной записи интеграции, имя группы ресурсов и имя сертификата.
Этот модуль поддерживает динамические параметры.
Чтобы использовать динамический параметр, введите его в команду.
Чтобы найти имена динамических параметров, введите дефис (-) после имени cmdlet, а затем нажимая клавишу TAB для циклическиго перебора доступных параметров.
Если параметр обязательного шаблона не задан, будет выдан запрос на его значение.

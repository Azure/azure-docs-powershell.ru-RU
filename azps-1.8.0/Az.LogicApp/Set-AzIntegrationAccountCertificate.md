# Set-AzIntegrationAccountCertificate

## КРАТКИй обзор
Изменение сертификата учетной записи интеграции.

## Максимальное

```
Set-AzIntegrationAccountCertificate -ResourceGroupName <String> -Name <String> -CertificateName <String>
 [-KeyName <String>] [-KeyVersion <String>] [-KeyVaultId <String>] [-PublicCertificateFilePath <String>]
 [-Metadata <Object>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzIntegrationAccountCertificate** изменяет сертификат учетной записи интеграции.
Этот командлет возвращает объект, представляющий сертификат учетной записи интеграции.
Указание имени учетной записи интеграции, имени группы ресурсов и имени сертификата.
Этот модуль поддерживает динамические параметры.
Чтобы использовать динамический параметр, введите его в командную команду.
Чтобы найти имена динамических параметров, введите дефис (-) после имени командлета, а затем несколько раз нажмите клавишу TAB, чтобы просмотреть все доступные параметры.
Если вы не пропустите требуемый параметр шаблона, командлет предложит вам указать значение.

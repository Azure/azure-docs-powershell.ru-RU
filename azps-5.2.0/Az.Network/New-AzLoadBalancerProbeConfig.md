# New-AzLoadBalancerProbeConfig

## SYNOPSIS
Создается конфигурация формы для балансиров нагрузки.

## СИНТАКСИС

```
New-AzLoadBalancerProbeConfig -Name <String> [-Protocol <String>] -Port <Int32> -IntervalInSeconds <Int32>
 -ProbeCount <Int32> [-RequestPath <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для балансиров нагрузки Azure создается конфигурация новой службы **AzLoadBalancerProbeConfig.**

## ПРИМЕРЫ

### Пример 1. Создание конфигурации формы
```powershell
PS C:\>New-AzLoadBalancerProbeConfig -Name "MyProbe" -Protocol "http" -Port 80 -IntervalInSeconds 15 -ProbeCount 15
```

Эта команда создает конфигурацию с именем MyProbe с использованием http-протокола.
Новая флама будет подключена к службе с балансиза нагрузкой через порт 80.

### Пример 2

Создается конфигурация формы для балансиров нагрузки. (autogenerated)

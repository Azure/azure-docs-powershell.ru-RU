# Модуль AzureRM. LocationBasedServices
## Nописание
В этом разделе приводится Справка для командлетов служб, основанных на расположении Azure.

## Командлеты AzureRM. LocationBasedServices
### [Get-AzureRmLocationBasedServicesAccount](Get-AzureRmLocationBasedServicesAccount.md)
Возвращает учетную запись.

### [Get-AzureRmLocationBasedServicesAccountKey](Get-AzureRmLocationBasedServicesAccountKey.md)
Возвращает ключи API для учетной записи. Эти ключи являются механизмом проверки подлинности, используемым при последующих звонках на службы расположения Azure.

### [New-AzureRmLocationBasedServicesAccount](New-AzureRmLocationBasedServicesAccount.md)
Создание учетной записи служб на основе расположения.

### [New-AzureRmLocationBasedServicesAccountKey](New-AzureRmLocationBasedServicesAccountKey.md)
Повторное создание ключа учетной записи.

### [Remove-AzureRmLocationBasedServicesAccount](Remove-AzureRmLocationBasedServicesAccount.md)
Удаление учетной записи служб на основе расположения.

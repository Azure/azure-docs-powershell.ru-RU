# New-AzureRmSqlServerVirtualNetworkRule

## КРАТКИй обзор
Создание правила виртуальной сети для Azure SQL Server. 

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Создание правила виртуальной сети для Azure SQL Server. Правила виртуальной сети используются для подключения сервера Azure SQL Server к определенной виртуальной сети, чтобы ограничить доступ к серверу Azure SQL Server только в пределах виртуальной сети. 

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> $virtualNetworkRule = New-AzureRmSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Создание правила виртуальной сети для Azure SQL Server

# Set-AzDeploymentManagerServiceTopology

## SYNOPSIS
Обновляет топологию службы.

## СИНТАКСИС

```
Set-AzDeploymentManagerServiceTopology [-InputObject] <PSServiceTopologyResource>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Set-AzDeploymentManagerServiceTopology** обновляет топологию службы с помощью указанного объекта топологии службы.
Cmdlet возвращает обновленный объект топологии службы.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Set-AzDeploymentManagerService -InputObject $serviceTopologyObject
```

Эта команда обновляет топологию службы, имя и группа ресурсов которой соответствуют свойствам Name и ResourceGroupName $serviceTopologyObject соответственно.
Команда возвращает обновленный объект топологии службы.

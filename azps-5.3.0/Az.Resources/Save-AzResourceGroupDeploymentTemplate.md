# Save-AzResourceGroupDeploymentTemplate

## SYNOPSIS
Сохранение шаблона развертывания группы ресурсов в файл.

## СИНТАКСИС

```
Save-AzResourceGroupDeploymentTemplate -ResourceGroupName <String> -DeploymentName <String> [-Path <String>]
 [-Force] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для сохранения шаблона развертывания группы ресурсов в файл JSON будет сохраняться cmdlet **Save-AzResourceGroupDeploymentTemplate.**

## ПРИМЕРЫ

### Пример 1. Сохранение шаблона развертывания
```powershell
PS C:\>Save-AzResourceGroupDeploymentTemplate -DeploymentName "TestDeployment" -ResourceGroupName "TestGroup"
```

Эта команда получает шаблон развертывания из TestDeployment и сохраняет его как файл JSON в текущем каталоге.

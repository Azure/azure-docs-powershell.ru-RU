# Remove-AzAutomationConnectionType

## SYNOPSIS
Удаляет тип соединения автоматизации.

## СИНТАКСИС

```
Remove-AzAutomationConnectionType [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого из автоматизации** Azure удаляется тип подключения.
Все подключения, связанные с типом удаляемого подключения, становятся недоступными.
Удалите их, если только вы не создайте новый тип подключения, который соответствует следующим условиям: 
- Этот тип имеет то же имя, что и исходный тип подключения. 
- Этот тип имеет те же определения полей, что и исходный тип подключения.
Оно может иметь дополнительные поля.

## ПРИМЕРЫ

### Пример 1. Удаление типа подключения
```
PS C:\>Remove-AzAutomationConnectionType -AutomationAccountName "Contoso17" -Name "ContosoConnectionType" -ResourceGroupName "ResourceGroup01"
```

Эта команда удаляет тип подключения ContosoConnectionType в учетной записи автоматизации с именем Contoso17.

# Remove-AzApplicationGatewayRewriteRuleSet

## КРАТКИй обзор
Удаляет набор правил перезаписи из шлюза приложений.

## Максимальное

```
Remove-AzApplicationGatewayRewriteRuleSet -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **rewrite-AzApplicationGatewayRewriteRuleSet** удаляет набор правил перезаписи из шлюза приложений Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayRewriteRuleSet -ApplicationGateway $AppGw -Name "RuleSet02"
```

Первая команда получает шлюз приложения и сохраняет его в переменной $AppGw.
Вторая команда удаляет из шлюза приложений, сохраненного в $AppGw, наборы правил перезаписи с именем RuleSet02.

# Get-AzureRmApplicationGatewayCustomError

## КРАТКИй обзор
Получает пользовательские ошибки из шлюза приложений.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmApplicationGatewayCustomError [-StatusCode <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmApplicationGatewayCustomError** получает пользовательские ошибки из шлюза приложений.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение настраиваемой ошибки в шлюзе приложений
```powershell
PS C:\> $ce = Get-AzureRmApplicationGatewayCustomError -ApplicationGateway $appgw -StatusCode HttpStatus502
```

Эта команда возвращает и возвращает ошибку 502 кода состояния HTTP из $appgw шлюза приложений.

### Пример 2: получение списка всех настраиваемых ошибок в шлюзе приложений
```powershell
PS C:\> $ces = Get-AzureRmApplicationGatewayCustomError -ApplicationGateway $appgw
```

Эта команда возвращает список всех настраиваемых ошибок из $appgw шлюза приложений.

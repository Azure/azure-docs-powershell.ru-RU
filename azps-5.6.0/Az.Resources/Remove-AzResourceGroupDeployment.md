# Remove-AzResourceGroupDeployment

## SYNOPSIS
Удаляет развертывание группы ресурсов и все связанные с ней операции.

## СИНТАКСИС

### RemoveByResourceGroupName (По умолчанию)
```
Remove-AzResourceGroupDeployment [-ResourceGroupName] <String> [-Name] <String> [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceGroupDeploymentId
```
Remove-AzResourceGroupDeployment -Id <String> [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ

Из-за выполнения всех связанных операций развертывание группы ресурсов Azure удаляется с использованием **cmdlet Remove-AzResourceGroupDeployment.**

## ПРИМЕРЫ

### Пример 1. Удаление развертывания группы ресурсов с помощью ResourceId

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceId /subscriptions/{subId}/resourceGroups/testGroup/providers/Microsoft.Resources/deployments/testDeployment1

True
```

Эта команда удаляет развертывание группы ресурсов с ее полного ИД.
Успешное удаление возвращает результат "Истина".

### Пример 2. Удаляет развертывание группы ресурсов с помощью resourceGroupName и ResourceName

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceGroupName testGroup -Name testDeployment1

True
```

Эта команда удаляет развертывание группы ресурсов с предоставленными resourceGroupName и ResourceName.
Успешное удаление возвращает результат "Истина".

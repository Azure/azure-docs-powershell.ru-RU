# Get-AzCdnProfileSsoUrl

## SYNOPSIS
Получает URL-адрес единого для регистрации в профиле CDN.

## СИНТАКСИС

### ByFieldsParameterSet (по умолчанию)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzCdnProfileSsoUrl** получает URL-адрес единого входного адреса профиля сети доставки содержимого (CDN) Azure.
Этот URL-адрес позволяет пользователям подключаться к дополнительному порталу и использовать дополнительные возможности сети CDN.

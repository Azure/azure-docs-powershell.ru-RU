# Remove-AzureVirtualNetworkGatewayConnection

## КРАТКИй обзор
Удаляет подключение к шлюзу виртуальной сети.

## Максимальное

```
Remove-AzureVirtualNetworkGatewayConnection -GatewayId <String> -ConnectedEntityId <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureVirtualNetworkGatewayConnection** удаляет подключение к шлюзу виртуальной сети.

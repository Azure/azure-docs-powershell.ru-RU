# Get-AzureRmSqlServerDnsAlias

## КРАТКИй обзор
Получает или перечисляет DNS-псевдоним Azure SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlServerDnsAlias [-Name <String>] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Получите конкретный DNS-псевдоним для Azure SQL Server или список всех DNS-псевдонимов сервера.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> $serverDNSAliases = Get-AzureRmSqlServerDNSAlias -ServerName servername -ResourceGroupName rgname

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
rgname             servername   dnsaliasname2
```

Вывод списка всех DNS-псевдонимов сервера для определенного сервера

### Пример 2
```
PS C:\> $serverDNSAliases = Get-AzureRmSqlServerDNSAlias -DnsAliasName dnsaliasname -ServerName servername -ResourceGroupName rgname

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Получает псевдоним DNS сервера, указанный именем сервера и псевдонимом.

# Get-AzApplicationGatewayAutoscaleConfiguration

## КРАТКИй обзор
Получает конфигурацию автоматического масштабирования для шлюза приложений.

## Максимальное

```
Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzApplicationGatewayAutoscaleConfiguration** получает конфигурацию автомасштабирования шлюза приложений.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $autoscaleConfiguration = Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $autoscaleConfiguration.MinCapacity
```

Первая команда получает шлюз приложения и сохраняет его в $gw переменной.
Вторая команда извлекает конфигурацию автомасштабирования из шлюза приложений.

# Get-AzSnapshot

## SYNOPSIS
Свойства моментального снимка

## СИНТАКСИС

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Свойства **снимка получает cmdlet Get-AzSnapshot.**

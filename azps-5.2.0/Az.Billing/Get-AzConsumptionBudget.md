# Get-AzConsumptionBudget

## SYNOPSIS
Получить список бюджетов в подписке или группе ресурсов.

## СИНТАКСИС

```
Get-AzConsumptionBudget [-DefaultProfile <IAzureContextContainer>] [-ResourceGroupName <String>]
 [-Name <String>] [<CommonParameters>]
```

## ОПИСАНИЕ
С Get-AzConsumptionBudget или группы ресурсов возвращается список бюджетов в подписке или группе ресурсов.

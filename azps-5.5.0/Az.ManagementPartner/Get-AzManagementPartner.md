# Get-AzManagementPartner

## SYNOPSIS
Получает ИД Microsoft Partner Network (MPN) текущего пользователя, который является авториентом или основной службой.

## СИНТАКСИС

```
Get-AzManagementPartner [[-PartnerId] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Получает идентификацию текущего пользователя, который является авториентом или главой службы, в microsoft Partner Network(MPN).

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzManagementPartner
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Получить текущий ИД партнера по управлению

### Пример 2
```powershell
PS C:\> Get-AzManagementPartner -PartnerId 4977985
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Получите текущий ИД партнера по управлению с помощью его ид, если он не совпадает, он не будет работать

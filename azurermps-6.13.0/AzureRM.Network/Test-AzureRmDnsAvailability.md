# Test-AzureRmDnsAvailability

## КРАТКИй обзор
Проверяет, доступен ли доменное имя в зоне cloudapp.azure.com для использования.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Test-AzureRmDnsAvailability -DomainNameLabel <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Проверяет, доступен ли доменное имя в зоне cloudapp.azure.com для использования.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Проверка доступности contoso.cloudapp.azure.com для использования.
```
Test-AzureRmDnsAvailability -DomainNameLabel contoso -Location westus
```

# Модуль AZS. Commerce. admin
## Nописание
Предварительный просмотр выпусков модуля администратора AzureStack Commerce, с помощью которого можно просматривать общее использование данных в AzureStack. 

## Командлеты AZS. Commerce. администратора
### [Get-AzsSubscriberUsage](Get-AzsSubscriberUsage.md)
Получение данных об использовании в течение указанного интервала времени.

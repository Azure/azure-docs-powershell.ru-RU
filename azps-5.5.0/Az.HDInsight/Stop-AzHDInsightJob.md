# Stop-AzHDInsightJob

## SYNOPSIS
Остановка заданного задания в кластере.

## СИНТАКСИС

```
Stop-AzHDInsightJob [-ClusterName] <String> [-JobId] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **запуска указанного задания в** кластере Azure HDInsight запуск задания прекращается.

## ПРИМЕРЫ

### Пример 1. Остановка задания в указанном кластере
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential
PS C:\> Stop-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
            -JobId $jobId
```

Эта команда останавливает задание на кластере "ваш-hadoop-001".

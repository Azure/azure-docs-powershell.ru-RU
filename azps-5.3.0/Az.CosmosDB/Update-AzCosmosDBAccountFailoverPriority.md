## СИНТАКСИС

### ByNameParameterSet (по умолчанию)
```

 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Update-AzCosmosDBAccountFailoverPriority -ResourceGroupName rg -Name dbname -FailoverPolicy "region1, region2, region3"
```

FailoverPolicies обновлено с регионом1 как FailoverPriority 1, region2 как FailoverPriority 2 и region3 как FailoverPriority 3.

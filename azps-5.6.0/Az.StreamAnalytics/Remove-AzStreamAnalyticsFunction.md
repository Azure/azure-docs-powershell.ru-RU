# Remove-AzStreamAnalyticsFunction

## SYNOPSIS
Удаляет функцию из задания анализа Stream.

## СИНТАКСИС

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Remove-AzStreamAnalyticsFunction** удаляет функцию асинхронно из задания Azure Stream Analytics.

## ПРИМЕРЫ

### Пример 1. Удаление функции Stream Analytics
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Эта команда удаляет функцию ScoreTweet из задания StreamJob22.

# Модуль Az.Search
## Описание
В этой теме вы можете найти разделы справки по cmdlets Azure Search.

## Az.Search Cmdlets
### [Get-AzSearchAdminKeyPair](Get-AzSearchAdminKeyPair.md)
Получает пару ключей администратора для службы поиска Azure.

### [Get-AzSearchQueryKey](Get-AzSearchQueryKey.md)
Получает ключи запросов службы поиска Azure.

### [Get-AzSearchService](Get-AzSearchService.md)
Получает службу поиска Azure.

### [New-AzSearchAdminKey](New-AzSearchAdminKey.md)
Повторно сгенерирует ключ администратора службы поиска Azure.

### [New-AzSearchQueryKey](New-AzSearchQueryKey.md)
Создайте ключ запроса для службы поиска Azure.

### [New-AzSearchService](New-AzSearchService.md)
Создает службу поиска Azure.

### [Remove-AzSearchQueryKey](Remove-AzSearchQueryKey.md)
Удалите ключ запроса из службы поиска Azure.

### [Remove-AzSearchService](Remove-AzSearchService.md)
Удалите службу поиска Azure.

### [Set-AzSearchService](Set-AzSearchService.md)
Обновив службу поиска Azure.

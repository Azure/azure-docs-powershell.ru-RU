# Clear-AzSqlServerAdvancedThreatProtectionSetting

## SYNOPSIS
Удаляет дополнительные параметры защиты от угроз с сервера.

## СИНТАКСИС

```
Clear-AzSqlServerAdvancedThreatProtectionSetting [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Clear-AzSqlServerAdvancedThreatProtectionSetting удаляет дополнительные параметры защиты от угроз с SQL Azure.
Чтобы использовать этот cmdlet, укажите параметры ResourceGroupName и ServerName, чтобы определить сервер, с которого этот cmdlet удаляет параметры.

## ПРИМЕРЫ

### Пример 1. Удаление дополнительных параметров защиты от угроз в базе данных
```
PS C:\> Clear-AzSqlServerAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Эта команда удаляет дополнительные параметры защиты от угроз с сервера Server01.

# New-AzureRmPolicySetDefinition

## КРАТКИй обзор
Создание определения множества политик.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmPolicySetDefinition -Name <String> [-DisplayName <String>] [-Description <String>]
 [-Metadata <String>] -PolicyDefinition <String> [-Parameter <String>] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmPolicySetDefinition** создает определение политики.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Создание определения политики с помощью файла Set политики
```
PS C:\>New-AzureRmPolicySetDefinition -Name "VMPolicyDefinition" -PolicyDefinition C:\VMPolicySet.json
```

Эта команда создает определение политики с именем VMPolicyDefinition, которое включает определения политик, указанные в C:\VMPolicy.json.

# Set-AzHDInsightClusterSize

## SYNOPSIS
Задает количество узлов рабочих в указанном кластере.

## СИНТАКСИС

```
Set-AzHDInsightClusterSize [-ClusterName] <String> [-TargetInstanceCount] <Int32> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Set-AzHDInsightClusterSize** задает количество узлов рабочих узлов в указанном кластере Azure HDInsight.

## ПРИМЕРЫ

### Пример 1. Настройка размера указанного кластера
```
PS C:\>Set-AzHDInsightClusterSize -ClusterName "your-hadoop-001" -TargetInstanceCount 6
```

Эта команда задает размер кластера с именем "ваш-hadoop-001".

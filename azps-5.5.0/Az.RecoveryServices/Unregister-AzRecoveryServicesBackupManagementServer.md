# Unregister-AzRecoveryServicesBackupManagementServer

## SYNOPSIS
Отозвит сервер SCDPM или сервер резервного копирования из хранилища.

## СИНТАКСИС

```
Unregister-AzRecoveryServicesBackupManagementServer [-AzureRmBackupManagementServer] <BackupEngineBase>
 [-PassThru] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Из хранилища на сервере **Unregister-AzRecoveryServicesBackupManagementServer** Data Protection Manager System Center Data Protection Manager (SCDPM) или сервер резервного копирования Azure.
Этот cmdlet removes references to the servers that are unregistered from the vault.
Прежде чем можно будет отопустить контейнер, необходимо удалить все связанные с ним защищенные данные.
Прежде чем использовать текущий Set-AzRecoveryServicesVaultContext, задате контекст хранилища с помощью Set-AzRecoveryServicesVaultContext.

## ПРИМЕРЫ

### Пример 1. Ото для регистрации сервера SCDPM из хранилища
```powershell
PS C:\>$BMS = Get-AzRecoveryServicesBackupManagementServer -Name "dpmserver01.contoso.com"
PS C:\> Unregister-AzRecoveryServicesBackupManagementServer -AzBackupManagementServer $BMS
```

Первая команда получает сервер резервного копирования с именем dpmserver01.contoso.com, а затем сохраняет его в переменной $BMS архивации.
Вторая команда отозвит сервер SCDPM из хранилища.

### Пример 2

Отозвит сервер SCDPM или сервер резервного копирования из хранилища. (autogenerated)

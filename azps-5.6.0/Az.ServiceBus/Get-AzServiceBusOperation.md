# Get-AzServiceBusOperation

## SYNOPSIS
Список поддерживаемых операций ServiceBus

## СИНТАКСИС

```
Get-AzServiceBusOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
В этом из списка поддерживаемых операций ServiceBusOperation есть список **Get-AzServiceBusOperation.**

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Get-AzServiceBusOperation
```

Списки поддерживаемых операций ServiceBus

# Get-AzureRmLogicAppTriggerHistory

## КРАТКИй обзор
Получает историю триггеров в логическом приложении.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmLogicAppTriggerHistory -ResourceGroupName <String> -Name <String> -TriggerName <String>
 [-HistoryName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmLogicAppTriggerHistory** получает историю триггеров в приложении логики в компоненте "логические приложения".
Этот командлет возвращает объект **WorkflowTriggerHistory** .
Укажите логическое приложение, группу ресурсов и триггер.

Этот модуль поддерживает динамические параметры.
Чтобы использовать динамический параметр, введите его в командную команду.
Чтобы найти имена динамических параметров, введите дефис (-) после имени командлета, а затем несколько раз нажмите клавишу TAB, чтобы просмотреть все доступные параметры.
Если вы не пропустите требуемый параметр шаблона, командлет предложит вам указать значение.

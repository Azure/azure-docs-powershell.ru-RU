# Restart-AzureRmWebApp

## КРАТКИй обзор
Перезапускает веб-приложение Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### Состояний
```
Restart-AzureRmWebApp [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S2
```
Restart-AzureRmWebApp [-WebApp] <Site> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **restart-AzureRmWebApp** останавливается, а затем запускает веб-приложение Azure.
Если веб-приложение находится в остановленном состоянии, используйте командлет Start-AzureRmWebApp.

## ИЛЛЮСТРИРУЮТ

### Пример 1: перезагрузка веб-приложения
```
PS C:\>Restart-AzureRmWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Эта команда останавливает веб-приложение Azure с именем ContosoSite, которое принадлежит группе ресурсов Default-Web-WestUS, а затем перезапускает ее.

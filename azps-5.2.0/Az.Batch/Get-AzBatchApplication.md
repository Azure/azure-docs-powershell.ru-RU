# Get-AzBatchApplication

## SYNOPSIS
Получает сведения об указанном приложении.

## СИНТАКСИС

```
Get-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [[-ApplicationName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Чтобы получить сведения о приложении в учетной записи Azure Batch, можно получить сведения о нем с использованием **cmdlet Get-AzBatchApplication.**

## ПРИМЕРЫ

### Пример 1. Отображение приложений в учетной записи пакета
```
PS C:\>Get-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup"
ApplicationName AllowUpdates DisplayName

------------- ------------ ----------------------------

litware       False        Litware Advanced Reticulator
```

Эта команда отображает все приложения в учетной записи ContosoBatch.

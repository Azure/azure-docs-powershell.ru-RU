# Get-AzCosmosDBCassandraKeyspace

## КРАТКИй обзор
Возвращает CosmosDB Cassandra keyspace.

## Максимальное

### ByNameParameterSet (по умолчанию)
```
Get-AzCosmosDBCassandraKeyspace -ResourceGroupName <String> -AccountName <String> [-Name <String>] [-Detailed]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBCassandraKeyspace [-Name <String>] -InputObject <PSDatabaseAccount> [-Detailed]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzCosmosDBCassandraKeyspace** создает новый или обновляет существующий CosmosDB Cassandra keyspace.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzCosmosDBCassandraKeyspace -ResourceGroupName {resourceGroupName} -AccountName {accountName} -Name {name}

Name    Id   Resource
{name}  {id} {resourceObject}
```

Get-AzCosmosDBCassandraKeyspace получает свойства существующего CassandraKeyspace. Вы можете развернуть ресурс, чтобы получить _etag, _ts _rid свойств.

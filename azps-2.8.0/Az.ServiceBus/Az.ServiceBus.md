# AZ. ServiceBus Module
## Nописание
В этом разделе отображаются разделы справки по командлетам служебной шины Azure.

## Командлеты AZ. ServiceBus
### [Complete-AzServiceBusMigration](Complete-AzServiceBusMigration.md)
Командлеты устанавливают переход с Standard на Premium Namespace как завершенные и строки подключения в стандартном пространстве имен теперь указывают на пространство имен Premium.

### [Get-AzServiceBusAuthorizationRule](Get-AzServiceBusAuthorizationRule.md)
Возвращает описание указанного правила авторизации для заданного пространства имен, очереди или темы или псевдонима (конфигурации GeoDR). 

### [Get-AzServiceBusGeoDRConfiguration](Get-AzServiceBusGeoDRConfiguration.md)
Извлекает псевдоним (конфигурацию аварийного восстановления) для основного или дополнительного пространства имен.

### [Get-AzServiceBusKey](Get-AzServiceBusKey.md)
Получает первичные и дополнительные строки подключения для заданного пространства имен, очереди или темы или псевдонима (конфигурации GeoDR).

### [Get-AzServiceBusMigration](Get-AzServiceBusMigration.md)
Извлекает MigrationConfiguration для пространства имен.

### [Get-AzServiceBusNamespace](Get-AzServiceBusNamespace.md)
Получает описание пространства имен служебной шины в группе ресурсов.

### [Get-AzServiceBusOperation](Get-AzServiceBusOperation.md)
Список поддерживаемых операций ServiceBus

### [Get-AzServiceBusQueue](Get-AzServiceBusQueue.md)
Возвращает описание для указанной очереди служебной шины.

### [Get-AzServiceBusRule](Get-AzServiceBusRule.md)
Создает новое правило для конкретной подписки на тему. 

### [Get-AzServiceBusSubscription](Get-AzServiceBusSubscription.md)
Возвращает описание подписки для указанной темы.

### [Get-AzServiceBusTopic](Get-AzServiceBusTopic.md)
Возвращает описание для указанной темы служебной шины.

### [New-AzServiceBusAuthorizationRule](New-AzServiceBusAuthorizationRule.md)
Создает новое правило авторизации для указанной шины обслуживания, которое заданное пространство имен или очередь или тема.

### [New-AzServiceBusAuthorizationRuleSASToken](New-AzServiceBusAuthorizationRuleSASToken.md)
Создание правила авторизации tolen для Azure serviucebus для пространства имен/очереди/темы. 

### [New-AzServiceBusGeoDRConfiguration](New-AzServiceBusGeoDRConfiguration.md)
Создание нового псевдонима (конфигурация аварийного восстановления)

### [New-AzServiceBusKey](New-AzServiceBusKey.md)
Повторно создает первичные или дополнительные строки подключения для пространства имен служебной шины или очереди или темы.

### [New-AzServiceBusNamespace](New-AzServiceBusNamespace.md)
Создает новое пространство имен служебной шины.

### [New-AzServiceBusQueue](New-AzServiceBusQueue.md)
Создание очереди служебной шины в указанном пространстве имен служебной шины.

### [New-AzServiceBusRule](New-AzServiceBusRule.md)
Создает новое правило для конкретной подписки на тему. 

### [New-AzServiceBusSubscription](New-AzServiceBusSubscription.md)
Создает подписку на определенную тему служебной шины.

### [New-AzServiceBusTopic](New-AzServiceBusTopic.md)
Создает новую тему служебной шины в указанном пространстве имен служебной шины.

### [Remove-AzServiceBusAuthorizationRule](Remove-AzServiceBusAuthorizationRule.md)
Удаление правила авторизации пространства имен или очереди или темы из указанной группы ресурсов.

### [Remove-AzServiceBusGeoDRConfiguration](Remove-AzServiceBusGeoDRConfiguration.md)
Удаление псевдонима (конфигурация аварийного восстановления)

### [Remove-AzServiceBusMigration](Remove-AzServiceBusMigration.md)
Командлет удаляет конфигурацию миграции для стандартных и расширенных пространств имен.

### [Remove-AzServiceBusNamespace](Remove-AzServiceBusNamespace.md)
Удаляет пространство имен из указанной группы ресурсов. 

### [Remove-AzServiceBusQueue](Remove-AzServiceBusQueue.md)
Удаляет очередь из указанного пространства имен служебной шины.

### [Remove-AzServiceBusRule](Remove-AzServiceBusRule.md)
Удаляет указанное правило данной подписки.

### [Remove-AzServiceBusSubscription](Remove-AzServiceBusSubscription.md)
Удаляет подписку на раздел из указанного пространства имен служебной шины.

### [Remove-AzServiceBusTopic](Remove-AzServiceBusTopic.md)
Удаляет раздел из указанного пространства имен служебной шины.

### [Set-AzServiceBusAuthorizationRule](Set-AzServiceBusAuthorizationRule.md)
Обновляет указанное описание правила авторизации для заданного пространства имен служебной шины или очереди или темы.

### [Set-AzServiceBusGeoDRConfigurationBreakPair](Set-AzServiceBusGeoDRConfigurationBreakPair.md)
Эта операция отключает восстановление после аварии и останавливает репликацию изменений из основного в дополнительные пространства имен.

### [Set-AzServiceBusGeoDRConfigurationFailOver](Set-AzServiceBusGeoDRConfigurationFailOver.md)
Вызывает отработку отказа географического восстановления и повторно конфигурирует псевдоним, чтобы он указывал на дополнительное пространство имен.

### [Set-AzServiceBusNamespace](Set-AzServiceBusNamespace.md)
Обновляет описание существующего пространства имен служебной шины.

### [Set-AzServiceBusQueue](Set-AzServiceBusQueue.md)
Обновляет описание очереди служебной шины в указанном пространстве имен служебной шины.

### [Set-AzServiceBusRule](Set-AzServiceBusRule.md)
Обновляет указанное описание правила для данной подписки.

### [Set-AzServiceBusSubscription](Set-AzServiceBusSubscription.md)
Обновляет описание подписки для темы служебной шины в указанном пространстве имен служебной шины.

### [Set-AzServiceBusTopic](Set-AzServiceBusTopic.md)
Обновляет описание раздела служебной шины в указанном пространстве имен служебной шины.

### [Start-AzServiceBusMigration](Start-AzServiceBusMigration.md)
Создание новой конфигурации миграции и начало миграции сущностей из стандартного в расширенное пространство имен

### [Остановить-AzServiceBusMigration](Stop-AzServiceBusMigration.md)
{Заполнение кратких описаний}

### [Test-AzServiceBusName](Test-AzServiceBusName.md)
Проверяет доступность указанного имени пространства имен или псевдонима (имя конфигурации DR).

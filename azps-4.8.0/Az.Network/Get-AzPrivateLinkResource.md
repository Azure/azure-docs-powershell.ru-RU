# Get-AzPrivateLinkResource

## КРАТКИй обзор
Возвращает ресурс частной ссылки.

## Максимальное

### ByPrivateLinkResourceId (по умолчанию)
```
Get-AzPrivateLinkResource -PrivateLinkResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzPrivateLinkResource** извлекает все ссылочные ресурсы, принадлежащие PrivateLinkResource.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
Get-AzPrivateLinkResource -PrivateLinkResourceId '/subscriptions/00000000-0000-0000-0000-000000000000/resourceGroups/TestResourceGroup/providers/Microsoft.Sql/servers/mySql'
```

В этом примере список всех ресурсов частной ссылки nbelong на сервер SQL Server с именем mySql.

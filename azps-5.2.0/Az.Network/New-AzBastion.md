## ОПИСАНИЕ
Создает ресурс на высвечи. Для этого потребуется общедоступный IP-адрес и VirtualNetwork. В этой службе VirtualNetwork должна быть подсеть с именем AzureBastionSubnet. Для создания PUBIC IP-адреса необходимо использовать стандартный SKU.

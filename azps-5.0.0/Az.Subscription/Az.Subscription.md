# AZ. модуль подписки
## Nописание
В этом разделе приведены командлеты Azure PowerShell для подписки Azure в платформе диспетчера ресурсов Azure (ARM). Командлеты существуют в пространстве имен Microsoft. Azure. Commands. Subscription.

## AZ. командлеты по подписке
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Получение сведений о псевдониме подписки

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Создание нового псевдонима и подписки

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Удаление псевдонима подписки

### [Update-AzSubscription](Update-AzSubscription.md)
Обновляет подписку Azure

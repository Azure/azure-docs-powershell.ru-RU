# New-AzDataFactoryHub

## КРАТКИй обзор
Создает центр для фабрики данных Azure.

## Максимальное

### ByFactoryName (по умолчанию)
```
New-AzDataFactoryHub [-Name] <String> [-File] <String> [-Force] [-DataFactoryName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryHub [-Name] <String> [-File] <String> [-Force] [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzDataFactoryHub** создает Hub для фабрики данных Azure в указанной группе ресурсов Azure и в указанной фабрике данных с указанным определением файла.
После того как вы создадите центр, вы можете использовать его для хранения и управления связанными службами в группе, а также добавлять конвейеры в этот центр.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание концентратора
```
PS C:\>New-AzDataFactoryHub -ResourceGroupName "ADFResourceGroup" -DataFactoryName "ADFDataFactory" -Name "ContosoDataHub" -File "C:\Hub.json"
```

Эта команда создает концентратор с именем ContosoDataHub в группе ресурсов ADFResourceGroup и фабрика данных с именем ADFDataFactory.

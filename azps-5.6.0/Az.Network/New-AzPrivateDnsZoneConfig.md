# New-AzPrivateDnsZoneConfig

## SYNOPSIS
Создает конфигурацию зоны DNS для закрытой группы зон DNS.

## СИНТАКСИС

```
New-AzPrivateDnsZoneConfig -Name <String> [-PrivateDnsZoneId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet New-AzPrivateDnsZoneConfig** можно создать новый объект конфигурации зоны DNS, который будет установлен в группе зон DNS.

## ПРИМЕРЫ

### Создает конфигурацию зоны DNS.
```powershell
PS C:\> $dnsZone = New-AzPrivateDnsZone -ResourceGroupName "rg" -Name "test.vault.azure.com"
PS C:\> $config = New-AzPrivateDnsZoneConfig -Name "test-vault-azure-com" -PrivateDnsZoneId $dnsZone.ResourceId
```

В примере выше создается зона DNS, а затем создается конфигурация зоны DNS. `New-AzPrivateDnsZone` Командлет доказуется модулем Az.PrivateDns.

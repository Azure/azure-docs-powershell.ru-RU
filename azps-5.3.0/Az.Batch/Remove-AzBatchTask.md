# Remove-AzBatchTask

## SYNOPSIS
Удаляет пакетную задачу.

## СИНТАКСИС

### ИД
```
Remove-AzBatchTask [-JobId] <String> [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzBatchTask [-InputObject] <PSCloudTask> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его использованием удаляется** задача пакета Azure.
Этот cmdlet запросит подтверждение, если не указан параметр *Force.*

## ПРИМЕРЫ

### Пример 1. Удаление пакетной задачи по ИД
```
PS C:\>Remove-AzBatchTask -JobId "Job-000001" -Id "Task23" -BatchContext $Context
```

Эта команда удаляет задачу с ИД задачи23 для задания с заданием 0000001.
Команда запросит подтверждение.
Чтобы назначить контекст переменной $Context, используйте $Context **AzBatchAccountKey.**

### Пример 2. Удаление задачи с пакетом с помощью конвейера без подтверждения
```
PS C:\>Get-AzBatchTask -JobId "Job-000001" -Id "Task26" -BatchContext $Context | Remove-AzBatchTask -Force -BatchContext $Context
```

Эта команда получает задачу с пакетной задачей, которая имеет задачу "Код26" для задания с заданием 000001 с помощью командлета **Get-AzBatchTask.**
Эта команда передает эту задачу текущему командлету с помощью оператора конвейера.
Команда удалит задачу.
Эта команда определяет параметр *Force.*
Поэтому команда не будет запроса на подтверждение.

## PARAMETERS

### -BatchContext
Указывает экземпляр **BatchAccountContext,** который этот cmdlet использует для взаимодействия со службой Batch.
Если для Get-AzBatchAccount BatchAccountContext используется Get-AzBatchAccount, при взаимодействии со службой batchy будет использоваться проверка подлинности Azure Active Directory. Чтобы использовать проверку подлинности общего ключа, используйте Get-AzBatchAccountKey для получения объекта BatchAccountContext с заполненными ключами доступа. При использовании проверки подлинности общего ключа первичный ключ доступа используется по умолчанию. Чтобы изменить ключ, за установите свойство BatchAccountContext.KeyInUse.

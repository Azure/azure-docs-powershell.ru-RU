# New-AzNetworkWatcherNetworkConfigurationDiagnosticProfile

## SYNOPSIS
Создание нового объекта диагностики конфигурации сети. Этот объект используется для ограничения конфигурации сети во время сеанса диагностики с использованием указанных критериев.

## СИНТАКСИС

```
New-AzNetworkWatcherNetworkConfigurationDiagnosticProfile -Direction <String> -Protocol <String>
 -Source <String> -Destination <String> -DestinationPort <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
С помощью New-AzNetworkWatcherNetworkConfigurationDiagnosticProfile-профиля создается новый объект профиля диагностики. Этот объект используется для ограничения конфигурации сети во время сеанса диагностики конфигурации сети с использованием указанных критериев.

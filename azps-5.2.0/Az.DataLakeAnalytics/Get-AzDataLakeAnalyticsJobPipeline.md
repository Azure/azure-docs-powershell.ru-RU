# Get-AzDataLakeAnalyticsJobPipeline

## SYNOPSIS
Позволяет получить конвейер работы или конвейеры анализа для озера данных.

## СИНТАКСИС

### GetAllInAccount (по умолчанию)
```
Get-AzDataLakeAnalyticsJobPipeline [-Account] <String> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetBySpecificJobPipeline
```
Get-AzDataLakeAnalyticsJobPipeline [-Account] <String> [-PipelineId] <Guid> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Get-AzDataLakeAnalyticsJobPipeline** получает определенный конвейер azure Data Lake Analytics или список конвейеров.

## ПРИМЕРЫ

### Пример 1. Получить указанный канал
```
PS C:\>Get-AzDataLakeAnalyticsJobPipeline -Account "contosoadla" -PipelineId 83cb7ad2-3523-4b82-b909-d478b0d8aea3
```

Эта команда получает указанный канал с ид '83cb7ad2-3523-4b82-b909-d478b0d8aea3' в учетной записи contosoadla.

### Пример 2. Получить список всех каналов в учетной записи
```
PS C:\>Get-AzDataLakeAnalyticsJobPipeline -AccountName "contosoadla"
```

Эта команда получает список всех каналов в учетной записи contosoadla.

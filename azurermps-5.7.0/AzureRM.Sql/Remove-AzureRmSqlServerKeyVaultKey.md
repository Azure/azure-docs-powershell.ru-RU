# Remove-AzureRmSqlServerKeyVaultKey

## КРАТКИй обзор
Удаление ключа хранилища ключей из SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzureRmSqlServerKeyVaultKey удаляет ключ хранилища ключей из указанного SQL Server.

Обратите внимание, что разрешения SQL Server для хранилища ключа не изменяются.
Чтобы изменить разрешения, используйте Set-AzureRmKeyVaultAccessPolicy.

Обратите внимание, что этот командлет не вносит изменения в хранилище ключей.
Чтобы удалить раздел из хранилища ключей, используйте Remove-AzureKeyVaultKey.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление ключа хранилища ключей
```
PS C:\> Remove-AzureRmSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Эта команда удаляет ключ хранилища ключей с идентификатором " https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 " с указанного сервера.

# Remove-AzRedisCacheDiagnostic

## SYNOPSIS
Отключает диагностику в кэше Azure Redis.

## СИНТАКСИС

```
Remove-AzRedisCacheDiagnostic [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Remove-AzRedisCacheDiagnostic** отключает диагностику в кэше Azure Redis.

## ПРИМЕРЫ

### Пример 1. Отключение диагностики
```
PS C:\>Remove-AzRedisCacheDiagnostic -ResourceGroupName "ContosoResourceGroup" -Name "PeakCache" -Force
```

Эта команда отключает диагностику для указанного кэша Azure Redis.
Это отключает диагностику для всех кэшей Azure Redis в одном регионе для подписки.

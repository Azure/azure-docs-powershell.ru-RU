# Get-AzureLocalNetworkGateway

## КРАТКИй обзор
Возвращает шлюз локальной сети.

## Максимальное

```
Get-AzureLocalNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureLocalNetworkGateway** получает локальный сетевой шлюз.

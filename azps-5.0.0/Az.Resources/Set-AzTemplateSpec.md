## NОПИСАНИЕ
Изменяет спецификацию Templace. Если спецификация шаблона с указанным именем и (или) определенной версией еще не существует, она будет создана.

При изменении содержимого шаблона ARM версии спецификации содержимое может либо поступать из необработанной строки JSON (с использованием параметра **UpdateVersionByNameFromJsonParameterSet** ), либо из указанного JSON-файла (с использованием **UpdateVersionByNameFromJsonFileParameterSet** Set).

## ИЛЛЮСТРИРУЮТ

### Пример 1:
```powershell
PS C:\> $templateJson = @"
{
    "$schema": "http://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
    "contentVersion": "1.0.0.0",
    "parameters": {},
    "resources": []
}
"@
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v1.0' -Location 'West US' -TemplateJson $templateJson
```

Изменяет версию "v 1.0" спецификации шаблона с именем "myTemplateSpec". Указанная версия будет $templateJson в качестве содержимого шаблона ARM. Если спецификации корневого шаблона и (или) версии еще не существуют, они будут созданы.


**Пуск** 

* Шаблон ARM в примере является отсутствием операций, так как он не содержит реальных ресурсов.
* Расположение требуется только в том случае, если спецификация шаблона еще не существует

### Пример 2:
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v2.0' -Location 'West US' -TemplateFile 'myTemplateContent.json'
```

Изменяет версию "v 2.0" спецификации шаблона с именем "myTemplateSpec". Указанная версия будет содержать содержимое из локального файла "myTemplateContent.json" в качестве содержимого шаблона ARM для версии. Если спецификации корневого шаблона и (или) версии еще не существуют, они будут созданы.

**Примечание.** Расположение требуется только в том случае, если спецификация шаблона еще не существует

### Пример 3:
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec'  -Location 'West US' -Description 'My updated Template Spec'
```

Изменение описания спецификации шаблона с именем "myTemplateSpec" в группе ресурсов "myRG". Если спецификаций шаблона еще не существует, она будет создана.

**Примечание.** Расположение требуется только в том случае, если спецификация шаблона еще не существует

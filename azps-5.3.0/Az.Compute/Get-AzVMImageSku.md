# Get-AzVMImageSku

## SYNOPSIS
Получает skus VMImage.

## СИНТАКСИС

```
Get-AzVMImageSku -Location <String> -PublisherName <String> -Offer <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **получения skus -VMImage возвращается cmdlet Get-AzVMImageSku.**

## ПРИМЕРЫ

### Пример 1. Получить skUs VMImage
```
PS C:\> Get-AzVMImageSku -Location "Central US" -PublisherName "Fabrikam" -Offer "LinuxServer"
```

Эта команда получает skus для указанного издателя и предложения.

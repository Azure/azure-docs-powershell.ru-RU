# Remove-AzLoadBalancerOutboundRuleConfig

## SYNOPSIS
Удаляет конфигурацию исходящие правила из балансиров нагрузки.

## СИНТАКСИС

```
Remove-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При **этом из балансиров** нагрузки Azure удаляется конфигурация исходящие правил.

## ПРИМЕРЫ

### Пример 1. Удаление правила исходящие сообщения из балансиров нагрузки Azure
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Remove-AzLoadBalancerOutboundRuleConfig -Name "RuleName" -LoadBalancer $slb
PS C:\>Set-AzLoadBalancer -LoadBalancer $slb
```

Первая команда получает балансировку нагрузки, связанную с конфигурацией правил исходящие, которую вы хотите удалить, и сохраняет ее в переменной $slb.
Вторая команда удаляет из балансиров нагрузки связанную конфигурацию исходящие правила.
Третья команда обновляет балансировку нагрузки.

# Set-AzureRmCdnProfile

## КРАТКИй обзор
Обновляет профиль CDN.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Set-AzureRmCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzureRmCdnProfile** обновляет профиль сети доставки содержимого (CDN) Azure.

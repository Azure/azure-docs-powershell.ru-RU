## ОПИСАНИЕ
Для развертывания шаблона в текущей области подписки ARM шаблон What-If **Get-AzDeploymentWhatIfResult.** Она возвращает список изменений, указывающий, какие ресурсы будут обновлены при применении развертывания, не внося никаких изменений в реальные ресурсы. Чтобы указать формат возвращаемго результата, используйте параметр *ResultFormat.*

## ПРИМЕРЫ

### Пример 1. Получите What-If по области действия подписки
```powershell
PS C:\> Get-AzDeploymentWhatIfResult `
    -DeploymentName "deploy-01" `
    -Location "West US" `
    -TemplateFile "D:\Azure\Templates\ServiceTemplate.json" `
    -TemplateParameterFile "D:\Azure\Templates\ServiceParameters.json" `
    -ResultFormat "FullResourcePayloads"
```

Эта команда возвращает What-If текущую область действия подписки с помощью настраиваемого файла шаблона и файла параметров на диске.
Для указания места *хранения* данных развертывания используется параметр Location.
Для указания файла шаблона используется параметр *TemplateFile.*
Для указания файла *параметра шаблона используется параметр TemplateParameterFile.*
Для этого используется параметр *ResultFormat* What-If результатов.

### Пример 2. Получите результат What-If области действия подписки с ResourceIdOnly
```powershell
PS C:\> Get-AzDeploymentWhatIfResult `
    -DeploymentName "deploy-01" `
    -Location "West US" `
    -TemplateFile "D:\Azure\Templates\ServiceTemplate.json" `
    -TemplateParameterFile "D:\Azure\Templates\ServiceParameters.json" `
    -ResultFormat "ResourceIdOnly"
```

Эта команда возвращает What-If текущую область действия подписки с помощью настраиваемого файла шаблона и файла параметров на диске.
Для указания места *хранения* данных развертывания используется параметр Location.
Для указания файла шаблона используется параметр *TemplateFile.*
Для указания файла *параметра шаблона используется параметр TemplateParameterFile.*
Команда использует параметр *ResultFormat,* чтобы у What-If результатов содержались только ИД ресурсов.

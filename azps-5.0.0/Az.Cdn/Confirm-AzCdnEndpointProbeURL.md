# Confirm-AzCdnEndpointProbeURL

## КРАТКИй обзор
Проверка пробного URL-адреса.

## Максимальное

```
Confirm-AzCdnEndpointProbeURL -ProbeUrl <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Confirm-AzCdnEndpointProbeURL** подтверждает, что указанный URL-адрес пробной проверки можно использовать для ускорения динамического сайта.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Confirm-AzCdnEndpointProbeURL -ProbeUrl "http://www.bing.com/images"
IsValid: true
ErrorCode: None
Message:
```

Проверка URL-адреса пробы " http://www.bing.com/images "

# Remove-AzureVirtualNetworkGateway

## КРАТКИй обзор
Удаление шлюза виртуальной сети.

## Максимальное

```
Remove-AzureVirtualNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureVirtualNetworkGateway** удаляет шлюз виртуальной сети.

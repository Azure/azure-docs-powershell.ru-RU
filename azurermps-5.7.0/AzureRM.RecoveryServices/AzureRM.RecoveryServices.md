# Модуль AzureRM. RecoveryServices
## Nописание
В этом разделе отображаются разделы справки для командлетов служб восстановления Azure.

## Командлеты AzureRM. RecoveryServices
### [Get-AzureRmRecoveryServicesBackupProperty](Get-AzureRmRecoveryServicesBackupProperty.md)
Получает свойства резервного копирования.

### [Get-AzureRmRecoveryServicesVault](Get-AzureRmRecoveryServicesVault.md)
Получает список хранилищ служб восстановления.

### [Get-AzureRmRecoveryServicesVaultSettingsFile](Get-AzureRmRecoveryServicesVaultSettingsFile.md)
Получает файл параметров хранилища Azure Site Recovery.

### [New-AzureRmRecoveryServicesVault](New-AzureRmRecoveryServicesVault.md)
Создание нового хранилища служб восстановления.

### [Remove-AzureRmRecoveryServicesVault](Remove-AzureRmRecoveryServicesVault.md)
Удаляет хранилище служб восстановления.

### [Set-AzureRmRecoveryServicesBackupProperties](Set-AzureRmRecoveryServicesBackupProperties.md)
Задает свойства для управления резервным копированием.

### [Set-AzureRmRecoveryServicesVaultContext](Set-AzureRmRecoveryServicesVaultContext.md)
Задает контекст хранилища.

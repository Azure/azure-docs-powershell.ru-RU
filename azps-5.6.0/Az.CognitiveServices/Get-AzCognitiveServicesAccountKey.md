# Get-AzCognitiveServicesAccountKey

## SYNOPSIS
Получает ключи API для учетной записи.

## СИНТАКСИС

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzCognitiveServicesAccountKey** получает ключи API для учетной записи provisioned Cognitive Services.
Учетная запись когнитивных служб имеет два ключа API: "Ключ1" и "Ключ2".
Эти клавиши обеспечивают взаимодействие с конечной точкой учетной записи "Когнитивные услуги".
Используйте New-AzCognitiveServicesAccountKey для повторного сгенерации ключа.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

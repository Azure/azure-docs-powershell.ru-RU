# Add-AzVmssWinRMListener

## SYNOPSIS
Добавляет в VMSS слушателя WinRM.

## СИНТАКСИС

```
Add-AzVmssWinRMListener [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Protocol] <ProtocolTypes>]
 [[-CertificateUrl] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого с** его Windows Remote Management в наборе виртуальных машин (VMSS) будет добавлен Windows Remote Management-прослушивания WinRM.

## ПРИМЕРЫ

### Пример 1. Добавление слушателя WinRM в VMSS
```
PS C:\> $VMSS = New-AzVmssConfig
PS C:\> Add-AzVmssWinRMListener -VirtualMachineScaleSet $VMSS -Protocol Https -CertificateUrl "http://keyVaultName.vault.contoso.net/secrets/secretName/secretVersion"
```

В этом примере в VMSS добавляется слушатель WinRM.
Первая команда использует командлет **New-AzVmssConfig** для создания объекта конфигурации VMSS и сохраняет результат в переменной с именем $VMSS.
Вторая команда добавляет в VMSS прослушки HTTP-протокола WinRM с сертификатом по указанному URL-адресу.

# Start-AzureRmJitNetworkAccessPolicy

## КРАТКИй обзор
Вызывает временный запрос на доступ к сети.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

### ResourceGroupLevelResource (по умолчанию)
```
Start-AzureRmJitNetworkAccessPolicy -ResourceGroupName <String> -Location <String> -Name <String>
 -VirtualMachine <PSSecurityJitNetworkAccessPolicyInitiateVirtualMachine[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Вызывает временный запрос на доступ к сети.
Запрос проверяются на соответствие настроенной политике доступа к сети JIT и, если permittet, открывает сетевое подключение в соответствии с запросом пользователя.
Запрос будет loged в политике для более поздней проверки и будет прекращен, когда будет превышена Указанная длительность.

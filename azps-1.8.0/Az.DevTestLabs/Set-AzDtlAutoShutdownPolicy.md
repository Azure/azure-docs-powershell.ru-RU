# Set-AzDtlAutoShutdownPolicy

## КРАТКИй обзор
Задает политику автоматического выключения лабораторных семинаров DevTest Labs.

## Максимальное

### Включить (по умолчанию)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Ключив
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzDtlAutoShutdownPolicy** задает политику автоматического отключения лаборатории, которая автоматически завершает работу всех виртуальных машин в лаборатории в заданное время суток.
Для настройки политики командлет использует указанную группу ресурсов и имя лаборатории.

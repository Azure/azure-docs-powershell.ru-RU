# Remove-AzAutomationSourceControl

## SYNOPSIS
Удаляет исходный контроль автоматизации Azure.

## СИНТАКСИС

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Remove-AzAutomationSourceControl удаляет исходный набор данных из автоматизации Azure.

## ПРИМЕРЫ

### Пример 1
Эта команда удаляет исходный контроль автоматизации с именем VSTSNative в учетной записи devAccount.
Эта команда определяет параметр *Force.* Поэтому запрос на подтверждение не будет.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```

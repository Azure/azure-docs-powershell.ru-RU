# New-AzIotHubSasToken

## КРАТКИй обзор
Создание маркера SAS для целевого центра Интернета IoT, устройства или модуля.

## Максимальное

### Источник данных (по умолчанию)
```
New-AzIotHubSasToken [-ResourceGroupName] <String> [-IotHubName] <String> [-DeviceId <String>]
 [-ModuleId <String>] [-KeyName <String>] [-KeyType <PSKeyType>] [-Duration <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
New-AzIotHubSasToken [-InputObject] <PSIotHub> [-DeviceId <String>] [-ModuleId <String>] [-KeyName <String>]
 [-KeyType <PSKeyType>] [-Duration <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdSet
```
New-AzIotHubSasToken [-ResourceId] <String> [-DeviceId <String>] [-ModuleId <String>] [-KeyName <String>]
 [-KeyType <PSKeyType>] [-Duration <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Для маркеров SAS устройства параметр политики используется для доступа только к реестру устройства. Таким образом, политика должна иметь доступ к реестру для чтения.
Для маркеров центра Интернета вещей политика входит в состав SAS.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzIotHubSasToken -ResourceGroupName "myresourcegroup" -IotHubName "myiothub"
```

Создание маркера SAS центра Интернета вещей с помощью политики iothubowner и первичного ключа.

### Пример 2
```powershell
PS C:\> New-AzIotHubSasToken -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -KeyName "registryRead" -KeyType "secondary"
```

Создание маркера SAS центра Интернета вещей с помощью политики registryRead и вторичного ключа.

### Пример 3
```powershell
PS C:\> New-AzIotHubSasToken -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -DeviceId "myDevice1"
```

Создание маркера SAS устройства с помощью политики iothubowner для доступа к реестру устройства {iothub_name}.

### Пример 4
```powershell
PS C:\> New-AzIotHubSasToken -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -DeviceId "myDevice1" -ModuleId "myModule1"
```

Создание маркера SAS модуля с помощью политики iothubowner для доступа к реестру устройства {iothub_name}.

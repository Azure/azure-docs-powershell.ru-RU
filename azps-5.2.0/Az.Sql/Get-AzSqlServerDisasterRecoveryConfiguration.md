# Get-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Получает конфигурацию восстановления системы сервера базы данных.

## СИНТАКСИС

```
Get-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для этого вы получаете SQL сервера базы данных **Get-AzSqlServerDisasterRecoveryConfiguration.**

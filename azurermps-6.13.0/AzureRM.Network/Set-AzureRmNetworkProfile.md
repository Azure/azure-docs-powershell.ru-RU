# Set-AzureRmNetworkProfile

## КРАТКИй обзор
Задает состояние цели для существующего профиля сети

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Set-AzureRmNetworkProfile -NetworkProfile <PSNetworkProfile> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzureRmPublicIpPrefix** задает состояние цели для профиля сети.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
$networkProfile = Get-AzureRmNetworkProfile -Name np1 -ResourceGroupName rg1

$networkProfile.Tags = "TestTag"

$networkProfile.ContainerNetworkInterfaceConfigurations = New-AzureRmNetworkProfileContainerNicConfig -Name cnicconfig1

$networkProfile | Set-AzureRmNetworkProfile
```

Первая команда получает существующий профиль сети. Вторая команда обновляет тег, а третья добавляет конфигурацию сетевого интерфейса в сетевом профиле. Четвертая команда обновляет профиль сети.

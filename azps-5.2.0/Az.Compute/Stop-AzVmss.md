# Stop-AzVmss

## SYNOPSIS
Остановка виртуальных машин или виртуальных машин в них.

## СИНТАКСИС

### DefaultParameter (по умолчанию)
```
Stop-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-Force]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### FriendMethod
```
Stop-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-Force]
 [-StayProvisioned] [-SkipShutdown] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для остановки всех виртуальных машин в наборе виртуальных машин (VMSS) или набора виртуальных машин используется **cmdlet Stop-AzVmss.**
С помощью параметра *InstanceId* можно выбрать набор виртуальных машин.

## ПРИМЕРЫ

### Пример 1. Остановка всех виртуальных машин в VMSS
```
PS C:\> Stop-AzVmss -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Эта команда останавливает все виртуальные машины, которые относятся к VMSS с именем ContosoVMSS.

### Пример 2. Остановка определенного набора виртуальных машин в VMSS
```
PS C:\> Stop-AzVmss -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS" -InstanceId "3","5"
```

Эта команда останавливает определенный набор виртуальных машин, заданный массивом строки "ИД экземпляра", относимых к VMSS ContosoVMSS.

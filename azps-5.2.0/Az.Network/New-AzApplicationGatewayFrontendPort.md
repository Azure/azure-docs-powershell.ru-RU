# New-AzApplicationGatewayFrontendPort

## SYNOPSIS
Создается стойкий порт для шлюза приложения.

## СИНТАКСИС

```
New-AzApplicationGatewayFrontendPort -Name <String> -Port <Int32> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для шлюза приложения Azure создается клиентский порт **New-AzApplicationGatewayFrontendPort.**

## ПРИМЕРЫ

### Пример 1. Пример 1. Создание переднего порта
```powershell
PS C:\>$FrontEndPort = New-AzApplicationGatewayFrontendPort -Name "FrontEndPort01" -Port 80
```

Эта команда создает стойкий порт FrontEndPort01 в порте 80 и сохраняет результат в переменной с $FrontEndPort.

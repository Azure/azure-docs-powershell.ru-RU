# Stop-AzDataMigrationService

## SYNOPSIS
Остановка экземпляра службы миграции баз данных Azure, которая находится в запущенном состоянии.

## СИНТАКСИС

### ComponentNameParameterSet (по умолчанию)
```
Stop-AzDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Stop-AzDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Stop-AzDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Stop-AzDataMigrationService останавливает экземпляр службы миграции баз данных Azure, запущенный в состоянии.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Stop-AzDataMigrationService -ResourceGroupName MyResourceGroup -ServiceName TestService
```

В примере выше остановка экземпляра службы миграции баз данных Azure TestService на основе имени службы, переданного в качестве входного параметра

### Пример 2
```
PS C:\> Stop-AzDataMigrationService -InputObject $TestService
```

В примере выше экземпляр службы миграции баз данных Azure останавливается на основе объекта PSDataMigrationService, переданного в качестве входного параметра.

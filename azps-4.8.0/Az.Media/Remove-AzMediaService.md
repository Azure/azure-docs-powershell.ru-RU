# Remove-AzMediaService

## КРАТКИй обзор
Удаление службы мультимедиа.

## Максимальное

```
Remove-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzMediaService** Удаляет службу мультимедиа.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление службы мультимедиа
```
PS C:\>Remove-AzMediaService -ResourceGroupName "ResourceGroup001" -AccountName "MediaService0011"
```

Эта команда удаляет службу мультимедиа с именем MediaService0011 в группе ресурсов с именем ResourceGroup001.

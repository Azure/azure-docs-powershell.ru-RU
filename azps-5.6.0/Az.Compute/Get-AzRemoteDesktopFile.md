# Get-AzRemoteDesktopFile

## SYNOPSIS
Возвращает RDP-файл.

## СИНТАКСИС

### Скачать
```
Get-AzRemoteDesktopFile [-ResourceGroupName] <String> [-Name] <String> [-LocalPath] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Запуск
```
Get-AzRemoteDesktopFile [-ResourceGroupName] <String> [-Name] <String> [[-LocalPath] <String>] [-Launch]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого с помощью cmdlet Get-AzRemoteDesktopFile** будет удален RDP-файл.

## ПРИМЕРЫ

### Пример 1. Получить файл удаленного рабочего стола
```
PS C:\> Get-AzRemoteDesktopFile -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07" -LocalPath "D:\RemoteDesktopFile07.rdp"
```

Эта команда получает файл удаленного рабочего стола для виртуальной машины с именем VirtualMachine07.
Команда сохраняет результат в файле D:\RemoteDesktopFile07.rdp.

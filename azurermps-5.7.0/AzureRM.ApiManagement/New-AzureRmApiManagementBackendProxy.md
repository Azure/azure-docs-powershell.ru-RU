# New-AzureRmApiManagementBackendProxy

## КРАТКИй обзор
Создает новый объект прокси-сервера в серверной части.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmApiManagementBackendProxy -Url <String> [-ProxyCredential <PSCredential>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Создает новый объект-посредник для базы данных, который может быть передан при создании новой серверной сущности.

## ИЛЛЮСТРИРУЮТ

### Создание объекта прокси-сервера в серверной In-Memory
```
PS C:\>$secpassword = ConvertTo-SecureString "PlainTextPassword" -AsPlainText -Force
PS C:\>$proxyCreds = New-Object System.Management.Automation.PSCredential ("foo", $secpassword)
PS C:\>$credential = New-AzureRmApiManagementBackendProxy -Url "http://12.168.1.1:8080" -ProxyCredential $proxyCreds
```

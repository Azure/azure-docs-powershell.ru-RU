# Stop-AzSqlSyncGroupSync

## SYNOPSIS
Остановка синхронизации группы синхронизации.

## СИНТАКСИС

```
Stop-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Командалет **Stop-AzSqlSyncGroupSync** останавливает синхронизацию группы синхронизации.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Stop-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Эта команда останавливает синхронизацию, которая продолжается для mysg группы синхронизации.

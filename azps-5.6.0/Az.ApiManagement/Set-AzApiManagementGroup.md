# Set-AzApiManagementGroup

## SYNOPSIS
Настраивает группу управления API.

## СИНТАКСИС

```
Set-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-Name <String>]
 [-Description <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для настройки группы управления API группа **Set-AzApiManagementGroup.**

## ПРИМЕРЫ

### Пример 1. Настройка группы управления
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementGroup -Context $apimContext -GroupId "0001" -Description "Updated Management Group" -Name "Group0001"
```

Эта команда настраивает группу управления с именем Group0001.

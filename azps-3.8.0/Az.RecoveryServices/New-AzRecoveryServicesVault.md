# New-AzRecoveryServicesVault

## КРАТКИй обзор
Создание нового хранилища служб восстановления.

## Максимальное

```
New-AzRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzRecoveryServicesVault** создает новое хранилище служб восстановления.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> New-AzRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Создание хранилища службы восстановления в группе ресурсов и указанном расположении.

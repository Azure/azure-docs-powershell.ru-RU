# Get-AzDtlAutoShutdownPolicy

## SYNOPSIS
Получает политику автоматического закрытия лабораторных анализов DevTest Labs.

## СИНТАКСИС

```
Get-AzDtlAutoShutdownPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **cmdlet get-AzDtlAutoShutdownPolicy** задана политика автоматического завершения работы лабораторных лаборатории, которая позволяет автоматически закрыть все виртуальные машины в заданное время суток.
Он возвращает состояние политики и время суток, задав для нее автоматическое завершение работы виртуальных машин.

# Get-AzAutoscaleSetting

## SYNOPSIS
Параметры автомасштабии.

## СИНТАКСИС

```
Get-AzAutoscaleSetting -ResourceGroupName <String> [-Name <String>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzAutoscaleSetting можно** получить все параметры автомасштабы, связанные с группой ресурсов или заданным параметром авто шкалы.

# Reset-AzureRmWebAppSlotPublishingProfile


## NОПИСАНИЕ
Командлет **Reset-AzureRmWebAppSlotPublishingProfile** сбрасывает профиль публикации для указанной области веб-приложения.

## ИЛЛЮСТРИРУЮТ

### 1:
```
PS C:\> Reset-AzureRmWebAppSlotPublishingProfile -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -Slot "slot001"
```

Эта команда сбрасывает профиль публикации для слота с именем slot001 для веб-приложения ContosoWebApp, связанного с группой ресурсов по умолчанию-Web-WestUS.

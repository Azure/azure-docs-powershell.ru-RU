# Enable-AzRecoveryServicesBackupProtection

## SYNOPSIS
Включает резервное копирование элемента с помощью указанной политики защиты от резервного копирования.

## СИНТАКСИС

### AzureVMComputeEnableProtection (по умолчанию)
```
Enable-AzRecoveryServicesBackupProtection [[-Policy] <PolicyBase>] [-Name] <String>
 [-ResourceGroupName] <String> [-InclusionDisksList <String[]>] [-ExclusionDisksList <String[]>]
 [-ExcludeAllDataDisks] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### AzureVMClassicComputeEnableProtection
```
Enable-AzRecoveryServicesBackupProtection [[-Policy] <PolicyBase>] [-Name] <String> [-ServiceName] <String>
 [-InclusionDisksList <String[]>] [-ExclusionDisksList <String[]>] [-ExcludeAllDataDisks] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AzureFileShareEnableProtection
```

 [-StorageAccountName] <String> [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]

```

### AzureWorkloadEnableProtection

```
 [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Enable-AzRecoveryServicesBackupProtection** включает резервное копирование путем связывания политики защиты с элементом.
Если код политики не задан или элемент резервной копии не связан с какой-либо политикой, эта команда будет ожидать код политики.
Прежде чем использовать текущий Set-AzRecoveryServicesVaultContext, задате контекст хранилища с помощью Set-AzRecoveryServicesVaultContext.

## ПРИМЕРЫ

### Пример 1. Защита резервных копий для элемента
```powershell
PS C:\> $Pol = Get-AzRecoveryServicesBackupProtectionPolicy -Name "DefaultPolicy"
PS C:\> $inclusionDiskLUNS = ("1", "2")
PS C:\> Enable-AzRecoveryServicesBackupProtection -Policy $Pol -Name "V2VM" -ResourceGroupName "RGName1" -InclusionDisksList $inclusionDiskLUNS
WorkloadName    Operation        Status          StartTime                  EndTime
------------    ---------        ------          ---------                  -------
co03-vm         ConfigureBackup  Completed       11-Apr-16 12:19:49 PM      11-Apr-16 12:19:54 PM
```

Первый cmdlet получает объект политики по умолчанию, а затем сохраняет его в $Pol переменной.
Второй cmdlet указывает, какие коды luNs диска необходимо сделать сархивом и которые будут $inclusionDiskLUNS переменной.
Третий cmdlet set the Backup protection policy for the ARM machine named V2VM using the policy in $Pol.

### Пример 2

Включает резервное копирование элемента с помощью указанной политики защиты от резервного копирования. (autogenerated)

# Get-AzureServiceRemoteDesktopExtension

## КРАТКИй обзор
Этот командлет получает расширение удаленного рабочего стола облачной службы, примененное ко всем ролям или именованным ролям в определенном слоте развертывания.

## Максимальное

```
Get-AzureServiceRemoteDesktopExtension [[-ServiceName] <String>] [[-Slot] <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureServiceRemoteDesktopExtension** получает расширение удаленного рабочего стола облачной службы, примененное ко всем ролям или именованным ролям в определенном слоте развертывания.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение расширения удаленного рабочего стола для указанной службы
```
PS C:\> Get-AzureServiceRemoteDesktopExtension -ServiceName $svc
```

Эта команда получает расширение удаленного рабочего стола для указанной службы.

# New-AzApplicationGatewayAutoscaleConfiguration

## КРАТКИй обзор
Создает конфигурацию автомасштабирования для шлюза приложений.

## Максимальное

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzApplicationGatewayAutoscaleConfiguration** создает конфигурацию автомасштабирования для шлюза приложений Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Первая команда создает конфигурацию автомасштабирования с минимальной емкостью 3.
Вторая команда создает шлюз приложений с конфигурацией автомасштабирования.

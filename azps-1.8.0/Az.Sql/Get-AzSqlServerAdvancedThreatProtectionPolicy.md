# Get-AzSqlServerAdvancedThreatProtectionPolicy

## КРАТКИй обзор
Возвращает политику Advanced Threat Protection для сервера.

## Максимальное

```
Get-AzSqlServerAdvancedThreatProtectionPolicy [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlServerAdvancedThreatProtectionPolicy** retrives расширенной политики защиты от угроз для сервера.

## ИЛЛЮСТРИРУЮТ

### Пример 1-получает сервер Advanced Threat protection
```powershell
PS C:\>  Get-AzSqlServerAdvancedThreatProtectionPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Пример 2: получение Advanced Threat protection из серверного ресурса
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Get-AzSqlServerAdvancedThreatProtectionPolicy

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

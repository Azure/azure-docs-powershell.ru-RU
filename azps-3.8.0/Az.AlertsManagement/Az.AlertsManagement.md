# AZ. AlertsManagement Module
## Nописание
Управление оповещениями, интеллектуальными группами и конфигурацией правила действий для платформы управления оповещениями.

## Командлеты AZ. AlertsManagement
### [Get-AzActionRule](Get-AzActionRule.md)
Получение сведений о правилах действия

### [Get-AzAlert](Get-AzAlert.md)
Получение сведений об оповещениях

### [Get-AzAlertObjectHistory](Get-AzAlertObjectHistory.md)
Получение сведений о журнале оповещений

### [Get-AzSmartGroup](Get-AzSmartGroup.md)
Возвращает сведения о смарт-группах

### [Get-AzSmartGroupHistory](Get-AzSmartGroupHistory.md)
Получение журнала Smart Group

### [Мера — AzAlertStatistic](Measure-AzAlertStatistic.md)
Получение сводных данных об оповещениях

### [Remove-AzActionRule](Remove-AzActionRule.md)
Удаление группы действий

### [Set-AzActionRule](Set-AzActionRule.md)
Создание или обновление правила действия.

### [Update-AzActionRule](Update-AzActionRule.md)
Обновляет свойства правила действия.

### [Update-AzAlertState](Update-AzAlertState.md)
Обновление состояния оповещения

### [Update-AzSmartGroupState](Update-AzSmartGroupState.md)
Обновляет состояние Smart Group

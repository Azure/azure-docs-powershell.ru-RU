# Remove-AzVpnGateway

## КРАТКИй обзор
Командлет Remove-AzVpnGateway удаляет шлюз Azure VPN. Это шлюз, специфичный для подключения по виртуальной глобальной сети Azure.

## Максимальное

### ByVpnGatewayName (по умолчанию)
```
Remove-AzVpnGateway -ResourceGroupName <String> -Name <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayObject
```
Remove-AzVpnGateway -InputObject <PSVpnGateway> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayResourceId
```
Remove-AzVpnGateway -ResourceId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzVpnGateway удаляет шлюз Azure VPN. Это шлюз, специфичный для подключения по виртуальной глобальной сети Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1

```powershell
PS C:\> New-AzResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Remove-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -Passthru
```

В этом примере создается группа ресурсов, Виртуальная глобальная сеть, виртуальный концентратор виртуальных частных сетей, масштабируемый шлюз VPN в центре US и сразу же удаляются. Чтобы отключить вывод запроса при удалении виртуального шлюза, используйте флаг-force.
Это приведет к удалению VpnGateway и всех VpnConnections, прикрепленных к нему.

### Пример 2

```powershell
PS C:\> New-AzResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Get-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" | Remove-AzVpnGateway-Passthru
```

В этом примере создается группа ресурсов, Виртуальная глобальная сеть, виртуальный концентратор виртуальных частных сетей, масштабируемый шлюз VPN в центре US и сразу же удаляются. Это удаление выполняется с помощью оболочки PowerShell, в которой используется объект VpnGateway, возвращенный командой "Get-AzVpnGateway".
Чтобы отключить вывод запроса при удалении виртуального шлюза, используйте флаг-force.
Это приведет к удалению VpnGateway и всех VpnConnections, прикрепленных к нему.

# Get-AzureRmManagedApplication

## КРАТКИй обзор
Получает управляемые приложения

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### Список всех наборов параметров управляемых приложений. По умолчанию
```
Get-AzureRmManagedApplication [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Заданный параметр имени управляемого приложения.
```
Get-AzureRmManagedApplication [-Name <String>] -ResourceGroupName <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Заданный параметр идентификатора управляемого приложения.
```
Get-AzureRmManagedApplication -Id <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmManagedApplication** получает управляемые приложения.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение всех управляемых приложений в группе ресурсов
```
PS C:\>Get-AzureRmManagedApplication -ResourceGroupName "MyRG"
```

Эта команда получает управляемые приложения в группе ресурсов "MyRG".

### Пример 2: получение всех управляемых приложений
```
PS C:\>Get-AzureRmManagedApplication
```

Эта команда получает все управляемые приложения в текущей подписке

# New-AzDataFactory

## КРАТКИй обзор
Создание фабрики данных.

## Максимальное

```
New-AzDataFactory [-Name] <String> [-Location] <String> [[-Tag] <Hashtable>] [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzDataFactory** создает фабрику данных с указанным именем группы ресурсов и расположением.
Для выполнения этих операций выполните указанные ниже действия. 
- Создание фабрики данных. 
- Создание связанных служб. 
- Создание наборов данных. 
- Создание конвейера.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Создание фабрики данных
```
PS C:\>New-AzDataFactory -ResourceGroupName "ADF" -Name "WikiADF" -Location "WestUS"
DataFactoryName   : WikiADF
ResourceGroupName : ADF
Location          : WestUS
Tags              : {}
Properties        : Microsoft.WindowsAzure.Commands.Utilities.PSDataFactoryConfiguration
```

Эта команда создает фабрику данных с именем WikiADF в группе ресурсов с именем ADF в расположении WestUS.

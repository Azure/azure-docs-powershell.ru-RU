# Get-AzVMDiagnosticsExtension

## SYNOPSIS
Получает параметры расширения "Диагностика" на виртуальной машине.

## СИНТАКСИС

```
Get-AzVMDiagnosticsExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzVMDiagnosticsExtension** можно получить параметры расширения Azure Diagnostics на виртуальной машине.

## ПРИМЕРЫ

### Пример 1. Расширение диагностики, примененное к виртуальной машине
```powershell
PS C:\> Get-AzVMDiagnosticsExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22"
```

Эта команда получает расширение диагностики, примененное к виртуальной машине с именем ContosoVM22.

### Пример 2

Получает параметры расширения "Диагностика" на виртуальной машине. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Get-AzVMDiagnosticsExtension -Name 'AgentPool01' -ResourceGroupName 'ResourceGroup11' -Status -VMName 'ContosoVM22'
```

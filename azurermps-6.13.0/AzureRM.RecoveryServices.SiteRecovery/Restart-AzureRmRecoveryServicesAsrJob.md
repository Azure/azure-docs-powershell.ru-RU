# Restart-AzureRmRecoveryServicesAsrJob

## КРАТКИй обзор
Перезапускает задание восстановления сайта Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### ByObject (по умолчанию)
```
Restart-AzureRmRecoveryServicesAsrJob -InputObject <ASRJob> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Restart-AzureRmRecoveryServicesAsrJob -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **restart-AzureRmRecoveryServicesAsrJob** перезапустит задание восстановления сайта Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> $currentJob = Restart-AzureRmRecoveryServicesAsrJob -Job $Job
```

Перезапускает указанное задание ASR и возвращает обновленный объект задания ASR для задания ASR.

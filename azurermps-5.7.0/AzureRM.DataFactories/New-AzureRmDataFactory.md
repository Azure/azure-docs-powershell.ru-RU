# New-AzureRmDataFactory

## КРАТКИй обзор
Создание фабрики данных.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmDataFactory [-Name] <String> [-Location] <String> [[-Tag] <Hashtable>] [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmDataFactory** создает фабрику данных с указанным именем группы ресурсов и расположением.

Для выполнения этих операций выполните указанные ниже действия. 

- Создание фабрики данных. 
- Создание связанных служб. 
- Создание наборов данных. 
- Создание конвейера.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Создание фабрики данных
```
PS C:\>New-AzureRmDataFactory -ResourceGroupName "ADF" -Name "WikiADF" -Location "WestUS"
DataFactoryName   : WikiADF
ResourceGroupName : ADF
Location          : WestUS
Tags              : {}
Properties        : Microsoft.WindowsAzure.Commands.Utilities.PSDataFactoryConfiguration
```

Эта команда создает фабрику данных с именем WikiADF в группе ресурсов с именем ADF в расположении WestUS.

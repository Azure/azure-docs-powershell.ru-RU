# Get-AzSqlInstanceDatabaseGeoBackup

## КРАТКИй обзор
Возвращает сведения об избыточной резервной копии базы данных SQL, управляемой с управлением Azure.

## Максимальное

```
Get-AzSqlInstanceDatabaseGeoBackup [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlInstanceDatabaseGeoBackup** получает из управляемого экземпляра базы данных SQL Azure одно или несколько резервных копий баз данных Azure SQL.

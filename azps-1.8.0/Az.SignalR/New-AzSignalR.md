# New-AzSignalR

## КРАТКИй обзор
Создание службы сигналов.

## Максимальное

```
New-AzSignalR [-ResourceGroupName <String>] [-Name] <String> [-Location <String>] [-Sku <String>]
 [-UnitCount <Int32>] [-Tag <System.Collections.Generic.IDictionary`2[System.String,System.String]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Создание службы сигналов.
Если не указано, для параметров будут использоваться следующие значения:
* `ResourceGroupName`: Стандартная группа ресурсов, установленная по умолчанию `Set-AzDefault -ResourceGroupName` .
* `Location`: расположение группы ресурсов
* `Sku`: `Standard_S1`

## ИЛЛЮСТРИРУЮТ

### Создание поsignalr
```powershell
PS C:\> New-AzSignalR -ResourceGroupName myResourceGroup1 -Name mysignalr1 -Location eastus -Sku Standard_S1

```

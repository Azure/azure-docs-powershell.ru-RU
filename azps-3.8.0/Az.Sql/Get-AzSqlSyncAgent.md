# Get-AzSqlSyncAgent

## КРАТКИй обзор
Возвращает сведения об агентах синхронизации Azure SQL.

## Максимальное

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlSyncAgent** возвращает сведения об одном или нескольких агентах синхронизации Azure SQL.
Укажите имя агента синхронизации, чтобы просмотреть сведения только для этого агента синхронизации.

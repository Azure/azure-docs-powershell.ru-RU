# Get-AzApplicationGatewayAuthenticationCertificate

## КРАТКИй обзор
Возвращает сертификат для проверки подлинности для шлюза приложения.

## Максимальное

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzApplicationGatewayAuthenticationCertificate** получает сертификат проверки подлинности для шлюза приложений Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение указанного сертификата для проверки подлинности
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $pool = Get-AzApplicationGatewayBackendAddressPool -Name "pool01" -ApplicationGateway $appgw
```

Первая команда получает шлюз приложения с именем appGwName и сохраняет его в переменной $appgw.
Вторая команда получает сертификат для проверки подлинности с именем pool01 и сохраняет его в переменной $pool.

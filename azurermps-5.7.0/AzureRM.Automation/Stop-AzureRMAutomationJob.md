# Stop-AzureRmAutomationJob

## КРАТКИй обзор
Остановка задания автоматизации.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Stop-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Stop-AzureRmAutomationJob** останавливает задание службы автоматизации Azure.
Укажите выполнение задания автоматизации.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Остановка задания
```
PS C:\>Stop-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Эта команда останавливает задание с указанным ИДЕНТИФИКАТОРом.

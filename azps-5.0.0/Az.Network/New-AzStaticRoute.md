# New-AzStaticRoute

## КРАТКИй обзор
Создает объект StaticRoute, который затем можно добавить в объект RoutingConfiguration.

## Максимальное

```powershell
New-AzStaticRoute -Name <String> -AddressPrefix <String[]> -NextHopIpAddress <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Создает объект StaticRoute.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzStaticRoute -Name "route1" -AddressPrefix @("10.20.0.0/16", "10.30.0.0/16") -NextHopIpAddress "10.90.0.5"

Name   AddressPrefixes              NextHopIpAddress
----   ---------------              ----------------
route1 {10.20.0.0/16, 10.30.0.0/16} 10.90.0.5
```

Вышеприведенная команда создаст объект StaticRoute, который затем можно будет добавить в объект RoutingConfiguration.

# Get-AzSynapseFirewallRule

## SYNOPSIS
Получает правило брандмауэра synapse Analytics.

## СИНТАКСИС

### GetByNameParameterSet (по умолчанию)
```
Get-AzSynapseFirewallRule [-ResourceGroupName <String>] -WorkspaceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseFirewallRule -WorkSpaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **брандмауэра Get-AzSynapseFirewallRule** вы получаете правило брандмауэра Azure Synapse Analytics.
Если не указать имя правила, этот cmdlet получит все правила.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzSynapseFirewallRule -ResourceGroupName ContosoResourceGroup -WorkspaceName ContosoWorkspace
```

Эта команда получает все правила брандмауэра в рабочей области.

### Пример 2
```powershell
PS C:\> Get-AzSynapseFirewallRule -ResourceGroupName ContosoResourceGroup -WorkspaceName ContosoWorkspace -Name ContosoFirewallRule
```

Эта команда получает правило брандмауэра в рабочей области ContosoWorkspace с именем ContosoFirewallRule.

### Пример 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseFirewallRule
```

Эта команда получает все правила брандмауэра в рабочей области через канал.

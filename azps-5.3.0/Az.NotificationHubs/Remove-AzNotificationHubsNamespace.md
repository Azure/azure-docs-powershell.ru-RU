# Remove-AzNotificationHubsNamespace

## SYNOPSIS
Удаляет пространство имен концентратора уведомлений.

## СИНТАКСИС

```
Remove-AzNotificationHubsNamespace [-ResourceGroup] <String> [-Namespace] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При **развертывании удаляется** пространство имен концентратора уведомлений.
Пространства имен — это логические контейнеры, которые помогают организовать концентраторы уведомлений и управлять ими.
При **развертывании удаляется** пространство имен концентратора уведомлений.
При запуске этого cmdlet указанное пространство имен удаляется вместе со всеми центрами уведомлений, связанными с этим пространством имен.

## ПРИМЕРЫ

### Пример 1. Удаление пространства имен концентратора уведомлений
```
PS C:\>Remove-AzNotificationHubsNamespace -Namespace "ContosoNamespace" -ResourceGroup "ContosoNotificationsGroup"
```

Эта команда удаляет пространство имен с именем ContosoNamespace.
Необходимо указать группу ресурсов, которая назначена области имен.

# Get-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Получает конфигурацию переднего IP-адреса в балансе нагрузки.

## СИНТАКСИС

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого вы** можете выбрать переднюю конфигурацию IP-адреса или список конфигураций переднего IP-адреса в балансе нагрузки.

## ПРИМЕРЫ

### Пример 1. Настройка переднего IP-адреса в балансире нагрузки
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

Первая команда получает балансировку нагрузки MyLoadBalancer, а затем сохраняет его в переменной $slb.
Вторая команда получает конфигурацию переднего IP-адреса, связанную с этим балансировом нагрузки.

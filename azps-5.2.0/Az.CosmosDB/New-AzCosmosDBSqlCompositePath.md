# New-AzCosmosDBSqlCompositePath

## SYNOPSIS
Создание объекта типа PSCompositePath. Его можно передать в качестве значения параметра для Set-AzCosmosDBSqlContainer.

## СИНТАКСИС

```
New-AzCosmosDBSqlCompositePath [-Path <String>] [-Order <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Объект, соответствующий Sql API CompositePath.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> New-AzCosmosDBSqlCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```

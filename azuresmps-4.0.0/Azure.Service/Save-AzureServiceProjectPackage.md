# Save-AzureServiceProjectPackage

## КРАТКИй обзор
Упаковывает проект службы в облачный пакет Azure.

## Максимальное

```
Save-AzureServiceProjectPackage [-Local] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
В этом разделе описан командлет в версии 0.8.10 модуля Microsoft Azure PowerShell.
Чтобы получить версию модуля, который вы используете, введите в командной консоли Azure PowerShell `(Get-Module -Name Azure).Version` .

Командлет **Save-AzureServiceProjectPackage** упаковывает проект службы в облачный пакет Azure (*. cspkg).

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание пакета проекта службы
```
PS C:\> Save-AzureServiceProjectPackage
```

В этом примере создается объект *. cspgk для проекта службы с именем MyAzureServiceProject.

# Suspend-AzAutomationJob

## КРАТКИй обзор
Приостанавливает задание автоматизации.

## Максимальное

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Suspend-AzAutomationJob** приостанавливает задание службы автоматизации Azure.
Укажите выполнение задания автоматизации.
Чтобы возобновить приостановленное задание, используйте командлет Resume-AzAutomationJob.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Приостановка задания
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Эта команда приостанавливает задание с указанным ИДЕНТИФИКАТОРом.

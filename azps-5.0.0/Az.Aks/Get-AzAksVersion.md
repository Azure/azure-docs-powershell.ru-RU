# Get-AzAksVersion

## КРАТКИй обзор
Список доступных версий для создания управляемого кластера Kubernetes.

## Максимальное

```
Get-AzAksVersion -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Список доступных версий для создания управляемого кластера Kubernetes.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzAksVersion -Location westus
```

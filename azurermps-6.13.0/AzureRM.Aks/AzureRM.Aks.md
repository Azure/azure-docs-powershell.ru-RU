# Модуль AzureRM. AKS
## Nописание
Команды для взаимодействия с Kubernetes кластерами, управляемыми с помощью Azure.

## Командлеты AzureRM. AKS
### [Get-AzureRmAks](Get-AzureRmAks.md)
Перечислите Kubernetes управляемые кластеры.

### [Import-AzureRmAksCredential](Import-AzureRmAksCredential.md)
Импорт и слияние конфигурации Kubectl для управляемого Kubernetes кластера.

### [New-AzureRmAks](New-AzureRmAks.md)
Создание нового управляемого кластера Kubernetes.

### [Remove-AzureRmAks](Remove-AzureRmAks.md)
Удаление управляемого кластера Kubernetes.

### [Set-AzureRmAks](Set-AzureRmAks.md)
Обновите или создайте управляемый кластер Kubernetes.

### [Start-AzureRmAksDashboard](Start-AzureRmAksDashboard.md)
Создайте туннель SSH Kubectl на панели мониторинга управляемого кластера.

### [Остановить-AzureRmAksDashboard](Stop-AzureRmAksDashboard.md)
Остановите туннель SSH Kubectl, созданный в Start-AzureRmKubernetesDashboard.

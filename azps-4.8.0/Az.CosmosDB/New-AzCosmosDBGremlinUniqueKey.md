# New-AzCosmosDBGremlinUniqueKey

## КРАТКИй обзор
Создает новый объект CosmosDB UniqueKeyPolicy.

## Максимальное

```
New-AzCosmosDBGremlinUniqueKey -Path <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzCosmosDBGremlinUniqueKeyPolicy** создает новый объект типа PSUniqueKeyPolicy.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzCosmosDBGremlinUniqueKey -Path "abc"
```

# Grant-AzureRmHDInsightRdpServicesAccess

## КРАТКИй обзор
Предоставление доступа к кластеру Windows для подключения к удаленному рабочему столу.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Grant-AzureRmHDInsightRdpServicesAccess [-ClusterName] <String> [-RdpCredential] <PSCredential>
 [-RdpAccessExpiry] <DateTime> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Grant-AzureRmHDInsightRdpServicesAccess** включает протокол удаленного рабочего стола (RDP) для доступа к кластеру Azure HDInsight на базе Windows.

## ИЛЛЮСТРИРУЮТ

### Пример 1: предоставление доступа к удаленному рабочему столу в кластере HDInsight Azure
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzureRmHDInsightRdpServicesAccess `
            -ClusterName $clusterName `
            -RdpCredential $newRdpCreds `
            -RdpAccessExpiry $newRdpExpiry
```

Эта команда предоставляет доступ к кластеру через RDP с именем-Hadoop-001.

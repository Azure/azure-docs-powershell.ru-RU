# Remove-AzureRmStorageAccount

## КРАТКИй обзор
Удаление учетной записи хранения из Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmStorageAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmStorageAccount** удаляет учетную запись хранения из Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление учетной записи хранения
```
PS C:\>Remove-AzureRmStorageAccount -ResourceGroupName "RG01" -AccountName "MyStorageAccount"
```

Эта команда удаляет указанную учетную запись хранения.

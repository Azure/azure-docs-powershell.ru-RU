# Модуль Az.ServiceBus
## Описание
В этой теме вы можете найти разделы справки для работы с клиентом службы Azure.

## Az.ServiceBus Cmdlets
### [Complete-AzServiceBusMigration](Complete-AzServiceBusMigration.md)
При настроив переход со стандартного на премиум-пространство имен как завершенный, а строки подключения в стандартном пространстве имен теперь указывают на пространство имен Premium.

### [Get-AzServiceBusAuthorizationRule](Get-AzServiceBusAuthorizationRule.md)
Описание указанного правила авторизации для определенного пространства имен, очереди или раздела или псевдонима (GeoDR Configurations). 

### [Get-AzServiceBusGeoDRConfiguration](Get-AzServiceBusGeoDRConfiguration.md)
Извлекает псевдоним (конфигурация аварийного восстановления) для основного или дополнительного пространства имен.

### [Get-AzServiceBusKey](Get-AzServiceBusKey.md)
Возвращает строки основного и дополнительного подключения для заданного пространства имен, очереди или раздела или псевдонима (GeoDR Configurations).

### [Get-AzServiceBusMigration](Get-AzServiceBusMigration.md)
Извлекает migrationConfiguration для пространства имен.

### [Get-AzServiceBusNamespace](Get-AzServiceBusNamespace.md)
Описание указанного пространства имен автобусов обслуживания в группе ресурсов.

### [Get-AzServiceBusOperation](Get-AzServiceBusOperation.md)
Список поддерживаемых операций ServiceBus

### [Get-AzServiceBusQueue](Get-AzServiceBusQueue.md)
Возвращает описание указанной очереди "Автобусы обслуживания".

### [Get-AzServiceBusRule](Get-AzServiceBusRule.md)
Создание правила для подписки на раздел. 

### [Get-AzServiceBusSubscription](Get-AzServiceBusSubscription.md)
Возвращает описание подписки для указанного раздела.

### [Get-AzServiceBusTopic](Get-AzServiceBusTopic.md)
Возвращает описание указанного раздела "Автобусы обслуживания".

### [New-AzServiceBusAuthorizationRule](New-AzServiceBusAuthorizationRule.md)
Создает новое правило авторизации для указанного автобусы обслуживания, указанного в пространстве имен, очереди или теме.

### [New-AzServiceBusAuthorizationRuleSASToken](New-AzServiceBusAuthorizationRuleSASToken.md)
Создает tolen SAS для правила авторизации Azure serviucebus пространства имен/ очереди/раздела. 

### [New-AzServiceBusGeoDRConfiguration](New-AzServiceBusGeoDRConfiguration.md)
Создание нового псевдонима (конфигурация аварийного восстановления)

### [New-AzServiceBusKey](New-AzServiceBusKey.md)
Regenerates the primary or secondary connection strings for the Service Bus namespace or queue or topic.

### [New-AzServiceBusNamespace](New-AzServiceBusNamespace.md)
Создание нового пространства имен «Шина обслуживания».

### [New-AzServiceBusQueue](New-AzServiceBusQueue.md)
Создание очереди "Автобус обслуживания" в указанном пространстве доменных имен "Автобус обслуживания".

### [New-AzServiceBusRule](New-AzServiceBusRule.md)
Создание правила для подписки на раздел. 

### [New-AzServiceBusSubscription](New-AzServiceBusSubscription.md)
Создает подписку на указанный раздел "Автобус обслуживания".

### [New-AzServiceBusTopic](New-AzServiceBusTopic.md)
Создает новую тему "Автобус обслуживания" в указанном пространстве имен автобусов обслуживания.

### [Remove-AzServiceBusAuthorizationRule](Remove-AzServiceBusAuthorizationRule.md)
Удаляет правило авторизации пространства имен или очереди или темы для группы ресурсов.

### [Remove-AzServiceBusGeoDRConfiguration](Remove-AzServiceBusGeoDRConfiguration.md)
Удаляет псевдоним (конфигурация аварийного восстановления)

### [Remove-AzServiceBusMigration](Remove-AzServiceBusMigration.md)
Cmdlet deletes the Migration configuration for Standard to Premium namespaces

### [Remove-AzServiceBusNamespace](Remove-AzServiceBusNamespace.md)
Удаляет пространство имен из указанной группы ресурсов. 

### [Remove-AzServiceBusQueue](Remove-AzServiceBusQueue.md)
Удаляет очередь из указанного пространства имен автобусов обслуживания.

### [Remove-AzServiceBusRule](Remove-AzServiceBusRule.md)
Удаляет указанное правило для указанной подписки.

### [Remove-AzServiceBusSubscription](Remove-AzServiceBusSubscription.md)
Удаляет подписку на раздел из указанного пространства имен автобусов обслуживания.

### [Remove-AzServiceBusTopic](Remove-AzServiceBusTopic.md)
Удаляет тему из указанного пространства имен автобусов обслуживания.

### [Set-AzServiceBusAuthorizationRule](Set-AzServiceBusAuthorizationRule.md)
Обновляет описание указанного правила авторизации для указанного пространства имен автобусов службы, очереди или раздела.

### [Set-AzServiceBusGeoDRConfigurationBreakPair](Set-AzServiceBusGeoDRConfigurationBreakPair.md)
Эта операция отключает аварийное восстановление и останавливает репликацию изменений из основного в дополнительное пространство имен.

### [Set-AzServiceBusGeoDRConfigurationFailOver](Set-AzServiceBusGeoDRConfigurationFailOver.md)
Вызывает отбой GEO DR и повторно настроит псевдоним, чтобы он указать на дополнительное пространство имен.

### [Set-AzServiceBusNamespace](Set-AzServiceBusNamespace.md)
Обновляет описание существующего пространства имен «Автобус обслуживания».

### [Set-AzServiceBusQueue](Set-AzServiceBusQueue.md)
Обновляет описание очереди "Автобусы обслуживания" в указанном пространстве имен "Автобус обслуживания".

### [Set-AzServiceBusRule](Set-AzServiceBusRule.md)
Обновляет описание указанного правила для данной подписки.

### [Set-AzServiceBusSubscription](Set-AzServiceBusSubscription.md)
Обновляет описание подписки для темы "Автобус обслуживания" в указанном пространстве имен "Автобус обслуживания".

### [Set-AzServiceBusTopic](Set-AzServiceBusTopic.md)
Обновляет описание темы "Автобус обслуживания" в указанном пространстве имен автобусов обслуживания.

### [Start-AzServiceBusMigration](Start-AzServiceBusMigration.md)
Создание новой конфигурации миграции и начало переноса сущностями из пространства имен Standard в premium

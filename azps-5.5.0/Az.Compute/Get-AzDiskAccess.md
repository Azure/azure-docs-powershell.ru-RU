# Get-AzDiskAccess

## SYNOPSIS
Свойства дискового доступа

## СИНТАКСИС

### DefaultParameterSet (по умолчанию)
```
Get-AzDiskAccess [[-ResourceGroupName] <String>] [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIDParameterSet
```
Get-AzDiskAccess [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения свойств дискового доступа к диску возвращается cmdlet **Get-AzDiskAccess.**

## ПРИМЕРЫ

### Пример 1. Использование набора параметров по умолчанию 
```
PS C:\> Get-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -Name 'DiskAccess01'

TimeCreated                : 8/13/2020 7:02:50 PM

Name                       : DiskAccess01
```

Эта команда получает свойства ресурса Диска Access с именем "ДискAccess01" в группе ресурсов "Группа ресурсов01".

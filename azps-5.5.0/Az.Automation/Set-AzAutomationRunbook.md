# Set-AzAutomationRunbook

## SYNOPSIS
Изменяет книгу.

## СИНТАКСИС

```
Set-AzAutomationRunbook [-Name] <String> [-Description <String>] [-Tag <IDictionary>] [-LogProgress <Boolean>]
 [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Set-AzAutomationRunbook** изменяет конфигурацию книги автоматизации Azure в APS.

## ПРИМЕРЫ

### Пример 1. Включить подробное ведение журнала для книги
```
PS C:\>Set-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook02" -LogVerbose $True -ResourceGroupName "ResourceGroup01"
```

Эта команда позволяет подробно вести журнал для заданий указанной книги в учетной записи автоматизации Azure Contoso17.

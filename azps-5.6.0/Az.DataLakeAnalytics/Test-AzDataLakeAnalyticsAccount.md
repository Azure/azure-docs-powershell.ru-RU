# Test-AzDataLakeAnalyticsAccount

## SYNOPSIS
Проверяет наличие учетной записи Data Lake Analytics.

## СИНТАКСИС

```
Test-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Test-AzDataLakeAnalyticsAccount** проверяет наличие учетной записи Data Lake Analytics.

## ПРИМЕРЫ

### Пример 1. Проверка на существования учетной записи
```
PS C:\>Test-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Эта команда проверяет, существует ли учетная запись с именем ContosoAdlAccount.

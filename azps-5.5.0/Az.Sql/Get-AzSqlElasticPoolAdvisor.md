# Get-AzSqlElasticPoolAdvisor

## SYNOPSIS
Получает одного или несколько консультантов по пулу Azure SQL.

## СИНТАКСИС

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
С **его** использованием можно получить один или несколько помощников по SQL Azure SQL для эластичного пула Azure SQL.

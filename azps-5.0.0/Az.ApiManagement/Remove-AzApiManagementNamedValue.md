# Remove-AzApiManagementNamedValue

## КРАТКИй обзор
Удаляет именованное значение для управления API.

## Максимальное

```
Remove-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApiManagementNamedValue** удаляет **именованное значение** управления API Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление именованного значения
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -PassThru
```

Эта команда удаляет именованное значение с ИДЕНТИФИКАТОРом Property11.

# Resume-AzSqlDatabase

## КРАТКИй обзор
Возобновляет базу данных хранилища данных SQL.

## Максимальное

```
Resume-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Resume-AzSqlDatabase** возобновляет базу данных хранилища данных SQL Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: возобновление базы данных хранилища данных SQL Azure
```
PS C:\>Resume-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда Возобновляет приостановленную базу данных хранилища данных SQL Azure.

## Пуск

# Get-AzStackEdgeStorageAccountCredential

## SYNOPSIS
Получает учетные данные учетной записи хранения, соответствующие учетной записи хранения на устройстве.

## СИНТАКСИС

### ListParameterSet (по умолчанию)
```
Get-AzStackEdgeStorageAccountCredential [-ResourceGroupName] <String> [-DeviceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzStackEdgeStorageAccountCredential -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByNameParameterSet
```
Get-AzStackEdgeStorageAccountCredential [-ResourceGroupName] <String> [-DeviceName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzStackEdgeStorageAccountCredential [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 -DeviceObject <PSStackEdgeDevice> [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzStackEdgeStorageAccountCredential** получает учетные данные учетной записи хранения для соответствующей учетной записи хранения на устройстве Stack Edge. Вы можете указать параметры "Имя", "Имя группы ресурсов" и "Имя устройства", чтобы получить сведения об учетных данных конкретной учетной записи хранения.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\>  Get-AzStackEdgeStorageAccountCredential -ResourceGroupName resourceGroupName -DeviceName deviceName
Name                          StorageAccount          SslStatus  ResourceGroupName
----------------------------- ---------------------- ---------- ---------------------
storageAccountCredentialName  StorageAccountName     Enabled    resourceGroupName
```

# New-AzPeeringRegisteredPrefix

## КРАТКИй обзор
Создание зарегистрированных префиксов для одноранговых объектов.

## Максимальное

### ByResourceGroupAndName (по умолчанию)
```
New-AzPeeringRegisteredPrefix [-ResourceGroupName] <String> [-PeeringName] <String> [-Name] <String>
 -Prefix <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
New-AzPeeringRegisteredPrefix -InputObject <PSPeering> [-Name] <String> -Prefix <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
New-AzPeeringRegisteredPrefix [-ResourceId] <String> [-Name] <String> -Prefix <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Создание зарегистрированных префиксов для одноранговых объектов.

## ИЛЛЮСТРИРУЮТ

### Получение для пиринга и создание зарегистрированного префикса
```powershell
PS C:\>$peering = Get-AzPeering -ResourceGroupName $resourceGroupName -Name $name
PS C:\>$peering | New-AzPeeringRegisteredPrefix -Name $asnName -Asn $asn
```

Получите одноранговый элемент, к которому вы хотите добавить зарегистрированный префикс. Затем передайте этот метод в unifiedgroup.

### Использование ИД ресурса пиринга для создания зарегистрированного ASN
```powershell
PS C:\>New-AzPeeringRegisteredPrefix -ResourceId $resourceId -Name $asnName -Asn $asn
```

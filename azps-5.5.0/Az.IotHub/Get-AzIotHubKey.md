# Get-AzIotHubKey

## SYNOPSIS
Возвращает ключ IotHub.

## СИНТАКСИС

### Набор ресурсов (по умолчанию)
```
Get-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [[-KeyName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubKey [-HubId] <String> [[-KeyName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Возвращает ключ IotHub.
Вы можете перечислить все ключи или отфильтровать список по определенному имени ключа.

## ПРИМЕРЫ

### Пример 1. Получить все ключи
```
PS C:\> Get-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Все клавиши для IotHub с именем "myiothub"

### Пример 2. Получить сведения для определенного ключа
```
PS C:\> Get-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner"
```

Возвращает сведения о ключе "iothubowner" для IotHub с именем Myiothub.

# Get-AzDefault

## SYNOPSIS
Установите значения по умолчанию для пользователя в текущем контексте.

## СИНТАКСИС

```
Get-AzDefault [-ResourceGroup] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С Get-AzDefault возвращается группа ресурсов, задавшаяся пользователем по умолчанию в текущем контексте.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Get-AzDefault

Id         : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/myResourceGroup
Name       : myResourceGroup
Properties : Microsoft.Azure.Management.Internal.Resources.Models.ResourceGroupProperties
Location   : eastus
ManagedBy  :
Tags       :
```

Эта команда возвращает текущие значения по умолчанию, если заданной по умолчанию, или ничего, если значение по умолчанию не заданной.

### Пример 2
```
PS C:\> Get-AzDefault -ResourceGroup

Id         : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/myResourceGroup
Name       : myResourceGroup
Properties : Microsoft.Azure.Management.Internal.Resources.Models.ResourceGroupProperties
Location   : eastus
ManagedBy  :
Tags       :
```

Эта команда возвращает текущую группу ресурсов по умолчанию, если она заданной по умолчанию, или ничего не возвращает, если она не заданной по умолчанию.

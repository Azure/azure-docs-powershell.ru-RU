# Add-AzLogProfile

## SYNOPSIS
Создание профиля журнала действий. Этот профиль используется для архивирования журнала действий в учетной записи хранилища Azure или его потокового передачи в центр событий Azure по той же подписке. 

## СИНТАКСИС

```
Add-AzLogProfile -Name <String> [-StorageAccountId <String>] [-ServiceBusRuleId <String>]
 [-RetentionInDays <Int32>] -Location <System.Collections.Generic.List`1[System.String]>
 [-Category <System.Collections.Generic.List`1[System.String]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его учетом** создается профиль журнала.
- **Учетная запись хранения** — поддерживается только стандартная учетная запись хранения (учетная запись премиум-хранилища не поддерживается). Это может быть один из ARM или классический. Если журнал регистрируется в учетной записи хранения, за хранение журнала действий выставлен счет по стандартным тарифам хранилища. Косвенным образом для экспорта журнала действий можно использовать только один профиль журнала на одну подписку. 
- **Центр событий.** Для экспорта журнала действий может быть только один профиль журнала на подписку. Если журнал действий потоковой передачи в центр событий, будет применяться стандартная цена в Центре событий. В журнале действий события могут относится к региону или могут быть "глобальными". Глобально это, по сути, означает, что эти события являются мгнистическими данными регионов и не зависят от региона, фактически большинство событий попадают в эту категорию. Если профиль журнала действий настроен на портале, он неявно добавляет глобальный профиль, а также любой другой регион, выбранный в пользовательском интерфейсе. При использовании этого cmdlet расположение должно быть явно упоминаться как глобальное. 
**Примечание.** Если не установить глобальный в расположениях, большинство журналов действий не будут **экспортироваться.** Этот cmdlet реализует шаблон ShouldProcess, то есть может запросить подтверждение у пользователя перед фактическим созданием, изменением или удалением ресурса.

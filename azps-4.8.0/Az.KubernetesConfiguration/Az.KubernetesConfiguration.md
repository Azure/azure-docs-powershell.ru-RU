# AZ. KubernetesConfiguration Module
## Nописание
Командлеты Microsoft Azure PowerShell: KubernetesConfiguration

## Командлеты AZ. KubernetesConfiguration
### [Get-AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
Получает сведения о настройке системы управления версиями.

### [New-AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
Создайте новую конфигурацию системы управления версиями Kubernetes.

### [Remove-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
Это приведет к удалению файла YAML, используемого для настройки конфигурации системы управления версиями, тем самым прекращая дальнейшую синхронизацию из исходного репозитория.

# Get-AzLoadBalancerInboundNatPoolConfig

## КРАТКИй обзор
Возвращает один или несколько конфигураций входящего пула NAT из подсистемы балансировки нагрузки.

## Максимальное

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzLoadBalancerInboundNatPoolConfig** получает один или несколько конфигураций ВХОДЯЩЕГО пула NAT из подсистемы балансировки нагрузки.

## ИЛЛЮСТРИРУЮТ

### 1: Get
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```

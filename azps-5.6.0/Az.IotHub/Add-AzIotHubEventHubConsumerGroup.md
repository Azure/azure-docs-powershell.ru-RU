# Add-AzIotHubEventHubConsumerGroup

## SYNOPSIS
Создает группу eventhub для потребителей.

## СИНТАКСИС

```
Add-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-EventHubConsumerGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Создает группу потребителей в Eventhub, связанном с указанным IotHub.

## ПРИМЕРЫ

### Пример 1. Добавление группы потребителей в е eventhub телеметрии
```
PS C:\> Add-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubConsumerGroupName "myconsumergroup"
```

Добавляет новую группу потребителей с именем myconsumergroup в еверху для событий телеметрии в iothub с именем myiothub.

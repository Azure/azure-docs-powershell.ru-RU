# Publish-AzAutomationRunbook

## КРАТКИй обзор
Публикует Runbook.

## Максимальное

```
Publish-AzAutomationRunbook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Publish-AzAutomationRunbook** публикует Runbook для использования в производственной среде службы автоматизации Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: публикация Runbook
```
PS C:\>Publish-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbk01" -ResourceGroupName "ResourceGroup01"
```

Эта команда публикует Runbook с именем Runbk01 в учетной записи службы автоматизации Azure с именем Contoso17.

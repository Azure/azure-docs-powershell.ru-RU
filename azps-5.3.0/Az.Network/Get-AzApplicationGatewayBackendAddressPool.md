# Get-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Возвращает пул конечных адресов для шлюза приложения.

## СИНТАКСИС

```
Get-AzApplicationGatewayBackendAddressPool [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzApplicationGatewayBackendAddressPool** можно получить одну или несколько конфигураций пула дополнительных адресов из шлюза приложения.

## ПРИМЕРЫ

### Пример 1. Получить указанный серверный пул серверов
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $BackendPool = Get-AzApplicationGatewayBackendAddressPool -Name "Pool01" -ApplicationGateway $AppGw
```

Первая команда получает шлюз приложения ApplicationGateway01 в группе ресурсов ResourceGroup01 и сохраняет его в переменной $AppGw ресурса.
Вторая команда получает пул конечных адресов, связанный с $AppGw Pool01, и сохраняет его в переменной $BackendPool.

### Пример 2. Получить список серверного пула серверов
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $BackendPools = Get-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw
```

Первая команда получает шлюз приложения ApplicationGateway01 в группе ресурсов ResourceGroup01 и сохраняет его в переменной $AppGw ресурса.
Вторая команда получает список пулов адресов, связанных с $AppGw, и сохраняет список в переменной $BackendPools адресов.

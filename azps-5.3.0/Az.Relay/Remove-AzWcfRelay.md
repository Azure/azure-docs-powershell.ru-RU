# Remove-AzWcfRelay

## SYNOPSIS
Удаляет WcfRelay из указанного пространства имен ретрансляции.

## СИНТАКСИС

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Командлет **Remove-AzWcfRelay** удаляет WcfRelay из указанного пространства имен ретрансляции.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Удаляет WcfRelay `TestWCFRelay1` из пространства `TestNameSpace-Relay1` имен.

# Remove-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Удаляет конфигурацию SQL сервера базы данных.

## СИНТАКСИС

```
Remove-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
С его SQL сервера базы данных удаляется конфигурация сервера базы данных **Remove-AzSqlServerDisasterRecoveryConfiguration.**

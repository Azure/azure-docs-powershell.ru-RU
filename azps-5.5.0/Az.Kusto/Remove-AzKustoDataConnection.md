# Remove-AzKustoDataConnection

## SYNOPSIS
Удаляет подключение к данным с заданным именем.

## СИНТАКСИС

### Delete (По умолчанию)
```
Remove-AzKustoDataConnection -ClusterName <String> -DatabaseName <String> -Name <String>
 -ResourceGroupName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzKustoDataConnection -InputObject <IKustoIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## ОПИСАНИЕ
Удаляет подключение к данным с заданным именем.

## ПРИМЕРЫ

### Пример 1. Удаление существующего подключения к данным по имени
```powershell
PS C:\> Remove-AzKustoDataConnection -ResourceGroupName "testrg" -ClusterName "testnewkustocluster" -DatabaseName "mykustodatabase" -DataConnectionName "mykustodataconnection"
```

Команда выше удаляет подключение к данным с именем "mykustodataconnection" в базе данных "mykustodatabase" существующего кластера testnewkustocluster, найденное в группе ресурсов testrg

# Get-AzStorageFileServiceProperty

## КРАТКИй обзор
Получение свойств службы для файловых служб хранилища Azure.

## Максимальное

### Имя_учетной_записи (по умолчанию)
```
Get-AzStorageFileServiceProperty [-ResourceGroupName] <String> [-StorageAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AccountObject
```
Get-AzStorageFileServiceProperty -StorageAccount <PSStorageAccount> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### FileServicePropertiesResourceId
```
Get-AzStorageFileServiceProperty [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzStorageFileServiceProperty** получает свойства службы для файловых служб хранилища Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение свойства файловых служб хранилища Azure для указанной учетной записи хранения
```powershell
PS C:\> Get-AzStorageFileServiceProperty -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount"

StorageAccountName ResourceGroupName ShareDeleteRetentionPolicy.Enabled ShareDeleteRetentionPolicy.Days
------------------ ----------------- ---------------------------------- -------------------------------
mystorageaccount   myresourcegroup   True                               5
```

Эта команда получает свойство файловых служб для указанной учетной записи хранения.

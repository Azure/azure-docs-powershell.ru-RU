# Remove-AzApplicationGatewaySslCertificate

## SYNOPSIS
Удаляет SSL-сертификат из шлюза приложения Azure.

## СИНТАКСИС

```
Remove-AzApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
При этом из шлюза приложения Azure удаляется **SSL-сертификат Remove-AzApplicationGateWaySslCertificate.**

## ПРИМЕРЫ

### Пример 1. Удаление SSL-сертификата из шлюза приложения
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

Первая команда получает шлюз приложения ApplicationGateway01 и сохраняет результат в переменной с $AppGW.
Вторая команда удаляет SSL-сертификат с именем Cert02 из шлюза приложения, $AppGW переменной.

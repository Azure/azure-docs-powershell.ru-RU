# Clear-AzDefault

## КРАТКИй обзор
Очищает значения по умолчанию, заданные пользователем в текущем контексте.

## Максимальное

```
Clear-AzDefault [-ResourceGroup] [-PassThru] [-Force] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Clear-AzDefault удаляет значения по умолчанию, заданные пользователем, в зависимости от параметров переключателей, указанных пользователем.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Clear-AzDefault
```

Эта команда удаляет все значения по умолчанию, заданные пользователем в текущем контексте.

### Пример 1
```
PS C:\> Clear-AzDefault -ResourceGroup
```

Эта команда удаляет группу ресурсов по умолчанию, установленную пользователем в текущем контексте.

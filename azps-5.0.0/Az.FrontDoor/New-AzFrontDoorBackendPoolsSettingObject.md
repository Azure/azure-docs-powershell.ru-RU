# New-AzFrontDoorBackendPoolsSettingObject

## КРАТКИй обзор
Создайте объект PSBackendPoolsSetting для создания передней дверцы.

## Максимальное

```
New-AzFrontDoorBackendPoolsSettingObject [-EnforceCertificateNameCheck <PSEnabledState>]
 [-SendRecvTimeoutInSeconds <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzFrontDoorBackendpoolsSettingObject** создает новый объект PSBackendPoolsSettings для создания передней дверцы.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание объекта BackendPoolsSettings с помощью значений по умолчанию
```powershell
PS C:\> New-AzFrontDoorBackendpoolsSettingObject

```

### Пример 2: создание объекта BackendPoolsSettings с указанными пользователем значениями
```powershell
PS C:\> New-AzFrontDoorBackendpoolsSettingObject -SendRecvTimeoutInSeconds 60 -EnforceCertificateNameCheck Enabled
```

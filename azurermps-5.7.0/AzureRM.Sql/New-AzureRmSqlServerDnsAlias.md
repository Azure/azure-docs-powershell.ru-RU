# New-AzureRmSqlServerDnsAlias

## КРАТКИй обзор
Эта команда создает новый DNS-псевдоним для Azure SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Создает новый DNS-псевдоним Azure SQL Server, указывающий на указанный сервер.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> $serverDNSAlias = New-AzureRmSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Эта команда создает псевдоним DNS для Azure SQL Server с именем aliasName, указывающим на сервер serverName.

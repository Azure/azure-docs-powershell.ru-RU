# Get-AzRelayHybridConnection

## КРАТКИй обзор
Возвращает описание указанного HybridConnection в пространстве имен ретрансляции.

## Максимальное

```
Get-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzRelayHybridConnection** Возвращает описание указанной HybridConnection в пространстве имен ретрансляции.

## ИЛЛЮСТРИРУЮТ

# Set-AzCognitiveServicesAccount

## КРАТКИй обзор
Изменение учетной записи.

## Максимальное

```
Set-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-SkuName <String>]
 [-Tag <Hashtable[]>] [-CustomSubdomainName <String>] [-NetworkRuleSet <PSNetworkRuleSet>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzCognitiveServicesAccount** изменяет SKU или теги указанной учетной записи службы.

## ИЛЛЮСТРИРУЮТ

# Get-AzPowerBIEmbeddedCapacity

## SYNOPSIS
Подробные сведения о внедренной емкости PowerBI.

## СИНТАКСИС

### ByCapacityOrResourceGroupOrSubscription (по умолчанию)
```
Get-AzPowerBIEmbeddedCapacity [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzPowerBIEmbeddedCapacity -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
С Get-AzPowerBIEmbeddedCapacity управления получают сведения о внедренной емкости PowerBI.

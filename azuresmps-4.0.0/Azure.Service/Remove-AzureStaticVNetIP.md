# Remove-AzureStaticVNetIP

## КРАТКИй обзор
Удаляет данные об IP-адресе статической виртуальной сети из объекта виртуальной машины, если таковые имеются.

## Максимальное

```
Remove-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureStaticVNetIP** удаляет статические сведения о IP-адресе виртуальной сети (VNet) из объекта виртуальной машины, если таковые имеются.

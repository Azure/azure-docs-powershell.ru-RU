# Stop-AzDeploymentManagerRollout

## КРАТКИй обзор
Прекращение выпуска.

## Максимальное

### Интерактивный (по умолчанию)
```
Stop-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Stop-AzDeploymentManagerRollout [-ResourceId] <String> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Stop-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Stop-AzDeploymentManagerRollout** останавливает процесс развертывания и возвращает объект, который представляет текущее состояние выпуска.
Укажите имя и имя группы ресурсов. Кроме того, можно предоставить объект выпуска или ИД ресурса.

Обратите внимание на то, что после прекращения выпуска приложение не может быть возобновлено или перезапущено. Вы можете создать только новый выпуск.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Stop-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout -SkipSucceeded
```

Эта команда останавливает выгрузку с именем ContosoRollout в ContosoResourceGroup. 

### Пример 2: прекращение выпуска с помощью идентификатора ресурса
```powershell
PS C:\> Restart-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Эта команда останавливает выгрузку с именем ContosoRollout в ContosoResourceGroup.

### Пример 3: прекращение выпуска с помощью объекта выпуска.
```powershell
PS C:\> Get-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Эта команда останавливает выгрузку, имя и значение ResourceGroup которой соответствуют свойствам Name и ResourceGroupName для $rolloutObject соответственно.

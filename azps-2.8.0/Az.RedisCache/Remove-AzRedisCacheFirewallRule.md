# Remove-AzRedisCacheFirewallRule

## КРАТКИй обзор
Удаление правила брандмауэра из кэша Redis.

## Максимальное

### NormalParameterSet (по умолчанию)
```
Remove-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> -RuleName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PSRedisFirewallRuleObject
```
Remove-AzRedisCacheFirewallRule -InputObject <PSRedisFirewallRule> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Удаление правила брандмауэра из кэша Redis.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление одного правила брандмауэра
```
PS C:\>Remove-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone" -PassThru
True
```

Эта команда удаляет правило брандмауэра с именем ruleone из кэша Redis с именем mycache.

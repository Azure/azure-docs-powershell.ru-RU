# Get-AzDeviceSecurityGroup

## SYNOPSIS
Получить группу безопасности устройств (безопасность IoT Hub)

## СИНТАКСИС

### ResourceIdScope (по умолчанию)
```
Get-AzDeviceSecurityGroup -HubResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdLevelResource
```
Get-AzDeviceSecurityGroup -HubResourceId <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Get-AzDeviceSecurityGroup возвращает группу безопасности устройств, определенное в решении для защиты iot.

# New-AzApplicationGatewayAuthenticationCertificate

## КРАТКИй обзор
Создание сертификата проверки подлинности для шлюза приложения.

## Максимальное

```
New-AzApplicationGatewayAuthenticationCertificate -Name <String> -CertificateFile <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzApplicationGatewayAuthenticationCertificate** создает сертификат для проверки подлинности для шлюза приложений Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание сертификата для проверки подлинности
```
PS C:\> $cert = New-AzApplicationGatewayAuthenticationCertificate -Name "cert01" -CertificateFile "C:\cert.cer"
```

Первая команда создает сертификат для проверки подлинности с именем cert01.

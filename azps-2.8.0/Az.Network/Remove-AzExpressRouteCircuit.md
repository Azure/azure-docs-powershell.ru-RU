# Remove-AzExpressRouteCircuit

## КРАТКИй обзор
Удаление канала ExpressRoute.

## Максимальное

```
Remove-AzExpressRouteCircuit -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzExpressRouteCircuit** удаляет канал ExpressRoute.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление канала ExpressRoute
```
Remove-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg
```

### Пример 2: удаление канала ExpressRoute с помощью конвейера
```
Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzExpressRouteCircuit
```

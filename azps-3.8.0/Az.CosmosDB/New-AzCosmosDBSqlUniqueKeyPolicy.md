# New-AzCosmosDBSqlUniqueKeyPolicy

## КРАТКИй обзор
Создает новый объект CosmosDB SqlUniqueKeyPolicy.

## Максимальное

```
New-AzCosmosDBSqlUniqueKeyPolicy -UniqueKey <PSSqlUniqueKey[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzCosmosDBSqlUniqueKeyPolicy** создает новый объект типа PSSqlUniqueKeyPolicy.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\>New-AzCosmosDBSqlUniqueKeyPolicy -UniqueKey {psUniqueKey1, psUniqueKey2}
```

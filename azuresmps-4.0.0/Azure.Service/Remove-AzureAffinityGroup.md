# Remove-AzureAffinityGroup

## КРАТКИй обзор
Удаляет территориальную группу в подписке.

## Максимальное

```
Remove-AzureAffinityGroup [-Name] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureAffinityGroup** удаляет территориальную группу Azure в текущей подписке.
Вы не можете удалить территориальную группу, которая содержит каких – либо пользователей.
Сначала необходимо удалить всех участников территориальной группы.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление территориальной группы
```
PS C:\> Remove-AzureAffinityGroup -Name "South01"
```

Эта команда удаляет территориальную группу South01 в текущей подписке.

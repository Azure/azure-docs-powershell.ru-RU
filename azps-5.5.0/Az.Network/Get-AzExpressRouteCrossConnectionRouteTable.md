# Get-AzExpressRouteCrossConnectionRouteTable

## SYNOPSIS
Получает таблицу маршрутов из перекрестного подключения ExpressRoute.

## СИНТАКСИС

### SpecifyByParameterValues
```
Get-AzExpressRouteCrossConnectionRouteTable -ResourceGroupName <String> -CrossConnectionName <String>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### SpecifyByReference
```
Get-AzExpressRouteCrossConnectionRouteTable -ExpressRouteCrossConnection <PSExpressRouteCrossConnection>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения подробной таблицы маршрутов из схемы ExpressRoute извлекается из таблицы маршрутов **Get-AzExpressRouteCrossConnectionRouteTable.** В таблице маршрутов будут отфильтроваться все маршруты или отфильтровать маршруты для определенного типа пиринга. Таблицу маршрутов можно использовать для проверки конфигурации пиринга и подключения.

## ПРИМЕРЫ

### Пример 1. Отображение таблицы маршрутов для основного пути
```
Get-AzExpressRouteCrossConnectionRouteTable -ResourceGroupName $RG -ExpressRouteCrossConnectionName $CircuitName -DevicePath 'Primary'
```

# Get-AzureRmServiceBusQueue

## КРАТКИй обзор
Возвращает описание для указанной очереди служебной шины.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmServiceBusQueue [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Возвращает описание для указанной очереди служебной шины.

Возвращает описание очереди.

### Пример 2
```
PS C:\> Get-AzureRmServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Возвращает список очередей для заданного пространства имен, по умолчанию возвращаются очереди 100, если больше 100 очередей, которые нужно вернуть, и используйте параметр-MaxCount.

### Пример 3
```
PS C:\> Get-AzureRmServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Возвращает список первых очередей 150 для заданного пространства имен.

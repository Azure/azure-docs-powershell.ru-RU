# Test-AzDataLakeStoreAccount

## SYNOPSIS
Проверяет наличие учетной записи Data Lake Store.

## СИНТАКСИС

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Test-AzDataLakeStoreAccount** проверяет наличие учетной записи Data Lake Store.

## ПРИМЕРЫ

### Пример 1. Проверка учетной записи
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Эта команда проверяет, существует ли учетная запись с именем ContosoADL.

# Add-AzIotHubEventHubConsumerGroup

## КРАТКИй обзор
Создает группу потребителей eventhub.

## Максимальное

```
Add-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-EventHubConsumerGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Создает группу потребителей в Eventhub, связанной с указанным IotHub.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Добавление группы потребителей к eventhub телеметрии
```
PS C:\> Add-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubConsumerGroupName "myconsumergroup"
```

Добавляет новый consumergroup с именем "myconsumergroup" для событий eventhub для телеметрии в iothub с именем "myiothub"

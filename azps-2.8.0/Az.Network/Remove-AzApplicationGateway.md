# Remove-AzApplicationGateway

## КРАТКИй обзор
Удаляет шлюз приложения.

## Максимальное

```
Remove-AzApplicationGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApplicationGateway** удаляет шлюз приложения.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление указанного шлюза приложения
```
PS C:\>Remove-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
```

Эта команда удаляет шлюз приложения с именем ApplicationGateway01 в группе ресурсов с именем ResourceGroup01.

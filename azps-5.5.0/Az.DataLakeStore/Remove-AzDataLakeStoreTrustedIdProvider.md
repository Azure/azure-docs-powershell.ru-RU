# Remove-AzDataLakeStoreTrustedIdProvider

## SYNOPSIS
Удаление указанного надежного поставщика удостоверений из указанного магазина Data Lake Store.

## СИНТАКСИС

```
Remove-AzDataLakeStoreTrustedIdProvider [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для удаления указанного надежного поставщика удостоверений из указанного магазина Data Lake Store удаляется **cmdlet Remove-AzDataLakeStoreTrustedIdProvider.**

## ПРИМЕРЫ

### Пример 1. Удаление надежного поставщика удостоверений.
```
PS C:\> Remove-AzDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider
```

Удаление поставщика "MyProvider" из учетной записи ContosoADL

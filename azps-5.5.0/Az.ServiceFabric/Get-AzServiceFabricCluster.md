# Get-AzServiceFabricCluster

## SYNOPSIS
Получите сведения о ресурсах кластера.

## СИНТАКСИС

### BySubscription (по умолчанию)
```
Get-AzServiceFabricCluster [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceGroup
```
Get-AzServiceFabricCluster [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzServiceFabricCluster [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Сведения **о ресурсах кластера будет получать get-AzServiceFabricCluster.**

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzServiceFabricCluster -ResourceGroupName 'Group1' -ClusterName 'Contoso01SFCluster'
```

Эта команда получит сведения о ресурсах кластера для кластера "МойCluster".

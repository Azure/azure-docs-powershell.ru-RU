# Get-AzPolicyRemediation

## SYNOPSIS
Возвращает исправление политики.

## СИНТАКСИС

### SubscriptionScope (по умолчанию)
```
Get-AzPolicyRemediation [-Top <Int32>] [-Filter <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-Top <Int32>] [-IncludeDetail] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GenericScope
```
Get-AzPolicyRemediation -Scope <String> [-Top <Int32>] [-Filter <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ManagementGroupScope
```
Get-AzPolicyRemediation -ManagementGroupName <String> [-Top <Int32>] [-Filter <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupScope
```
Get-AzPolicyRemediation -ResourceGroupName <String> [-Top <Int32>] [-Filter <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzPolicyRemediation -ResourceId <String> [-Top <Int32>] [-IncludeDetail]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **получения всех исправлений** политики в области или в определенном исправлении с еготем вы получаете все изменения политики.

## ПРИМЕРЫ

### Пример 1. Получить все меры политики в текущей подписке
```
PS C:\> Select-AzSubscription -Subscription "My Subscription"
PS C:\> Get-AzPolicyRemediation
```

Эта команда получает все меры, созданные в подписке "Моя подписка" или под ней.

### Пример 2. Получите сведения об определенном исправлении политики и развертывании
```
PS C:\> Get-AzPolicyRemediation -ResourceGroupName "myResourceGroup" -Name "remediation1" -IncludeDetail
```

Эта команда получает исправление с именем "исправление1" из группы ресурсов "myResourceGroup". Будут включены сведения о том, что ресурсы будут исправлены.

### Пример 3. Получите 10 исправлений политики в группе управления с необязательными фильтрами
```
PS C:\> Get-AzPolicyRemediation -ManagementGroupName "mg1" -Top 10 -Filter "PolicyAssignmentId eq '/providers/Microsoft.Management/managementGroups/mg1/providers/Microsoft.Authorization/policyAssignments/pa1'"
```

Эта команда получает не более 10 исправлений политики из группы управления "mg1". Будут восстановлены только меры политики для данного назначения политики.

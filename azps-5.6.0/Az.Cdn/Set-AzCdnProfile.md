# Set-AzCdnProfile

## SYNOPSIS
Обновляет профиль сети CDN.

## СИНТАКСИС

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
**Настройка Set-AzCdnProfile** обновляет профиль сети доставки содержимого (CDN) Azure.

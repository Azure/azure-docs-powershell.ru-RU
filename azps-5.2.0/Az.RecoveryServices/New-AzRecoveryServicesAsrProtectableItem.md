# New-AzRecoveryServicesAsrProtectableItem

## SYNOPSIS
Добавление физического сервера в список защищенных элементов.

## СИНТАКСИС

```
New-AzRecoveryServicesAsrProtectableItem -ProtectionContainer <ASRProtectionContainer> -FriendlyName <String>
 -IPAddress <String> -OSType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
**New-AzRecoveryServicesAsrProtectableItem** добавляет новый защищенный элемент в список обнаруженных защищенных элементов в контейнере защиты в материале ASR (применимо только для типа ткань VMware).

## ПРИМЕРЫ

### Пример 1

Добавление физического сервера в список защищенных элементов. (autogenerated)

# Disable-AzSqlInstanceAdvancedDataSecurity

## КРАТКИй обзор
Отключение расширенной защиты данных на управляемом экземпляре.

## Максимальное

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Disable-AzSqlInstanceAdvancedDataSecurity** отключает расширенную защиту данных на управляемом экземпляре.

## ИЛЛЮСТРИРУЮТ

### Пример 1: отключение расширенной защиты данных управляемого экземпляра
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### Пример 2: отключение расширенной защиты данных сервера от управляемого ресурса экземпляра
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

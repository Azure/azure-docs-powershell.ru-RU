# Remove-AzSqlServerCommunicationLink

## КРАТКИй обзор
Удаляет связь для транзакций эластичных баз данных между двумя серверами.

## Максимальное

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzSqlServerCommunicationLink** удаляет связь "сервер-сервер" для транзакций эластичных баз данных в базе данных Azure SQL.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление коммуникационной ссылки
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

Эта команда удаляет связь "сервер-сервер" с именем Link01 на ContosoServer17.

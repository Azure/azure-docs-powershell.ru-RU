# Stop-AzSqlDatabaseActivity

## КРАТКИй обзор
Отменяет операцию асинхронных обновлений для базы данных.

## Максимальное

```
Stop-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Stop-AzSqlDatabaseActivity** отменяет операцию асинхронных обновлений для базы данных.

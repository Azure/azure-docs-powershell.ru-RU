# Get-AzBlueprintArtifact

## SYNOPSIS
Извлекать артефакты из определения схемы.

## СИНТАКСИС

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Извлекать артефакты из определения схемы. Если версия определения документа не указана, черновик будет извлечен. Если черновик версии не существует, возвращается последнее опубликованное определение документа.

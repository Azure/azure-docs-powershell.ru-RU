# Set-AzSqlDatabaseFailoverGroup

## SYNOPSIS
Изменяет конфигурацию группы отбойных SQL базы данных Azure.

## СИНТАКСИС

```
Set-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String>
 [-FailoverPolicy <FailoverPolicy>] [-GracePeriodWithDataLossHours <Int32>]
 [-AllowReadOnlyFailoverToPrimary <AllowReadOnlyFailoverToPrimary>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Эта команда изменяет конфигурацию группы отбойных SQL базы данных Azure.
Для выполнения команды следует использовать основной сервер группы отбойных команд.
Для управления набором баз данных в группе используйте add-AzSqlDatabaseToFailoverGroup и Remove-AzSqlDatabaseFromFailoverGroup.
Во время предварительного просмотра функции "Группы отката" для параметра "-GracePeriodWithDataLossHours" поддерживаются только значения, которые больше или равны 1 часу.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> $failoverGroup = Set-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg -FailoverPolicy Automatic -GracePeriodWithDataLossHours 1
```

Устанавливает для группы отключки политику отключки "Автоматически".

### Пример 2
```
PS C:\> $failoverGroup = Get-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg | Set-AzSqlDatabaseFailoverGroup -FailoverPolicy Manual
```

Устанавливает для группы отбойных отбойов политику отс неудачной передачи "Вручную", перенаправив в группу отбойных передач.

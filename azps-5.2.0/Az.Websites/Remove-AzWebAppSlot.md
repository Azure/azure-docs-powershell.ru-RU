# Remove-AzWebAppSlot

## SYNOPSIS

## СИНТАКСИС

### S1
```
Remove-AzWebAppSlot [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String> [-Slot] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzWebAppSlot [-Force] [-AsJob] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Remove-AzWebAppSlot** удаляется слот Azure Web App при условии, что группа ресурсов и имя веб-приложения были присвоены.
Этот cmdlet по умолчанию также удаляет все слоты и метрики.

## ПРИМЕРЫ

### Пример 1. Удаление слота веб-приложения
```
PS C:\>Remove-AzWebAppSlot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite" -Slot "Slot001"
```

Эта команда удаляет слот Slot001, связанный с веб-приложением ContosoSite, который принадлежит к группе ресурсов Default-Web-WestUS.

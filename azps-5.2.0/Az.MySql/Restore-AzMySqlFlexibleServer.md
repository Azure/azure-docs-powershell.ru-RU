# Restore-AzMySqlFlexibleServer

## SYNOPSIS
Восстановление сервера из существующей резервной копии

## СИНТАКСИС

```
Restore-AzMySqlFlexibleServer -Name <String> -ResourceGroupName <String> -InputObject <IServerAutoGenerated>
 -RestorePointInTime <DateTime> [-SubscriptionId <String>] [-Sku <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## ОПИСАНИЕ
Восстановление сервера из существующей резервной копии

## ПРИМЕРЫ

### Пример 1. Восстановление сервера MySql с помощью восстановления PointInTime
```powershell
PS C:\> $restorePointInTime = (Get-Date).AddMinutes(-10)
PS C:\> Get-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -ServerName mysql-test | Restore-AzMySqlFlexibleServer -Name mysql-test-restore -ResourceGroupName PowershellMySqlTest -RestorePointInTime $restorePointInTime 

Name               Location AdministratorLogin Version StorageProfileStorageMb SkuName   SkuTier       
----               -------- ------------------ ------- ----------------------- -------   -------       
mysql-test-restore eastus   mysql_test         5.7     10240                   Standard_D2ds_v4 GeneralPurpose
```

Эти cmdlets restore MySql server using PointInTime Restore.

## ПРИМЕЧАНИЯ

ALIASES

СЛОЖНЫЕ СВОЙСТВА ПАРАМЕТРОВ

Чтобы создать описанные ниже параметры, создайте hash table, содержащую нужные свойства. Чтобы получить сведения о hash tables, запустите Get-Help about_Hash_Tables.


INPUTOBJECT: <IServerAutoGenerated> исходный объект сервера, из который нужно восстановить.
  - `Location <String>`: географическое расположение ресурса
  - `SkuName <String>`: название SKU, например Standard_D32s_v3.
  - `SkuTier <SkuTier>`. Уровень конкретного SKU, например GeneralPurpose.
  - `[Tag <ITrackedResourceTags>]`: теги ресурсов.
    - `[(Any) <String>]`— это означает, что к этому объекту можно добавить любое свойство.
  - `[AdministratorLogin <String>]`: имя администратора для входа на сервер. Может быть указано только при создании сервера (и требуется для создания).
  - `[AdministratorLoginPassword <String>]`: пароль для входа администратора (требуется для создания сервера).
  - `[AvailabilityZone <String>]`: сведения о зоне доступности сервера.
  - `[CreateMode <CreateMode?>]`Режим создания нового сервера MySQL.
  - `[DelegatedSubnetArgumentSubnetArmResourceId <String>]`: делегированная ид ресурса подсети.
  - `[HaEnabled <HaEnabledEnum?>]`: включить HA или не использовать для сервера.
  - `[IdentityType <ResourceIdentityType?>]`: Тип удостоверения.
  - `[InfrastructureEncryption <InfrastructureEncryptionEnum?>]`Состояние, показывающая, включено ли шифрование инфраструктуры сервера.
  - `[MaintenanceWindowCustomWindow <String>]`: указывает, включено ли настраиваемое окно
  - `[MaintenanceWindowDayOfWeek <Int32?>]`: день недели в окне обслуживания
  - `[MaintenanceWindowStartHour <Int32?>]`: время начала обслуживания в окне обслуживания
  - `[MaintenanceWindowStartMinute <Int32?>]`: время начала технического обслуживания
  - `[PropertiesTag <IServerPropertiesTags>]`: метаданные конкретного приложения в виде пар ключевых значений.
    - `[(Any) <String>]`— это означает, что к этому объекту можно добавить любое свойство.
  - `[ReplicationRole <String>]`: Роль репликации.
  - `[RestorePointInTime <DateTime?>]`: время создания точки (формат ISO8601), указывав время для восстановления.
  - `[SourceServerId <String>]`: исходный код сервера MySQL.
  - `[SslEnforcement <SslEnforcementEnum?>]`: включить или не включить ssl-при подключении к серверу.
  - `[StorageProfileBackupRetentionDay <Int32?>]`: дней резервного копирования для хранения на сервере.
  - `[StorageProfileStorageAutogrow <StorageAutogrow?>]`: включить автоматическое рост хранилища.
  - `[StorageProfileStorageIop <Int32?>]`: IOPS хранилища для сервера.
  - `[StorageProfileStorageMb <Int32?>]`: Максимальное хранилище, разрешенное для сервера.
  - `[Version <ServerVersion?>]`: Версия сервера.

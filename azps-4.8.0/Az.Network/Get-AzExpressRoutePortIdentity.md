# Get-AzExpressRoutePortIdentity

## КРАТКИй обзор
Получение удостоверения, назначенного ExpressRoutePort.

## Максимальное

```
Get-AzExpressRoutePortIdentity -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzExpressRoutePortIdentity** получает удостоверение, назначенное локальному объекту Azure ExpressRoutePort.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $exrPort = Get-AzExpressRoutePort -Name $exrPortName -ResourceGroupName $resgpName
PS C:\> $identity = Get-AzExpressRoutePortIdentity -ExpressRoutePort $exrPort
```

# Get-AzureNetworkSecurityGroupConfig

## КРАТКИй обзор
Получение сведений о группе безопасности сети.

## Максимальное

```
Get-AzureNetworkSecurityGroupConfig [-Detailed] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureNetworkSecurityGroupConfig** возвращает сведения о сетевой группе безопасности.
Укажите *подробный* параметр для отображения правил сетевой безопасности.

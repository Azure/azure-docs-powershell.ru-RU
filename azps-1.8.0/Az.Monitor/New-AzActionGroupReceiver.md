# New-AzActionGroupReceiver

## КРАТКИй обзор
Создает нового приемника группы действий.

## Максимальное

### NewEmailReceiver (по умолчанию)
```
New-AzActionGroupReceiver -Name <String> [-EmailReceiver] -EmailAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### NewSmsReceiver
```
New-AzActionGroupReceiver -Name <String> [-SmsReceiver] [-CountryCode <String>] -PhoneNumber <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### NewWebhookReceiver
```
New-AzActionGroupReceiver -Name <String> [-WebhookReceiver] -ServiceUri <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzActionGroupReceiver** создает в памяти нового приемника группы действий.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание нового получателя электронной почты в памяти.
```
PS C:\>$emailReceiver = New-AzActionGroupReceiver -Name 'emailReceiver1' -EmailReceiver -EmailAddress 'user1@example.com'
```

Эта команда создает новый ресивер электронной почты в памяти.

### Пример 2: создание нового приемника SMS в памяти.
```
PS C:\>$smsReceiver = New-AzActionGroupReceiver -Name 'smsReceiver1' -SmsReceiver -CountryCode '1' -PhoneNumber '5555555555'
```

Эта команда создает новый ресивер SMS в памяти.

### Пример 3: создание в памяти нового приемника веб-перехватчиков.
```
PS C:\>$webhookReceiver = New-AzActionGroupReceiver -Name 'webhookReceiver1' -WebhookReceiver -ServiceUri 'http://test.com'
```

Эта команда создает в памяти нового приемника веб-перехватчиков.

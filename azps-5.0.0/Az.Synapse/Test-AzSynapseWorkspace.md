# Test-AzSynapseWorkspace

## КРАТКИй обзор
Проверка существования рабочей области аналитики Synapse.

## Максимальное

```
Test-AzSynapseWorkspace [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Test-AzSynapseWorkspace** проверяет наличие рабочей области для аналитики Synapse.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Test-AzSynapseWorkspace -Name ContosoWorkspace
```

Эта команда проверяет наличие рабочей области Analytics Synapse.

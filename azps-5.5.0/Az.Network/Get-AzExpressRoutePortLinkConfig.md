# Get-AzExpressRoutePortLinkConfig

## SYNOPSIS
Получает конфигурацию связи ExpressRoutePort.

## СИНТАКСИС

### ResourceNameParameterSet (по умолчанию)
```
Get-AzExpressRoutePortLinkConfig -ExpressRoutePort <PSExpressRoutePort> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePortLinkConfig -ResourceId <String> -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzExpressRoutePortLinkConfig** извлекает конфигурацию связи ExpressRoutePort.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -Name Link1
```

Получает конфигурацию Link1 для ExpressRoutePort $erport

### Пример 2
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -ResourceId $id
```

Получает конфигурацию связи с resourceId $id ExpressRoutePort $erport

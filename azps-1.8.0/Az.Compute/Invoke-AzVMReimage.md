# Invoke-AzVMReimage

## КРАТКИй обзор
Переобразировать виртуальную машину Azure.

## Максимальное

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Invoke-AzVMReimage** переобразует виртуальную машину Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Эта команда переобразует виртуальную машину с именем VirtualMachine07 в ResourceGroup11.

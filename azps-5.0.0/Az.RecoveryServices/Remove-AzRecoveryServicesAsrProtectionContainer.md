# Remove-AzRecoveryServicesAsrProtectionContainer

## КРАТКИй обзор
Удаляет указанный контейнер защиты из его структуры.

## Максимальное

```
Remove-AzRecoveryServicesAsrProtectionContainer -InputObject <ASRProtectionContainer>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzRecoveryServicesAsrProtectionContainer удаляет указанный контейнер защиты Azure Site Recovery.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Get-AzRecoveryServicesAsrProtectionContainer -Name xxxxx  -Fabric $fabric
PS C:\> Remove-AzRecoveryServicesAsrProtectionContainer -InputObject $protectionContainer
```

Запускает удаление указанного контейнера защиты и возвращает задание ASR, используемое для отслеживания операции удаления.

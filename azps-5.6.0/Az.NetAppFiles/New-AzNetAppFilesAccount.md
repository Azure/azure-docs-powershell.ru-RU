# New-AzNetAppFilesAccount

## SYNOPSIS
Создает новую учетную запись Azure NetApp Files (ANF).

## СИНТАКСИС

```
New-AzNetAppFilesAccount -ResourceGroupName <String> -Location <String> -Name <String>
 [-ActiveDirectory <PSNetAppFilesActiveDirectory[]>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания учетной записи ANF создается новый **cmdlet AzNetAppFilesAccount.**

## ПРИМЕРЫ

### Пример 1. Создание учетной записи ANF
```
PS C:\>New-AzNetAppFilesAccount -ResourceGroupName "MyRG" -Name "MyAnfAccount" -l "westus2"

Output:

Location          : westus2
Id                : /subscriptions/mySubs/resourceGroups/MyRG/providers/Microsoft.NetApp/netAppAccounts/MyAnfAccount
Name              : MyAnfAccount
Type              : Microsoft.NetApp/netAppAccounts
Tags              :
ProvisioningState : Succeeded
```

Эта команда создает новую учетную запись ANF "MyAnfAccount".

# Remove-AzApplicationGatewayUrlPathMapConfig

## КРАТКИй обзор
Удаление сопоставлений URL-путей с пулом серверов внутренней базы данных.

## Максимальное

```
Remove-AzApplicationGatewayUrlPathMapConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApplicationGatewayUrlPathMapConfig** Удаляет сопоставления URL-путей с пулом серверов внутренней базы данных.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление сопоставления URL-пути из шлюза приложений
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway $appgw -Name "map01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Первая команда получает шлюз приложения с именем appGwName и сохраняет результат в переменной $appgw.
Вторая команда удаляет сопоставление URL-пути с именем map01 из шлюза приложений.
Третья команда обновляет шлюз приложения.

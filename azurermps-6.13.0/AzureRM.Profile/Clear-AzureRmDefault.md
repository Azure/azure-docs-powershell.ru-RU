# Clear-AzureRmDefault

## КРАТКИй обзор
Очищает значения по умолчанию, заданные пользователем в текущем контексте.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Clear-AzureRmDefault [-ResourceGroup] [-PassThru] [-Force] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Clear-AzureRmDefault удаляет значения по умолчанию, заданные пользователем, в зависимости от параметров переключателей, указанных пользователем.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Clear-AzureRmDefault
```

Эта команда удаляет все значения по умолчанию, заданные пользователем в текущем контексте.

### Пример 1
```
PS C:\> Clear-AzureRmDefault -ResourceGroup
```

Эта команда удаляет группу ресурсов по умолчанию, установленную пользователем в текущем контексте.

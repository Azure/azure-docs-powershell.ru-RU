# Add-AzApplicationGatewayIPConfiguration

## КРАТКИй обзор
Добавление IP-конфигурации к шлюзу приложений.

## Максимальное

### SetByResourceId
```
Add-AzApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-SubnetId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResource
```
Add-AzApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-Subnet <PSSubnet>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Add-AzApplicationGatewayIPConfiguration** добавляет конфигурацию IP-адресов в шлюз приложений.
IP-конфигурации содержат подсеть, в которой развернут шлюз приложений.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Добавление конфигурации виртуальной сети к шлюзу приложений
```
PS C:\>$Vnet = Get-AzVirtualNetwork -Name "Vnet01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Subnet = Get-AzVirtualNetworkSubnetConfig -Name "Subnet01" -VirtualNetwork $Vnet 
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Appgwsubnet01" -Subnet $Subnet
```

Первая команда получает виртуальную сеть.
Вторая команда получает подсеть, используя ранее созданную виртуальную сеть.
Третья команда получает шлюз приложения и сохраняет его в переменной $AppGw.
Четвертая команда добавляет конфигурацию IP-адресов в шлюз приложений, хранящийся в $AppGw.

# Get-AzAvailableServiceAlias

## КРАТКИй обзор
Получить доступные псевдонимы служб в регионе.

## Максимальное

```
Get-AzAvailableServiceAlias -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzAvailableServiceAlias** позволяет получить все доступные псевдонимы служб для подсети в указанном расположении.

# Install-AzsUpdate

## КРАТКИй обзор
Применение определенного обновления на расположении обновления.

## Максимальное

### Apply (по умолчанию)
```
Install-AzsUpdate -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### ApplyViaIdentity
```
Install-AzsUpdate -InputObject <IUpdateAdminIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## NОПИСАНИЕ
Применение определенного обновления на расположении обновления.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Install-AzsUpdate по имени
```powershell
PS C:\> Install-AzsUpdate -Name Microsoft1.1907.0.10
```

Unifiedgroup позволяет устанавливать конкретные обновления по имени.
Обратите внимание на то, что версия обновления строго превышает текущую версию.

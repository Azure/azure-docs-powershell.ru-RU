# Get-AzDataShare

## SYNOPSIS
Получите сведения о информационных акциях.

## СИНТАКСИС

### ByFieldsParameterSet (по умолчанию)
```
Get-AzDataShare -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShare -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его использованием** можно получить сведения об акциях данных в accoount в azure Data Share.
Если указать имя для нужного поделиться данными, этот cmdlet получит сведения об этом поделиться данными.
Если имя не указано, этот cmdlet получает сведения обо всех данных в учетной записи azure data share.

## ПРИМЕРЫ

### Пример 1. Получить определенный поделиться данными
```
PS C:\>Get-AzDataShare -ResourceGroupName "ADS" -AccountName "WikiAdsAccount" -Name "AdsShare"
Name                : AdsShare
Id                  : /subscriptions/f3ead1ff-d0ab-4cf4-9a5a-86f1661d4685/resourceGroups/ADS/providers/Microsoft.DataShare/accounts/WikiAdsAccount/shares/AdsShare
Type                : Microsoft.DataShare/shares
CreatedAt           : 6/11/2019 12:00:00 AM
CreatedBy           : Contoso ADS
ShareKind           : CopyBased
Description         : Example of description  
ProvisioningState   : Succeeded
Terms               : This should not be shared
```

Эта команда отображает сведения о совместном использование данных AdsShare в учетной записи обмена данными Azure WikiAdsAccount и группе ресурсов ADS.

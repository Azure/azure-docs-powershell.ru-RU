# Remove-AzRedisCacheLink

## SYNOPSIS
Удаление связи геокопии между двумя кэшами Redis.

## СИНТАКСИС

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Удаление связи геокопии между двумя кэшами Redis.

## ПРИМЕРЫ

### Пример 1. Удаление ссылки на геокопию
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Эта команда удаляет ссылки гео-репликации, где кэш Redis с именем mycache1 является основным, а кэш Redis с именем mycache2 — дополнительным.

# Remove-AzResourceGroupDeployment

## КРАТКИй обзор
Удаляет развертывание группы ресурсов и связанные с ней операции.

## Максимальное

### RemoveByResourceGroupName (по умолчанию)
```
Remove-AzResourceGroupDeployment [-ResourceGroupName] <String> [-Name] <String> [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceGroupDeploymentId
```
Remove-AzResourceGroupDeployment -Id <String> [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ

Командлет **Remove-AzResourceGroupDeployment** удаляет развертывание группы ресурсов Azure и все связанные операции.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление развертывания группы ресурсов с ResourceId

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceId /subscriptions/{subId}/resourceGroups/testGroup/providers/Microsoft.Resources/deployments/testDeployment1

True
```

Эта команда удаляет развертывание группы ресурсов с полным идентификатором ресурса для развертывания.
Успешное удаление возвращает значение истина.

### Пример 2: Удаление развертывания группы ресурсов с ResourceGroupName и ResourceName

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceGroupName testGroup -Name testDeployment1

True
```

Эта команда удаляет развертывание группы ресурсов с предоставленными ResourceGroupName и ResourceName.
Успешное удаление возвращает значение истина.

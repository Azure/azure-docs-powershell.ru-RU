# New-AzIotSecuritySolutionUserDefinedResourcesObject

## КРАТКИй обзор
Создание новых ресурсов, определенных пользователем, для решения безопасности IOT

## Максимальное

```
New-AzIotSecuritySolutionUserDefinedResourcesObject -Query <String> -QuerySubscriptionList <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет AzIotSecuritySolutionUserDefinedResourcesObject создает новый объект ресурсов, определяемый пользователем, для решения безопасности IOT.

## ИЛЛЮСТРИРУЮТ

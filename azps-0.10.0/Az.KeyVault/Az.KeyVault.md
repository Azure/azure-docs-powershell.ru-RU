# AZ. KeyVault Module
## Nописание
В этом разделе отображаются разделы справки для командлетов хранилища ключей Azure.

## Командлеты AZ. KeyVault
### [Add-AzKeyVaultCertificate](Add-AzKeyVaultCertificate.md)
Добавляет сертификат в хранилище ключей.

### [Add-AzKeyVaultCertificateContact](Add-AzKeyVaultCertificateContact.md)
Добавляет контакт для уведомлений сертификата.

### [Add-AzKeyVaultKey](Add-AzKeyVaultKey.md)
Создание ключа в хранилище ключей или импорт ключа в хранилище ключей.

### [Add-AzKeyVaultManagedStorageAccount](Add-AzKeyVaultManagedStorageAccount.md)
Добавляет существующую учетную запись хранилища Azure в заданное хранилище ключей для ключей, управляемых службой хранилища ключей.

### [Backup-AzKeyVaultKey](Backup-AzKeyVaultKey.md)
Резервное копирование ключа в хранилище ключей.

### [Backup-AzKeyVaultSecret](Backup-AzKeyVaultSecret.md)
Резервное копирование секрета в хранилище ключей.

### [Get-AzKeyVaultCertificate](Get-AzKeyVaultCertificate.md)
Получает сертификат из хранилища ключей.

### [Get-AzKeyVaultCertificateContact](Get-AzKeyVaultCertificateContact.md)
Возвращает контакты, которые зарегистрированы для уведомлений сертификата для хранилища ключей.

### [Get-AzKeyVaultCertificateIssuer](Get-AzKeyVaultCertificateIssuer.md)
Возвращает поставщика сертификата для хранилища ключей.

### [Get-AzKeyVaultCertificateOperation](Get-AzKeyVaultCertificateOperation.md)
Возвращает состояние операции с сертификатом.

### [Get-AzKeyVaultCertificatePolicy](Get-AzKeyVaultCertificatePolicy.md)
Возвращает политику для сертификата в хранилище ключей.

### [Get-AzKeyVaultKey](Get-AzKeyVaultKey.md)
Получает ключи хранилища ключей.

### [Get-AzKeyVaultManagedStorageAccount](Get-AzKeyVaultManagedStorageAccount.md)
Получение управляемых учетных записей хранилища Azure для хранилища ключей.

### [Get-AzKeyVaultManagedStorageSasDefinition](Get-AzKeyVaultManagedStorageSasDefinition.md)
Получение определений SAS хранилища, управляемого основным хранилищем.

### [Get-AzKeyVaultSecret](Get-AzKeyVaultSecret.md)
Возвращает секреты в хранилище ключей.

### [Get-AzKeyVault](Get-AzKeyVault.md)
Получает ключевые хранилища.

### [Import-AzKeyVaultCertificate](Import-AzKeyVaultCertificate.md)
Импорт сертификата в хранилище ключей.

### [New-AzKeyVaultCertificateAdministratorDetails](New-AzKeyVaultCertificateAdministratorDetails.md)
Создает объект подробных данных администратора сертификата в памяти.

### [New-AzKeyVaultCertificateOrganizationDetails](New-AzKeyVaultCertificateOrganizationDetails.md)
Создает объект подробных сведений об организации сертификата в памяти.

### [New-AzKeyVaultCertificatePolicy](New-AzKeyVaultCertificatePolicy.md)
Создает объект политики сертификата в памяти.

### [New-AzKeyVault](New-AzKeyVault.md)
Создание хранилища ключей.

### [Remove-AzKeyVaultCertificate](Remove-AzKeyVaultCertificate.md)
Удаляет сертификат из хранилища ключей.

### [Remove-AzKeyVaultCertificateContact](Remove-AzKeyVaultCertificateContact.md)
Удаляет контакт, зарегистрированный для уведомлений о сертификате из хранилища ключей.

### [Remove-AzKeyVaultCertificateIssuer](Remove-AzKeyVaultCertificateIssuer.md)
Удаление поставщика сертификата из хранилища ключей.

### [Remove-AzKeyVaultCertificateOperation](Remove-AzKeyVaultCertificateOperation.md)
Удаляет операцию с сертификатом из хранилища ключей.

### [Remove-AzKeyVaultKey](Remove-AzKeyVaultKey.md)
Удаление ключа из хранилища ключей.

### [Remove-AzKeyVaultManagedStorageAccount](Remove-AzKeyVaultManagedStorageAccount.md)
Удаляет управляемую учетную запись хранилища Azure с основным хранилищем и все связанные определения SAS.

### [Remove-AzKeyVaultManagedStorageSasDefinition](Remove-AzKeyVaultManagedStorageSasDefinition.md)
Удаляет определения SAS из хранилища ключей для управляемых хранилищ Azure.

### [Remove-AzKeyVaultSecret](Remove-AzKeyVaultSecret.md)
Удаление секрета в хранилище ключей.

### [Remove-AzKeyVault](Remove-AzKeyVault.md)
Удаление хранилища ключей.

### [Remove-AzKeyVaultAccessPolicy](Remove-AzKeyVaultAccessPolicy.md)
Удаляет все разрешения для пользователя или приложения из хранилища ключей.

### [Restore-AzKeyVaultKey](Restore-AzKeyVaultKey.md)
Создание ключа в хранилище ключей из резервной копии ключа.

### [Restore-AzKeyVaultSecret](Restore-AzKeyVaultSecret.md)
Создает секретный ключ из резервного хранилища ключей.

### [Set-AzKeyVaultCertificateAttribute](Set-AzKeyVaultCertificateAttribute.md)
Изменяет редактируемые атрибуты сертификата.

### [Set-AzKeyVaultCertificateIssuer](Set-AzKeyVaultCertificateIssuer.md)
Задает поставщика сертификата в хранилище ключей.

### [Set-AzKeyVaultCertificatePolicy](Set-AzKeyVaultCertificatePolicy.md)
Создание или обновление политики сертификата в хранилище ключей.

### [Set-AzKeyVaultKeyAttribute](Set-AzKeyVaultKeyAttribute.md)
Обновляет атрибуты ключа в хранилище ключей.

### [Set-AzKeyVaultManagedStorageSasDefinition](Set-AzKeyVaultManagedStorageSasDefinition.md)
Задает определение сигнатуры общего доступа (SAS) с помощью хранилища ключей для указанной учетной записи хранения для хранилища ключей в управляемом хранилище Azure.

### [Set-AzKeyVaultSecret](Set-AzKeyVaultSecret.md)
Создание или обновление секрета в хранилище ключей.

### [Set-AzKeyVaultSecretAttribute](Set-AzKeyVaultSecretAttribute.md)
Обновляет атрибуты секрета в хранилище ключей.

### [Set-AzKeyVaultAccessPolicy](Set-AzKeyVaultAccessPolicy.md)
Предоставление или изменение существующих разрешений для пользователя, приложения или группы безопасности для выполнения операций с помощью хранилища ключей.

### [Остановить-AzKeyVaultCertificateOperation](Stop-AzKeyVaultCertificateOperation.md)
Отменяет операцию с сертификатом в хранилище ключей.

### [Undo-AzKeyVaultCertificateRemoval](Undo-AzKeyVaultCertificateRemoval.md)
Восстановление удаленного сертификата из хранилища ключей в активном состоянии.

### [Undo-AzKeyVaultKeyRemoval](Undo-AzKeyVaultKeyRemoval.md)
Восстановление удаленного ключа из хранилища ключей в активном состоянии.

### [Undo-AzKeyVaultSecretRemoval](Undo-AzKeyVaultSecretRemoval.md)
Восстановление удаленного секрета из хранилища ключей в активном состоянии.

### [Undo-AzKeyVaultRemoval](Undo-AzKeyVaultRemoval.md)
Восстановление удаленного хранилища ключей в активном состоянии.

### [Update-AzKeyVaultManagedStorageAccount](Update-AzKeyVaultManagedStorageAccount.md)
Обновление редактируемых атрибутов хранилища ключей управляемой учетной записи хранилища Azure.

### [Update-AzKeyVaultManagedStorageAccountKey](Update-AzKeyVaultManagedStorageAccountKey.md)
Повторное создание указанного ключа учетной записи хранения для хранилища ключей из управляемого хранилища Azure.

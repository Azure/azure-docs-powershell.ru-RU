# Get-AzureRmApplicationGatewayAuthenticationCertificate

## КРАТКИй обзор
Возвращает сертификат для проверки подлинности для шлюза приложения.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmApplicationGatewayAuthenticationCertificate [-Name <String>]
 -ApplicationGateway <PSApplicationGateway> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmApplicationGatewayAuthenticationCertificate** получает сертификат проверки подлинности для шлюза приложений Azure.

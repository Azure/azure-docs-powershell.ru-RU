# Remove-AzureRmSchedulerJobCollection

## КРАТКИй обзор
Удаляет коллекцию заданий.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmSchedulerJobCollection -ResourceGroupName <String> -JobCollectionName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmSchedulerJobCollection** удаляет коллекцию заданий в планировщике Azure.

# Get-AzSqlSyncAgent

## SYNOPSIS
Возвращает сведения об агентах синхронизации Azure SQL Azure.

## СИНТАКСИС

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Get-AzSqlSyncAgent** возвращает сведения об одном или SQL агентов синхронизации Azure.
Укажите имя агента синхронизации, чтобы увидеть сведения только для этого агента.

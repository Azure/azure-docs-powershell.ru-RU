# Disable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
Отключение advanced Data Security для управляемого экземпляра.

## СИНТАКСИС

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Из-за cmdlet **Disable-AzSqlInstanceAdvancedDataSecurity** в управляемом экземпляре отключена служба Advanced Data Security.

## ПРИМЕРЫ

### Пример 1. Отключение управляемого экземпляра Advanced Data Security
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### Пример 2. Отключение advanced Data Security сервера от ресурса управляемого экземпляра
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

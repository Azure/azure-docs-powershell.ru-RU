# Модуль AZ. резервирования
## Nописание
В этом разделе приведены командлеты PowerShell Azure для резервирований Azure в платформе диспетчера ресурсов Azure (ARM). Командлеты существуют в пространстве имен Microsoft. Azure. Commands. резервирования.

## Командлеты AZ. резервирования
### [Get-AzReservation](Get-AzReservation.md)
Получить `Reservation` s в заданном заказе на резервирование

### [Get-AzReservationCatalog](Get-AzReservationCatalog.md)
Получение каталога доступных резервирований

### [Get-AzReservationHistory](Get-AzReservationHistory.md)
Получение `Reservation` журнала изменений.

### [Get-AzReservationOrder](Get-AzReservationOrder.md)
Получить `ReservationOrder`

### [Get-AzReservationOrderId](Get-AzReservationOrderId.md)
Получить список приемлемых `ReservationOrder` идентификаторов.

### [Merge-AzReservation](Merge-AzReservation.md)
Объединение двух `Reservation` s.

### [Разделение — AzReservation](Split-AzReservation.md)
Разделение а `Reservation` .

### [Update-AzReservation](Update-AzReservation.md)
Обновление `Reservation` .

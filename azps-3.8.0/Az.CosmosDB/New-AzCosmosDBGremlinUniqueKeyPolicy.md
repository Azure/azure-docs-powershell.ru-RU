# New-AzCosmosDBGremlinUniqueKeyPolicy

## КРАТКИй обзор
Создает новый объект CosmosDB UniqueKeyPolicy.

## Максимальное

```
New-AzCosmosDBGremlinUniqueKeyPolicy -UniqueKey <PSUniqueKey[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzCosmosDBGremlinUniqueKeyPolicy** создает новый объект типа PSUniqueKeyPolicy.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $uk = New-AzCosmosDBGremlinUniqueKey -Path "abc"

 New-AzCosmosDBGremlinUniqueKeyPolicy -UniqueKey $uk,$uk
```

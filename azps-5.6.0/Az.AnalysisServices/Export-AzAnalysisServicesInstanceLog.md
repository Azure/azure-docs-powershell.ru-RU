# Export-AzAnalysisServicesInstanceLog

## SYNOPSIS
Экспорт журнала с сервера Analysis Services в текущий логин в среде, как указано в Add-AzAnalysisServicesAccount

## СИНТАКСИС

```
Export-AzAnalysisServicesInstanceLog -OutputPath <String> [-Force] [-Instance] <String> [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Export-AzAnalysisServicesInstance-файл экспортирует журнал из экземпляра сервера Azure Analysis Services в файл

## ПРИМЕРЫ

### Пример 1
```
PS C:\>Export-AzAnalysisServicesInstanceLog -Instance testserver -OutputPath C:\path\to\log\testserver.log
```

Эта команда экспортирует журнал с сервера "testserver" в среде, указанной в команде Add-AzAnalysisServicesAccount, и сохраняет его в файл, указанный в OutputPath 'C:\path\to\log\testserver.log"

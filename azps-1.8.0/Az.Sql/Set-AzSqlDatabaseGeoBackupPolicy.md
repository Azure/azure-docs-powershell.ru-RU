# Set-AzSqlDatabaseGeoBackupPolicy

## КРАТКИй обзор
Задает политику географического резервного копирования базы данных.

## Максимальное

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzSqlDatabaseGeoBackupPolicy** задает политику географического резервного копирования, зарегистрированную в базе данных.
Это ресурс резервного копирования Azure, который используется для определения политики хранения резервных копий.

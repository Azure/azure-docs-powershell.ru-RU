# Get-AzApplicationGatewayRewriteRuleSet

## КРАТКИй обзор
Возвращает набор правил перезаписи для шлюза приложения.

## Максимальное

```
Get-AzApplicationGatewayRewriteRuleSet [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Возвращает набор правил перезаписи для шлюза приложения.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение определенного набор правил для перезаписи
```
PS C:\> $AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Rule = Get-AzApplicationGatewayRewriteRuleSet -Name "RuleSet01" -ApplicationGateway $AppGW
```

Первая команда получает шлюз приложения с именем ApplicationGateway01 и сохраняет результат в переменной с именем $AppGW.
Вторая команда получает наборы правил перезаписи с именем RuleSet01 из шлюза приложения, который хранится в переменной с именем $AppGW.

# Get-AzSqlInstance

## КРАТКИй обзор
Возвращает сведения о экземпляре управляемой базы данных Azure SQL.

## Максимальное

```
Get-AzSqlInstance [[-Name] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlInstance** возвращает сведения об одном или нескольких экземплярах, УПРАВЛЯЕМЫХ Azure SQL.
Укажите имя экземпляра, чтобы просмотреть сведения только для этого экземпляра.

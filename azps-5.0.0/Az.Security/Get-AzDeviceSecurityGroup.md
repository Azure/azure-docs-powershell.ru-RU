# Get-AzDeviceSecurityGroup

## КРАТКИй обзор
Получение группы безопасности устройства (безопасность в центре IoT)

## Максимальное

### ResourceIdScope (по умолчанию)
```
Get-AzDeviceSecurityGroup -HubResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdLevelResource
```
Get-AzDeviceSecurityGroup -HubResourceId <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Get-AzDeviceSecurityGroup возвращает группу безопасности устройства, определенную в решении системы безопасности IOT

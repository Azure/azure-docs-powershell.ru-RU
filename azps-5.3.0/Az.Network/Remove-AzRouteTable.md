# Remove-AzRouteTable

## SYNOPSIS
Удаляет таблицу маршрутов.

## СИНТАКСИС

```
Remove-AzRouteTable -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При этом таблица маршрутов Azure удаляется с использованием **cmdlet Remove-AzRouteTable.**

## ПРИМЕРЫ

### Пример 1. Удаление таблицы маршрутов
```
PS C:\>Remove-AzRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Эта команда удаляет таблицу маршрутов с именем RouteTable01 в группе ресурсов "Группа ресурсов11".
Перед удалением таблицы будет предложено подтвердить удаление.

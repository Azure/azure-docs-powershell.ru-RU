# Get-AzVirtualNetworkTap

## SYNOPSIS
Получает виртуальное сетевое касание

## СИНТАКСИС

### ListParameterSet (по умолчанию)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Командлет **Get-AzVirtualNetworkTap** получает виртуальный сетевой касание Azure или список виртуальных сетей Azure, которые коснитесь группы ресурсов.

## ПРИМЕРЫ

### Пример 1. Виртуальный сетевой касание
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

Эта команда получает ссылку на tap VirtualNetwork для данного "VirtualTap1" в "ResourceGroup1".

### Пример 2. Все виртуальные сетевые касания с помощью фильтрации
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

Эта команда получает все ссылки на virtualNetwork, которые начинаются с VirtualTap.

# Get-AzDataLakeStoreItem

## SYNOPSIS
Получает сведения о файле или папке в Data Lake Store.

## СИНТАКСИС

```
Get-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения сведений о файле или папке в магазине Data Lake Store можно получить подробные сведения о файле или папке **Get-AzDataLakeStoreItem.**

## ПРИМЕРЫ

### Пример 1. Просмотр сведений о файле из Магазина "Озеро данных"
```
PS C:\>Get-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/MyFiles/Test.csv"
```

Эта команда получает сведения о файлеTest.csv из магазина Data Lake Store.

# New-AzApplicationGatewayFirewallPolicyManagedRule

## SYNOPSIS
Создайте ManagedRules для политики брандмауэра.

## СИНТАКСИС

```
New-AzApplicationGatewayFirewallPolicyManagedRule
 [-ManagedRuleSet <PSApplicationGatewayFirewallPolicyManagedRuleSet[]>]
 [-Exclusion <PSApplicationGatewayFirewallPolicyExclusion[]>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
**Новое-AzApplicationGatewayFirewallPolicyManagedRule** создает управляемые правила для политики брандмауэра.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallPolicyManagedRule -ManagedRuleSet $managedRuleSet -Exclusion $exclusion1,$exclusion2
```

Команда создает список управляемых правил ManagedRuleSet со списком $managedRuleSet и списком исключений с записями $exclusion 1, $exclusion 2.

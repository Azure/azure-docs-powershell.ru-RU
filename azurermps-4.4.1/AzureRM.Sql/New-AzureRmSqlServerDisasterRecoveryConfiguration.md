# New-AzureRmSqlServerDisasterRecoveryConfiguration

## КРАТКИй обзор
Создает конфигурацию восстановления системы сервера баз данных.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmSqlServerDisasterRecoveryConfiguration -VirtualEndpointName <String>
 -PartnerResourceGroupName <String> -PartnerServerName <String> [-FailoverPolicy <String>]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmSqlServerDisasterRecoveryConfiguration** создает конфигурацию восстановления системы для сервера базы данных SQL.

# Remove-AzSqlServerMSSupportAudit

## SYNOPSIS
Удаляет параметры аудита операций поддержки Майкрософт для azure SQL сервера.

## СИНТАКСИС

### ServerParameterSet (по умолчанию)
```
Remove-AzSqlServerMSSupportAudit [-ResourceGroupName] <String> [-ServerName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ServerObjectParameterSet
```
Remove-AzSqlServerMSSupportAudit -ServerObject <AzureSqlServerModel> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для удаления параметров аудита операций поддержки microsoft для сервера Azure SQL удаляется **cmdlet Remove-AzSqlServerMSSupportAudit.**
Чтобы использовать этот cmdlet, используйте параметры *ResourceGroupName* и *ServerName* для определения сервера.

## ПРИМЕРЫ

### Пример 1. Удаление параметров аудита операций поддержки Майкрософт для сервера Azure SQL
```powershell
PS C:\>Remove-AzSqlServerMSSupportAudit -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

### Пример 2. Удаление (через конвейер) параметров аудита сервера Azure SQL Майкрософт
```
PS C:\> Get-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Remove-AzSqlServerMSSupportAudit
```

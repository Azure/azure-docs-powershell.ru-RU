# New-AzP2sVpnGateway

## SYNOPSIS
Создайте новую веб-часть P2SVpnGateway в VirtualHub, чтобы указать на возможность подключения к сайту.

## ОПИСАНИЕ
С помощью **cmdlet New-AzP2sVpnGateway** можно создать в VirtualHub новую технологию P2SVpnGateway для подключения point к сайту от Point к клиентам сайта к Azure VirtualWan.

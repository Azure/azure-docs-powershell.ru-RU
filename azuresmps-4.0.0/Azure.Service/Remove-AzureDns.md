# Remove-AzureDns

## КРАТКИй обзор
Удаляет DNS-сервер из службы Azure.

## Максимальное

```
Remove-AzureDns [-Name] <String> [-ServiceName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureDns** удаляет DNS-сервер из службы Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление DNS-сервера из службы
```
PS C:\> Remove-AzureDns -ServiceName "ContosoService" -Name "Dns07"
```

Эта команда удаляет DNS-сервер с именем Dns07 из ContosoService.

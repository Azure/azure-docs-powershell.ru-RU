# Set-AzSynapseDataFlow

## КРАТКИй обзор
Создает или обновляет поток данных в рабочей области.

## Максимальное

### SetByName (по умолчанию)
```
Set-AzSynapseDataFlow -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataFlow -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzSynapseDataFlow** создает поток данных или обновляет существующий поток данных в рабочей области.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Set-AzSynapseDataFlow -WorkspaceName ContosoWorkspace -Name ContosoDataFlow -DefinitionFile "C:\\samples\\DataFlow.json"
```

Эта команда создает поток данных с именем ContosoDataFlow в рабочей области с именем ContosoWorkspace.
В команде поток данных заосновывается на сведениях в DataFlow.jsфайле.

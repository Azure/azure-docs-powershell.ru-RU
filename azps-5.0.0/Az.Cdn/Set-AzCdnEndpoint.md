# Set-AzCdnEndpoint

## КРАТКИй обзор
Обновляет конечную точку CDN.

## Максимальное

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzCdnEndpoint** обновляет конечную точку сети доставки содержимого (CDN) для Azure.

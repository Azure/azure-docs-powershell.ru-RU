# Get-AzDiskAccess

## КРАТКИй обзор
Возвращает свойства доступа к диску.

## Максимальное

### DefaultParameterSet (по умолчанию)
```
Get-AzDiskAccess [[-ResourceGroupName] <String>] [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIDParameterSet
```
Get-AzDiskAccess [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzDiskAccess** получает свойства доступа к диску.

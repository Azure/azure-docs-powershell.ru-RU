# Set-AzSqlServerVirtualNetworkRule

## SYNOPSIS
Изменяет конфигурацию правила виртуальной SQL Server Azure.

## СИНТАКСИС

```
Set-AzSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Эта команда изменяет конфигурацию правила виртуальной сети Azure SQL Server.
Для управления набором виртуальных правил сети на сервере используйте add-AzSqlServerVirtualNetworkRule и Remove-AzSqlServerVirtualNetworkRule.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> $virtualNetworkRule = Set-AzSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Изменяет существующее правило виртуальной сети с помощью нового id подсети виртуальной сети, который содержит сведения о новой виртуальной сети.

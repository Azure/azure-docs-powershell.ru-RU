# Get-AzAutomationCredential

## КРАТКИй обзор
Получает учетные данные автоматизации.

## Максимальное

### ByAll (по умолчанию)
```
Get-AzAutomationCredential [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationCredential [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzAutomationCredential** получает один или несколько учетных данных службы автоматизации Azure.
По умолчанию возвращаются все учетные данные.
Укажите имя учетных данных для получения определенных учетных данных.
В целях безопасности этот командлет не возвращает пароли учетных данных.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение всех учетных данных
```
PS C:\>Get-AzAutomationCredential -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17"
```

Эта команда получает метаданные для всех учетных данных в учетной записи автоматизации с именем Contoso17.

### Пример 2: получение учетных данных
```
PS C:\>Get-AzAutomationCredential -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17" -Name "ContosoCredential"
```

Эта команда получает метаданные для учетных данных с именем ContosoCredential в учетной записи автоматизации с именем Contoso17.

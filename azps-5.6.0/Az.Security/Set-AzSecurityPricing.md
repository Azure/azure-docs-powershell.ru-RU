# Set-AzSecurityPricing

## SYNOPSIS

Включает или отключает планы Защитника Azure для подписки в Центре безопасности Azure.

## СИНТАКСИС

### SubscriptionLevelResource (по умолчанию)

```powershell
Set-AzSecurityPricing -Name <String> -PricingTier <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObject

```powershell
Set-AzSecurityPricing -InputObject <PSSecurityPricing> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ

Включив или отключив любой из планов Защитника Azure для подписки.

Дополнительные сведения о Защитнике Azure и доступных планах см. [в обзоре защитника Azure.](https://docs.microsoft.com/azure/security-center/azure-defender)

## ПРИМЕРЫ

### Пример 1

```powershell
PS C:\> Set-AzSecurityPricing -Name "virtualmachines" -PricingTier "Standard"
```

Включает Защитник **Azure для серверов** для подписки.

"Стандартный" означает состояние "В этом" для плана Azure Defender, как показано в области цен и параметров Центра безопасности Azure на портале Azure.

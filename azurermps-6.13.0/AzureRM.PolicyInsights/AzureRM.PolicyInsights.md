# Модуль AzureRM. PolicyInsights
## Nописание
В этом разделе отображается справка по командлетам политики Azure Insights.

## Командлеты AzureRM. PolicyInsights
### [Get-AzureRmPolicyEvent](Get-AzureRmPolicyEvent.md)
Получение событий оценки политики, возникающих при создании или обновлении ресурсов.

### [Get-AzureRmPolicyRemediation](Get-AzureRmPolicyRemediation.md)
Получение исправлений политики.

### [Get-AzureRmPolicyState](Get-AzureRmPolicyState.md)
Возвращает состояние соответствия политике для ресурсов.

### [Get-AzureRmPolicyStateSummary](Get-AzureRmPolicyStateSummary.md)
Получает актуальные сведения о состояниях соответствия политике для ресурсов.

### [Remove-AzureRmPolicyRemediation](Remove-AzureRmPolicyRemediation.md)
Удаление исправления политики.

### [Start-AzureRmPolicyRemediation](Start-AzureRmPolicyRemediation.md)
Создание и запуск исправления политики для назначения политики.

### [Остановить-AzureRmPolicyRemediation](Stop-AzureRmPolicyRemediation.md)
Отмена выполняющегося исправления политики.

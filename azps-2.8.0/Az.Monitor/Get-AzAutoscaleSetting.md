# Get-AzAutoscaleSetting

## КРАТКИй обзор
Получает параметры автомасштабирования.

## Максимальное

```
Get-AzAutoscaleSetting -ResourceGroupName <String> [-Name <String>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzAutoscaleSetting** получает все параметры автомасштабирования, связанные с группой ресурсов или с указанным параметром автомасштабирования.

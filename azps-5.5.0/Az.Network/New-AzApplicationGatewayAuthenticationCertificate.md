# New-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Создает сертификат проверки подлинности для шлюза приложения.

## СИНТАКСИС

```
New-AzApplicationGatewayAuthenticationCertificate -Name <String> -CertificateFile <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Новый-AzApplicationGatewayAuthenticationCertificate** создает сертификат проверки подлинности для шлюза приложения Azure.

## ПРИМЕРЫ

### Пример 1. Создание сертификата проверки подлинности
```
PS C:\> $cert = New-AzApplicationGatewayAuthenticationCertificate -Name "cert01" -CertificateFile "C:\cert.cer"
```

Первая команда создает сертификат проверки подлинности с именем cert01.

# Get-AzMySqlFirewallRule

## SYNOPSIS
Сведения о правиле брандмауэра сервера.

## СИНТАКСИС

### Список (по умолчанию)
```
Get-AzMySqlFirewallRule -ResourceGroupName <String> -ServerName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Получить
```
Get-AzMySqlFirewallRule -Name <String> -ResourceGroupName <String> -ServerName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzMySqlFirewallRule -InputObject <IMySqlIdentity> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## ОПИСАНИЕ
Сведения о правиле брандмауэра сервера.

## ПРИМЕРЫ

### Пример 1. Список всех правил брандмауэра на указанном сервере MySql
```powershell
PS C:\> Get-AzMySqlFirewallRule -ResourceGroupName PowershellMySqlTest -ServerName mysql-test

Name Type
---- ----
rule Microsoft.DBforMySQL/servers/firewallRules
```

Этот cmdlet перечисляет все правило брандмауэра на указанном сервере MySql.

### Пример 2. Получить правило брандмауэра по имени
```powershell
PS C:\> Get-AzMySqlFirewallRule -Name rule -ResourceGroupName PowershellMySqlTest -ServerName mysql-test

Name Type
---- ----
rule Microsoft.DBforMySQL/servers/firewallRules
```

Этот cmdlet получает правило брандмауэра по имени.

### Пример 3. Получить правило брандмауэра по удостоверениям
```powershell
PS C:\> $ID = "/subscriptions/<SubscriptionId>/resourceGroups/PowershellMySqlTest/providers/Microsoft.DBforMySQL/servers/mysql-test/firewallRules/rule"
PS C:\> Get-AzMySqlFirewallRule -InputObject $ID

Name Type
---- ----
rule Microsoft.DBforMySQL/servers/firewallRules
```

Этот cmdlet получает правило брандмауэра по удостоверениям.

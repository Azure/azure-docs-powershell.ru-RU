# New-AzVirtualHubRoute

## SYNOPSIS
Создает объект виртуального концентратора Azure Route.

## СИНТАКСИС

```
New-AzVirtualHubRoute -AddressPrefix <String[]> -NextHopIpAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Создает объект виртуального концентратора Azure Route.

## ПРИМЕРЫ

### Пример 1

```powershell
PS C:\> $route1 = New-AzVirtualHubRoute -AddressPrefix @("10.0.0.0/16", "11.0.0.0/16") -NextHopIpAddress "12.0.0.5"

AddressPrefixes            NextHopIpAddress
---------------            ----------------
{10.0.0.0/16, 11.0.0.0/16} 12.0.0.5
```

Выше будет создаваться объект виртуального центрального маршрута, который можно включить в таблицу маршрутов виртуального концентратора.

Маршрут виртуального концентратора — это объект в памяти, который можно использовать для создания объекта VirtualHubRouteTable.

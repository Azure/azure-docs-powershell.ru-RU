# Get-AzureRmSqlSyncAgentLinkedDatabase

## КРАТКИй обзор
Возвращает сведения о базах данных SQL Server, связанных агентом синхронизации.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlSyncAgentLinkedDatabase [-ServerName] <String> [-SyncAgentName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSqlSyncAgentLinkedDatabases** возвращает сведения о базах данных SQL Server, связанных агентом синхронизации.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение связанных баз данных SQL Server для агента синхронизации Azure SQL.
```
PS C:\> Get-AzureRmSqlSyncAgentLinkedDatabases -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01" | Format-List
SeverName                 : sever01
DatabaseId                : databaseId
DatabaseName              : database01
DatabaseType              : SQLServerDatabase
Description               : 
UserName                  : myAccount
```

Эта команда возвращает связанные базы данных SQL Server, связанные с агентом синхронизации Azure SQL.

# Remove-AzDataLakeStoreAccount

## КРАТКИй обзор
Удаление учетной записи Data Lake Store без возможности восстановления.

## Максимальное

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzDataLakeStoreAccount** удаляет учетную запись Data Lake Store без возможности восстановления.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление учетной записи Data Lake Store
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Эта команда удаляет учетную запись с именем ContosoADL из Data Lake Store.

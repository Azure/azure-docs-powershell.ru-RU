# Get-AzEventHub

## SYNOPSIS
Возвращает сведения об одном центре событий или список концентраторов событий.

## СИНТАКСИС

```
Get-AzEventHub [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>] [-MaxCount <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Get-AzEventHub возвращает сведения об концентраторе событий или список всех концентраторов событий в текущем пространстве имен.
Если в центре событий есть имя, возвращаются сведения об одном центре событий.
Если имя центра событий не указано, возвращается список всех концентраторов событий в указанном пространстве имен.

## ПРИМЕРЫ

### Пример 1: указанный EventHub
```powershell
PS C:\> Get-AzEventHub -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName -EventHubName MyEventHubName
```

Возвращает сведения о Центре событий \` MyEventHubName. \`

### Пример 2. Список EventHub в указанном пространстве имен
```powershell
PS C:\> Get-AzEventHub -ResourceGroup MyResourceGroupName -NamespaceName MyNamespaceName
```

Возвращает список концентраторов событий в области имен \` MyNamespaceName. \`

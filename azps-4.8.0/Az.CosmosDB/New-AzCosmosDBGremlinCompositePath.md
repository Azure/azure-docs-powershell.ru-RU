# New-AzCosmosDBGremlinCompositePath

## КРАТКИй обзор
Создает новый объект типа PSCompositePath. Она может быть передана в качестве значения параметра Set-AzCosmosDBGremlinGraph.

## Максимальное

```
New-AzCosmosDBGremlinCompositePath [-Path <String>] [-Order <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Объект, соответствующий CompositePath API Gremlin.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzCosmosDBGremlinCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```

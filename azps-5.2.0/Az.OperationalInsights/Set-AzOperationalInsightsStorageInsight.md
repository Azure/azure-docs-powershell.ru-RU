# Set-AzOperationalInsightsStorageInsight

## SYNOPSIS
Обновляет сведения о хранилище.

## СИНТАКСИС

### ByWorkspaceName (по умолчанию)
```
Set-AzOperationalInsightsStorageInsight [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-Name] <String> [[-StorageAccountKey] <String>] [[-Tables] <String[]>] [[-Containers] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByWorkspaceObject
```
Set-AzOperationalInsightsStorageInsight [-Workspace] <PSWorkspace> [-Name] <String>
 [[-StorageAccountKey] <String>] [[-Tables] <String[]>] [[-Containers] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Set-AzOperationalInsightsStorageInsight** изменяет конфигурацию аналитики хранилища.

## ПРИМЕРЫ

### Пример 1. Изменение данных хранилища по имени
```
PS C:\>Set-AzOperationalInsightsStorageInsight -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "MyWorkspace" -Name "MyStorageInsight" -Tables @("WADWindowsEventLogsTable")
```

Эта команда изменяет таблицы, из которых будет прочитана информация о хранилище MyStorageInsight.

### Пример 2. Изменение статистики хранилища с помощью объекта рабочей области
```
PS C:\>$Workspace = Get-AzOperationalInsightsWorkspace -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace"

PS C:\>Set-AzOperationalInsightsStorageInsight -Workspace $Workspace -Name "MyStorageInsight" -Containers @("wad-iis-logfiles")
```

Первая команда использует Get-AzOperationalInsightsWorkspace для получения рабочей области MyWorkspace, а затем сохраняет ее в $Workspace переменной.
Вторая команда изменяет контейнеры, из которых вы читаете "Сведения о хранилище" с именем MyStorageInsight.

# Get-AzureRmRedisCache

## КРАТКИй обзор
Возвращает кэш Redis.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmRedisCache [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmRedisCache** получает указанный кэш Azure Redis.
Если вы укажете параметр "нет", эта операция возвращает каждый кэш Redis для текущей подписки.

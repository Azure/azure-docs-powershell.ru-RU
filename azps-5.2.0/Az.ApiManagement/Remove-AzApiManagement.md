# Remove-AzApiManagement

## SYNOPSIS
Удаляет службу управления API.

## СИНТАКСИС

```
Remove-AzApiManagement -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Из-за выполнения **команды Remove-AzApiManagement** служба управления API Azure удаляется.

## ПРИМЕРЫ

### Пример 1. Удаление службы управления API
```
PS C:\>Remove-AzApiManagement -ResourceGroupName "ContosoGroup02" -Name "ContosoApi"
```

Эта команда удаляет службу управления API под названием ContosoApi.

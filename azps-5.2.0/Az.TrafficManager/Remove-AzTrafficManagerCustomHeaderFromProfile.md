# Remove-AzTrafficManagerCustomHeaderFromProfile

## SYNOPSIS
Удаляет пользовательские данные загона из локального объекта профиля Диспетчера трафика.

## СИНТАКСИС

```
Remove-AzTrafficManagerCustomHeaderFromProfile -Name <String> -TrafficManagerProfile <TrafficManagerProfile>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **удаления azTrafficManagerCustomHeaderFromProfile** удаляются пользовательские данные заголовок из локального объекта профиля Диспетчера трафика Azure.
Вы можете получить профиль с помощью New-AzTrafficManagerProfile или Get-AzTrafficManagerProfile.

Этот cmdlet выполняется с объектом локального профиля.
Зафиксировать изменения в профиле диспетчера трафика с помощью Set-AzTrafficManagerProfile управления.

## ПРИМЕРЫ

### Пример 1. Удаление пользовательских данных загона из профиля
```
PS C:\> $TrafficManagerProfile = Get-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11"
PS C:\> Remove-AzTrafficManagerCustomHeaderFromEndpoint -TrafficManagerProfile $TrafficManagerProfile -Name "host"
PS C:\> Set-AzTrafficManagerProfile -TrafficManagerProfile $TrafficManagerProfile
```

Первая команда получает профиль Диспетчера трафика Azure с помощью командлета **Get-AzTrafficManagerProfile.**
Вторая команда удаляет пользовательские данные загона из профиля, храняного в $TrafficManagerProfile.
Конечная команда обновляет профиль Диспетчера трафика в соответствие с локальным значением в $TrafficManagerProfile.

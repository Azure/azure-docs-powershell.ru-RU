# New-AzSqlSyncAgentKey

## SYNOPSIS
Создает ключ агента синхронизации SQL Azure.

## СИНТАКСИС

```
New-AzSqlSyncAgentKey [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания ключа агента синхронизации Azure SQL **AzSqlSyncAgentKey.**

## ПРИМЕРЫ

### Пример 1. Создание ключа агента синхронизации для агента синхронизации Azure SQL.
```
PS C:\> New-AzSqlSyncAgentKey -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01"
SyncAgentKey                  : Key
```

Эта команда создает ключ агента синхронизации для azure SQL синхронизации.

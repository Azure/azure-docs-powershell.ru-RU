# Add-AzureDns

## КРАТКИй обзор
Добавляет DNS-сервер в службу Azure.

## Максимальное

```
Add-AzureDns [-Name] <String> [-IPAddress] <String> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Add-AzureDns** добавляет DNS-сервер в службу Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Добавление DNS-сервера
```
PS C:\> Add-AzureDns -ServiceName "ContosoService" -IPAddress 10.1.2.4 -Name "Dns07"
```

Эта команда добавляет DNS-сервер с адресом 10.1.2.4 в ContosoService.

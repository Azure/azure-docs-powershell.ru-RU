# Remove-AzureRmServiceFabricNode

## КРАТКИй обзор
Удаление узлов из кластера с определенным типом узла.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
С помощью **Remove-AzureRmServiceFabricNode** удалите узлы из кластера определенного типа. Удаление выполняется только в том случае, если оно соответствует метрикам работоспособности кластера.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS c:> Remove-AzureRmServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1' -NumberOfNodesToRemove 2
```

Эта команда удалит 2 узла из NodeType "NT1".

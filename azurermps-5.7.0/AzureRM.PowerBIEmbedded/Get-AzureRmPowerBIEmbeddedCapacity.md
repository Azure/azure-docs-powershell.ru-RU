# Get-AzureRmPowerBIEmbeddedCapacity

## КРАТКИй обзор
Получает сведения о емкости встроенных данных PowerBI.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmPowerBIEmbeddedCapacity [[-ResourceGroupName] <String>] 
    [<CommonParameters>]

Get-AzureRmPowerBIEmbeddedCapacity [-ResourceGroupName] <String> [-Name] <String> 
    [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Get-AzureRmPowerBIEmbeddedCapacity получает сведения о емкости встроенных данных PowerBI.

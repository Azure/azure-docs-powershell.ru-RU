# Get-AzCloudServiceRoleInstanceView

## SYNOPSIS
Извлекает сведения о состоянии времени запуска экземпляра роли в облачной службе.

## СИНТАКСИС

```
Get-AzCloudServiceRoleInstanceView -CloudServiceName <String> -ResourceGroupName <String>
 -RoleInstanceName <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## ОПИСАНИЕ
Извлекает сведения о состоянии времени запуска экземпляра роли в облачной службе.

## ПРИМЕРЫ

### Пример 1. Получить сведения о представлении экземпляра для экземпляра роли облачной службы
```powershell
PS C:\> Get-AzCloudServiceRoleInstanceView -ResourceGroupName "ContosOrg" -CloudServiceName "ContosoCS" -RoleInstanceName "ContosoFrontEnd_IN_0"

Statuses           PlatformFaultDomain PlatformUpdateDomain
--------           ------------------- --------------------
{RoleStateStarted} 0                   0

```

Этот cmdlet возвращает представление экземпляра роли ContosoFrontEnd_IN_0 облачной службы ContosoCS, которая принадлежит к группе ресурсов ContosOrg.

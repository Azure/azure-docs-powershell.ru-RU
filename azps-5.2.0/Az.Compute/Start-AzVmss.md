# Start-AzVmss

## SYNOPSIS
Запускает VMSS или набор виртуальных машин в составе VMSS.

## СИНТАКСИС

```
Start-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **запуска всех виртуальных** машин в наборе масштаба виртуальных машин (VMSS) или набора виртуальных машин запускается этот набор.
С помощью параметра *InstanceId* можно выбрать набор виртуальных машин.

## ПРИМЕРЫ

### Пример 1. Запуск определенного набора виртуальных машин в VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"-InstanceId "0", "1"
```

Эта команда запускает набор виртуальных машин, заданный массивом строки "ИД экземпляра", относимых к VMSS ContosoVMSS.

### Пример 2. Запуск всех виртуальных машин в VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"
```

Эта команда запускает все виртуальные машины, которые относятся к VMSS ContosoVMSS.

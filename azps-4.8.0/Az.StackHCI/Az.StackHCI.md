# AZ. StackHCI Module
## Nописание
Командлеты Microsoft Azure PowerShell: StackHCI

## Командлеты AZ. StackHCI
### [Регистрация — AzStackHCI](Register-AzStackHCI.md)
Register-AzStackHCI создает облачный ресурс Microsoft. AzureStackHCI, представляющий локальный кластер, и регистрирует локальный кластер с помощью Azure.

### [Отмена регистрации — AzStackHCI](Unregister-AzStackHCI.md)
Unregister-AzStackHCI удаляет облачный ресурс Microsoft. AzureStackHCI, представляющий локальный кластер, и отменяет регистрацию локального кластера с помощью Azure.

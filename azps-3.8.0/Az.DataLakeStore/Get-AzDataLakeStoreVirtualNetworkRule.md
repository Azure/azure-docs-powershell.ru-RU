# Get-AzDataLakeStoreVirtualNetworkRule

## КРАТКИй обзор
Получает указанные правила виртуальной сети в указанном Data Lake Store.
Если правило для виртуальной сети не задано, выводится список всех правил для учетной записи в сети.

## Максимальное

```
Get-AzDataLakeStoreVirtualNetworkRule [-Account] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Get-AzDataLakeStoreVirtualNetworkRule получает указанные правила для виртуальных сетей в указанном Data Lake Store.
Если правило для виртуальной сети не задано, выводится список всех правил для учетной записи в сети.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"

ResourceGroupName                :
AccountName                      :
VirtualNetworkRuleName           : myVNET
VirtualNetworkSubnetId           : /subscriptions/<subscriptionId>/resourceGroups/<resourceGroup>/providers/Microsoft.Network/virtualNetworks/myVNET/subnets/testId
IgnoreMissingVnetServiceEndpoint :
State                            :
```

Возвращает правило виртуальной сети с именем "myVNET" из учетной записи "списки рассылки".

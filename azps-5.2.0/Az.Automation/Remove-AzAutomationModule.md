# Remove-AzAutomationModule

## SYNOPSIS
Модуль удаляется из автоматизации.

## СИНТАКСИС

```
Remove-AzAutomationModule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Командлет **Remove-AzAutomationModule** удаляет модуль из учетной записи автоматизации в Azure Automation.

## ПРИМЕРЫ

### Пример 1. Удаление модуля
```
PS C:\>Remove-AzAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule" -ResourceGroupName "ResourceGroup01"
```

Эта команда удаляет модуль ContosoModule из учетной записи автоматизации с именем Contoso17.

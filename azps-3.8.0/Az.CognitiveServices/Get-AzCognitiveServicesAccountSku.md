# Get-AzCognitiveServicesAccountSku

## КРАТКИй обзор
Возвращает доступные SKU для учетной записи.

## Максимальное

```
Get-AzCognitiveServicesAccountSku [-Type <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzCognitiveServicesAccountSku** получает доступные SKU для учетной записи службы.
SKU — это ярусный план для учетной записи.
Она определяет цену, ограничение на звонки и ставку для счета.
F0 SKU является бесплатным уровнем.
Оплаченные уровни включают в себя S0, S1, S2 и т. д.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> (Get-AzCognitiveServicesAccountSku -Type 'TextAnalytics' -Location "westus").Value | Select-Object -E
xpandProperty Sku;

Name     Tier
----     ----
F0       Free
S0   Standard
```

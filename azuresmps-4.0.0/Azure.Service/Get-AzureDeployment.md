# Get-AzureDeployment

## КРАТКИй обзор
Получает сведения о развертывании.

## Максимальное

```
Get-AzureDeployment [-ServiceName] <String> [[-Slot] <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureDeployment** получает сведения о развертывании Azure.
Укажите имя службы Azure и ячейку развертывания.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение сведений о рабочем развертывании
```
PS C:\> Get-AzureDeployment -ServiceName "ContosoService"
```

Эта команда возвращает сведения о развертывании для службы с именем ContosoService.
В этой команде не указана ячейка.
Поэтому в команде используется значение по умолчанию "производство".

### Пример 2: получение подробных сведений о промежуточном развертывании
```
PS C:\> Get-AzureDeployment -ServiceName "ContosoService" -Slot "Staging"
```

Эта команда возвращает сведения о промежуточном развертывании ContosoService.

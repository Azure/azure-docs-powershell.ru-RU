# New-AzSqlDatabaseInstanceFailoverGroup

## SYNOPSIS
Эта команда создает новую группу отбойных SQL экземпляра базы данных Azure.

## СИНТАКСИС

```
New-AzSqlDatabaseInstanceFailoverGroup [-Name] <String> [-PartnerResourceGroupName <String>]
 -PartnerRegion <String> -PrimaryManagedInstanceName <String> -PartnerManagedInstanceName <String>
 [-PartnerSubscriptionId <String>] [-FailoverPolicy <String>] [-GracePeriodWithDataLossHours <Int32>]
 [-AllowReadOnlyFailoverToPrimary <String>] [-ResourceGroupName] <String> [-Location] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Создает новую группу отбойного SQL экземпляра базы данных Azure между указанными регионами с указанной парой "Управляемый экземпляр".

В доменах Name.SqlDataDnsSuffix (например, Name.database.windows.net) и Name.secondary.SqlDataBaseDnsSuffix создаются две конечные точки базы данных Azure SQL. Эти конечные точки могут использоваться для подключения к основной и дополнительным областям группы отключении соответственно. Если на основной регион влияет сбой, автоматический отключок конечных точек и баз данных активируется так, как это диктуется политикой отключки и льготным периодом для группы отключания экземпляров.

Во время предварительного просмотра функции "Группы отката" для параметра "-GracePeriodWithDataLossHours" поддерживаются только значения, которые больше или равны 1 часу.

## ПРИМЕРЫ

### Пример 1
```powershell
C:\> $failoverGroup = New-AzSqlDatabaseInstanceFailoverGroup -Name fgName -Location location -ResourceGroupName rg -PrimaryManagedInstanceName $managedInstance.Name -PartnerRegion $partnerRegion -PartnerManagedInstanceName $partnerManagedInstance.Name -FailoverPolicy Automatic -GracePeriodWithDataLossHours 1

PartnerManagedInstanceName            : managedInstance2
```

```powershell
PartnerRegion                         : West US
PrimaryManagedInstanceName            : managedInstance1
```

Эта команда создает новую группу отбойных экземпляров с политикой отбойного управления "Вручную" для пары управляемых экземпляров.

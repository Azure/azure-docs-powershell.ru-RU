# Get-AzPolicyDefinition

## SYNOPSIS
Получает определения политик.

## СИНТАКСИС

### NameParameterSet (по умолчанию)
```
Get-AzPolicyDefinition [-Name <String>] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ManagementGroupNameParameterSet
```
Get-AzPolicyDefinition [-Name <String>] -ManagementGroupName <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionIdParameterSet
```
Get-AzPolicyDefinition [-Name <String>] -SubscriptionId <Guid> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdParameterSet
```
Get-AzPolicyDefinition -Id <String> [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### BuiltinFilterParameterSet
```
Get-AzPolicyDefinition [-ManagementGroupName <String>] [-SubscriptionId <Guid>] [-Builtin]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### CustomFilterParameterSet
```
Get-AzPolicyDefinition [-ManagementGroupName <String>] [-SubscriptionId <Guid>] [-Custom]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzPolicyDefinition** можно получить набор определений политики или определение политики, определенное по имени или коду.

## ПРИМЕРЫ

### Пример 1. Получить все определения политик
```
PS C:\> Get-AzPolicyDefinition
```

Эта команда получает все определения политики.

### Пример 2. Определение политики из текущей подписки по имени
```
PS C:\> Get-AzPolicyDefinition -Name 'VMPolicyDefinition'
```

Эта команда получает определение политики VMPolicyDefinition из текущей подписки по умолчанию.

### Пример 3. Определение политики из группы управления по имени
```
PS C:\> Get-AzPolicyDefinition -Name 'VMPolicyDefinition' -ManagementGroupName 'Dept42'
```

Эта команда получает определение политики VMPolicyDefinition из группы управления "Отдел42".

### Пример 4. Получить все встроенные определения политик из подписки
```
PS C:\> Get-AzPolicyDefinition -SubscriptionId '3bf44b72-c631-427a-b8c8-53e2595398ca' -Builtin
```

Эта команда получает все встроенные определения политик из подписки с помощью ID 3bf44b72-c631-427a-b8c8-53e2595398ca.

### Пример 5. Получать определения политик из данной категории
```
PS C:\> Get-AzPolicyDefinition | where-object {$_.Properties.metadata.category -eq "Virtual Machine"}
```

Эта команда получает все определения политики в категории "Виртуальная машинка".

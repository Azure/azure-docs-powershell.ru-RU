# Remove-AzSqlServerDnsAlias

## КРАТКИй обзор
Удаляет DNS-псевдоним Azure SQL Server.

## Максимальное

### Удаление DNS-псевдонима сервера из входных параметров командлета
```
Remove-AzSqlServerDnsAlias -Name <String> -ServerName <String> [-ResourceGroupName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Удаление псевдонима DNS сервера из определения экземпляра AzureSqlServerDnsAliasModel
```
Remove-AzSqlServerDnsAlias -InputObject <AzureSqlServerDnsAliasModel> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Удаление DNS-псевдонима сервера из идентификатора ресурса Azure
```
Remove-AzSqlServerDnsAlias -ResourceId <String> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Эти команды удаляют псевдоним DNS для Azure SQL Server с сервера, оставив сервер неизменным.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Remove-AzSqlServerDnsAlias -DnsAliasName aliasName -ServerName serverName -ResourceGroupName rg
```

Удаляет DNS-псевдоним Azure SQL Server с именем aliasName с сервера с именем serverName.

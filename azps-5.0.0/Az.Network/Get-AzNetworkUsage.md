# Get-AzNetworkUsage

## КРАТКИй обзор
Список использования сети для подписки

## Максимальное

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Get-AzNetworkUsage получает ограничения и текущее использование сетевых ресурсов.

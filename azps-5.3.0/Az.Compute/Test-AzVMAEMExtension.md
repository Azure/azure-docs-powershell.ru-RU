# Test-AzVMAEMExtension

## SYNOPSIS
Проверяет конфигурацию расширения AEM.

## СИНТАКСИС

```
Test-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-OSType] <String>]
 [[-WaitTimeInMinutes] <Int32>] [-SkipStorageCheck] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Test-AzVMAEMExtension** проверяет конфигурацию расширения Azure Enhanced Monitoring (AEM).
Расширение AEM собирает данные о производительности.
Этот cmdlet проверяет, доступны ли данные о производительности.

## ПРИМЕРЫ

### Пример 1. Проверка конфигурации расширения AEM
```
PS C:\> Test-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Эта команда проверяет конфигурацию расширения AEM для виртуальной машины с именем contoso-server.

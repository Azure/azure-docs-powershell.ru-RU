# New-AzRedisCachePatchSchedule

## SYNOPSIS
Добавляет расписание исправлений.

## СИНТАКСИС

```
New-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **кэша в кэше Azure Redis Cache** добавляется расписание исправлений.

## ПРИМЕРЫ

### Пример 1. Создание и добавление графика исправлений в кэше
```
PS C:\>New-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

Эта команда добавляет расписание исправлений в кэш RedisCache06.
Параметр Entries принимает в качестве значения команду, которая использует **New-AzRedisCacheScheduleEntry** для создания расписания.

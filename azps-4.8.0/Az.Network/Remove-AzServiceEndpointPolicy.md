# Remove-AzServiceEndpointPolicy

## КРАТКИй обзор
Удаление политики конечной точки службы.

## Максимальное

### RemoveByNameParameterSet (по умолчанию)
```
Remove-AzServiceEndpointPolicy -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzServiceEndpointPolicy -ResourceId <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzServiceEndpointPolicy -InputObject <PSServiceEndpointPolicy> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzServiceEndpointPolicy** удаляет политику конечной точки службы.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление политики конечной точки службы с помощью имени
```
Remove-AzServiceEndpointPolicy -Name "Policy1" -ResourceGroup "resourcegroup1"
```

Эта команда удаляет политику конечной точки службы с именем Policy1, которая принадлежит к параметру resourcegroup с именем "resourcegroup1".

### Пример 2: Удаление политики конечной точки службы с помощью объекта input
```
Remove-AzServiceEndpointPolicy -InputObject $Policy1 -ResourceGroup "resourcegroup1"
```

Эта команда удаляет объект политики конечной точки службы Policy1, который принадлежит к параметру resourcegroup с именем "resourcegroup1".

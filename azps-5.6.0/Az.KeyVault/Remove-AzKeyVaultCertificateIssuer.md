# Remove-AzKeyVaultCertificateIssuer

## SYNOPSIS
Удаляет выпускаемого сертификата из хранилища ключей.

## СИНТАКСИС

### По умолчанию (по умолчанию)
```
Remove-AzKeyVaultCertificateIssuer [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzKeyVaultCertificateIssuer [-InputObject] <PSKeyVaultCertificateIssuerIdentityItem> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Remove-AzKeyVaultCertificateIssuer** удаляет проблему с сертификатом из хранилища ключей.

## ПРИМЕРЫ

### Пример 1. Удаление выпуска сертификата
```powershell
PS C:\> Remove-AzKeyVaultCertificateIssuer -VaultName "ContosoKV01" -Name "TestIssuer01" -Force

AccountId           :
ApiKey              :
OrganizationDetails :
Name                : TestIssuer01
IssuerProvider      : test
VaultName           : ContosoKV01
```

Эта команда удаляет выпускаемого сертификата TestIssuer01 из хранилища ключей ContosoKV01.

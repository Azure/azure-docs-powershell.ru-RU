# Get-AzDataLakeStoreAccount

## КРАТКИй обзор
Получение сведений об учетной записи для Data Lake Store.

## Максимальное

### GetAllInSubscription (по умолчанию)
```
Get-AzDataLakeStoreAccount [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceGroup
```
Get-AzDataLakeStoreAccount [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetBySpecificAccount
```
Get-AzDataLakeStoreAccount [[-ResourceGroupName] <String>] [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzDataLakeStoreAccount** получает сведения о учетной записи Data Lake Store.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение учетной записи для Data Lake Store
```
PS C:\>Get-AzDataLakeStoreAccount -Name "ContosoADL"
```

Эта команда получает учетную запись с именем ContosoADL.

# Get-AzSynapseLinkedService

## КРАТКИй обзор
Получение сведений о связанных службах в рабочей области.

## Максимальное

### GetByName (по умолчанию)
```
Get-AzSynapseLinkedService -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseLinkedService -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSynapseLinkedService** получает сведения о связанных службах в рабочей области.
Если указать имя связанной службы, этот командлет получает сведения о связанной службе.
Если имя не задано, этот командлет получает сведения обо всех связанных службах в рабочей области.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace
```

Эта команда получает сведения обо всех связанных службах в рабочей области с именем ContosoWorkspace.

### Пример 2
```powershell
PS C:\> Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService
```

Эта команда получает сведения о связанной службе с именем ContosoLinkedService в рабочей области с именем ContosoWorkspace.

### Пример 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseLinkedService -Name ContosoLinkedService
```

Эта команда получает сведения о связанной службе с именем ContosoLinkedService в рабочей области с именем ContosoWorkspace с помощью конвейера.

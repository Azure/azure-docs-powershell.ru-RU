# New-AzPublicIpPrefix

## SYNOPSIS
Создание префикса для общедоступных IP-адресов

## СИНТАКСИС

```
New-AzPublicIpPrefix -Name <String> -ResourceGroupName <String> [-Location <String>] [-Sku <String>]
 [-Tier <String>] -PrefixLength <UInt16> [-IpAddressVersion <String>] [-IpTag <PSPublicIpPrefixTag[]>]
 [-Zone <String[]>] [-CustomIpPrefix <PSCustomIpPrefix>] [-Tag <Hashtable>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания общего IP-префикса создается **cmdlet New-AzPublicIpPrefix.**

## ПРИМЕРЫ

### Пример 1. Создание префикса для общедоступных IP-адресов
```powershell
PS C:\> $publicIpPrefix = New-AzPublicIpPrefix -Name $prefixName -ResourceGroupName $rgName -PrefixLength 30
```

Эта команда создает общедоступный IP-префикс ресурса. 

### Пример 2. Создание нового глобального префикса общедоступных IP-адресов
```powershell
PS C:\> $publicIpPrefix = New-AzPublicIpPrefix -ResourceGroupName $rgname -name $rname -location $location -Tier Global -PrefixLength 30
```

Эта команда создает новый глобальный общедоступный IP-префикс ресурса.

# Set-AzureRmServiceBusTopic

## КРАТКИй обзор
Обновляет описание раздела служебной шины в указанном пространстве имен служебной шины.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Set-AzureRmServiceBusTopic [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-InputObject] <TopicAttributes> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzureRmServiceBusTopic** обновляет объект описания для темы служебной шины в указанном пространстве имен служебной шины.

PS C:\> $topicObj.EnableExpress = $True

# Get-AzApiManagementAuthorizationServerClientSecret

## SYNOPSIS
Получает секрет клиента авторизации управления API.

## СИНТАКСИС

### ContextParameterSet (по умолчанию)
```
Get-AzApiManagementAuthorizationServerClientSecret -Context <PsApiManagementContext> [-ServerId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementAuthorizationServerClientSecret [-ServerId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzApiManagementAuthorizationServerClientSecret** получает секрет клиента сервера авторизации управления API Azure.

## ПРИМЕРЫ

### Пример 1. Получите секрет определенного клиента авторизации по id
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServerClientSecret -Context $ApiMgmtContext -ServerId "0123456789"
```

Эта команда получает указанный секрет сервера авторизации.

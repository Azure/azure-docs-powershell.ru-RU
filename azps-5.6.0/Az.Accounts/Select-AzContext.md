# Select-AzContext

## SYNOPSIS
Выбор подписки и учетной записи для целевой аудитории в azure PowerShell

## СИНТАКСИС

### SelectByInputObject (по умолчанию)
```
Select-AzContext -InputObject <PSAzureContext> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SelectByName
```
Select-AzContext [-Scope <ContextModificationScope>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [-Name] <String> [<CommonParameters>]
```

## ОПИСАНИЕ
Выберите подписку, которая будет целевой (или учетная запись или клиент) в azure PowerShell.  После этого cmdlet,future cmdlets will target the selected context.

## ПРИМЕРЫ

### Пример 1. Целевое имя контекста
```powershell
PS C:\> Select-AzContext "Work"

Name    Account             SubscriptionName    Environment         TenantId
----    -------             ----------------    -----------         --------
Work    test@outlook.com    Subscription1       AzureCloud          xxxxxxxx-x...
```

Целевые будущие наборы azure PowerShell для учетной записи, клиента и подписки в контексте "Работа".

### Пример 2

В этом сеансе powerShell выберите подписку и учетную запись, которые нужно подбирать для целевой аудитории в azure PowerShell. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Select-AzContext -Name TestEnvironment -Scope Process
```

# Remove-AzDataLakeAnalyticsAccount

## SYNOPSIS
Удаляет учетную запись Data Lake Analytics.

## СИНТАКСИС

```
Remove-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Remove-AzDataLakeAnalyticsAccount** окончательно удаляет учетную запись Azure Data Lake Analytics.

## ПРИМЕРЫ

### Пример 1. Удаление учетной записи
```
PS C:\>Remove-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Эта команда удаляет указанную учетную запись Data Lake Analytics.

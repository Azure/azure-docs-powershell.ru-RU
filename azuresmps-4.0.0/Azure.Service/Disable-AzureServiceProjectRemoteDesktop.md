# Disable-AzureServiceProjectRemoteDesktop

## КРАТКИй обзор
Запрещает удаленному рабочему столу доступ к облачной службе.

## Максимальное

```
Disable-AzureServiceProjectRemoteDesktop [-PassThru] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
В этом разделе описан командлет в версии 0.8.10 модуля Microsoft Azure PowerShell.
Чтобы получить версию модуля, который вы используете, введите в командной консоли Azure PowerShell `(Get-Module -Name Azure).Version` .

**Disable-AzureServiceProjectRemoteDesktop** отключает удаленный рабочий стол для доступа к размещенной службе.
Чтобы изменения вступили в силу, необходимо опубликовать службу с помощью командлета **Publish-AzureServiceProject** после отключения доступа к удаленному рабочему столу.

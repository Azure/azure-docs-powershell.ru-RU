# New-AzureRmStreamAnalyticsFunction

## КРАТКИй обзор
Создает или заменяет функцию в задании Stream Analytics.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmStreamAnalyticsFunction** создает функцию в задании Azure Stream Analytics или заменяет существующую функцию.
Определите функцию в файле нотации JavaScript Object Notation (JSON).
Вы можете указать имя функции с помощью параметра *Name (имя* ) или в файле. JSON.
Если указать имя обоими способами, имена должны совпадать.
Чтобы заменить существующую функцию, укажите имя существующей функции.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание функции Stream Analytics
```
PS C:\>New-AzureRmStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob07" -File "C:\Function07.json"
```

Эта команда создает функцию из файла Function07.json.
Имя функции хранится в JSON – файле.

### Пример 2: создание функции Stream Analytics с именем ScoreTweet
```
PS C:\>New-AzureRmStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet"
```

Эта команда создает функцию для задания с именем ScoreTweet.

### Пример 3: замена функции Stream Analytics
```
PS C:\>New-AzureRmStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet" -Force
```

Эта команда заменяет определение существующей функции с именем ScoreTweet с определением в Function22.json.

# New-AzureRmNetworkWatcherProtocolConfiguration

## КРАТКИй обзор
Создает новый объект конфигурации протокола.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmNetworkWatcherProtocolConfiguration -Protocol <String> [-Method <String>] [-Header <IDictionary>]
 [-ValidStatusCode <Int32[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет New-AzureRmNetworkWatcherProtocolConfiguration создает новый объект конфигурации протокола. Этот объект используется для ограничения confiuration протокола во время проверки connecitivity сеанса с использованием указанных условий.

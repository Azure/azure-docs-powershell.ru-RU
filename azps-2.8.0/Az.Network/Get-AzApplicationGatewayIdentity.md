# Get-AzApplicationGatewayIdentity

## КРАТКИй обзор
Получение удостоверения, назначенного шлюзу приложений.

## Максимальное

```
Get-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzApplicationGatewayIdentity** получает удостоверение, назначенное шлюзу приложений.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $identity = Get-AzApplicationGatewayIdentity -ApplicationGateway $gw
```

В этом примере показано, как получить удостоверение шлюза приложения из шлюза приложений.

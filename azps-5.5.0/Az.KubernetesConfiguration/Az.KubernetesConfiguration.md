# Модуль Az.KubernetesConfiguration
## Описание
Microsoft Azure PowerShell: cmdlets KubernetesConfiguration

## Az.KubernetesConfiguration Cmdlets
### [Get-AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
Сведения о конфигурации управления источником.

### [New-AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
Создайте новую конфигурацию управления исходным источником в Кубберсетях.

### [Remove-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
При этом будет удален файл YAML, используемый для настройки конфигурации управления источником, что останавливает его последующие синхронизации.

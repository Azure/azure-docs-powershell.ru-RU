# Set-AzStorageQueueStoredAccessPolicy

## КРАТКИй обзор
Задает политику сохранения для доступа для очереди хранилища Azure.

## Максимальное

```
Set-AzStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-NoStartTime] [-NoExpiryTime] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzStorageQueueStoredAccessPolicy** задает политику сохраненного доступа для очереди хранилища Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Настройка политики доступа в очереди с полным разрешением
```
PS C:\> Set-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy07" -Permission arup
```

Эта команда задает политику доступа с именем Policy07 для очереди хранения с именем MyQueue.

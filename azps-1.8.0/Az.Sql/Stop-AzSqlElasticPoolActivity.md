# Stop-AzSqlElasticPoolActivity

## КРАТКИй обзор
Отменяет асинхронную операцию обновления в пуле эластичных БД.

## Максимальное

```
Stop-AzSqlElasticPoolActivity [-PassThru] [-ServerName] <String> [-ElasticPoolName] <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Stop-AzSqlElasticPoolActivity** отменяет асинхронную операцию обновления в пуле эластичных БД.

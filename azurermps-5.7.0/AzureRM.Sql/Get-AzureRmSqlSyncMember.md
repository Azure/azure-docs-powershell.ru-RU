# Get-AzureRmSqlSyncMember

## КРАТКИй обзор
Возвращает сведения о членах синхронизации базы данных Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlSyncMember [-Name <String>] [-SyncGroupName] <String> [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSqlSyncMember** возвращает сведения об одном или нескольких членах синхронизации базы данных Azure SQL.
Укажите имя члена синхронизации, чтобы просмотреть сведения только для этого члена синхронизации.

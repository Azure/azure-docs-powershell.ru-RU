# Remove-AzDataLakeStoreItemAcl

## SYNOPSIS
Очищает ACL файла или папки в магазине Data Lake Store.

## СИНТАКСИС

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При наложении cmdlet **Remove-AzDataLakeStoreItemAcl** список управления доступом очищается из списка управления доступом файла или папки в магазине Data Lake Store.

## ПРИМЕРЫ

### Пример 1. Удаление ACL из папки
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Эта команда удаляет ACL для корневого каталога учетной записи ContosoADL.

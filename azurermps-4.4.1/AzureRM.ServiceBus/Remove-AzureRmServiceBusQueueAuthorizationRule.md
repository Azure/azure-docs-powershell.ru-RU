# Remove-AzureRmServiceBusQueueAuthorizationRule

## КРАТКИй обзор
Удаляет правило авторизации очереди из указанного пространства имен служебной шины.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmServiceBusQueueAuthorizationRule [-ResourceGroup] <String> -Namespace <String> -Queue <String>
 -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmServiceBusQueueAuthorizationRule** удаляет правило авторизации очереди из указанного пространства имен служебной шины.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Remove-AzureRmServiceBusQueueAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -QueueName SB-Queue_exampl1 -AuthorizationRuleName SBAuthoRule1
```

Удаляет правило авторизации `SBAuthoRule1` очереди `SB-Queue_exampl1` из пространства имен `SB-Example1` .

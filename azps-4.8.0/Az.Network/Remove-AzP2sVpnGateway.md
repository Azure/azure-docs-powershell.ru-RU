# Remove-AzP2sVpnGateway

## КРАТКИй обзор
Удаление существующего P2SVpnGateway.

## Максимальное

### ByP2SVpnGatewayName (по умолчанию)
```
Remove-AzP2sVpnGateway -ResourceGroupName <String> -Name <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByP2SVpnGatewayObject
```
Remove-AzP2sVpnGateway -InputObject <PSP2SVpnGateway> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByP2SVpnGatewayResourceId
```
Remove-AzP2sVpnGateway -ResourceId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzP2sVpnGateway** позволяет удалить существующий P2SVpnGateway в разделе VirtualHub. После удаления P2SVpnGateway все клиенты, которые указывают на возможности подключения клиентов сайта, будут завершаться сбоем.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Remove-AzP2sVpnGateway -Name 683482ade8564515aed4b8448c9757ea-westus-gw-ResourceGroupName P2SCortexGATesting -Force -PassThru
```

Командлет **Remove-AzP2sVpnGateway** позволяет удалить существующий P2SVpnGateway в разделе VirtualHub.

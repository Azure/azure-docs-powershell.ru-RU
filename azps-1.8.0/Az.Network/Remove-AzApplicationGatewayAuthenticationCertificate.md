# Remove-AzApplicationGatewayAuthenticationCertificate

## КРАТКИй обзор
Удаляет сертификат проверки подлинности из шлюза приложений.

## Максимальное

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApplicationGatewayAuthenticationCertificate** Удаляет сертификат для проверки подлинности из шлюза приложений Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление сертификата для проверки подлинности из шлюза приложений
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Первая команда получает шлюз приложения с именем appGwName и сохраняет результат в переменной $appgw.
Вторая команда удаляет сертификат проверки подлинности с именем cert01 из шлюза приложений.
Третья команда обновляет шлюз приложения.

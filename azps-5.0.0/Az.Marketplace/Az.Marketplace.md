# Модуль AZ. Marketplace
## Nописание
Команды для взаимодействия с Azure Marketplace.

## Командлеты AZ. Marketplace
### [Get-AzMarketplacePrivateStoreOffer](Get-AzMarketplacePrivateStoreOffer.md)
Получение одного или нескольких предложений частного магазина.

### [Get-AzMarketplacePrivateStore](Get-AzMarketplacePrivateStore.md)
Получение списка частных хранилищ

### [Remove-AzMarketplacePrivateStoreOffer](Remove-AzMarketplacePrivateStoreOffer.md)
Удаление предложения из частного магазина.

### [Set-AzMarketplacePrivateStoreOffer](Set-AzMarketplacePrivateStoreOffer.md)
Обновление или создание предложения для частного магазина.

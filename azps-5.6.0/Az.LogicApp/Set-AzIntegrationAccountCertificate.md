# Set-AzIntegrationAccountCertificate

## SYNOPSIS
Изменяет сертификат учетной записи интеграции.

## СИНТАКСИС

```
Set-AzIntegrationAccountCertificate -ResourceGroupName <String> -Name <String> -CertificateName <String>
 [-KeyName <String>] [-KeyVersion <String>] [-KeyVaultId <String>] [-PublicCertificateFilePath <String>]
 [-Metadata <Object>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
**Set-AzIntegrationAccountCertificate cmdlet** изменяет сертификат учетной записи интеграции.
Этот cmdlet возвращает объект, который представляет сертификат учетной записи интеграции.
Укажите имя учетной записи интеграции, имя группы ресурсов и имя сертификата.
Этот модуль поддерживает динамические параметры.
Чтобы использовать динамический параметр, введите его в команду.
Чтобы найти имена динамических параметров, введите дефис (-) после имени cmdlet, а затем нажимая клавишу TAB для циклическиго перебора доступных параметров.
Если параметр обязательного шаблона не задан, будет выдан запрос на его значение.

Эта команда изменяет сертификат учетной записи интеграции в указанной группе ресурсов.

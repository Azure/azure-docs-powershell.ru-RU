# Set-AzureAffinityGroup

## КРАТКИй обзор
Изменяет свойства территориальной группы.

## Максимальное

```
Set-AzureAffinityGroup [-Name] <String> -Label <String> [-Description <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzureAffinityGroup** изменяет свойства территориальной группы Azure.
Вы можете изменить метку и описание.

## ИЛЛЮСТРИРУЮТ

### Пример 1: изменение территориальной группы
```
PS C:\> Set-AzureAffinityGroup -Name "South01" -Label "SouthUSProduction" -Description "Production applications for Southern US locations"
```

Эта команда изменяет метку территориальной группы с именем South01 на SouthUSProduction, она также изменяет описание.

# Set-AzureRmRouteTable

## КРАТКИй обзор
Задает состояние цели для таблицы маршрутов.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Set-AzureRmRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzureRmRouteTable** задает состояние цели для таблицы маршрутов Azure.

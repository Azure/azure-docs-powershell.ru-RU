# Remove-AzAutoscaleSetting

## SYNOPSIS
Удаляет параметры автомасштаби.

## СИНТАКСИС

```
Remove-AzAutoscaleSetting -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **удаления параметра авто шкалы AzAutoscaleSetting** удаляются все параметры.
Необходимо указать имя параметра и имя группы ресурсов, которой он назначен.
Этот cmdlet реализует шаблон ShouldProcess, то есть может запросить подтверждение у пользователя перед созданием, изменением или удалением ресурса.

## ПРИМЕРЫ

### Пример 1

Удаляет параметры автомасштаби. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzAutoscaleSetting -Name 'LogAlertRule1' -ResourceGroupName MyResourceGroup
```

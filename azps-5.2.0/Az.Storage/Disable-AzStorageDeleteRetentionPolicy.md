# Disable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Отключите политику хранения удаления для службы BLOB-документов хранилища Azure.

## СИНТАКСИС

```
Disable-AzStorageDeleteRetentionPolicy [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Disable-AzStorageDeleteRetentionPolicy** отключает политику хранения удаления для службы BLOB-документов хранилища Azure.

## ПРИМЕРЫ

### Пример 1. Отключение политики хранения удаления для службы BLOB-документов
```
C:\PS>Disable-AzStorageDeleteRetentionPolicy
```

Эта команда отключает политику хранения удаления для службы BLOB-документов.

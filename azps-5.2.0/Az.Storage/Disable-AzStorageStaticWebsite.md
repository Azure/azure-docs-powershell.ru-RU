# Disable-AzStorageStaticWebsite

## SYNOPSIS
Отключение статического веб-сайта для учетной записи хранилища Azure.

## СИНТАКСИС

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для учетной записи хранилища Azure отключен статический **веб-сайт Disable-AzStorageStaticWebsite.**

## ПРИМЕРЫ

### Пример 1. Отключение статического веб-сайта для учетной записи хранилища Azure
```
C:\PS>Disable-AzStorageStaticWebsite
```

Эта команда отключает статический веб-сайт для учетной записи хранилища Azure.

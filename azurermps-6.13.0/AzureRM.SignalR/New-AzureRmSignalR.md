# New-AzureRmSignalR

## КРАТКИй обзор
Создание службы сигналов.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmSignalR [-ResourceGroupName <String>] [-Name] <String> [-Location <String>] [-Sku <String>]
 [-UnitCount <Int32>] [-Tag <System.Collections.Generic.IDictionary`2[System.String,System.String]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Создание службы сигналов.
Если не указано, для параметров будут использоваться следующие значения:
* `ResourceGroupName`: Стандартная группа ресурсов, установленная по умолчанию `Set-AzureRmDefault -ResourceGroupName` .
* `Location`: расположение группы ресурсов
* `Sku`: `Standard_S1`

## ИЛЛЮСТРИРУЮТ

### Создание поsignalr

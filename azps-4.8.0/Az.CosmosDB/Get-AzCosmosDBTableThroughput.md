# Get-AzCosmosDBTableThroughput

## КРАТКИй обзор
Возвращает пропускную способность таблицы CosmosDB.

## Максимальное

### ByNameParameterSet (по умолчанию)
```
Get-AzCosmosDBTableThroughput -ResourceGroupName <String> -AccountName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBTableThroughput -Name <String> -InputObject <PSDatabaseAccountGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzCosmosDBTableThroughput** возвращает пропускную способность таблицы CosmosDB.

## ИЛЛЮСТРИРУЮТ

### Пример 1

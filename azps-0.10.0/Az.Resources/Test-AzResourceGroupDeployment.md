# Test-AzResourceGroupDeployment

## КРАТКИй обзор
Проверяет развертывание группы ресурсов.

 [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]

## NОПИСАНИЕ
Командлет **Test-AzResourceGroupDeployment** определяет, являются ли шаблон развертывания группы ресурсов Azure и значения его параметров допустимыми.

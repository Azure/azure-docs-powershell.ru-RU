# New-AzureRmApplicationInsights

## КРАТКИй обзор
Создание нового ресурса Application Insights

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmApplicationInsights [-ResourceGroupName] <String> [-Name] <String> [-Location] <String>
 [-Kind <String>] [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Создание нового ресурса Application Insights

## ИЛЛЮСТРИРУЮТ

### Пример 1. Создание нового ресурса Application Insights

Добавьте новый ресурс Application Insights под названием "тест" в группе ресурсов "testgroup" с помощью типа "Java".

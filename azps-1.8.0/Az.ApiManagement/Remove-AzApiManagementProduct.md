# Remove-AzApiManagementProduct

## КРАТКИй обзор
Удаляет существующий продукт для управления API.

## Максимальное

```
Remove-AzApiManagementProduct -Context <PsApiManagementContext> -ProductId <String> [-DeleteSubscriptions]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApiManagementProduct** удаляет существующий продукт для управления API.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление существующего продукта и всех подписок
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementProduct -Context $apimContext -ProductId "0123456789" -DeleteSubscriptions
```

Эта команда удаляет существующий продукт и все подписки.

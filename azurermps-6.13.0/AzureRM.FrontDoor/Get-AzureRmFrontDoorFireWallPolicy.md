# Get-AzureRmFrontDoorFireWallPolicy

## КРАТКИй обзор
Получение политики WAF

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Параметр **Get-AzureRmFrontDoorFireWallPolicy** CMDLETGET получает WAF политику в группе ресурсов в рамках текущей подписки.

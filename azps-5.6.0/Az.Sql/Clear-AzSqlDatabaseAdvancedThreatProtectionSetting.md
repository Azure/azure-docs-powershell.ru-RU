# Clear-AzSqlDatabaseAdvancedThreatProtectionSetting

## SYNOPSIS
Удаляет дополнительные параметры защиты от угроз из базы данных.

## СИНТАКСИС

```
Clear-AzSqlDatabaseAdvancedThreatProtectionSetting [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого** из базы данных AzureAzure SQL расширенные параметры защиты от угроз.
Чтобы использовать этот cmdlet, укажите параметры *ResourceGroupName* и *ServerName,* чтобы определить базу данных, из которой этот cmdlet удаляет параметры.

## ПРИМЕРЫ

### Пример 1. Удаление дополнительных параметров защиты от угроз в базе данных
```
PS C:\>Clear-AzSqlDatabaseAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда удаляет дополнительные параметры защиты от угроз из базы данных Database01 на сервере с именем Server01.

# Remove-AzSqlServerCommunicationLink

## SYNOPSIS
Удаляет связь для эластичных транзакций базы данных между двумя серверами.

## СИНТАКСИС

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
С его использованием в azure **SQL** Database удаляется связь между сервером и сервером для транзакций в базе данных Azure SQL.

## ПРИМЕРЫ

### Пример 1. Удаление связи
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

Эта команда удаляет связь между сервером и сервером с именем Link01 в ContosoServer17.

# Join-AzDataLakeStoreItem

## SYNOPSIS
Соединяет один или несколько файлов, чтобы создать один файл в Data Lake Store.

## СИНТАКСИС

```
Join-AzDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]>
 [-Destination] <DataLakeStorePathInstance> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания одного файла в магазине Data Lake Store к нему присоединяется **cmdlet Join-AzDataLakeStoreItem.**

## ПРИМЕРЫ

### Пример 1. Соединить два пункта
```
PS C:\>Join-AzDataLakeStoreItem -AccountName "ContosoADL" -Paths "/MyFiles/File01.txt","/MyFiles/File02.txt" -Destination "/MyFiles/CombinedFile.txt"
```

Эта команда соединяет File01.txt File02.txt, чтобы создать файл CombinedFile.txt.

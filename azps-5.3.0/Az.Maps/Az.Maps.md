# Модуль Az.Maps
## Описание
В этой теме отображается справка по cmdlets Azure Maps.

## Az.Maps Cmdlets
### [Get-AzMapsAccount](Get-AzMapsAccount.md)
Получает учетную запись.

### [Get-AzMapsAccountKey](Get-AzMapsAccountKey.md)
Получает ключи API для учетной записи.
Эти ключи являются механизмом проверки подлинности, который используется при последующих звонках на карты Azure.

### [New-AzMapsAccount](New-AzMapsAccount.md)
Создает учетную запись Azure Maps.

### [New-AzMapsAccountKey](New-AzMapsAccountKey.md)
Повторно сгенерирует ключ учетной записи.

### [Remove-AzMapsAccount](Remove-AzMapsAccount.md)
Удаляет учетную запись Azure Maps.

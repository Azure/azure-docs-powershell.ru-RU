# Start-AzAksDashboard

## SYNOPSIS
Создайте ASH-туннель Kubectl на панели мониторинга управляемого кластера.

## СИНТАКСИС

### GroupNameParameterSet (по умолчанию)
```
Start-AzAksDashboard [-ResourceGroupName] <String> [-Name] <String> [-DisableBrowser] [-ListenPort <Int32>]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObjectParameterSet
```
Start-AzAksDashboard [-InputObject] <PSKubernetesCluster> [-DisableBrowser] [-ListenPort <Int32>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdParameterSet
```
Start-AzAksDashboard [-Id] <String> [-DisableBrowser] [-ListenPort <Int32>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Создайте ASH-туннель Kubectl на панели мониторинга управляемого кластера. SSH-туннель настроен в задание PowerShell Kubectl-Tunnel и его можно найти с помощью `Get-Job` запуска. Туннель должен быть доступен через [http://127.0.0.1:8001](http://127.0.0.1:8001) .

## ПРИМЕРЫ

### Запуск SSH-туннель и открытие браузера на панели мониторинга Kubernetes
```
PS C:\> Start-AzAksDashboard -ResourceGroupName group -Name myCluster
```

# Remove-AzStreamAnalyticsOutput

## SYNOPSIS
Удаляет выходные данные из задания Stream Analytics.

## СИНТАКСИС

```
Remove-AzStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его учетом** можно асинхронно удалить выходные данные из задания Stream Analytics в Azure.

## ПРИМЕРЫ

### ПРИМЕР 1. Удаление результата из задания
```
PS C:\>Remove-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Эта команда удаляет выходные данные из задания StreamingJob.

# AZ. CustomProviders Module
## Nописание
Microsoft Azure PowerShell: командлеты для настраиваемых поставщиков

## Командлеты AZ. CustomProviders
### [Get-AzCustomProvider](Get-AzCustomProvider.md)
Получает манифест настраиваемого поставщика ресурсов.

### [Get-AzCustomProviderAssociation](Get-AzCustomProviderAssociation.md)
Получить связь.

### [New-AzCustomProvider](New-AzCustomProvider.md)
Создает или обновляет настраиваемый поставщик ресурсов.

### [New-AzCustomProviderAssociation](New-AzCustomProviderAssociation.md)
Создание или обновление связи.

### [Remove-AzCustomProvider](Remove-AzCustomProvider.md)
Удаление настраиваемого поставщика ресурсов.

### [Remove-AzCustomProviderAssociation](Remove-AzCustomProviderAssociation.md)
Удаление связи.

### [Update-AzCustomProvider](Update-AzCustomProvider.md)
Обновляет существующий настраиваемый поставщик ресурсов.
Единственным значением, которое может быть обновлено посредством исправления, является тег.

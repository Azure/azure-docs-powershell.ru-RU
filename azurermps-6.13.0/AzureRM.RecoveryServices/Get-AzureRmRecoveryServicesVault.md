# Get-AzureRmRecoveryServicesVault

## КРАТКИй обзор
Получает список хранилищ служб восстановления.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmRecoveryServicesVault [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmRecoveryServicesVault** получает список хранилищ служб восстановления в текущей подписке.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Get-AzureRmRecoveryServicesVault
```

Получение списка хранилищ в выбранной подписке.

### Пример 2
```
PS C:\> Get-AzureRmRecoveryServicesVault -ResourceGroupName "resourceGroup"
```

Получение списка хранилищ в группе ресурсов в выбранной подписке.

### Пример 3
```
PS C:\> Get-AzureRmRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
```

Получение хранилища в группе ресурсов с указанным именем.

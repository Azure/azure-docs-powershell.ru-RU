# Модуль Az.ADDomainServices
## Описание
Microsoft Azure PowerShell: cmdDomainServices

## Az.ADDomainServices Cmdlets
### [Get-AzADDomainService](Get-AzADDomainService.md)
Операция get Domain Service (Получить доменную службу) извлекает json-представление службы домена.

### [New-AzADDomainService](New-AzADDomainService.md)
В ходе операции создания службы домена создается новая служба домена с указанными параметрами.
Если определенная служба уже существует, все исправления будут обновлены и неизменяемые свойства останутся без изменений.

### [New-AzADDomainServiceForestTrust](New-AzADDomainServiceForestTrust.md)
Создание объекта в памяти для ForestTrust

### [New-AzADDomainServiceReplicaSet](New-AzADDomainServiceReplicaSet.md)
Создание в памяти объекта для ReplicaSet

### [Remove-AzADDomainService](Remove-AzADDomainService.md)
В ходе операции удаления доменной службы удаляется существующая доменная служба.

### [Update-AzADDomainService](Update-AzADDomainService.md)
Операцию обновления доменной службы можно использовать для обновления существующего развертывания.
При вызове обновления поддерживаются только свойства, указанные в теле исправления.

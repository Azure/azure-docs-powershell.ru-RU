# Stop-AzApplicationGateway

## SYNOPSIS
Остановка шлюза приложения

## СИНТАКСИС

```
Stop-AzApplicationGateway -ApplicationGateway <PSApplicationGateway> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для шлюза приложения прекращается доступ к шлюзу приложения с применением **cmdlet Stop-AzApplicationGateway.**

## ПРИМЕРЫ

### Пример 1. Остановка шлюза приложения
```
PS C:\>Stop-AzApplicationGateway -ApplicationGateway $AppGw
```

Эта команда останавливает хранение шлюза приложения в переменной $AppGw приложения.

# Remove-AzApiManagement

## КРАТКИй обзор
Удаляет службу управления API.

## Максимальное

```
Remove-AzApiManagement -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApiManagement** Удаляет службу управления API Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление службы управления API
```
PS C:\>Remove-AzApiManagement -ResourceGroupName "ContosoGroup02" -Name "ContosoApi"
```

Эта команда удаляет службу управления API с именем ContosoApi.

# Remove-AzureRmPublicIpAddress

## КРАТКИй обзор
Удаление общедоступного IP-адреса.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmPublicIpAddress** удаляет общедоступный IP-адрес Azure.

## ИЛЛЮСТРИРУЮТ

### 1: Удаление ресурса общедоступного IP-адреса
```
Remove-AzureRmPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Эта команда удаляет ресурс общедоступного IP-адреса с именем $publicIpName в группе ресурсов $rgName.

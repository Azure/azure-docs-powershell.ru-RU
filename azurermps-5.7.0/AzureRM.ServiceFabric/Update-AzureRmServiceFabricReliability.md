# Update-AzureRmServiceFabricReliability

## КРАТКИй обзор
Обновите уровень надежности основного типа узла в кластере.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Update-AzureRmServiceFabricReliability [-ResourceGroupName] <String> [-Name] <String>
 -ReliabilityLevel <ReliabilityLevel> [-AutoAddNode] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Используйте **Update-AzureRmServiceFabricReliability** , чтобы обновить надежность типа основного узла в кластере.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS c:> Add-AzureRmServiceFabricReliability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -ReliabilityLevel Silver
```

Эта команда изменяет уровень надежности основного узла на серебро.

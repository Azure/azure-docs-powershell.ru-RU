# Approve-AzPrivateEndpointConnection

## КРАТКИй обзор
Утверждает подключение к частной конечной точке.

## Максимальное

### ByResourceId (по умолчанию)
```
Approve-AzPrivateEndpointConnection -ResourceId <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResource
```
Approve-AzPrivateEndpointConnection -Name <String> -ServiceName <String> -ResourceGroupName <String>
 [-Description <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **reодобряет-AzPrivateEndpointConnection** утверждает подключение к частной конечной точке.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
Approve-AzPrivateEndpointConnection -Name TestPrivateEndpointConnection -ResourceGroupName TestResourceGroup -ServiceName TestPrivateLinkService
```

В этом примере утверждается подключение к частной конечной точке.

# New-AzUserAssignedIdentity

## SYNOPSIS
Создает новое удостоверение, назначенное пользователю, или обновляет существующее удостоверение.

## СИНТАКСИС

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания нового пользовательского удостоверения создается новый идентификатор **New-AzUserAssignedIdentity.** При его использования с уже существующим удостоверением оно обновлялось.
Чтобы добавить теги Диспетчера ресурсов Azure в удостоверение, воспользуйтесь Set-AzResource командлетом.

# Publish-AzBlueprint

## КРАТКИй обзор
Опубликуйте новую версию чертежа.

## Максимальное

```
Publish-AzBlueprint -Version <String> [-ChangeNote <String>] -Blueprint <PSBlueprint>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Опубликуйте новую версию определения чертежа.

## ИЛЛЮСТРИРУЮТ

### Пример 1

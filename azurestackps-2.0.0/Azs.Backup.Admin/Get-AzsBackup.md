# Get-AzsBackup

## КРАТКИй обзор
Возвращает резервное копирование из расположения на основе имени.

## Максимальное

### Список (по умолчанию)
```
Get-AzsBackup [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String[]>] [-Skip <String>]
 [-Top <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Получить
```
Get-AzsBackup -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzsBackup -InputObject <IBackupAdminIdentity> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## NОПИСАНИЕ
Возвращает резервное копирование из расположения на основе имени.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание резервных копий списка
```powershell
PS C:\> Get-AzsBackup

```

Получение сведений sbout всех резервных копий стеков Azure.

### Пример 2: получение конкретной резервной копии
```powershell
PS C:\> Get-AzsBackup -Name 'backupName'

```

Получение сведений для указанной резервной копии стека Azure.

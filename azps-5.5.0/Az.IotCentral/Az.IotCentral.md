# Модуль Az.IotCentral
## Описание
Управление CRUD для центрального приложения IoT.

## Az.IotCentral Cmdlets
### [Get-AzIotCentralApp](Get-AzIotCentralApp.md)
Свойства одного или нескольких центрального приложения IoT.

### [New-AzIotCentralApp](New-AzIotCentralApp.md)
Создает новое центральное приложение IoT.

### [Remove-AzIotCentralApp](Remove-AzIotCentralApp.md)
Удаляет центральное приложение IoT.

### [Set-AzIotCentralApp](Set-AzIotCentralApp.md)
Обновляет метаданные центрального приложения IoT.

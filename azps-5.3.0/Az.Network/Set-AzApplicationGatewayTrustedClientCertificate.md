# Set-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Изменяет цепочку сертификатов доверенных клиентских ЦС шлюза приложения.

## СИНТАКСИС

```
Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
The **Set-AzApplicationGatewayTrustedClientCertificate cmdificate** the trusted client CA certificate chain of an application gateway.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\clientCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

В примерах выше показано, как обновить существующий объект цепочки сертификатов доверенных клиентов. Первая команда получает шлюз приложения и сохраняет его в переменной $gw. Вторая команда изменяет существующий объект цепочки сертификатов доверенного клиента с помощью нового файла цепочки сертификатов ЦС. Третья команда обновляет шлюз приложения в Azure.

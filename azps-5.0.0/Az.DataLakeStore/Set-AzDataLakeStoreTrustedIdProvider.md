# Set-AzDataLakeStoreTrustedIdProvider

## КРАТКИй обзор
Изменяет указанного доверенного поставщика удостоверений в указанном Data Lake Store.

## Максимальное

```
Set-AzDataLakeStoreTrustedIdProvider [-Account] <String> [-Name] <String> [-ProviderEndpoint] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzDataLakeStoreTrustedIdProvider** изменяет указанный доверенный поставщик удостоверений в указанном Data Lake Store.

## ИЛЛЮСТРИРУЮТ

### Пример 1: обновление доверенной конечной точки поставщика удостоверений
```
PS C:\> Set-AzDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider -ProviderEndpoint "https://sts.windows.net/6b04908c-b91f-40ce-8024-7ee8a4fd6150"
```

Это действие обновит конечную точку поставщика для правила брандмауэра "MyProvider" в учетной записи "ContosoADL" в " https://sts.windows.net/6b04908c-b91f-40ce-8024-7ee8a4fd6150 "

# Get-AzSqlElasticPoolAdvisor

## КРАТКИй обзор
Получает один или несколько консультантов для пула эластичных БД SQL Azure.

## Максимальное

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlElasticPoolAdvisor** получает один или несколько советников пула эластичных БД Azure SQL для пула эластичных БД Azure SQL.

# Unregister-AzResourceProvider

## КРАТКИй обзор
Отмена регистрации поставщика ресурсов.

## Максимальное

```
Unregister-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Unregister-AzResourceProvider** отменяет регистрацию поставщика ресурсов Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Отмена регистрации поставщика ресурсов с помощью ProviderNamespace

```powershell
PS C:\>Unregister-AzResourceProvider -ProviderNamespace "Microsoft.support"
```

Эта команда отменяет регистрацию поставщика ресурсов "Microsoft. support".

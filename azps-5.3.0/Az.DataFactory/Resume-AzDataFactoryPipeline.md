# Resume-AzDataFactoryPipeline

## SYNOPSIS
Возобновление приостановленного конвейера в фабрике данных.

## СИНТАКСИС

### ByFactoryName (По умолчанию)
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для приостановленного конвейера в фабрике данных Azure возобновляется **cmdlet AzDataFactoryPipeline.**

## ПРИМЕРЫ

### Пример 1. Возобновление конвейера
```powershell
PS C:\>Resume-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to resume pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Эта команда возобновляет конвейер под названием DPWikisample в фабрике данных с именем WikiADF.
Для **приостановки конвейера используйте cmdlet Suspend-AzDataFactoryPipeline.**
Команда возвращает значение $True.

### Пример 2

Возобновление приостановленного конвейера в фабрике данных. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Resume-AzDataFactoryPipeline -DataFactory $DataFactory -Name 'DPWikisample'
```

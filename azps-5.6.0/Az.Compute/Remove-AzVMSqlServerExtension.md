# Remove-AzVMSqlServerExtension

## SYNOPSIS
Удаляет расширение SQL Server с виртуальной машины.

## СИНТАКСИС

```
Remove-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **удаления расширения** AzureSQL Server из виртуальной машины удаляется расширение AzureSQL Server.

## ПРИМЕРЫ

### Пример 1. Удаление SQL Server расширения
```
PS C:\> Remove-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" -Name "SqlIaaSAgent"
```

Эта команда удаляет расширение SQL Server с виртуальной машины с именем ContosoVM22.

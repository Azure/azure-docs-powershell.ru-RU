### [Update-AzSqlSyncSchema](Update-AzSqlSyncSchema.md)
Обновите схему синхронизации для базы данных участника синхронизации или базы данных концентратора синхронизации.
Она будет получать последнюю схему базы данных из реальной базы данных, а затем использовать ее для обновления схемы, кэшированной с помощью базы данных метаданных синхронизации.
Если задано значение "SyncMemberName", будет обновлена схема базы данных участника; в противном случае будет обновлена схема базы данных основного сервера.

# New-AzApiManagementGroup

## SYNOPSIS
Создает группу управления API.

## СИНТАКСИС

```
New-AzApiManagementGroup -Context <PsApiManagementContext> [-GroupId <String>] -Name <String>
 [-Description <String>] [-Type <PsApiManagementGroupType>] [-ExternalId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания группы управления API создается группа **New-AzApiManagementGroup.**

## ПРИМЕРЫ

### Пример 1. Создание группы управления
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>New-AzApiManagementGroup -Context $apimContext -Name "Group0001"
```

Эта команда создает группу управления.

### Пример 2

Создает группу управления API. (autogenerated)

```powershell
<!-- Aladdin Generated Example --> 
New-AzApiManagementGroup -Context <PsApiManagementContext> -Description 'Create Echo Api V4' -GroupId '0001' -Name 'Group0001' -Type Custom
```

### -Type
Тип группы. Настраиваемая группа — это пользовательская группа. В состав system Group входят администратор, разработчики и гости. Создать или обновить группу системы невозможно.

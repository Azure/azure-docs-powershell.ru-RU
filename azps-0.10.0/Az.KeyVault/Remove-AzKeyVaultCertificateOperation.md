# Remove-AzKeyVaultCertificateOperation

## КРАТКИй обзор
Удаляет операцию с сертификатом из хранилища ключей.

## Максимальное

```
Remove-AzKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzKeyVaultCertificateOperation** удаляет операцию с сертификатом из хранилища ключей.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление операции с сертификатом
```
PS C:\>Remove-AzKeyVaultCertificateOperation -VaultName "ContosoKV01" -Name "TestCert01" -Force
```

Эта команда удаляет операцию с сертификатом с именем TestCert01 из хранилища ключей ContosoKV01 без запроса подтверждения.

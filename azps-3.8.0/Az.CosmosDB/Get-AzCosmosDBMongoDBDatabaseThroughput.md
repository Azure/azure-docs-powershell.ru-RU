# Get-AzCosmosDBMongoDBDatabaseThroughput

## КРАТКИй обзор
Возвращает свойства пропускной способности CosmosDB для базы данных MongoDB.

## Максимальное

### ByNameParameterSet (по умолчанию)
```
Get-AzCosmosDBMongoDBDatabaseThroughput -ResourceGroupName <String> -AccountName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBMongoDBDatabaseThroughput -Name <String> -InputObject <PSDatabaseAccount>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzCosmosDBMongoDBDatabaseThroughput** получает свойства пропускной способности базы данных MongoDB.

## ИЛЛЮСТРИРУЮТ

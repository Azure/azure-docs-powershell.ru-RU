# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## КРАТКИй обзор
Получает расширенную политику безопасности данных управляемого экземпляра.

## Максимальное

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlInstanceAdvancedDataSecurityPolicy** извлекает политику повышенной безопасности данных управляемого экземпляра.

## ИЛЛЮСТРИРУЮТ

### Пример 1 — получение расширенной защиты данных управляемого экземпляра
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Пример 2: получение расширенной защиты данных управляемых экземпляров от ресурса управляемых экземпляров
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

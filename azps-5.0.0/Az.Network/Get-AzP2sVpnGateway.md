# Get-AzP2sVpnGateway

## КРАТКИй обзор
Возвращает существующий P2SVpnGateway в разделе VirtualHub.

## Максимальное

### ListBySubscriptionId (по умолчанию)
```
Get-AzP2sVpnGateway [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ListByResourceGroupName
```
Get-AzP2sVpnGateway [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzP2sVpnGateway** позволяет получить существующий P2SVpnGateway в разделе VirtualHub.

# Get-AzureRmSqlElasticPool

## КРАТКИй обзор
Возвращает эластичные пулы и значения их свойств в базе данных Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSqlElasticPool** получает эластичные пулы и значения их свойств.
Укажите имя существующего эластичного пула, чтобы просмотреть значения свойств только для этого пула.

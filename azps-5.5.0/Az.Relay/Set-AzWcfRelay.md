# Set-AzWcfRelay

## SYNOPSIS
Обновляет описание WcfRelay в указанном пространстве имен ретрансляции.

## СИНТАКСИС

### WcfRelayInputObjectSet
```
Set-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-InputObject <PSWcfRelayAttributes>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### WcfRelayPropertiesSet
```
Set-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String> [-UserMetadata <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Командлет Set-AzWcfRelay обновляет описание WcfRelay в указанном пространстве имен ретрансляторов.

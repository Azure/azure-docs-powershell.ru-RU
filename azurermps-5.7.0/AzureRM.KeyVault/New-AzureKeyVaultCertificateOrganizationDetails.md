# New-AzureKeyVaultCertificateOrganizationDetails

## КРАТКИй обзор
Создает объект подробных сведений об организации сертификата в памяти.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureKeyVaultCertificateOrganizationDetails [-Id <String>]
 [-AdministratorDetails <System.Collections.Generic.List`1[Microsoft.Azure.Commands.KeyVault.Models.PSKeyVaultCertificateAdministratorDetails]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureKeyVaultCertificateOrganizationDetails** создает объект подробных сведений об организации сертификата в памяти.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание объекта сведений об Организации
```
PS C:\>$AdminDetails = New-AzureKeyVaultCertificateAdministratorDetails -FirstName "Patti" -LastName "Fuller" -EmailAddress "Patti.Fuller@contoso.com" -PhoneNumber "1234567890"
$OrgDetails = New-AzureKeyVaultCertificateOrganizationDetails -Name "Contoso" -Address1 "1 Redmond Way" -Address2 "Suite 906" -City "Redmond" -State "WA" -Zip 98052 -Country "US" -AdministratorDetails $AdminDetails
```

В первой команде создается объект сведений администратора сертификата, который затем сохраняется в переменной $AdminDetails.

Вторая команда создает объект сведений об организации сертификата и сохраняет его в переменной $OrgDetails.

# Get-AzAdvisorConfiguration

## SYNOPSIS
Получите конфигурации помощника Azure для данной подписки или группы ресурсов.

## СИНТАКСИС

```
Get-AzAdvisorConfiguration [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Конфигурации, связанные с подпиской, имеют два типа:

Конфигурация на уровне подписки: для подписки может быть только одна конфигурация этого типа. LowCpuThreshold и Exclude являются единственным свойством этого типа конфигурации.

Конфигурация уровня ResourceGroup: для каждой группы ресурсов в подписке может быть только одна конфигурация. Единственным свойством такого типа конфигурации является exclude.

### Пример 1
```powershell
PS C:\>$data = Get-AzAdvisorConfiguration

Name       : {user_subscription}
Properties : Microsoft.Azure.Commands.Advisor.Cmdlets.Models.PsAzureAdvisorConfigurationProperties
Type       : Microsoft.Advisor/Configurations

Id         : /subscriptions/{user_subscription}/providers/Microsoft.Advisor/configurations/{user_subscription}-{resourceGroupName}
Name       : {user_subscription}-{resourceGroupName}
Properties : Microsoft.Azure.Commands.Advisor.Cmdlets.Models.PsAzureAdvisorConfigurationProperties
Type       : Microsoft.Advisor/Configurations

PS C:\>$data[0].Properties
AdditionalProperties :
Exclude              : False

AdditionalProperties :
Exclude              : True
LowCpuThreshold      : null

```
Извлекает список конфигураций помощников Azure.

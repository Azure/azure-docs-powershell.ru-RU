# Get-AzureRmSiteRecoveryNetwork

## КРАТКИй обзор
Возвращает сведения о сетях, управляемых восстановлением сайта для текущего хранилища.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

### По умолчанию (по умолчанию)


## NОПИСАНИЕ
Командлет **Get-AzureRmSiteRecoveryNetwork** получает сведения об сетях Azure Site Recovery для текущего хранилища Azure Site Recovery.

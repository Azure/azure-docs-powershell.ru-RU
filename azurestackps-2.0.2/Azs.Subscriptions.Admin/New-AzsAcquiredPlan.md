Свойства сложных параметров для создания описанных ниже параметров создайте хэш-таблицу, содержащую соответствующие свойства. Для получения сведений о хэш-таблицах запустите Get-Help about_Hash_Tables.

ACQUIREDPLANDEFINITION <IPlanAcquisition> : 
  - `[AcquisitionId <String>]`: Идентификатор приобретения.
  - `[AcquisitionTime <DateTime?>]`: Время приобретения.
  - `[ExternalReferenceId <String>]`: Идентификатор внешней ссылки.
  - `[Id <String>]`: Идентификатор в контексте подписки клиента.
  - `[PlanId <String>]`: Идентификатор плана в контексте подписки клиента.
  - `[ProvisioningState <ProvisioningState?>]`: Состояние подготовки.

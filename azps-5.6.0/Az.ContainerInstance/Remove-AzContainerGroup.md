# Remove-AzContainerGroup

## SYNOPSIS
Удаляет группу контейнеров.

## СИНТАКСИС

### RemoveContainerGroupByResourceGroupAndNameParamSet (по умолчанию)
```
Remove-AzContainerGroup [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveContainerGroupByPSContainerGroupParamSet
```
Remove-AzContainerGroup -InputObject <PSContainerGroup> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveContainerGroupByResourceIdParamSet
```
Remove-AzContainerGroup -ResourceId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При **этом удаляется** группа контейнеров.

## ПРИМЕРЫ

### Пример 1. Удаление группы контейнеров
```
PS C:\> Remove-AzContainerGroup -ResourceGroupName MyResourceGroup -Name MyContainer
```

Эта команда удаляет указанную группу контейнеров.

### Пример 2. Удаляет группу контейнеров с помощью piping
```
PS C:\> Get-AzContainerGroup -ResourceGroupName MyResourceGroup -Name MyContainer | Remove-AzContainerGroup
```

Эта команда удаляет группу контейнеров с помощью piping.

### Пример 3. Удаляет группу контейнеров по ИД ресурсов.
```
PS C:\> Find-AzResource -ResourceGroupEquals MyResourceGroup -ResourceNameEquals MyContainer | Remove-AzContainerGroup
```

Эта команда удаляет группу контейнеров по ИД ресурсов.

# Remove-AzKeyVault

## SYNOPSIS
Удаляет хранилище ключей.

## СИНТАКСИС

### ByAvailableVault (по умолчанию)
```
Remove-AzKeyVault [-VaultName] <String> [[-ResourceGroupName] <String>] [[-Location] <String>] [-Force]
 [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByDeletedVault
```
Remove-AzKeyVault [-VaultName] <String> [-Location] <String> [-InRemovedState] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectByAvailableVault
```
Remove-AzKeyVault [-InputObject] <PSKeyVault> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectByDeletedVault
```
Remove-AzKeyVault [-InputObject] <PSKeyVault> [-InRemovedState] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdByAvailableVault
```
Remove-AzKeyVault [-ResourceId] <String> [[-Location] <String>] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdByDeletedVault
```
Remove-AzKeyVault [-ResourceId] <String> [-Location] <String> [-InRemovedState] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При **нажатии на клавиши Remove-AzKeyVault** указанное хранилище ключа удаляется.
При этом также удаляются все ключи и секреты, содержащиеся в этом экземпляре.
Имейте в виду, что хотя для этого cmdlet не требуется указывать группу ресурсов, это необходимо для улучшения производительности.

## ПРИМЕРЫ

### Пример 1. Удаление сейфа ключа
```powershell
PS C:\> Remove-AzKeyVault -VaultName "Contoso03Vault" -PassThru

True
```

Эта команда удаляет из текущей подписки хранилище ключей Contoso03Vault.

### Пример 2. Удаление сейфа ключа из указанной группы ресурсов
```powershell
PS C:\> Remove-AzKeyVault -Name "Contoso03Vault" -ResourceGroupName "Group14" -PassThru

True
```

Эта команда удаляет хранилище ключей Contoso03Vault из именоваемой группы ресурсов.
Если имя группы ресурсов не указано, выполняется поиск именоваемого хранилища ключей для удаления в текущей подписке.

### Пример 3. Удаление управляемого hsm
```powershell
PS C:\>  Remove-AzKeyVault -Name "testManagedHsm" -Hsm -PassThru

True
```

Эта команда удаляет управляемый hsm testManagedHsm из текущей подписки.

# Register-AzureRmResourceProvider

## КРАТКИй обзор
Регистрация поставщика ресурсов.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Register-AzureRmResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Register-AzureRmResourceProvider** регистрирует поставщик ресурсов Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: регистрация поставщика
```
PS C:\>Register-AzureRmResourceProvider -ProviderNamespace Microsoft.Network
```

В этом случае регистрируется служба Microsoft. Network для вашей учетной записи.

# Start-AzVM

## КРАТКИй обзор
Запуск виртуальной машины Azure.

## Максимальное

### ResourceGroupNameParameterSetName (по умолчанию)
```
Start-AzVM [-Name] <String> [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Start-AzVM [-Name] <String> [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Start-AzVM** запускает виртуальную машину Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Запуск виртуальной машины
```
PS C:\> Start-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Эта команда запускает виртуальную машину с именем VirtualMachine07 в ResourceGroup11.

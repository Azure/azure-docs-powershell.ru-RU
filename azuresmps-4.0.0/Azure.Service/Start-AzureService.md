# Start-AzureService

## КРАТКИй обзор
Запускает указанную размещенную службу в Windows Azure.

## Максимальное

```
Start-AzureService [-ServiceName <String>] [-Slot <String>] [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
В этом разделе описан командлет в версии 0.8.10 модуля Microsoft Azure PowerShell.
Чтобы получить версию модуля, который вы используете, введите в командной консоли Azure PowerShell `(Get-Module -Name Azure).Version` .

Командлет **Start-AzureService** запускает указанную размещенную службу в Windows Azure, если она находится в остановленном состоянии.
Обратите внимание, что командлет **Publish-AzureServiceProject** автоматически пытается запустить службу.

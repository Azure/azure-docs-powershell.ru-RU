# Reset-AzureVirtualNetworkGateway

## КРАТКИй обзор
Сброс шлюза виртуальной сети.

## Максимальное

```
Reset-AzureVirtualNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Reset-AzureVirtualNetworkGateway** сбрасывает шлюз виртуальной сети.

# Get-AzSqlDatabaseExpanded

## SYNOPSIS
Получает базу данных и значения ее расширенных свойств.

## СИНТАКСИС

```
Get-AzSqlDatabaseExpanded [-ServerName] <String> [[-DatabaseName] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzSqlDatabaseExpanded** получает базу данных и значения ее расширенных свойств.
Этот cmdlet также поддерживается службой SQL Server Stretch Database в Azure.

## ПРИМЕРЫ

### Пример 1. Получить объект базы данных со сведениями консультанта уровня обслуживания
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда возвращает базу данных с расширенными свойствами, которые содержат сведения помощника для уровня обслуживания.

### Пример 2. Список объектов базы данных с использованием фильтрации
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database*"
```

Эта команда возвращает расширенный объект базы данных для всех баз данных Сервера01, которые начинаются с "База данных".

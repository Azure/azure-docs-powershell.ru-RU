# Remove-AzureRmHDInsightPersistedScriptAction

## КРАТКИй обзор
Удаляет сохраненное действие сценария из кластера HDInsight.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmHDInsightPersistedScriptAction [-ClusterName] <String> [-Name] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmHDInsightPersistedScriptAction** удаляет сохраненное действие сценария из списка сохраненных действий сценария в Azure HDInsight Cluster.
Удаленный сценарий больше не будет выполняться при масштабировании кластера.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление действия сценария из списка сохраненных действий сценария в кластере
```
PS C:\>Remove-AzureRmHDInsightPersistedScriptAction `
            -ClusterName "your-hadoop-001" `
            -Name "Scriptaction"
```

Эта команда удаляет действие сценария с именем Scriptaction из списка сохраненных действий сценария в указанном кластере.

# Get-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Получает конфигурацию пула адресов для резерва нагрузки.

## СИНТАКСИС

```
Get-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для этого с помощью cmdlet **Get-AzLoadBalancerBackendAddressPoolConfig** можно получить один пул адресов backend или список пулов адресов в балансе нагрузки.

## ПРИМЕРЫ

### Пример 1. Получить пул адресов для backend
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" -LoadBalancer $loadbalancer
```

Первая команда получает существующий балансировка нагрузки MyLoadBalancer в группе ресурсов MyResourceGroup, а затем сохраняет ее в переменной $loadbalancer загрузки.
Вторая команда получает связанную конфигурацию пула адресов backend с именем BackendAddressPool02 для балансиров нагрузки в $loadbalancer.

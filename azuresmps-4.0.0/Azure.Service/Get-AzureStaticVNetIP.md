# Get-AzureStaticVNetIP

## КРАТКИй обзор
Получает данные статического IP-адреса VNet из объекта виртуальной машины, если таковые имеются.

## Максимальное

```
Get-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureStaticVNetIP** получает сведения о статических IP-адресах виртуальных сетей (VNet) из объекта виртуальной машины, если таковые имеются.

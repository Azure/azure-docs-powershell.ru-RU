# Get-AzureRmApiManagementApiVersionSet

## КРАТКИй обзор
Получение подробных сведений о наборах версий API

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmApiManagementApiVersionSet -Context <PsApiManagementContext> [-ApiVersionSetId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmApiManagementApiVersionSet** получает сведения о наборах версий API, настроенных в контексте управления API.

# Remove-AzStorageShareStoredAccessPolicy

## SYNOPSIS
Удаляет хранимую политику доступа из хранилища.

## СИНТАКСИС

```
Remove-AzStorageShareStoredAccessPolicy [-ShareName] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для удаления из хранилища Azure политика доступа **remove-AzStorageShareStoredAccessPolicy** удаляет хранимую политику доступа.

## ПРИМЕРЫ

### Пример 1. Удаление хранимой политики доступа из хранилища Azure
```
PS C:\>Remove-AzStorageShareStoredAccessPolicy -ShareName "ContosoShare" -Policy "GeneralPolicy"
```

Эта команда удаляет из ContosoShare хранимую политику доступа с именем GeneralPolicy.

### -ConcurrentTaskCount
Указывает максимальное число сетевых звонков.
С помощью этого параметра можно ограничить одновременное использование локального ЦП и пропускной способности, указав максимальное количество сетевых звонков одновременно.
Указанное значение является абсолютным и не умножается на основное.
Этот параметр помогает уменьшить количество проблем с сетевым подключением при низкой пропускной способности, например 100 КБ в секунду.
Значение по умолчанию — 10.

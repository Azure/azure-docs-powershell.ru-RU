# Get-AzCosmosDBCassandraKeyspace

## SYNOPSIS
Получает ключ "Еста" Климова Вадима.

## СИНТАКСИС

### ByNameParameterSet (по умолчанию)
```
Get-AzCosmosDBCassandraKeyspace -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBCassandraKeyspace [-Name <String>] -ParentObject <PSDatabaseAccountGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzCosmosDBCassandraKeyspace** создает или обновляет существующую клавишную область Вадима Кузьмина.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzCosmosDBCassandraKeyspace -ResourceGroupName {resourceGroupName} -AccountName {accountName} -Name {name}

Name    Id   Resource
{name}  {id} {resourceObject}
```

Get-AzCosmosDBCassandraKeyspace получает свойства существующего пространства ДляРки. Вы можете развернуть ресурс, чтобы получить _etag, _ts и _rid свойства.

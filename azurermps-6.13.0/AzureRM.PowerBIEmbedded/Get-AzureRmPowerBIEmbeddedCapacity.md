# Get-AzureRmPowerBIEmbeddedCapacity

## КРАТКИй обзор
Получает сведения о емкости встроенных данных PowerBI.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### ByCapacityOrResourceGroupOrSubscription (по умолчанию)
```
Get-AzureRmPowerBIEmbeddedCapacity [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzureRmPowerBIEmbeddedCapacity -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Get-AzureRmPowerBIEmbeddedCapacity получает сведения о емкости встроенных данных PowerBI.

# Remove-AzBatchApplication

## КРАТКИй обзор
Удаляет приложение из пакетной учетной записи.

## Максимальное

```
Remove-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationId] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzBatchApplication** удаляет приложение из пакетной учетной записи Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление приложения из пакетной учетной записи
```
PS C:\>Remove-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "Litware"
```

Эта команда удаляет приложение беседа Litware из учетной записи ContosoBatch.
Команда не выполняется, если приложение включает в себя пакеты.

# New-AzFirewallHubIpAddress

## SYNOPSIS
Ip addresses assoicated to the firewall on virtual hub

## СИНТАКСИС

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Ip-адреса, которые ассоциированы с брандмауэром в виртуальном центре. Это могут быть как общедоступные, так и частные адреса.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

В этом примере создается IP-адрес концентратора с количеством общедоступных IP-адресов (2). Объект HubIPAddress передается в брандмауэр в виртуальном центре.

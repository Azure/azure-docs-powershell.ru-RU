# Get-AzRelayHybridConnection

## SYNOPSIS
Возвращает описание указанного гибридного подключения в области имен ретрансляции.

## СИНТАКСИС

```
Get-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Командлет **Get-AzRelayHybridConnection** получает описание указанного гибридного подключения в области имен ретрансляции.

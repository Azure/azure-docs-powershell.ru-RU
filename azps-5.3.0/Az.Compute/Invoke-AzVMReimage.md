# Invoke-AzVMReimage

## SYNOPSIS
Повторное использование виртуальной машины Azure.

## СИНТАКСИС

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Invoke-AzVMReimage** повторно вызывает виртуальную машину Azure.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Эта команда переимежает виртуальную машину с именем VirtualMachine07 в ResourceGroup11.

# Add-AzsPlatformImage

## КРАТКИй обзор
Создает новый образ платформы с заданными издателем, предложением, SKU и версией.

## Максимальное

### CreateExpanded (по умолчанию)
```
Add-AzsPlatformImage -Offer <String> -Publisher <String> -Sku <String> -Version <String> [-Location <String>]
 [-SubscriptionId <String>] [-BillingPartNumber <String>] [-DataDisks <IDataDisk[]>] [-OsType <OSType>]
 [-OsUri <String>] [-ProvisioningState <ProvisioningState>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Заново
```
Add-AzsPlatformImage -Offer <String> -Publisher <String> -Sku <String> -Version <String>
 -NewImage <IPlatformImageParameters> [-Location <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentity
```
Add-AzsPlatformImage -InputObject <IComputeAdminIdentity> -NewImage <IPlatformImageParameters>
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentityExpanded
```
Add-AzsPlatformImage -InputObject <IComputeAdminIdentity> [-BillingPartNumber <String>]
 [-DataDisks <IDataDisk[]>] [-OsType <OSType>] [-OsUri <String>] [-ProvisioningState <ProvisioningState>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## NОПИСАНИЕ
Создает новый образ платформы с заданными издателем, предложением, SKU и версией.

## Пуск

Свойства сложных параметров для создания описанных ниже параметров создайте хэш-таблицу, содержащую соответствующие свойства. Для получения сведений о хэш-таблицах запустите Get-Help about_Hash_Tables.

ДИСКИ с данными <IDataDisk [] >: диски данных, используемые образом платформы.
  - `[Lun <Int32?>]`: Логический номер устройства.
  - `[Uri <String>]`: Расположение шаблона диска.

INPUTOBJECT <IComputeAdminIdentity> : параметр Identity
  - `[DiskId <String>]`: Идентификатор GUID диска.
  - `[Id <String>]`: Путь к удостоверению ресурса
  - `[Location <String>]`: Расположение ресурса.
  - `[MigrationId <String>]`: Имя GUID задания миграции.
  - `[Offer <String>]`: Название предложения.
  - `[Publisher <String>]`: Имя издателя.
  - `[QuotaName <String>]`: Имя квоты.
  - `[Sku <String>]`: Имя SKU.
  - `[SubscriptionId <String>]`: Учетные данные подписки, которые однозначно определяют подписку Microsoft Azure. Идентификатор подписки образует часть URI для каждого вызова службы.
  - `[Type <String>]`: Тип расширения.
  - `[Version <String>]`: Версия ресурса.

NEWIMAGE <IPlatformImageParameters> : параметры, используемые для создания нового образа платформы.
  - `[DataDisk <IDataDisk[]>]`: Диски данных, используемые образом платформы.
    - `[Lun <Int32?>]`: Логический номер устройства.
    - `[Uri <String>]`: Расположение шаблона диска.
  - `[DetailBillingPartNumber <String>]`: Номер детали используется для выставления счетов за программные расходы.
  - `[OSDiskOstype <OSType?>]`: Тип операционной системы.
  - `[OSDiskUri <String>]`: Расположение диска.
  - `[ProvisioningState <ProvisioningState?>]`: Состояние подготовки образа платформы.

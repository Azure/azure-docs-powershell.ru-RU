# Disable-AzureWebsiteDebug

## КРАТКИй обзор
Отключает отладку веб-сайта.

## Максимальное

```
Disable-AzureWebsiteDebug [-PassThru] [-Name <String>] [-Slot <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Отключает отладку веб-сайта в Visual Studio.

## ИЛЛЮСТРИРУЮТ

### --------------Отключить отладку веб-сайта--------------
```
PS C:\> Disable-AzureWebsiteDebug -Name MyWebsite
```

Отключение отладки веб-сайтов на веб-сайте MyWebsite

# New-AzCosmosDBCassandraClusterKey

## SYNOPSIS
Создает новый ключ кластера Климова (Сергей Климов).

## СИНТАКСИС

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Новая **клавиша AzCosmosDBCassandraClusterKey** создает новый ключ кластера Кузьмина Кузьмина Вадима.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```

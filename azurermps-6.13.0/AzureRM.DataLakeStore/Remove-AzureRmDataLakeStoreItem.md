# Remove-AzureRmDataLakeStoreItem

## КРАТКИй обзор
Удаление файла или папки в Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]> [-Recurse] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmDataLakeStoreItem** удаляет файл или папку в Data Lake Store.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление нескольких элементов
```
PS C:\>Remove-AzureRmDataLakeStoreItem -AccountName "ContosoADL" -Paths "/File01.txt","/MyFiles/File.csv"
```

Эта команда удаляет файлы File01.txt и File.csv из Data Lake Store.

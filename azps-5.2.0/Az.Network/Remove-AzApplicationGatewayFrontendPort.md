# Remove-AzApplicationGatewayFrontendPort

## SYNOPSIS
Удаление переднего порта из шлюза приложения.

## СИНТАКСИС

```
Remove-AzApplicationGatewayFrontendPort -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для удаления переднего порта из шлюза приложения Azure удаляется клиентский порт **Remove-AzApplicationGatewayFrontendPort.**

## ПРИМЕРЫ

### Пример: удаление переднего порта из шлюза приложения
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort02"
```

Первая команда получает шлюз приложения ApplicationGateway01, который принадлежит группе ресурсов ResourceGroup01, и сохраняет шлюз в $AppGw переменной.
Вторая команда удаляет порт FrontEndPort02 из шлюза приложения.

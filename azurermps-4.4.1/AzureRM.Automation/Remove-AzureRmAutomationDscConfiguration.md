# Remove-AzureRmAutomationDscConfiguration

## КРАТКИй обзор
Удаляет конфигурации DSC из автоматизации.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmAutomationDscConfiguration [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmAutomationDscConfiguration** удаляет из Azure Automation параметры необходимой конфигурации состояния (DSC).

# Start-AzVM

## SYNOPSIS
Запускает виртуальную машину Azure.

## СИНТАКСИС

### ResourceGroupNameParameterSetName (по умолчанию)
```
Start-AzVM [-Name] <String> [-NoWait] [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Start-AzVM [-NoWait] [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для запуска виртуальной машины Запускается клиент **Start-AzVM.**

## ПРИМЕРЫ

### Пример 1. Запуск виртуальной машины
```
PS C:\> Start-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Эта команда запускает виртуальную машину с именем VirtualMachine07 в ResourceGroup11.

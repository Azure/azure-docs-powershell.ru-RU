# Remove-AzureAvailabilitySet

## КРАТКИй обзор
Удаляет группу доступности из виртуальной машины Azure.

## Максимальное

```
Remove-AzureAvailabilitySet -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureAvailabilitySet** удаляет группу доступности из виртуальной машины Azure.

# New-AzIotHubKey

## КРАТКИй обзор
Создание ключа центра Azure IoT.

## Максимальное

### Источник данных (по умолчанию)
```
New-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String> [-RenewKey] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
New-AzIotHubKey [-HubId] <String> [-KeyName] <String> [-RenewKey] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Создание ключа центра Azure IoT.

## ИЛЛЮСТРИРУЮТ

### Пример 1 повторное создание первичного ключа
```
PS C:\> New-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "testKey" -RenewKey "primary"
```

Повторно создан первичный ключ для политики авторизации "testKey" центра Azure IOT.

### Пример 2 клавиши для замены
```

```

Переключение клавиш для политики авторизации "testKey" центра Azure IOT.

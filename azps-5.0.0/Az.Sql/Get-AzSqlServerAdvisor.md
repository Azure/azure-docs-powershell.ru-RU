# Get-AzSqlServerAdvisor

## КРАТКИй обзор
Получает один или несколько консультантов для Azure SQL Server.

## Максимальное

```
Get-AzSqlServerAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlServerAdvisor** получает один или несколько консультантов Azure SQL Server для Azure SQL Server.

# Get-AzRedisCache

## КРАТКИй обзор
Возвращает кэш Redis.

## Максимальное

```
Get-AzRedisCache [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzRedisCache** получает указанный кэш Azure Redis.
Если вы укажете параметр "нет", эта операция возвращает каждый кэш Redis для текущей подписки.

## ИЛЛЮСТРИРУЮТ

        Type               : Microsoft.Cache/Redis
        HostName           : myexists.redis.cache.windows.net

        RedisConfiguration : {}
        EnableNonSslPort   : False

Эта команда получает кэш Redis с именем myexists.

# Clear-AzSqlDatabaseAdvancedThreatProtectionSettings

## КРАТКИй обзор
Удаление расширенных параметров защиты от угроз из базы данных.

## Максимальное

```
Clear-AzSqlDatabaseAdvancedThreatProtectionSettings [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **clear-AzSqlDatabaseAdvancedThreatProtectionSettings** удаляет расширенные параметры защиты от угроз из базы данных SQL AzureAzure.
Чтобы использовать этот командлет, укажите параметры *ResourceGroupName* и *ServerName* , чтобы определить базу данных, из которой этот командлет удаляет параметры.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление расширенных параметров защиты от угроз для базы данных
```
PS C:\>Clear-AzSqlDatabaseAdvancedThreatProtectionSettings -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда удаляет расширенные параметры защиты от угроз из базы данных с именем Database01 на сервере под названием Server01.

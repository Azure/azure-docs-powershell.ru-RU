# Enter-AzWebAppContainerPSSession

## КРАТКИй обзор
Открытие удаленного сеанса PowerShell в контейнере Windows, указанном на данном сайте или в области, и данной группе ресурсов.

## Максимальное

### S1 (по умолчанию)
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [[-SlotName] <String>] [-ResourceGroupName] <String>
 [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
открытие удаленного сеанса PowerShell в контейнере Windows, указанном на данном сайте или в области, и данной группе ресурсов.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Enter-AzWebAppContainerPSSession -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Эта команда открывает удаленный сеанс PowerShell в приложении контейнера Windows ContosoASP

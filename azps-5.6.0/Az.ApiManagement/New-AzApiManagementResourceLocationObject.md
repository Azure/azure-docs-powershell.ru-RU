# New-AzApiManagementResourceLocationObject

## SYNOPSIS
Создать новый контракт с расположением ресурса (используется в шлюзах).

## СИНТАКСИС

```
New-AzApiManagementResourceLocationObject -Name <String> [-City <String>] [-District <String>]
 [-CountryOrRegion <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания нового контракта с расположением ресурса (используется в шлюзах) используется новый проект **AzApiManagementResourceLocationObject.**

## ПРИМЕРЫ

### Пример 1. Создание контракта с расположением ресурса
```powershell
PS C:\>$location = New-AzApiManagementResourceLocationObject -Name "n1" -City "c1" -District "d1" -CountryOrRegion "r1"
```

Эта команда создает расположение ресурса.

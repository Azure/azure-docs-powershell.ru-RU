# New-AzDataLakeStoreItem

## КРАТКИй обзор
Создание нового файла или папки в Data Lake Store.

## Максимальное

```
New-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance> [[-Value] <Object>]
 [[-Encoding] <Encoding>] [-Folder] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzDataLakeStoreItem** создает новый файл или папку в Data Lake Store.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание нового файла и новой папки
```
PS C:\>New-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/NewFile.txt"
PS C:\> New-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/NewFolder" -Folder
```

Первая команда создает файл NewFile.txt для указанной учетной записи.
Вторая команда создает папку NewFolder в корневой папке.

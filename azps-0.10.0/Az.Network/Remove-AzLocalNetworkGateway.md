# Remove-AzLocalNetworkGateway

## КРАТКИй обзор
Удаление шлюза локальной сети

## Максимальное

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Шлюз локальной сети — это объект, представляющий локальное устройство VPN.

Командлет **Remove-AzLocalNetworkGateway** удаляет объект, представляющий локальный шлюз, на основе имени и имени группы ресурсов.

## ИЛЛЮСТРИРУЮТ

### 1: Удаление шлюза локальной сети
```
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Удаляет объект шлюза локальной сети с именем "myLocalGW" в группе ресурсов "myRG"

Примечание. необходимо сначала удалить все подключения к шлюзу локальной сети с помощью командлета **Remove-AzVirtualNetworkGatewayConnection** .

# Remove-AzureVMCustomScriptExtension

## КРАТКИй обзор
Удаляет настраиваемое расширение сценария из виртуальной машины.

## Максимальное

```
Remove-AzureVMCustomScriptExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureVMCustomScriptExtension** Удаляет настраиваемое расширение сценария из виртуальной машины.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление расширения настраиваемого сценария для виртуальной машины
```
PS C:\> Remove-AzureVMCustomScriptExtension -VM $VM;
```

Эта команда удаляет расширение настраиваемого сценария виртуальной машины Azure из указанной виртуальной машины, как оно хранится в $VM переменной.

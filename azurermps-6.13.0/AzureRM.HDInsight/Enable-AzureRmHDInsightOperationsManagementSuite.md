# Enable-AzureRmHDInsightOperationsManagementSuite

## КРАТКИй обзор
Включает набор Operations Management Suite (OMS) в кластере HDInsight, и необходимые журналы будут отправлены в рабочую область OMS, указанную во время включения.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Enable-AzureRmHDInsightOperationsManagementSuite [-Name] <String> [-WorkspaceId] <String>
 [-PrimaryKey] <String> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Enable-AzureRmHDInsightOperationsManagementSuite** включает набор Operations Management Suite (OMS) в кластере Azure HDInsight.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Enable-AzureRmHDInsightOMS -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

В кластере HDInsight будет включена поддержка Operations Management Suite (OMS), и необходимые журналы будут отправлены в рабочую область OMS с ИД 1d364e89-bb71-4503-aa3d-a23535aea7bd

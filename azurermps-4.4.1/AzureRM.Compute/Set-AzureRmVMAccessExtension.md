# Set-AzureRmVMAccessExtension

## КРАТКИй обзор
Добавляет расширение VMAccess на виртуальную машину.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Set-AzureRmVMAccessExtension [-UserName <String>] [-Password <String>] [-ResourceGroupName] <String>
 [-VMName] <String> [-Name <String>] [-TypeHandlerVersion <String>] [-Location <String>]
 [-DisableAutoUpgradeMinorVersion] [-ForceRerun <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzureRmVMAccessExtension** добавляет расширение виртуальной машины для доступа к виртуальной машине (VMAccess) к виртуальной машине.
VMAccess может сбросить имя пользователя и пароль для виртуальной машины.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Добавление расширения VMAccess
```
PS C:\> Set-AzureRmVMAccessExtension -ResourceGroupName "ResrouceGroup11" -Location "Central US" -VMName "VirtualMachine07" -Name "ContosoTest" -TypeHandlerVersion "2.0" -UserName "PFuller" -Password "Password"
```

Эта команда добавляет расширение VMAccess для виртуальной машины с именем VirtualMachine07 в ResrouceGroup11.
Команда задает версию обработчика имени и типа для VMAccess.

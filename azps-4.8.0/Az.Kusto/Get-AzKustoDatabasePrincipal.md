# Get-AzKustoDatabasePrincipal

## КРАТКИй обзор
Возвращает список участников базы данных для заданного кластера Kusto и базы данных.

## Максимальное

```
Get-AzKustoDatabasePrincipal -ClusterName <String> -DatabaseName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## NОПИСАНИЕ
Возвращает список участников базы данных для заданного кластера Kusto и базы данных.

## ИЛЛЮСТРИРУЮТ

### Пример 1: список участников базы данных, указанных в кластере и базе данных Kusto

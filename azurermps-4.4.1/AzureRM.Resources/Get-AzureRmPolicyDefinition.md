# Get-AzureRmPolicyDefinition

## КРАТКИй обзор
Получение определений политики.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### Установленный список всех параметров определений политики. По умолчанию
```
Get-AzureRmPolicyDefinition [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

### Заданный параметр имени определения политики.
```
Get-AzureRmPolicyDefinition -Name <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

### Заданный параметр идентификатора определения политики.
```
Get-AzureRmPolicyDefinition -Id <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmPolicyDefinition** получает все определения политик или определенную политику, определяемую именем или идентификатором.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение определения политики
```
PS C:\>Get-AzureRmPolicyDefinition
```

Эта команда получает все определения политики.

### Пример 2: получение определения политики по имени
```
PS C:\>Get-AzureRmPolicyDefinition -Name "VMPolicyDefinition"
```

Эта команда получает определение политики с именем VMPolicyDefinition.

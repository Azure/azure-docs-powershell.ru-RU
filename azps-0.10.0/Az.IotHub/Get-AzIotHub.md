# Get-AzIotHub

## КРАТКИй обзор
Получение сведений о IotHubs в подписке.

## Максимальное

### ListIotHubsByResourceGroup (по умолчанию)
```
Get-AzIotHub [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetIotHubByName
```
Get-AzIotHub [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Получение сведений о IotHubs в подписке.
Вы можете просмотреть все экземпляры IotHub в подписке или отфильтровать результаты по группе ресурсов или определенному имени IotHub.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Get-AzIotHub
```

Получает все IotHubs в подписке.

### Пример 2
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup"
```

Получает все IotHubs в подписке, относящиеся к группе ресурсов "myresourcegroup".

### Пример 3
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Получает сведения о IotHub с именем "myiothub".

# Set-AzSqlServerFirewallRule

## SYNOPSIS
Изменяет правило брандмауэра в Azure SQL database server.

## СИНТАКСИС

```
Set-AzSqlServerFirewallRule [-FirewallRuleName] <String> -StartIpAddress <String> -EndIpAddress <String>
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Set-AzSqlServerFirewallRule** изменяет правило брандмауэра на сервере базы данных Azure SQL.

## ПРИМЕРЫ

### Пример 1. Изменение правила брандмауэра
```
PS C:\>Set-AzSqlServerFirewallRule -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -FirewallRuleName "Rule01" -StartIpAddress "192.168.0.197" -EndIpAddress "192.168.0.199"
ResourceGroupName : ResourceGroup01
ServerName        : Server01
StartIpAddress    : 192.168.0.199
EndIpAddress      : 192.168.0.200
FirewallRuleName  : Rule01
```

Эта команда изменяет правило брандмауэра с именем Rule01 на сервере Server01.
Команда изменяет IP-адреса начала и окончания.

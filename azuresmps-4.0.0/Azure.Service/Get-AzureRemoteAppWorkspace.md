# Get-AzureRemoteAppWorkspace

## КРАТКИй обзор
Получение сведений о рабочей области Azure RemoteApp.

## Максимальное

```
Get-AzureRemoteAppWorkspace [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRemoteAppWorkspace** извлекает сведения о рабочей области Azure RemoteApp.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение сведений о рабочей области
```
PS C:\> Get-AzureRemoteAppWorkspace
ClientUrl                               EndUserFeedName
---------                               ---------------
https://www.remoteapp.windowsazure.com/ Contoso Work Applications
```

Эта команда извлекает сведения о рабочей области Azure RemoteApp.

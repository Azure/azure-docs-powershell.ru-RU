# Get-AzApplicationGatewayUrlPathMapConfig

## SYNOPSIS
Возвращает массив сопоставлений URL-путей с пулом серверов серверов.

## СИНТАКСИС

```
Get-AzApplicationGatewayUrlPathMapConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzApplicationGatewayURLPathMapConfig** возвращает массив сопоставлений URL-путей с пулом серверов.

## ПРИМЕРЫ

### Пример 1. Настройка карты URL-адреса
```
PS C:\>Get-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway Gateway
```

Эта команда получает конфигурации URL-пути из сервера серверов серверов приложений, расположенного на шлюзе приложения "Шлюз".

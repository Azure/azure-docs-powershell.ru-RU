# New-AzAnalysisServicesFirewallRule

## SYNOPSIS
Создание правила брандмауэра Служб Analysis Services

## СИНТАКСИС

```
New-AzAnalysisServicesFirewallRule [-FirewallRuleName] <String> [-RangeStart] <String> [-RangeEnd] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Система New-AzAnalysisServicesFirewallRule создает объект правила брандмауэра.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> New-AzAnalysisServicesFirewallRule -FirewallRuleName rule1 -RangeStart 0.0.0.0 -RangeEnd 255.255.255.255
```

Создает правило брандмауэра с именем "Правило1" с диапазоном начала 0.0.0.0 и конечным диапазоном 255.255.255.255.

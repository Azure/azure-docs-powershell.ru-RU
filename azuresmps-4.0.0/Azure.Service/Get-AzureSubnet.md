# Get-AzureSubnet

## КРАТКИй обзор
Получает список подсетей, связанных с указанной виртуальной машиной Azure.

## Максимальное

```
Get-AzureSubnet -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureSubnet** возвращает список подсетей, связанных с указанной виртуальной машиной.
Укажите виртуальную машину с помощью **Get-AzureVM** .

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение подсетей для виртуальной машины
```
PS C:\> $VM = Get-AzureVM -ServiceName "ContosoService03" -Name "VirtualMachine01"
C:\PS> Get-AzureSubnet -VM $VM
```

Первая команда получает виртуальную машину с именем VirtualMachine01 в службе с именем ContosoService03 и сохраняет ее в переменной $VM.

Вторая команда получает подсети Azure для виртуальной машины в $VM.

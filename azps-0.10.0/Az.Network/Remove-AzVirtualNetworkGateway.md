# Remove-AzVirtualNetworkGateway

## КРАТКИй обзор
Удаление шлюза виртуальной сети

## Максимальное

```
Remove-AzVirtualNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Шлюз виртуальной сети — это объект, представляющий ваш шлюз в Azure.

Командлет **Get-AzVirtualNetworkGateway** возвращает объект шлюза в Azure на основе имени и имени группы ресурсов.

## ИЛЛЮСТРИРУЮТ

### 1: Удаление шлюза виртуальной сети
```
Remove-AzVirtualNetworkGateway -Name myGateway -ResourceGroupName myRG
```

Удаляет объект шлюза виртуальной сети с именем "myGateway" в группе ресурсов "myRG"

Примечание. необходимо сначала удалить все подключения к шлюзу виртуальной сети с помощью командлета **Remove-AzVirtualNetworkGatewayConnection** .

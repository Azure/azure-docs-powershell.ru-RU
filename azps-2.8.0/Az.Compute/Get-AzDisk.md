# Get-AzDisk

## КРАТКИй обзор
Получает свойства управляемого диска.

## Максимальное

```
Get-AzDisk [[-ResourceGroupName] <String>] [[-DiskName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzDisk** получает свойства управляемого диска.

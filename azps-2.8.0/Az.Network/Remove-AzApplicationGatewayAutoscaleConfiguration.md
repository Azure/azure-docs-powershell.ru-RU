# Remove-AzApplicationGatewayAutoscaleConfiguration

## КРАТКИй обзор
Удаляет конфигурацию автомасштабирования из шлюза приложений.

## Максимальное

```
Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApplicationGatewayAutoscaleConfiguration** удаляет конфигурацию автомасштабирования из существующего шлюза приложений.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Первая команда получает шлюз приложения и сохраняет его в $gw переменной.
Вторая команда удаляет конфигурацию автомасштабирования из шлюза приложений.
Третья команда обновляет шлюз приложения в Azure.

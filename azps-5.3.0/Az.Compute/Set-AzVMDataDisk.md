# Set-AzVMDataDisk

## SYNOPSIS
Изменяет свойства диска данных виртуальной машины.

## СИНТАКСИС

### ChangeWithName
```
Set-AzVMDataDisk [-VM] <PSVirtualMachine> [-Name] <String> [[-Caching] <CachingTypes>]
 [[-DiskSizeInGB] <Int32>] [-StorageAccountType <String>] [-DiskEncryptionSetId <String>] [-WriteAccelerator]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ChangeWithLun
```
Set-AzVMDataDisk [-VM] <PSVirtualMachine> [-Lun] <Int32> [[-Caching] <CachingTypes>] [[-DiskSizeInGB] <Int32>]
 [-StorageAccountType <String>] [-DiskEncryptionSetId <String>] [-WriteAccelerator]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Set-AzVMDataDisk** изменяет свойства диска данных виртуальной машины.

## ПРИМЕРЫ

### Пример 1. Изменение режима кэшации диска данных
```powershell
PS C:\> $VM = Get-AzVM -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07"
PS C:\> Set-AzVMDataDisk -VM $VM -Name "DataDisk01" -Caching ReadWrite | Update-AzVM
```

Первая команда получает виртуальную машину с именем ContosoVM07 с помощью **Get-AzVM.**
Команда сохраняет ее в переменной $VM.
Вторая команда изменяет режим кэшации диска данных DataDisk01 на виртуальной машине в $VM.
Команда передает результат командлету Update-AzVM, который реализует изменения.
Изменение режима обналички приводит к перезапуску виртуальной машины.

# New-AzCosmosDBGremlinUniqueKeyPolicy

## SYNOPSIS
Создание нового объекта UniqueKeyPolicy вНося в Результате ДВ.

## СИНТАКСИС

```
New-AzCosmosDBGremlinUniqueKeyPolicy -UniqueKey <PSSqlUniqueKey[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания объекта psUniqueKeyPolicy с новым объектом psUniqueKeyPolicy создается новый объект — **AzCosmosDBGremlinUniqueKeyPolicy.**

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $uk = New-AzCosmosDBGremlinUniqueKey -Path "abc"

 New-AzCosmosDBGremlinUniqueKeyPolicy -UniqueKey $uk,$uk
```

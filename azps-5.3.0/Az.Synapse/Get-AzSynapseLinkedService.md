# Get-AzSynapseLinkedService

## SYNOPSIS
Получает сведения о связанных службах в рабочей области.

## СИНТАКСИС

### GetByName (по умолчанию)
```
Get-AzSynapseLinkedService -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseLinkedService -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Чтобы получить сведения о связанных службах в рабочей области, можно получить сведения о **get-AzSynapseLinkedService.**
Если указать имя связанной службы, этот cmdlet получит сведения о связанной службе.
Если имя не указано, этот cmdlet получает сведения обо всех связанных службах в рабочей области.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace
```

Эта команда получает сведения обо всех связанных службах в рабочей области ContosoWorkspace.

### Пример 2
```powershell
PS C:\> Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService
```

Эта команда получает сведения о связанной службе ContosoLinkedService в рабочей области с именем ContosoWorkspace.

### Пример 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseLinkedService -Name ContosoLinkedService
```

Эта команда получает сведения о связанной службе ContosoLinkedService в рабочей области ContosoWorkspace через канал.

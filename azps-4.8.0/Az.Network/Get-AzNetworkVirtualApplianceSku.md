# Get-AzNetworkVirtualApplianceSku

## КРАТКИй обзор
Получение или перечисление доступных конфигураций сетевого устройства виртуальных устройств на складе.

## Максимальное

```
Get-AzNetworkVirtualApplianceSku [-SkuName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Get-AzNetworkVirtualApplianceSku получает или перечисляет доступные SKU сетевых виртуальных устройств в инвентаризации Microsoft Azure.

# Get-AzSqlDatabase

## КРАТКИй обзор
Возвращает одну или несколько баз данных.

## Максимальное

```
Get-AzSqlDatabase [[-DatabaseName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlDatabase** получает одну или несколько баз данных SQL Azure с сервера базы данных Azure SQL.
Этот командлет также поддерживается службой SQL Server Stretch Database Service в Azure.

# Get-AzApiManagementLogger

## SYNOPSIS
Получает объекты регистратора управления API.

## СИНТАКСИС

### GetAllLoggers (по умолчанию)
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByLoggerId
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого с** его использованием можно получить журнал управления API Azure или все журналы. 

## ПРИМЕРЫ

### Пример 1. Получить всех регистраторов
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext
```

Эта команда возвращает всех регистраторов для указанного контекста.

### Пример 2. Получить определенный журнал
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123"
```

Эта команда удаляет регистратор с ИД 123.

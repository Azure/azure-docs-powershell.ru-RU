# Модуль Az.DataBox
## Описание
Модуль для создания, отмены, удаления и получения заданий работы с почтовым ящиком данных

## Az.DataBox Cmdlets
### [Get-AzDataBoxCredential](Get-AzDataBoxCredential.md)
Получает учетные данные для почтового ящика определенного задания.

### [Get-AzDataBoxJob](Get-AzDataBoxJob.md)
Сведения о заданиях работы с данными

### [New-AzDataBoxJob](New-AzDataBoxJob.md)
Создание задания для почтового ящика с использованием указанных параметров

### [Remove-AzDataBoxJob](Remove-AzDataBoxJob.md)
Удаляет задание для работы с почтовым ящиком

### [Stop-AzDataBoxJob](Stop-AzDataBoxJob.md)
Если задание отменено, задание отменяется.

# Restart-AzServiceFabricManagedNodeType

## КРАТКИй обзор
Перезагрузите определенные узлы из типа узла.

## Максимальное

```
Restart-AzServiceFabricManagedNodeType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 -NodeName <String[]> [-ForceRestart] [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Перезагрузите определенные узлы из типа узла. После повторного запуска виртуальных компьютеров они будут отключены и снова включены после обратного. Если это делается для основных типов узлов, может потребоваться некоторое время, так как может не перезапускаться сразу все узлы. Функция-ForceRestart принудительно выполняет операцию, даже если фабрика служб не может отключить узлы, но используется с осторожностью, так как это может привести к потере данных, если на узле выполняются нагрузки с отслеживанием состояния.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
$rgName = "testRG"
$clusterName = "testCluster"
$NodeTypeName = "nt1"
Restart-AzServiceFabricManagedNodeType -ResourceGroupName $rgName -ClusterName $clusterName  -Name $NodeTypeName -NodeName nt1_0, nt1_3
```

Перезапустите узлы 0 и 3 для типа узла.

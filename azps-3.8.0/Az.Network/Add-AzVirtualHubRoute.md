# Add-AzVirtualHubRoute

## КРАТКИй обзор
Создает объект VirtualHubRoute, который можно передать как параметр для команды Add-AzVirtualHubRouteTable. 

## Максимальное

```
Add-AzVirtualHubRoute -Destination <String[]> -DestinationType <String> -NextHop <String[]>
 -NextHopType <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Создает объект VirtualHubRoute.

## ИЛЛЮСТРИРУЮТ

Вышеприведенная команда создаст объект VirtualHubRoute, который затем можно добавить в ресурс VirtualHubRouteTable и задать в VirtualHub.

# Get-AzSqlDatabaseDataMaskingRule

## SYNOPSIS
Получает правила маскровки данных из базы данных.

## СИНТАКСИС

```
Get-AzSqlDatabaseDataMaskingRule [-SchemaName <String>] [-TableName <String>] [-ColumnName <String>]
 [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого** можно получить определенное правило или все правила маскровки данных для базы данных Azure SQL.
Чтобы использовать этот cmdlet, используйте *параметры ResourceGroupName,* *ServerName* и *DatabaseName,* чтобы идентифицировать базу данных, и параметр *RuleId,* чтобы указать, какое правило возвращает этот cmdlet.
Если не предоставить *RuleId,* возвращаются все правила маскровки данных для базы SQL Azure.
Этот cmdlet также поддерживается службой SQL Server Stretch Database в Azure.

# Модуль AzureRM. Maps
## Nописание
В этом разделе отображается справка по командлетам карт Azure.

## Командлеты AzureRM. Maps
### [Get-AzureRmMapsAccount](Get-AzureRmMapsAccount.md)
Возвращает учетную запись.

### [Get-AzureRmMapsAccountKey](Get-AzureRmMapsAccountKey.md)
Возвращает ключи API для учетной записи.
Эти ключи являются механизмом проверки подлинности, используемым при последующих звонках на карты Azure.

### [New-AzureRmMapsAccount](New-AzureRmMapsAccount.md)
Создание учетной записи карт Azure.

### [New-AzureRmMapsAccountKey](New-AzureRmMapsAccountKey.md)
Повторное создание ключа учетной записи.

### [Remove-AzureRmMapsAccount](Remove-AzureRmMapsAccount.md)
Удаление учетной записи карт Azure.

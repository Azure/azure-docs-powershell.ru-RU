# Set-AzureRmHDInsightClusterSize

## КРАТКИй обзор
Задает количество рабочих узлов в указанном кластере.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Set-AzureRmHDInsightClusterSize [-ClusterName] <String> [-TargetInstanceCount] <Int32>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzureRmHDInsightClusterSize** задает количество узлов рабочих процессов в указанном кластере Azure HDInsight.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Установка размера указанного кластера
```
PS C:\>Set-AzureRmHDInsightClusterSize -ClusterName "your-hadoop-001" -TargetInstanceCount 6
```

Эта команда задает размер кластера с именем-Hadoop-001.

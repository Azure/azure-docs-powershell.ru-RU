# Remove-AzStorageSyncService

## SYNOPSIS
Эта команда удалит указанную службу синхронизации хранилища.

## СИНТАКСИС

### StringParameterSet (по умолчанию)
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Эта команда удалит указанную службу синхронизации хранилища. Службу синхронизации хранилища можно удалить только в том случае, если сначала удаляются все содержащиеся в ней группы синхронизации и зарегистрированные серверы.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

Эта команда удалит службу синхронизации хранилища.

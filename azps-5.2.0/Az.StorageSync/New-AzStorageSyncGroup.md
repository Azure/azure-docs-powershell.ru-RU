# New-AzStorageSyncGroup

## SYNOPSIS
Эта команда создает новую группу синхронизации в указанной службе синхронизации хранилища.

## СИНТАКСИС

### StringParameterSet (по умолчанию)
```
New-AzStorageSyncGroup [-ResourceGroupName] <String> [-StorageSyncServiceName] <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectParameterSet
```
New-AzStorageSyncGroup [-ParentObject] <PSStorageSyncService> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ParentStringParameterSet
```
New-AzStorageSyncGroup [-ParentResourceId] <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Эта команда создает новую группу синхронизации в указанной службе синхронизации хранилища. Группа синхронизации используется для описания топологии местоположений, которые называются конечными точками, и синхронизируются все файлы, хранимые в одной из конечных точек. Группа синхронизации содержит облачные конечные точки, ссылаясь на файловые папки Azure, а также конечные точки сервера, которые ссылаются на определенный локальный путь на зарегистрированном сервере.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> New-AzStorageSyncGroup -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -Name "mySyncGroupName"
```

Эта команда создает новую группу синхронизации в указанной службе синхронизации хранилища.

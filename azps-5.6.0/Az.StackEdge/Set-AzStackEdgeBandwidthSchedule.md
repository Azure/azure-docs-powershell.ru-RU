## ОПИСАНИЕ
**Cmdlet Set-AzStackEdgeBandwidthSchedule** обновляет расписание пропускной способности для устройства Stack Edge.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Set-AzStackEdgeBandwidthSchedule  -ResourceGroupName resourceGroupName -DeviceName deviceName -Name bandwidthSchedule -UnlimitedBandwidth

----                ----------                    ---------- --------- --------
bandwidthSchedule  Sunday, Tuesday, Saturday      Unlimited  11:00:00  12:00:00
```

### Пример 2
```powershell

----                ----------                    ---------- --------- --------
bandwidthSchedule  Sunday, Tuesday, Saturday      Unlimited  11:00:00  21:00:00
```

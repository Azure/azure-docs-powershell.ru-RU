# Resume-AzAutomationJob

## КРАТКИй обзор
Возобновление приостановленного задания автоматизации.

## Максимальное

```
Resume-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Resume-AzAutomationJob** возобновляет приостановленное задание службы автоматизации Azure.
Укажите приостановленное задание.
Чтобы приостановить задание, используйте командлет Suspend-AzAutomationJob.

## ИЛЛЮСТРИРУЮТ

### Пример 1: возобновление приостановленного задания
```
PS C:\>Resume-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Эта команда возобновляет задание с указанным ИДЕНТИФИКАТОРом.

# Get-AzureVirtualNetworkGatewayConnection

## КРАТКИй обзор
Возвращает подключение к шлюзу виртуальной сети.

## Максимальное

```
Get-AzureVirtualNetworkGatewayConnection [-GatewayId <String>] [-ConnectedEntityId <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureVirtualNetworkGatewayConnection** получает подключение к шлюзу виртуальной сети Azure.

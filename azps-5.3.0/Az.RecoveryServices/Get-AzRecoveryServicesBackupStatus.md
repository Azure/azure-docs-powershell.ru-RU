# Get-AzRecoveryServicesBackupStatus

## SYNOPSIS
Проверяет, имеется ли ARM ресурса.

## СИНТАКСИС

### Имя (по умолчанию)
```
Get-AzRecoveryServicesBackupStatus -Name <String> -ResourceGroupName <String> -Type <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdWorkload
```
Get-AzRecoveryServicesBackupStatus -Type <String> -ResourceId <String> -ProtectableObjectName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ИД
```
Get-AzRecoveryServicesBackupStatus -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Если указанный ресурс не защищен ни в каких хранилищах служб восстановления в подписке, возвращается пустое или пустое NULL. Если она защищена, возвращаются сведения о соответствующем сейфе.

## ПРИМЕРЫ

### Пример 1

Проверяет, имеется ли ARM ресурса. (autogenerated)


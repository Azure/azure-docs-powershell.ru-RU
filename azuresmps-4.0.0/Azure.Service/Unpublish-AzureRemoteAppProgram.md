# Unpublish-AzureRemoteAppProgram

## КРАТКИй обзор
Отменяет публикацию удаленного приложения RemoteApp для Azure.

## Максимальное

```
Unpublish-AzureRemoteAppProgram [-CollectionName] <String> [[-Alias] <String[]>] [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **UNPUBLISH-AzureRemoteAppProgram** отменяет публикацию программы RemoteApp Azure.
После отмены публикации программы она станет недоступна для пользователей коллекции Azure RemoteApp.

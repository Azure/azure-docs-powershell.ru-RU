# Remove-AzApplicationGatewayFrontendIPConfig

## SYNOPSIS
Удаляет конфигурацию IP-адреса переднего ip-адреса из шлюза приложения.

## СИНТАКСИС

```
Remove-AzApplicationGatewayFrontendIPConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для удаления IP-адреса из шлюза приложения Azure удаляется ip-адрес переднего входа в **AzApplicationGatewayFrontendIPConfig.**

## ПРИМЕРЫ

### Пример 1. Удаление конфигурации IP-адреса переднего
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendIPConfig -ApplicationGateway $AppGw -Name "FrontEndIP02"
```

Первая команда получает шлюз приложения ApplicationGateway01 и сохраняет его в переменной $AppGw.
Вторая команда удаляет ip-конфигурацию front-end с именем FrontEndIP02 из шлюза приложения, $AppGw.

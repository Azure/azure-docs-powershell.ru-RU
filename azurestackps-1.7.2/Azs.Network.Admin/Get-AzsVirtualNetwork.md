# Get-AzsVirtualNetwork

## КРАТКИй обзор
Получение списка всех виртуальных сетей.

## Максимальное

```
Get-AzsVirtualNetwork [[-Filter] <String>] [[-OrderBy] <String>] [[-Skip] <Int32>] [[-Top] <Int32>]
 [[-InlineCount] <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Получение списка всех виртуальных сетей.

# Save-AzureWebsiteLog

## КРАТКИй обзор
Загружает и сохраняет журналы для указанного веб-сайта.

## Максимальное

```
Save-AzureWebsiteLog [-Output <String>] [-PassThru] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
В этом разделе описан командлет в версии 0.8.10 модуля Microsoft Azure PowerShell.
Чтобы получить версию модуля, который вы используете, введите в командной консоли Azure PowerShell `(Get-Module -Name Azure).Version` .

Командлет **Save-AzureWebsiteLog** загружает журналы для указанного веб-сайта.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Загрузка и сохранение журналов для веб-сайта
```
PS C:\> Save-AzureWebsiteLogs -Name mySite -Output .\logs.zip
```

В этом примере выполняется загрузка журналов среды выполнения и развертывания для личного сайта в файл logs.zip в текущем каталоге.

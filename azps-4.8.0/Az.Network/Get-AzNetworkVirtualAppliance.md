# Get-AzNetworkVirtualAppliance

## КРАТКИй обзор
Получить или Просмотреть виртуальные сетевые устройства.

## Максимальное

### ResourceNameParameterSet (по умолчанию)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Команды Get-AzNetworkVirtualAppliance выводят на экран сетевые устройства виртуальных устройств.

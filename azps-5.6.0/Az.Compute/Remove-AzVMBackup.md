# Remove-AzVMBackup

## SYNOPSIS
Удаляет резервную копию с виртуальной машины.

## СИНТАКСИС

```
Remove-AzVMBackup [-ResourceGroupName] <String> [-VMName] <String> [-Tag] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

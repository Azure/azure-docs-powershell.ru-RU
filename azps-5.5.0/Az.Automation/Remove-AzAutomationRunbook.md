# Remove-AzAutomationRunbook

## SYNOPSIS
Удаляет книгу.

## СИНТАКСИС

```
Remove-AzAutomationRunbook [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для удаления книги из автоматизации Azure удаляется ее **cmdlet Remove-AzAutomationRunbook.**

## ПРИМЕРЫ

### Пример 1. Удаление книги
```
PS C:\>Remove-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook03" -ResourceGroupName "ResourceGroup01"
```

Эта команда удаляет книгу Runbook03 в учетной записи автоматизации Azure с именем Contoso17.

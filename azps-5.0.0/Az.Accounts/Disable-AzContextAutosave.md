# Disable-AzContextAutosave

## КРАТКИй обзор
Отключите Автосохранение учетных данных Azure.  Ваши регистрационные данные будут утрачены при следующем открытии окна PowerShell.

## Максимальное

```
Disable-AzContextAutosave [-Scope <ContextModificationScope>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Отключите Автосохранение учетных данных Azure.  Ваши регистрационные данные будут утрачены при следующем открытии окна PowerShell.

## ИЛЛЮСТРИРУЮТ

### Пример 1: отключение автосохранения контекста
```powershell
PS C:\> Disable-AzContextAutosave
```

Отключите Автосохранение для текущего пользователя.

### Пример 2

Отключите Автосохранение учетных данных Azure в этом сеансе PowerShell. (автоматически сформированные)

```powershell <!-- Aladdin Generated Example --> 
Disable-AzContextAutosave -Scope Process
```

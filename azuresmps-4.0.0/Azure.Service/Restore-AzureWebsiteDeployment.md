# Restore-AzureWebsiteDeployment

## КРАТКИй обзор
Повторное развертывание предыдущего развертывания веб-сайта в Azure.

## Максимальное

```
Restore-AzureWebsiteDeployment [-CommitId <String>] [-Force] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
В этом разделе описан командлет в версии 0.8.10 модуля Microsoft Azure PowerShell.
Чтобы получить версию модуля, который вы используете, введите в командной консоли Azure PowerShell `(Get-Module -Name Azure).Version` .

Командлет **redeploy-AzureWebsiteDeployment** повторно развертывает предыдущее развертывание веб-сайта в Azure.
Этот процесс заменяет текущее развертывание выбранным развертыванием.

## ИЛЛЮСТРИРУЮТ

### Пример 1: повторное развертывание сайта
```
PS C:\> Restore-AzureWebsiteDeployment -Name "ContosoSite" -CommitId "f876543210"
```

Эта команда повторно разворачивает развертывание с ИД f876543210 для веб-сайта с именем ContosoSite.

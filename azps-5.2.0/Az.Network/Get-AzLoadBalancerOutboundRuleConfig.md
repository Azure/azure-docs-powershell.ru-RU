# Get-AzLoadBalancerOutboundRuleConfig

## SYNOPSIS
Получает конфигурацию исходящие правила в балансире нагрузки.

## СИНТАКСИС

```
Get-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого можно** получить конфигурацию правил исходяния или список конфигураций правил исходящие в балансе нагрузки.

## ПРИМЕРЫ

### Пример 1. Настройка правил исходящие в балансе нагрузки
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Get-AzLoadBalancerOutboundRuleConfig -LoadBalancer $slb -Name "MyRule"
```

Первая команда получает балансировку нагрузки MyLoadBalancer, а затем сохраняет его в переменной $slb.
Вторая команда получает конфигурацию исходящие правила MyRule, связанную с этим балансировом нагрузки.

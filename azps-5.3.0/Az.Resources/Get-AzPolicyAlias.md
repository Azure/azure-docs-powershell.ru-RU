# Get-AzPolicyAlias

## SYNOPSIS
Get-AzPolicyAlias извлекает и выводит типы ресурсов поставщика Azure, которые определяются псевдонимами и соответствуют заданным значениям параметров. Если параметры не заданы, будут выводться все типы ресурсов поставщика, содержащие псевдоним.
Переключатель -ListAvailable изменяет это поведение, перечисляя все совпадающие типы ресурсов, включая те, у которых нет псевдонимов.

## СИНТАКСИС

```
Get-AzPolicyAlias [-NamespaceMatch <String>] [-ResourceTypeMatch <String>] [-AliasMatch <String>]
 [-PathMatch <String>] [-ApiVersionMatch <String>] [-LocationMatch <String>] [-ListAvailable]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **получения псевдонимов** политик возвращается список псевдонимов политик.
Псевдонимы политик используются политикой Azure для ссылки на свойства типа ресурса.
Предоставляются параметры, ограничивающие элементы в списке путем совпадения различных свойств типа ресурса или его псевдонимов.
Заданное значение совпадает, если целевая строка содержит его с учетом значения дела.

## ПРИМЕЧАНИЯ

* Чтобы развернуть псевдонимы или любое другое свойство, перетастите выходной канал в нужное `select -ExpandProperty <property>` время. Например: `Get-AzPolicyAlias -NamespaceMatch 'Microsoft.Cache' -ApiVersionMatch 'alpha' | select -ExpandProperty Aliases | select -Property Name -ExpandProperty Paths`

* Дополнительные свойства доступны в выходных данных и могут отображаться путем перенакаки выходных `Format-List` данных. Например: `Get-AzPolicyAlias -NamespaceMatch 'Web' -ResourceTypeMatch site -PathMatch cert | Format-List`

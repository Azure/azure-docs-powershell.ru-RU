# Get-AzureSubnetRouteTable

## КРАТКИй обзор
Возвращает таблицу маршрутов для подсети.

## Максимальное

```
Get-AzureSubnetRouteTable -VirtualNetworkName <String> -SubnetName <String> [-Detailed]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureSubnetRouteTable** получает таблицу маршрутов, связанную с подсетью.

## ИЛЛЮСТРИРУЮТ

### Пример 1: отображение маршрутов для подсети
```
PS C:\> Get-AzureSubnetRouteTable -VirtualNetworkName "VNetUSCentral" -SubnetName "ContosoSubnet" -Detailed
Routes                        Name                          Location                      Label
------                        ----                          --------                      -----
{internetroute}               PublicRT                      Central US                    Sample RT in Central US
```

Эта команда отображает маршруты в имени таблицы маршрутов, связанной с подсетью с именем ContosoSubnet.

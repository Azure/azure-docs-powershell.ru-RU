# Remove-AzBatchApplication

## SYNOPSIS
Удаляет приложение из учетной записи пакета.

## СИНТАКСИС

```
Remove-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его использованием** можно удалить приложение из учетной записи Azure Batch.

## ПРИМЕРЫ

### Пример 1. Удаление приложения из учетной записи пакета
```
PS C:\>Remove-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "Litware"
```

Эта команда удаляет приложение Litware из учетной записи ContosoBatch.
Команда не справилась, если приложение содержит какие-либо пакеты.

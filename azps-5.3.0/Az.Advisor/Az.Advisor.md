# Модуль Az.Advisor
## Описание
Команды для взаимодействия с помощником Azure.

## Az.Advisor Cmdlets
### [Get-AzAdvisorRecommendation](Get-AzAdvisorRecommendation.md)
Получает список рекомендаций консультанта Azure.

### [Disable-AzAdvisorRecommendation](Disable-AzAdvisorRecommendation.md)
Отключать рекомендации консультанта Azure.

### [Enable-AzAdvisorRecommendation](Enable-AzAdvisorRecommendation.md)
Включает рекомендации консультанта Azure.

### [Get-AzAdvisorConfiguration](Get-AzAdvisorConfiguration.md)
Получите конфигурации помощника Azure для данной подписки или группы ресурсов.

### [Set-AzAdvisorConfiguration](Set-AzAdvisorConfiguration.md)
Обновляет или создает конфигурацию помощника Azure.

## NОПИСАНИЕ
Командлет **New-AzureServiceADDomainExtensionConfig** создает конфигурацию для расширения домена Active Directory (AD) для облачных служб.

## ИЛЛЮСТРИРУЮТ

### Пример 1: указание конфигурации домена доменных служб
```
PS C:\> $ExtensionCfg = New-AzureServiceADDomainExtensionConfig -Role WorkerRole1 -DomainName $Domain -Credential $Cred -JoinOption 35;

PS C:\> New-AzureDeployment -ServiceName $CloudSvc -Slot "Production" -Package $Pkg -Configuration $Config -ExtensionConfiguration $ExtensionCfg;
```

Эта команда создает конфигурацию для расширения доменной рекламы.

# New-AzApiManagementHttpMessageDiagnostic

## SYNOPSIS
Создание экземпляра **PsApiManagementHttpMessageDiagnostic,** который является параметром диагностики http-сообщения.

## СИНТАКСИС

```
New-AzApiManagementHttpMessageDiagnostic [-HeadersToLog <String[]>] [-BodyBytesToLog <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для этого создается параметр диагностики **http-сообщения.**

## ПРИМЕРЫ

### Пример 1. Создание базового параметра диагностики http-сообщения
```powershell
PS C:\>  New-AzApiManagementHttpMessageDiagnostic -Headers 'Content-Type', 'UserAgent' -BodyBytes 100

Headers                   Body
-------                   ----
{Content-Type, UserAgent} Microsoft.Azure.Commands.ApiManagement.ServiceManagement.Models.PsApiManagementBodyDiagnosticSetting
```

Создание параметра диагностики http-сообщения для журнала и создания его заглавных полей наряду с `Content-Type` `User-Agent` 100байтами `body`

# Clear-AzSqlServerAdvancedThreatProtectionSetting

## КРАТКИй обзор
Удаление дополнительных параметров защиты от угроз с сервера.

## Максимальное

```
Clear-AzSqlServerAdvancedThreatProtectionSetting [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Clear-AzSqlServerAdvancedThreatProtectionSetting удаляет расширенные параметры защиты от угроз с сервера Azure SQL Server.
Чтобы использовать этот командлет, укажите параметры ResourceGroupName и ServerName для идентификации сервера, с которого этот командлет удаляет параметры.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление расширенных параметров защиты от угроз для базы данных
```
PS C:\> Clear-AzSqlServerAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Эта команда удаляет расширенные параметры защиты от угроз с сервера с именем Server01.

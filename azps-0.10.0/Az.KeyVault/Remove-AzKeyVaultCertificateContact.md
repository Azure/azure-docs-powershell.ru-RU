# Remove-AzKeyVaultCertificateContact

## КРАТКИй обзор
Удаляет контакт, зарегистрированный для уведомлений о сертификате из хранилища ключей.

## Максимальное

```
Remove-AzKeyVaultCertificateContact [-VaultName] <String> [-EmailAddress] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzKeyVaultCertificateContact** удаляет контакт, зарегистрированный для уведомлений о сертификате из хранилища ключей.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление контакта сертификата
```
PS C:\>Remove-AzKeyVaultCertificateContact -VaultName "Contoso01" -EmailAddress "patti.fuller@contoso.com"
```

Эта команда удаляет Patti Белова в качестве контакта сертификата для хранилища ключей Contoso01.

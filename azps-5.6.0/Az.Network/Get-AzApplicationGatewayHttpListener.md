# Get-AzApplicationGatewayHttpListener

## SYNOPSIS
Получение программы HTTP-прослушивания шлюза приложения.

## СИНТАКСИС

```
Get-AzApplicationGatewayHttpListener [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzApplicationGatewayHttpListener** можно получить http-прослушивание шлюза приложения.

## ПРИМЕРЫ

### Пример 1. Найдите конкретного http-прослушивание
```
PS C:\>$Appgw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listener = Get-AzApplicationGatewayHttpListener -Name "Listener01" -ApplicationGateway $Appgw
```

Эта команда получает http-прослушивание с именем Listener01.

### Пример 2. Получить список http-загона
```
PS C:\>$Appgw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listeners = Get-AzApplicationGatewayHttpListener -ApplicationGateway $Appgw
```

Эта команда возвращает список http-версий.

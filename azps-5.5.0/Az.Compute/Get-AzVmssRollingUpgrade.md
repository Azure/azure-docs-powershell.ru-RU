# Get-AzVmssRollingUpgrade

## SYNOPSIS
Показывает состояние последнего обновления в масштабе виртуальной машины.

## СИНТАКСИС

```
Get-AzVmssRollingUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Показывает состояние последнего обновления виртуальной машины.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Get-AzVmssRollingUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Эта команда показывает состояние последнего ветвного обновления VMSS под названием VMSS001, которое принадлежит группе ресурсов Group001.

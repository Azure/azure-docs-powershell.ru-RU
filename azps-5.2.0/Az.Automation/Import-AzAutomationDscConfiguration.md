# Import-AzAutomationDscConfiguration

## SYNOPSIS
Импорт конфигурации DSC в автоматизацию.

## СИНТАКСИС

```
Import-AzAutomationDscConfiguration -SourcePath <String> [-Tags <IDictionary>] [-Description <String>]
 [-Published] [-Force] [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Import-AzAutomationDscConfiguration** импортирует конфигурацию APS Desired State Configuration (DSC) в автоматизацию Azure.
Укажите путь к сценарию APS, который содержит одну конфигурацию DSC.

## ПРИМЕРЫ

### Пример 1. Импорт конфигурации DSC в автоматизацию
```powershell
PS C:\>Import-AzAutomationDscConfiguration -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -SourcePath "C:\DSC\client.ps1" -Force
```

Эта команда импортирует конфигурацию DSC в файле с именем client.ps1 в учетную запись автоматизации с именем Contoso17. Команда определяет параметр *Force.* При существующей конфигурации DSC эта команда заменяет ее.

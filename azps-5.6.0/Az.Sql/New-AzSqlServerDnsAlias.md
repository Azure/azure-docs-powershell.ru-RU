# New-AzSqlServerDnsAlias

## SYNOPSIS
Эта команда создает новый псевдоним DNS для Azure SQL Server Azure.

## СИНТАКСИС

```
New-AzSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Создает новый псевдоним DNS для Azure SQL Server, который указывает на указанный сервер.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> $serverDNSAlias = New-AzSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Эта команда создает псевдоним DNS Azure SQL Server с именем, которое назовет имя сервера

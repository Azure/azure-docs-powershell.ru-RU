# Get-AzCdnEndpointNameAvailability

## SYNOPSIS
Состояние доступности конечной точки CDN.

## СИНТАКСИС

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения состояния доступности конечной точки сети доставки содержимого (CDN) Azure возвращается состояние доступности для **cmdnEndpointNameAvailability.**

# Remove-AzureVMImageOSDiskConfig

## КРАТКИй обзор
Удаляет конфигурацию диска операционной системы из объекта DiskConfigSet.

## Максимальное

```
Remove-AzureVMImageOSDiskConfig [-DiskConfig] <VirtualMachineImageDiskConfigSet>
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureVMImageOSDiskConfig** удаляет конфигурацию диска операционной системы из объекта DiskConfigSet.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление конфигурации диска операционной системы из DiskConfigSet
```
PS C:\> $Disk = New-AzureDiskConfigSet
PS C:\> $Disk = Set-AzureOSDiskConfig -DiskConfig $Disk -HostCaching ReadWrite
PS C:\> Remove-AzureVMImageOSDiskConfig -DiskConfig $Disk
```

Эта команда удаляет конфигурацию диска операционной системы из объекта DiskConfigSet

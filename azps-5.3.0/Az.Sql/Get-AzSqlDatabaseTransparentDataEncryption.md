# Get-AzSqlDatabaseTransparentDataEncryption

## SYNOPSIS
Возвращает состояние TDE для базы данных.

## СИНТАКСИС

```
Get-AzSqlDatabaseTransparentDataEncryption [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для базы данных Azure SQL состояние шифрования данных (TDE) используется для cmdlet **Get-AzSqlDataBaseTransparentDataEncryption.**
Дополнительные сведения см. в теме "Прозрачное шифрование данных с помощью SQL Azure "База данных" https://msdn.microsoft.com/library/dn948096 https://msdn.microsoft.com/library/dn948096) (Microsoft Developer Network библиотеки).
Этот cmdlet получает текущее состояние TDE, но шифрование и расшифровка могут быть длительными.
Чтобы увидеть ход проверки шифрования, запустите Get-AzSqlDatabaseTransparentDataEncryptionActivity выполнения.
Этот cmdlet также поддерживается службой SQL Server Stretch Database в Azure.

## ПРИМЕРЫ

### Пример 1. Получить состояние TDE для базы данных
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryption -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName             ServerName                    DatabaseName                                          State
-----------------             ----------                    ------------                                          -----
resourcegroup01               server01                      database01                                            Disabled
```

Эта команда получает состояние TDE для базы данных Database01 на сервере с именем server01.

# Remove-AzureReservedIP

## КРАТКИй обзор
Удаляет зарезервированный IP-адрес по его имени.

## Максимальное

```
Remove-AzureReservedIP [-ReservedIPName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureReservedIP** удаляет зарезервированный IP-адрес по его имени.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление зарезервированного IP-адреса с помощью его имени
```
PS C:\> Remove-AzureReservedIP -ReservedIPName $name
```

Эта команда удаляет зарезервированный IP-адрес по его имени.

# Get-AzRedisCachePatchSchedule

## SYNOPSIS
Получает расписание исправлений.

## СИНТАКСИС

```
Get-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Get-AzRedisCachePatchSchedule** получает расписание исправлений для кэша в кэше Azure Redis.

## ПРИМЕРЫ

### Пример 1. Получить расписание исправлений
```
PS C:\>Get-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Эта команда получает расписание исправлений из кэша RedisCache06.

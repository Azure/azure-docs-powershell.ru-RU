# Update-AzAlertState

## КРАТКИй обзор
Обновление состояния оповещения

## Максимальное

### ByAlertId (по умолчанию)
```
Update-AzAlertState -AlertId <String> -State <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Update-AzAlertState -State <String> -InputObject <PSAlert> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
**Update-AzAlertState** . состояние предупреждения обновления командлетов.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Update-AzAlertState -AlertId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" -State "Closed"
```

Этот командлет обновляет состояние предупреждения на Closed.

# Remove-AzAutomationHybridWorkerGroup

## КРАТКИй обзор
Удаляет группу гибридного рабочего процесса из автоматизации.

## Максимальное

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzAutomationHybridWorkerGroup удаляет группу гибридного рабочего процесса из автоматизации.

## ИЛЛЮСТРИРУЮТ

### Пример 1
Эта команда удаляет гибридный рабочий процесс по имени.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```

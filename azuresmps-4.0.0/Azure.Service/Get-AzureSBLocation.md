# Get-AzureSBLocation

## КРАТКИй обзор
Возвращает расположение служебной шины.

## Максимальное

```
Get-AzureSBLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
В этом разделе описан командлет в версии 0.8.10 модуля Microsoft Azure PowerShell.
Чтобы получить версию модуля, который вы используете, введите в командной консоли Azure PowerShell `(Get-Module -Name Azure).Version` .

Командлет **Get-AzureSBLocation** возвращает расположение, из которого доступна служебная шина.

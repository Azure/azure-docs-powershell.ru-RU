# New-AzureRmSqlDatabaseRestorePoint

## КРАТКИй обзор
Создает новую точку восстановления, из которой можно восстановить базу данных SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmSqlDatabaseRestorePoint -RestorePointLabel <String> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmSqlDatabaseRestorePoint** создает новую точку восстановления, из которой может быть восстановлено хранилище данных Azure SQL.
Этот командлет в настоящее время поддерживается для хранилища данных Azure SQL.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание точки восстановления
```
PS C:\>New-AzureRmSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -RestorePointLabel "RestorePoint01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : DISCRETE
RestorePointCreationDate : 8/12/2015 12:00:00 AM
EarliestRestoreDate      : 
RestorePointLabel        : RestorePoint01
```

Эта команда создает точку восстановления для хранилища данных Azure SQL и возвращает сведения о точке восстановления.

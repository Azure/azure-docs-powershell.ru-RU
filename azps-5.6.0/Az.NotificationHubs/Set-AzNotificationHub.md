# Set-AzNotificationHub

## SYNOPSIS
Задает значения свойств для концентратора уведомлений.

## СИНТАКСИС

### InputFileParameterSet
```
Set-AzNotificationHub [-ResourceGroup] <String> [-Namespace] <String> [-InputFile] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### NotificationHubParameterSet
```
Set-AzNotificationHub [-ResourceGroup] <String> [-Namespace] <String>
 [-NotificationHubObj] <NotificationHubAttributes> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Set-AzNotificationHub** изменяет значения свойств концентратора уведомлений.
Значение свойства концентратора уведомлений можно изменить двумя способами.
Например, вы можете создать экземпляр объекта **NotificationHubAttributes,** а затем настроить его со значениями свойств, которые вы хотите использовать для нового концентратора.
Это можно сделать с помощью платформа .NET Framework.
Затем вы можете скопировать значения этих свойств в свой центр с помощью *параметра NotificationHubObj.*
Кроме того, можно создать файл нотации объектов JSON (JavaScript), содержащий значения конфигурации, и применить их с помощью параметра *InputFile.*
JSON-файл — это текстовый файл с синтаксисом, аналогичным следующему: {  
    "Name": "ContosoNotificationHub",  
    "Location": "West US",  
} При использовании в сочетании с **cmdlet Set-AzNotificationHub** в предыдущем примере JSON задается значение расположения концентратора уведомлений ContosoNotificationHub на запад США.

## ПРИМЕРЫ

### Пример 1. Изменение значений свойств для концентратора уведомлений
```powershell
PS C:\>Set-AzNotificationHub -Namespace "ContosoNamespace" -ResourceGroup "ContosoNotificationsGroup" -InputFile "C:\Configuration\Hubs.json"
```

Эта команда изменяет значения свойств концентратора уведомлений, найденного в области имен ContosoNamespace, и назначена группе ресурсов ContosoNotificationsGroup.
Значения свойств, а также имя центра, который нужно изменить, не указаны в команде.
Эти сведения будут содержаться во входных файлах, C:\Configuration\Hubs.jsних.

### Пример 2

Задает значения свойств для концентратора уведомлений. (autogenerated)

```powershell
Set-AzNotificationHub -Namespace 'ContosoNamespace' -NotificationHubObj <NotificationHubAttributes> -ResourceGroup 'ContosoNotificationsGroup'
```

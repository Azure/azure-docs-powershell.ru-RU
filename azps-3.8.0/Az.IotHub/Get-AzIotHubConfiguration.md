# Get-AzIotHubConfiguration

## КРАТКИй обзор
В этой статье перечислены все или некоторые параметры автоматической настройки управления устройствами в IoT.

## Максимальное

### Источник данных (по умолчанию)
```
Get-AzIotHubConfiguration [-ResourceGroupName] <String> [-IotHubName] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObjectSet
```
Get-AzIotHubConfiguration [-InputObject] <PSIotHub> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubConfiguration [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Узнайте о том, как настроить автоматическое управление устройствами в Интернет-центре Интернета вещей и включить в него автоматические конфигурации управления устройствами.
https://docs.microsoft.com/azure/iot-hub/iot-hub-automatic-device-managementДля получения дополнительных сведений ознакомьтесь с дополнительными сведениями.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "config1"
```

Получение подробных сведений о настройке автоматического управления устройствами IoT.

### Пример 2
```powershell
PS C:\> Get-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub"
```

Список автоматических конфигураций управления устройствами в центре Интернета вещей.

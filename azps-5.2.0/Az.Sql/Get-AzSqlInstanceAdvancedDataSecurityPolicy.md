# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## SYNOPSIS
Получает политику advanced Data Security для управляемого экземпляра.

## СИНТАКСИС

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzSqlInstanceAdvancedDataSecurityPolicy** извлекает политику Advanced Data Security для управляемого экземпляра.

## ПРИМЕРЫ

### Пример 1. Получает управляемый экземпляр Advanced Data Security
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Пример 2. Получает управляемый экземпляр Advanced Data Security от ресурса управляемого экземпляра
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

# Stop-AzureRmDeployment

## КРАТКИй обзор
Cancal запущенное развертывание

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

```

 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```


## NОПИСАНИЕ
Командлет **Stop-AzureRmDeployment** отменяет развертывание в области подписки, которая была запущена, но не завершена.
Чтобы остановить развертывание, необходимо, чтобы развертывание имело неполное состояние подготовки, такое как подготовка, а не завершенное состояние, например подготовка или сбой.

Чтобы создать развертывание в области подписки, используйте командлет New-AzureRmDeployment.

Этот командлет останавливает только одно запущенное развертывание. Используйте параметр *Name* для остановки определенного развертывания.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\>Stop-AzureRmDeployment -Name "deployment01"
```

Эта команда отменяет запущенное развертывание "deployment01" в текущей области подписки.

### Пример 2
```
PS C:\>Get-AzureRmDeployment -Name "deployment01" | Stop-AzureRmDeployment
```

Эта команда получает развертывание "deployment01" в текущей области подписки и отменяет ее.

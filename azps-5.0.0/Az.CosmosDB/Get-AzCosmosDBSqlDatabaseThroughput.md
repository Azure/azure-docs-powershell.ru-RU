# Get-AzCosmosDBSqlDatabaseThroughput

## КРАТКИй обзор
Получает параметры пропускной способности, соответствующие базе данных SQL CosmosDB.

## Максимальное

### ByNameParameterSet (по умолчанию)
```
Get-AzCosmosDBSqlDatabaseThroughput -ResourceGroupName <String> -AccountName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBSqlDatabaseThroughput -Name <String> -InputObject <PSDatabaseAccountGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzCosmosDBSqlDatabaseThroughput** получает параметры пропускной способности, соответствующие базе данных CosmosDB SQL.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzCosmosDBSqlDatabaseThroughput -AccountName {accountName} -ResourceGroupName {resourceGroupName} -Name {databaseName}

```

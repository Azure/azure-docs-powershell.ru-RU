# Get-AzureRmSqlElasticPoolDatabase

## КРАТКИй обзор
Возвращает эластичные базы данных в пуле эластичных БД и их значения свойств.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSqlElasticPoolDatabase** получает эластичные базы данных в пуле эластичных БД и значения их свойств.
В базе данных SQL Azure можно указать имя эластичной базы данных, чтобы просмотреть значения свойств только для этой базы данных.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение всех баз данных в пуле эластичных БД
```
PS C:\>Get-AzureRmSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Эта команда получает все базы данных в пуле эластичных БД с именем ElasticPool01.

# New-AzSqlSyncAgentKey

## КРАТКИй обзор
Создание ключа агента синхронизации SQL Azure.

## Максимальное

```
New-AzSqlSyncAgentKey [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzSqlSyncAgentKey** создает ключ агента синхронизации Azure SQL.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание ключа агента синхронизации для агента синхронизации Azure SQL.
```
PS C:\> New-AzSqlSyncAgentKey -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01"
SyncAgentKey                  : Key
```

Эта команда создает ключ агента синхронизации для агента Azure SQL Sync.

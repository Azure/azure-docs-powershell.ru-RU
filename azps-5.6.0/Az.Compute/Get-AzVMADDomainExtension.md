# Get-AzVMADDomainExtension

## SYNOPSIS
Сведения о расширении домена AD.

## СИНТАКСИС

```
Get-AzVMADDomainExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzVMADDomainExtension** получает сведения об указанном расширении домена Azure Active Directory (AD).

## ПРИМЕРЫ

### Пример 1

Сведения о расширении домена AD. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Get-AzVMADDomainExtension -Name 'AgentPool01' -ResourceGroupName myresourcegroup -VMName 'VM01'
```

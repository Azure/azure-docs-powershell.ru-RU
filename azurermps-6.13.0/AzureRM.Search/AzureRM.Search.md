# Модуль поиска AzureRM.
## Nописание
В этом разделе отображаются разделы справки по командлетам службы поиска Azure.

## Командлеты поиска AzureRM.
### [Get-AzureRmSearchAdminKeyPair](Get-AzureRmSearchAdminKeyPair.md)
Возвращает пару ключей администратора службы поиска Azure.

### [Get-AzureRmSearchQueryKey](Get-AzureRmSearchQueryKey.md)
Возвращает ключи запроса службы поиска Azure.

### [Get-AzureRmSearchService](Get-AzureRmSearchService.md)
Получает службу поиска Azure.

### [New-AzureRmSearchAdminKey](New-AzureRmSearchAdminKey.md)
Повторное создание ключа администратора для службы поиска Azure.

### [New-AzureRmSearchQueryKey](New-AzureRmSearchQueryKey.md)
Создание нового ключа запроса для службы поиска Azure.

### [New-AzureRmSearchService](New-AzureRmSearchService.md)
Создает службу поиска Azure.

### [Remove-AzureRmSearchQueryKey](Remove-AzureRmSearchQueryKey.md)
Удалите ключ запроса из службы поиска Azure.

### [Remove-AzureRmSearchService](Remove-AzureRmSearchService.md)
Удалите службу поиска Azure.

### [Set-AzureRmSearchService](Set-AzureRmSearchService.md)
Обновите службу поиска Azure.

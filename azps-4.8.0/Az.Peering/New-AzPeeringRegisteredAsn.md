# New-AzPeeringRegisteredAsn

## КРАТКИй обзор
Создание зарегистрированных ASN для пиринга

## Максимальное

### ByResourceGroupAndName (по умолчанию)
```
New-AzPeeringRegisteredAsn [-ResourceGroupName] <String> [-PeeringName] <String> [-Name] <String> -Asn <Int32>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
New-AzPeeringRegisteredAsn -InputObject <PSPeering> [-Name] <String> -Asn <Int32> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
New-AzPeeringRegisteredAsn [-ResourceId] <String> [-Name] <String> -Asn <Int32> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Создание зарегистрированных ASNs для одноранговых объектов.

## ИЛЛЮСТРИРУЮТ

### Получение однорангового соединения и создание зарегистрированной учетной записи ASN
```powershell
PS C:\>$peering = Get-AzPeering -ResourceGroupName $resourceGroupName -Name $name
PS C:\>$peering | New-AzPeeringRegisteredAsn -Name $asnName -Asn $asn
```

Получите пиринг, на который вы хотите добавить зарегистрированный ASN. Затем передайте этот метод в unifiedgroup.

### Использование ИД ресурса пиринга для создания зарегистрированного ASN
```powershell
PS C:\>New-AzPeeringRegisteredAsn -ResourceId $resourceId -Name $asnName -Asn $asn
```

# Get-AzKustoClusterLanguageExtension

## КРАТКИй обзор
Возвращает список расширений языка, которые могут выполняться в запросах KQL.

## Максимальное

```
Get-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## NОПИСАНИЕ
Возвращает список расширений языка, которые могут выполняться в запросах KQL.

## ИЛЛЮСТРИРУЮТ

### Пример 1: список всех расширений языка, заданных для кластера
```powershell
PS C:\> Get-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster

Name
----
R
PYTHON
```

Приведенная выше команда возвращает список расширений языка, которые могут выполняться в запросах KQL.

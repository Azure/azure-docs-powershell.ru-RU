# Remove-AzureRmApplicationGatewayBackendAddressPool

## КРАТКИй обзор
Удаляет пул серверных адресов из шлюза приложений.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmApplicationGatewayBackendAddressPool -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmApplicationGatewayBackendAddressPool** удаляет пул серверных адресов из шлюза приложений Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление пула конечных адресов из шлюза приложений
```
PS C:\>$AppGw = Get-AzureRmApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzureRmApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "BackEndPool02"
```

Первая команда получает шлюз приложения с именем ApplicationGateway01, принадлежащий группе ресурсов с именем ResourceGroup01, и сохраняет ее в переменной $AppGw.

Вторая команда удаляет пул серверных адресов с именем BackEndPool02 из шлюза приложений.

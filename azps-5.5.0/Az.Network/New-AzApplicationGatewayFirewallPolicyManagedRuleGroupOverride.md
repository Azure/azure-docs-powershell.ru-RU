# New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride

## SYNOPSIS
Создает запись RuleGroupOverride в ManagedRuleSets для политики брандмауэра.

## СИНТАКСИС

```
New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName <String>
 -Rule <PSApplicationGatewayFirewallPolicyManagedRuleOverride[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
**New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride** создает запись ruleGroupOverride в managedRuleSet для политики брандмауэра.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $overrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName $ruleName -Rules $rule1,$rule2
```

Создание записи RuleGroupOverride с именем группы в $ruleName и правилами $rule 1, $rule 2. Назначает то же самое $overrideEntry

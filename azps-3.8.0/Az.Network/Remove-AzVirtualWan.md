# Remove-AzVirtualWan

## КРАТКИй обзор
Удаляет виртуальную глобальную сеть Azure.

## Максимальное

### ByVirtualWanName (по умолчанию)
```
Remove-AzVirtualWan -ResourceGroupName <String> -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanObject
```
Remove-AzVirtualWan -InputObject <PSVirtualWan> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanResourceId
```
Remove-AzVirtualWan -ResourceId <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Удаляет виртуальную глобальную сеть Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Passthru
```

Этот пример создает виртуальную глобальную сеть в группе ресурсов, а затем немедленно удаляет ее. Чтобы отключить вывод запроса при удалении виртуальной глобальной сети, используйте флаг-force.

### Пример 2

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -InputObject $virtualWan -Passthru
```

Этот пример создает виртуальную глобальную сеть в группе ресурсов, а затем немедленно удаляет ее. Это удаление происходит при использовании объекта виртуальной глобальной сети, возвращенного командой New-AzVirtualWan.
Чтобы отключить вывод запроса при удалении виртуальной глобальной сети, используйте флаг-force.

### Пример 3

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -ResourceId $virtualWan.Id -Passthru
```

Этот пример создает виртуальную глобальную сеть в группе ресурсов, а затем немедленно удаляет ее. Это удаление происходит при использовании идентификатора виртуального глобального ресурса, возвращенного командой New-AzVirtualWan.
Чтобы отключить вывод запроса при удалении виртуальной глобальной сети, используйте флаг-force.

# Revoke-AzureRmDiskAccess

## КРАТКИй обзор
Отменяет доступ к диску.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Revoke-AzureRmDiskAccess [-ResourceGroupName] <String> [-DiskName] <String> [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **REVOKE-AzureRmDiskAccess** отменяет доступ к диску.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Revoke-AzureRmDiskAccess -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'
```

Отмена доступа к диску с именем "Disk01" в группе ресурсов с именем "ResourceGroup01"

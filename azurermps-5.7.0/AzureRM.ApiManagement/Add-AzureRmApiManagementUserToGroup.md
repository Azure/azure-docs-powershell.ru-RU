# Add-AzureRmApiManagementUserToGroup

## КРАТКИй обзор
Добавляет пользователя в группу.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Add-AzureRmApiManagementUserToGroup -Context <PsApiManagementContext> -GroupId <String> -UserId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Add-AzureRmApiManagementUserToGroup** добавляет пользователя в группу.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Добавление пользователя в группу
```
PS C:\>$apimContext = New-AzureRmApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Add-AzureRmApiManagementUserToGroup -Context $apimContext -GroupId "0001" -UserId "0123456789"
```

Эта команда добавляет существующего пользователя в существующую группу.

# Export-AzAutomationRunbook

## SYNOPSIS
Экспорт книги автоматизации.

## СИНТАКСИС

```
Export-AzAutomationRunbook [-Name] <String> [-Slot <String>] [-OutputFolder <String>] [-Force]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С помощью cmdlet **Export-AzAutomationRunbook** можно экспортировать книгу автоматизации Azure в файл сценария wps_2 (PS1), в wps_2 или wps_2 workflow runbook, а также в графический файл runbook (Graphrunbook) для графических книг.
Имя книги становится именем экспортируемого файла.

## ПРИМЕРЫ

### Пример 1. Экспорт книги
```
PS C:\>Export-AzAutomationRunbook -ResourceGroupName "ResourceGroup01" -AutomationAccountName "ContosoAutomationAccount" -Name "Runbook03" -Slot "Published" -OutputFolder "C:\Users\PattiFuller\Desktop"
```

Эта команда экспортирует опубликованную версию книги автоматизации на рабочий стол пользователя.

# Remove-AzFirewall

## КРАТКИй обзор
Удаление брандмауэра.

## Максимальное

```
Remove-AzFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzFirewall** удаляет брандмауэр Azure.

## ИЛЛЮСТРИРУЮТ

### 1: создание и удаление брандмауэра
```
New-AzFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

В этом примере создается брандмауэр, а затем он удаляется. Чтобы отключить вывод запроса при удалении брандмауэра, используйте флаг-force.

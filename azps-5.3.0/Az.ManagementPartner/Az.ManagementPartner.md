# Модуль Az.ManagementPartner
## Описание
Управляющих партнеров для Диспетчера ресурсов Azure.

## Az.ManagementPartner Cmdlets
### [Get-AzManagementPartner](Get-AzManagementPartner.md)
Получает идентификацию текущего пользователя, который является авториентом или главой службы, в microsoft Partner Network(MPN). 

### [New-AzManagementPartner](New-AzManagementPartner.md)
Связывает ИД Microsoft Partner Network(MPN) с текущим пользователем, который является авторификацией, или с главой службы.

### [Remove-AzManagementPartner](Remove-AzManagementPartner.md)
Удалите ИД Microsoft Partner Network (MPN) текущего пользователя, который является авториентом или основной службой.

### [Update-AzManagementPartner](Update-AzManagementPartner.md)
Обновляет идентификацию текущего пользователя или основного пользователя службы Microsoft Partner Network(MPN).

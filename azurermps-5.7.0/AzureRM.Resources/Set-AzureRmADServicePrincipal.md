# Set-AzureRmADServicePrincipal

## КРАТКИй обзор
Обновление существующего субъекта-службы Azure Active Directory.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### SpObjectIdWithDisplayNameParameterSet (по умолчанию)
```
Set-AzureRmADServicePrincipal -ObjectId <String> -DisplayName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SPNWithDisplayNameParameterSet
```
Set-AzureRmADServicePrincipal -ServicePrincipalName <String> -DisplayName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Обновление существующего субъекта-службы Azure Active Directory. Чтобы обновить учетные данные, связанные с этим субъектом-службой, используйте командлет New-AzureRmADSpCredential. Чтобы обновить свойства, связанные с основным приложением, используйте командлет Set-AzureRmADApplication.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
Set-AzureRmADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 -DisplayName "UpdatedNameForSp"
```

Обновляет отображаемое имя субъекта-службы с указанным идентификатором объекта.

### Пример 2
```
Set-AzureRmADServicePrincipal -ServicePrincipalName "http://MyApp1" -DisplayName "UpdatedNameforSp"
```

Обновление отображаемого имени субъекта-службы с указанным именем субъекта-службы.

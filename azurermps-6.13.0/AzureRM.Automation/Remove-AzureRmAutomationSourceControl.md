# Remove-AzureRmAutomationSourceControl

## КРАТКИй обзор
Удаляет элемент управления источником автоматизации Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzureRmAutomationSourceControl удаляет исходный элемент управления из Azure Automation.

## ИЛЛЮСТРИРУЮТ

### Пример 1
Эта команда удаляет элемент управления источником автоматизации с именем VSTSNative в учетной записи с именем devAccount.
Эта команда задает параметр *Force* . Поэтому он не запрашивает подтверждение.

```powershell
PS C:\> Remove-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```

# Start-AzRecoveryServicesAsrSwitchProcessServerJob

## SYNOPSIS
Переключение репликации с одного сервера процессов на другой для балансировки нагрузки.

## СИНТАКСИС

```
Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric <ASRFabric> -SourceProcessServer <ASRProcessServer>
 -TargetProcessServer <ASRProcessServer> [-ReplicationProtectedItem <ASRReplicationProtectedItem[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Start-AzRecoveryServicesAsrSwserviceProcessServerJob** переключает перемещение данных репликации для указанных виртуальных машин или указанного сервера процесса на указанный сервер целевого процесса. Используется для балансировки нагрузки или переключения репликации между серверами process.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric $fabric -SourceProcessServer $sourceProcessServer  -TargetProcessServer $TargetProcessServer
```

Задание для отслеживания сервера переключения для всех элементов, защищенных репликацией, из источника в целевой сервер процесса.

### Пример 2
```
PS C:\> Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric $fabric -SourceProcessServer $sourceProcessServer  -TargetProcessServer $TargetProcessServer -ReplicatedItem $rpList
```

Задание для отслеживания сервера переключения для защищенного элемента с репликацией из источника в целевой сервер процесса.

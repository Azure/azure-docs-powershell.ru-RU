# Remove-AzWvdRegistrationInfo

## SYNOPSIS
Удалите сведения о регистрации виртуальных рабочих стола Windows.

## СИНТАКСИС

```
Remove-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## ОПИСАНИЕ
Удалите сведения о регистрации виртуальных рабочих стола Windows.

## ПРИМЕРЫ

### Пример 1. Удаление маркера регистрации виртуального рабочего стола Windows
```powershell
PS C:\> Remove-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName
```

Эта команда удаляет маркер регистрации виртуального рабочего стола Windows в пуле хостов.

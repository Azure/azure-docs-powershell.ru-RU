# Get-AzPrivateEndpointConnection

## КРАТКИй обзор
Получает ресурс подключения к частной конечной точке.

## Максимальное

### ByResourceId (по умолчанию)
```
Get-AzPrivateEndpointConnection -ResourceId <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResource
```
Get-AzPrivateEndpointConnection -Name <String> -ServiceName <String> -ResourceGroupName <String>
 [-Description <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzPrivateEndpointConnection** извлекает ресурс подключения к частной конечной точке.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
Get-AzPrivateEndpointConnection -Name MyPrivateEndpointConnection1 -ResourceGroupName TestResourceGroup -ServiceName MyPrivateLinkServiceName
```

В этом примере показано, как получить подключение к частной конечной точке с именем MyPrivateEndpointConnection1

# Remove-AzApplicationGatewayIdentity

## КРАТКИй обзор
Удаляет удостоверение из шлюза приложений.

## Максимальное

```
Remove-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApplicationGatewayIdentity** удаляет удостоверение из шлюза приложений.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $appgw = Remove-AzApplicationGatewayIdentity -ApplicationGateway $appgw
PS C:\> $updatedgateway = Set-AzApplicationGateway -ApplicationGateway $appgw
```

В этом примере мы удалим удостоверение из существующего шлюза приложений.
Примечание. Если шлюз ссылается на секрет keyvault, важно также удалить эти ссылки на сертификаты SSL в этой операции.

# Resize-AzureVirtualNetworkGateway

## КРАТКИй обзор
Изменение размера шлюза виртуальной сети.

## Максимальное

```
Resize-AzureVirtualNetworkGateway -GatewayId <String> -GatewaySKU <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Resize-AzureVirtualNetworkGateway изменяет размер шлюза виртуальной сети.

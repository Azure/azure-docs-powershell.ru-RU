# Get-AzDataFactorySlice

## SYNOPSIS
Получает срезы данных для наборов данных в фабрике данных Azure.

## СИНТАКСИС

### ByFactoryName (По умолчанию)
```
Get-AzDataFactorySlice [[-EndDateTime] <DateTime>] [-DataFactoryName] <String> [-DatasetName] <String>
 [-StartDateTime] <DateTime> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactorySlice [[-EndDateTime] <DateTime>] [-DataFactory] <PSDataFactory> [-DatasetName] <String>
 [-StartDateTime] <DateTime> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для наборов данных в фабрике данных Azure срезы получаются срезами для **get-AzDataFactorySlice.**
Укажите время начала и время окончания для определения диапазона срезов данных, которые нужно просмотреть.
Состояние среза данных — одно из следующих значений: 
- PendingExecution.
Обработка данных не начата. 
- InProgress.
Идет обработка данных. 
- "Готово".
Обработка данных завершена.
Срез данных готов к обработке зависимых срезов. 
- Не удалось.
Сбой запуска среза. 
- Пропустить.
Фабрика данных пропускает обработку среза. 
- Повторить.
Фабрика данных искомый запуск, который создает срез. 
- "По времени". Обработка данных уже из-за времени. 
- PendingValidation.
Срез данных ожидает проверки перед обработкой. 
- Повторить проверку.
Фабрика данных искомые проверки для среза. 
- Сбой проверки.
Сбой проверки среза.
Для каждого из срезов можно получить дополнительные сведения о запуске, который создается с помощью Get-AzDataFactoryRun.

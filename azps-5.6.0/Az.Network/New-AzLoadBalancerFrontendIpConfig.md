# New-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Создает конфигурацию переднего IP-адреса для балансиров нагрузки.

## СИНТАКСИС

```

 [-PrivateIpAddressVersion <String>] [-Zone <String[]>] -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Новый-AzLoadBalancerFrontendIpConfig** создает конфигурацию переднего IP-адреса для балансиров нагрузки Azure.

## ПРИМЕРЫ

### Пример 1. Создание конфигурации переднего IP-адреса для балансиров нагрузки
```
PS C:\> $publicip = New-AzPublicIpAddress -ResourceGroupName "MyResourceGroup" -Name "MyPublicIP" -Location "West US" -AllocationMethod "Dynamic"
PS C:\> New-AzLoadBalancerFrontendIpConfig -Name "FrontendIpConfig01" -PublicIpAddress $publicip
```

Первая команда создает динамический общедоступный IP-адрес MyPublicIP в группе ресурсов MyResourceGroup, а затем сохраняет его в переменной $publicip ресурса.
Вторая команда создает конфигурацию переднего IP-адреса FrontendIpConfig01, используя общедоступный IP-адрес в $publicip.

### Пример 2. Создание передней конфигурации IP-адреса для балансировки нагрузки с помощью префикса IP-адреса
```
PS C:\> $publicipprefix = New-AzPublicIpPrefix -ResourceGroupName "MyResourceGroup" -name "MyPublicIPPrefix" -location "West US" -Sku Standard -PrefixLength 28
PS C:\> $frontend = New-AzLoadBalancerFrontendIpConfig -Name "FrontendIpConfig01" -PublicIpAddressPrefix $publicipprefix
```

Первая команда создает общедоступный IP-префикс MyPublicIP длины 28 в группе ресурсов MyResourceGroup, а затем сохраняет его в переменной $publicipprefix.
Вторая команда создает конфигурацию переднего IP-адреса с именем FrontendIpConfig01, используя префикс общего IP-адреса в $publicipprefix.

# Get-AzAutomationJobOutput

## SYNOPSIS
Возвращает результат задания автоматизации.

## СИНТАКСИС

```
Get-AzAutomationJobOutput [-Id] <Guid> [-Stream <StreamType>] [-StartTime <DateTimeOffset>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Get-AzAutomationJobOutput** получает результат задания автоматизации Azure.

## ПРИМЕРЫ

### Пример 1. Вывод задания автоматизации
```
PS C:\>Get-AzAutomationJobOutput -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01" -Stream "Any"
```

Эта команда получает все выходные данные задания с указанным ИД.

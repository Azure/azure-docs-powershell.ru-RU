# Модуль Az.Subscription
## Описание
В разделах этой статьи также ются командылеты Azure PowerShell для подписки на Azure в платформе Azure Resource Manager (ARM). Командлеты существуют в области имен Microsoft.Azure.Commands.Subscription.

## Az.Subscription Cmdlets
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Получает сведения об псевдониме подписки

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Создание псевдонима и подписки

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Удаление псевдонима подписки

### [Update-AzSubscription](Update-AzSubscription.md)
Обновление подписки Azure

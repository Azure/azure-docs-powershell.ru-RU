# Get-AzCosmosDBCassandraTable

## КРАТКИй обзор
Возвращает таблицу CosmosDB Cassandra.

## Максимальное

### ByNameParameterSet (по умолчанию)
```
Get-AzCosmosDBCassandraTable -AccountName <String> -KeyspaceName <String> -ResourceGroupName <String>
 [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBCassandraTable [-Name <String>] -ParentObject <PSCassandraKeyspaceGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzCosmosDBCassandraTable** создает новый или обновляет существующий CosmosDB Cassandra keyspace.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $table = Get-AzCosmosDBCassandraTable -ResourceGroupName {resourceGroupName} -AccountName {accountName} -Keyspace {keyspaceName} -Name {name}

Name    Id   Resource
{name}  {id} {resourceObject}
```

{{Get-AzCosmosDBCassandraTable получает свойства существующего CassandraKeyspace. Вы можете развернуть ресурс, чтобы получить DefaultTtl, схему, _etag, _ts, _rid свойства.}}

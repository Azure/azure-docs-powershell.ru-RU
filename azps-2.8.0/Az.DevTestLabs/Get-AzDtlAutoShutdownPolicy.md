# Get-AzDtlAutoShutdownPolicy

## КРАТКИй обзор
Получает политику автоматического отключения лаборатории в DevTest Labs.

## Максимальное

```
Get-AzDtlAutoShutdownPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzDtlAutoShutdownPolicy** получает политику автоматического отключения лаборатории, которая позволяет автоматически завершать работу всех виртуальных машин в лаборатории в заданное время суток.
Командлет возвращает значение, указывающее, включено ли состояние политики, и время, в течение которого вы захотите автоматически завершать работу виртуальных машин лаборатории.

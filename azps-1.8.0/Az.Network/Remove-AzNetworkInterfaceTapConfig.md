# Remove-AzNetworkInterfaceTapConfig

## КРАТКИй обзор
Удаление конфигурации TAP из данного сетевого интерфейса

## Максимальное

### RemoveByNameParameterSet (по умолчанию)
```
Remove-AzNetworkInterfaceTapConfig -ResourceGroupName <String> -NetworkInterfaceName <String> -Name <String>
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzNetworkInterfaceTapConfig -ResourceId <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzNetworkInterfaceTapConfig -InputObject <PSNetworkInterfaceTapConfiguration> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzNetworkInterfaceTapConfig** удаляет конфигурацию TAP для Azure из списка сетевых интерфейсов.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление конфигурации TAP
```
PS C:\>Remove-AzNetworkInterfaceTapConfig -Name "TapConfiguration" -NetworkInterfaceName "NetworkInterface1" -ResourceGroup "ResourceGroup1"
```

Эта команда удаляет TapConfiguration из NetworkInterface1 в группе ресурсов ResourceGroup1.
Так как параметр *Force* не используется, пользователю будет предложено подтвердить это действие.

### Пример 2: Удаление сетевого интерфейса
```
PS C:\>Get-AzNetworkInterfaceTapConfig -Name "TapConfiguration" -NetworkInterfaceName "NetworkInterface1" -ResourceGroup "ResourceGroup1" | Remove-AzNetworkInterfaceTapConfig -Force
```

Эта команда удаляет TapConfiguration из NetworkInterface1 в группе ресурсов ResourceGroup1.
Поскольку используется параметр *Force* , пользователь не получает запрос на подтверждение.

# AZ. DedicatedHsm Module
## Nописание
Командлеты Microsoft Azure PowerShell: DedicatedHsm

## Командлеты AZ. DedicatedHsm
### [Get-AzDedicatedHsm](Get-AzDedicatedHsm.md)
Получает указанный специальный HSM службы Azure.

### [New-AzDedicatedHsm](New-AzDedicatedHsm.md)
Создайте или обновите Специальный HSM-аппарат в указанной подписке.

### [Remove-AzDedicatedHsm](Remove-AzDedicatedHsm.md)
Удаляет указанный выделенный HSM службы Azure.

### [Update-AzDedicatedHsm](Update-AzDedicatedHsm.md)
Обновите Специальный HSM в указанной подписке.

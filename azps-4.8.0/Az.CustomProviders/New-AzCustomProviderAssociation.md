# New-AzCustomProviderAssociation

## КРАТКИй обзор
Создание или обновление связи.

## Максимальное

```
New-AzCustomProviderAssociation -Name <String> -Scope <String> [-TargetResourceId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## NОПИСАНИЕ
Создание или обновление связи.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Создание связи с настраиваемым поставщиком
```powershell
PS C:\> $provider = Get-AzCustomProvider -ResourceGroupName myRg -Name Namespace.Type
PS C:\> New-AzCustomProviderAssociation -Scope $resourceId -Name MyAssoc -TargetResourceId $provider.Id

Location  Name     Type
--------  ----     ----
East US 2 MyAssoc  Microsoft.CustomProviders/associations
```

Создание связи с настраиваемым поставщиком. соответствующие целевые provioder должны быть правильно настроены для маршрута "Associations"

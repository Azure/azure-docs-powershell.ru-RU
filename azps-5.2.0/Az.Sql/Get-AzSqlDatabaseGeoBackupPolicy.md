# Get-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Возвращает политику геоархивации базы данных.

## СИНТАКСИС

```
Get-AzSqlDatabaseGeoBackupPolicy [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения политики геоархивации, зарегистрированной в этой базе данных, возвращается **cmdlet Get-AzSqlDatabaseGeoBackupPolicy.**
Это ресурс резервной копии Azure, который используется для определения политики хранения резервных копий.

## ПРИМЕРЫ

### Пример 1

Возвращает политику геоархивации базы данных. (autogenerated)

```powershell

```

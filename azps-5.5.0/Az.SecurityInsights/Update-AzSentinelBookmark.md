## SYNOPSIS
Обновление закладки.

 [-Label <System.Collections.Generic.IList`1[System.String]>] [-Notes <String>] [-Query <String>]

## ОПИСАНИЕ
Для закладки в указанной рабочей области обновляется **cmdlet Update-AzSentinelBook.**
Объект **Bookmark** можно передать в качестве параметра или с помощью оператора конвейера. Кроме того, можно указать необходимые параметры *BookmarkId.*
Параметр Confirm  и переменная $ConfirmPreference Windows PowerShell можно использовать для управления запросом на подтверждение с помощью cmdlet.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Update-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceNAme" -BookmarkId "MyBookmarkId" -Notes "Found something interesting"
```

Команда обновляет закладку, настраивая свойство *"Заметки".*  Все остальные возможности остаются без сбоя.

### Пример 2
```powershell
PS C:\> $Bookmark = Get-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceNAme" -BookmarkId "MyBookmarkId"
PS C:\> $Bookmark | Set-AzSentinelBookmark -Notes "Found something interesting"
```

Первая команда получает закладку *bookmarkId* из указанной рабочей области, а затем сохраняет ее в переменной $Bookmark.
Вторая команда обновляет свойство "Заметки".   Все остальные возможности остаются без сбоя.

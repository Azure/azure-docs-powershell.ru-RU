# Update-AzADServicePrincipal

## КРАТКИй обзор
Обновление существующего субъекта-службы Azure Active Directory.

## Максимальное

### SpObjectIdWithDisplayNameParameterSet (по умолчанию)
```
Update-AzADServicePrincipal -ObjectId <String> [-DisplayName <String>] [-Homepage <String>]
 [-IdentifierUri <String[]>] [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SpApplicationIdWithDisplayNameParameterSet
```
Update-AzADServicePrincipal -ApplicationId <Guid> [-Homepage <String>] [-IdentifierUri <String[]>]
 [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SPNWithDisplayNameParameterSet
```
Update-AzADServicePrincipal -ServicePrincipalName <String> [-DisplayName <String>] [-Homepage <String>]
 [-IdentifierUri <String[]>] [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectWithDisplayNameParameterSet
```
Update-AzADServicePrincipal -InputObject <PSADServicePrincipal> [-DisplayName <String>] [-Homepage <String>]
 [-IdentifierUri <String[]>] [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Обновление существующего субъекта-службы Azure Active Directory. Чтобы обновить учетные данные, связанные с этим субъектом-службой, используйте командлет New-AzADSpCredential. Чтобы обновить свойства, связанные с основным приложением, используйте командлет Update-AzADApplication.

## ИЛЛЮСТРИРУЮТ

### Пример 1. Изменение отображаемого имени субъекта-службы

```
PS C:\> Update-AzADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 -DisplayName MyNewDisplayName
```

Обновление отображаемого имени субъекта-службы с идентификатором объекта "784136ca-3ae2-4fdd-A388-89d793e7c780" и "MyNewDisplayName".

### Пример 2. Изменение отображаемого имени субъекта-службы с помощью трубопроводов

```
PS C:\> Get-AzADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 | Update-AzADServicePrincipal -DisplayName MyNewDisplayName
```

Возвращает субъекта-службы с идентификатором объекта "784136ca-3ae2-4fdd-A388-89d793e7c780" и каналами, которые должны использовать командлет Update-AzADServicePrincipal, чтобы изменить отображаемое имя субъекта-службы на "MyNewDisplayName".

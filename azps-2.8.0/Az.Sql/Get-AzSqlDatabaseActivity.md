# Get-AzSqlDatabaseActivity

## КРАТКИй обзор
Возвращает состояние операций с базой данных.

## Максимальное

```
Get-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlDatabaseActivity** получает состояние операций с базой данных в базе данных SQL Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение состояния всех экземпляров базы данных SQL
```
PS C:\>Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Эта команда возвращает состояние операции всех экземпляров базы данных SQL в пуле эластичных БД с именем ElasticPool01.

### Пример 2: получение состояния всех операций с базой данных SQL
```
PS C:\>Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда возвращает состояние всех операций базы данных SQL в базе данных.

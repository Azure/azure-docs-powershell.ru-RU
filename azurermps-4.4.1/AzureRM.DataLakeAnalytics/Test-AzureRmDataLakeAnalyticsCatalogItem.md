# Test-AzureRmDataLakeAnalyticsCatalogItem

## КРАТКИй обзор
Проверка существования элемента каталога.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Test-AzureRmDataLakeAnalyticsCatalogItem [-Account] <String> [-ItemType] <CatalogItemType>
 [-Path] <CatalogPathInstance> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Test-AzureRmDataLakeAnalyticsCatalogItem** проверяет наличие элемента каталога Azure Data Lake Analytics.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Проверка наличия элемента каталога
```
PS C:\>Test-AzureRmDataLakeAnalyticsCatalogItem -Account "ContosoAdlAccount" -ItemType Schema -Path "databaseName.schemaName"
```

Эта команда проверяет, существует ли указанный элемент схемы.

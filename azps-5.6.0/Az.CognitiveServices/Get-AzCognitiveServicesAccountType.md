# Get-AzCognitiveServicesAccountType

## SYNOPSIS
Возвращает доступные типы учетных записей когнитивных служб.

## СИНТАКСИС

### GetAccountTypes (по умолчанию)
```
Get-AzCognitiveServicesAccountType [-Location <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetAccountTypeWithName
```
Get-AzCognitiveServicesAccountType -TypeName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для этой подписки доступны типы учетных записей Учетных записей **Get-AzCognitiveServicesAccountType.**

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountType
```

Получить список доступных типов.

### Пример 2
```powershell
PS C:\> Get-AzCognitiveServicesAccountType -Location westus
```

Получить список доступных типов на westus.

### Пример 3
```powershell
PS C:\> Get-AzCognitiveServicesAccountType -TypeName Face

Face
```

Проверьте, допустимо ли имя типа, оно будет возвращено, если `Face` оно допустимо.

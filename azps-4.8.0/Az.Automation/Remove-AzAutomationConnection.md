# Remove-AzAutomationConnection

## КРАТКИй обзор
Удаляет подключение автоматизации.

## Максимальное

```
Remove-AzAutomationConnection [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzAutomationConnection** удаляет подключение из Azure Automation.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление соединения
```
PS C:\>Remove-AzAutomationConnection -AutomationAccountName "Contoso17" -Name "ContosoConnection" -ResourceGroupName "ResourceGroup01"
```

Эта команда удаляет сертификат с именем ContosoConnection в учетной записи автоматизации с именем Contoso17.

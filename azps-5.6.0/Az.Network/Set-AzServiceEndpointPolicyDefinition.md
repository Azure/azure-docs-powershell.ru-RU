# Set-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Обновляет определение политики конечной точки службы.

## СИНТАКСИС

```
Set-AzServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <String[]>] [-Service <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания определения политики конечной точки службы будет создан **cmdlet Set-AzServiceEndpointPolicyDefinition.**

## ПРИМЕРЫ

### Пример 1. Обновление определения политики конечной точки службы в политике конечных точек службы
```
$serviceEndpointPolicy = Set-AzServiceEndpointPolicyDefinition -Name "Policydef1" -ServiceEndpointPolicy $serviceEndpointPolicy
```

Эта команда обновляет определение политики конечной точки службы с именем Policydef1 в политике конечных точек службы, заданной объектом $ServiceEndpointPolicy.

# Remove-AzSqlInstanceKeyVaultKey

## SYNOPSIS
Удаление ключа сейфа из SQL управляемого экземпляра

## СИНТАКСИС

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (по умолчанию)
```
Remove-AzSqlInstanceKeyVaultKey [-ResourceGroupName] <String> [-InstanceName] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyInputObjectParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-Instance] <AzureSqlManagedInstanceModel> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-InstanceResourceId] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Remove-AzSqlInstanceKeyVaultKey удаляет ключ сейфа из указанного управляемого экземпляра. Обратите внимание SQL что разрешения управляемого экземпляра для хранилища ключа не меняются. Чтобы изменить разрешения, используйте Set-AzKeyVaultAccessPolicy. Обратите внимание, что этот cmdlet не изменяет хранилище ключей. Чтобы удалить ключ из key Vault, используйте Remove-AzureKeyVaultKey.

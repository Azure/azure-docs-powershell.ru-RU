# Add-AzKeyVaultCertificate

## КРАТКИй обзор
Добавляет сертификат в хранилище ключей.

## Максимальное

```
Add-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String>
 [[-CertificatePolicy] <KeyVaultCertificatePolicy>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Add-AzKeyVaultCertificate** запускает процесс регистрации сертификата в хранилище ключей Azure Key Vault.

Вторая команда использует **Add-AzKeyVaultCertificate** , чтобы запустить процесс для создания сертификата.

Третья команда использует командлет Get-AzKeyVaultCertificateOperation для опроса операции, чтобы убедиться, что она завершена.

В последней команде для получения сертификата используется командлет Get-AzKeyVaultCertificate.

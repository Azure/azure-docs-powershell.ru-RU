# Get-AzApiManagementTenantAccessSecret

## КРАТКИй обзор
Получает ключи конфигурации доступа для клиента.

## Максимальное

```
Get-AzApiManagementTenantAccessSecret -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Получает ключи конфигурации доступа для клиента.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение ключей конфигурации для доступа к клиенту
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccessSecret -Context $apimContext
```

Эта команда получает ключи конфигурации для доступа клиента к заданному контексту.

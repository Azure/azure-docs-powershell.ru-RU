# Add-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Добавляет сертификат проверки подлинности к шлюзу приложения.

## СИНТАКСИС

```
Add-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Add-AzApplicationGatewayAuthenticationCertificate** добавляет сертификат проверки подлинности в шлюз приложения Azure.

## ПРИМЕРЫ

### Пример 1. Добавление сертификата проверки подлинности к шлюзу приложения
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Add-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01" -CertificateFile "C:\cert.cer"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Первая команда получает шлюз приложения appGwName и сохраняет его в $appgw переменной.
Вторая команда добавляет сертификат проверки подлинности с именем cert01 в шлюз приложения.
Третья команда обновляет шлюз приложения.

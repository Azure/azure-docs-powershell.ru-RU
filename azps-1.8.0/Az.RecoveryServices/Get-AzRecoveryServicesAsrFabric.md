# Get-AzRecoveryServicesAsrFabric

## КРАТКИй обзор
Получение сведений о структуре восстановления сайта Azure.

## Максимальное

### По умолчанию (по умолчанию)
```
Get-AzRecoveryServicesAsrFabric [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzRecoveryServicesAsrFabric -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFriendlyName
```
Get-AzRecoveryServicesAsrFabric -FriendlyName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzRecoveryServicesAsrFabric** получает свойства указанной структуры восстановления сайта Azure или всех структур восстановления сайта Azure в хранилище службы восстановления.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> $fabrics = Get-AzRecoveryServicesAsrFabric
```

Возвращает все структуры восстановления сайта Azure в хранилище.

### Пример 2
```
PS C:\> $fabric = Get-AzRecoveryServicesAsrFabric -Name xxxx

SiteIdentifier        : XXXXXXXXxxxxxxxxxxx
```

Возвращают структуру восстановления сайта Azure с именем XXXX.

# New-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Создает конфигурацию пула адресов для резерва нагрузки.

## СИНТАКСИС

```
New-AzLoadBalancerBackendAddressPoolConfig -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet New-AzLoadBalancerBackendAddressPoolConfig** создается конфигурация пула адресов для резервирования нагрузки Azure.

## ПРИМЕРЫ

### Пример 1. Создание конфигурации пула адресов для резерва нагрузки
```
PS C:\>New-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02"
```

Эта команда создает конфигурацию пула адресов для дополнительных адресов с именем BackendAddressPool02 для балансиров нагрузки.

# Remove-AzVMDiagnosticsExtension

## SYNOPSIS
Удаляет расширение diagnostics (Диагностика) с виртуальной машины.

## СИНТАКСИС

```
Remove-AzVMDiagnosticsExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Remove-AzVMDiagnosticsExtension** можно удалить расширение для диагностики Azure с виртуальной машины.
Для реализации изменений необходимо передать результат этого Update-AzVM в этот Update-AzVM.

## ПРИМЕРЫ

### Пример 1. Удаление расширения диагностики с виртуальной машины
```
PS C:\> Remove-AzVMDiagnosticsExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" | Update-AzVM
```

Эта команда удаляет расширение диагностики с виртуальной машины с именем ContosoVM22.
Команда передает результат командлету Update-AzVM с помощью оператора конвейера.
Эта команда обновляет виртуальную машину.

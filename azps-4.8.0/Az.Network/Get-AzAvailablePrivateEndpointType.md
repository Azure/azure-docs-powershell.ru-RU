# Get-AzAvailablePrivateEndpointType

## КРАТКИй обзор
Возвращают доступные типы частных конечных точек в расположении.

## Максимальное

```
Get-AzAvailablePrivateEndpointType -Location <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzAvailablePrivateEndpointType** возвращает все доступные типы частных конечных точек в расположении.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
Get-AzAvailablePrivateEndpointType -Location eastus
```

# Stop-AzureService

## КРАТКИй обзор
Останавливает текущую размещенную службу.

## Максимальное

```
Stop-AzureService [-ServiceName <String>] [-Slot <String>] [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
В этом разделе описан командлет в версии 0.8.10 модуля Microsoft Azure PowerShell.
Чтобы получить версию модуля, который вы используете, введите в командной консоли Azure PowerShell `(Get-Module -Name Azure).Version` .

Командлет **Stop-AzureService** останавливает текущую размещенную службу в указанном слоте в Windows Azure.
Если ячейка не указана, командлет останавливает службу в производственном слоте.

# New-AzApiManagementContext

## SYNOPSIS
Создает экземпляр PsAzureApiManagementContext.

## СИНТАКСИС

### ContextParameterSet (по умолчанию)
```
New-AzApiManagementContext -ResourceGroupName <String> -ServiceName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
New-AzApiManagementContext -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **создания экземпляра PsAzureApiManagementContext** создается экземпляр **PsAzureApiManagementContext.**
Этот контекст используется для всех cmdlets службы управления API.

## ПРИМЕРЫ

### Пример 1. Создание экземпляра PsApiManagementContext
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "ContosoResources" -ServiceName "Contoso"
```

Эта команда создает экземпляр **PsApiManagementContext.**

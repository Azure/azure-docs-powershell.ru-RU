# Remove-AzPrivateEndpoint

## SYNOPSIS
Удаляет частную конечную точку.

## СИНТАКСИС

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **удаления закрытой** конечной точки удаляется личная конечная точка. 

## ПРИМЕРЫ

### Пример 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

В этом примере удаляется частная конечная точка с именем MyPrivateEndpoint1.

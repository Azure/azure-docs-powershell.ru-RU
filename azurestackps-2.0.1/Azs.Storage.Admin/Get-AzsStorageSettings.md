# Get-AzsStorageSettings

## КРАТКИй обзор
Возвращает параметры поставщика ресурсов хранилища.

## Максимальное

```
Get-AzsStorageSettings [-Location <String>] [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Возвращает параметры поставщика ресурсов хранилища.

## ИЛЛЮСТРИРУЮТ

### Пример 1:
```powershell
PS C:\> Get-AzsStorageSettings
```

Получение параметров хранилища.

# Test-AzCdnCustomDomain

## КРАТКИй обзор
Проверяет, можно ли добавить в конечную точку настраиваемый домен.

## Максимальное

### ByFieldsParameterSet (по умолчанию)
```
Test-AzCdnCustomDomain -EndpointName <String> -ProfileName <String> -ResourceGroupName <String>
 -CustomDomainHostName <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Test-AzCdnCustomDomain -CdnEndpoint <PSEndpoint> -CustomDomainHostName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Test-AzCdnCustomDomain** проверяет, можно ли добавить настраиваемый домен в конечную точку, проверив сопоставление CNAME.

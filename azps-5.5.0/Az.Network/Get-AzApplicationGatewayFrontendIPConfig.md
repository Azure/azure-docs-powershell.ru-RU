# Get-AzApplicationGatewayFrontendIPConfig

## SYNOPSIS
Получает конфигурацию переднего IP-адреса шлюза приложения.

## СИНТАКСИС

```
Get-AzApplicationGatewayFrontendIPConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для шлюза приложения будет задана конфигурация IP-адреса переднего ip-адреса **Get-AzApplicationGatewayFrontendIPConfig.**

## ПРИМЕРЫ

### Пример 1. Получить указанную конфигурацию переднего IP-адреса
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndIP= Get-AzApplicationGatewayFrontendIPConfig -Name "FrontEndIP01" -ApplicationGateway $AppGw
```

Первая команда получает шлюз приложения ApplicationGateway01 из группы ресурсов ResourceGroup01 и сохраняет его в переменной $AppGw ресурса. Вторая команда получает от имени FrontEndIP01 конфигурацию переднего IP-адреса из $AppGw и сохраняет ее в переменной $FrontEndIP.

### Пример 2. Получить список конфигураций IP-адресов переднего
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndIPs= Get-AzApplicationGatewayFrontendIPConfig  -ApplicationGateway $AppGw
```

Первая команда получает шлюз приложения ApplicationGateway01 из группы ресурсов ResourceGroup01 и сохраняет его в переменной $AppGw ресурса. Вторая команда получает список конфигураций IP-адресов переднего $AppGw и сохраняет его в переменной $FrontEndIPs.

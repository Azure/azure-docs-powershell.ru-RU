# Remove-AzServiceFabricNode

## КРАТКИй обзор
Удаление узлов из кластера с определенным типом узла.

## Максимальное

```
Remove-AzServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
С помощью **Remove-AzServiceFabricNode** удалите узлы из кластера определенного типа. Удаление выполняется только в том случае, если оно соответствует метрикам работоспособности кластера.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS c:> Remove-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1' -NumberOfNodesToRemove 2
```

Эта команда удалит 2 узла из NodeType "NT1".

# New-AzRedisCachePatchSchedule

## КРАТКИй обзор
Добавление расписания исправлений.

## Максимальное

```
New-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzRedisCachePatchSchedule** добавляет расписание исправлений в кэш в Azure Redis Cache.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание и Добавление расписания исправлений в кэш
```
PS C:\>New-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

Эта команда добавляет расписание исправлений в кэш с именем RedisCache06.
Параметр "записи" принимает в качестве значения команду, которая использует **New-AzRedisCacheScheduleEntry** для создания расписания.

# Remove-AzVmssDiagnosticsExtension

## КРАТКИй обзор
Удаляет расширение диагностики из VMSS.

## Максимальное

```
Remove-AzVmssDiagnosticsExtension [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzVmssDiagnosticsExtension** удаляет расширение диагностики из установленной шкалы виртуальных машин (VMSS).

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление расширения диагностики из VMSS
```
PS C:\> Remove-AzVmssDiagnosticsExtension -VirtualMachineScaleSet $VMSS -Name $extName
```

Эта команда удаляет расширение диагностики из VMSS.

# Get-AzKustoClusterLanguageExtension

## SYNOPSIS
Возвращает список языковых расширений, которые можно использовать в запросах KQL.

## СИНТАКСИС

```
Get-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## ОПИСАНИЕ
Возвращает список языковых расширений, которые можно использовать в запросах KQL.

## ПРИМЕРЫ

### Пример 1. Список всех языковых расширений, установленных для кластера
```powershell
PS C:\> Get-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster

Name
----
R
PYTHON
```

Эта команда возвращает список языковых расширений, которые можно использовать в запросах KQL.

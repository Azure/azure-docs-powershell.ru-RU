# Set-AzCdnProfile

## КРАТКИй обзор
Обновляет профиль CDN.

## Максимальное

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzCdnProfile** обновляет профиль сети доставки содержимого (CDN) Azure.

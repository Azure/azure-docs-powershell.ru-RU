# New-AzPrivateLinkServiceIpConfig

## SYNOPSIS
Создайте конфигурацию IP-адреса службы частных ссылок.

## СИНТАКСИС

```
New-AzPrivateLinkServiceIpConfig -Name <String> [-PrivateIpAddressVersion <String>]
 [-PrivateIpAddress <String>] [-PublicIpAddress <PSPublicIpAddress>]
 [-Subnet <PSSubnet>] [-Primary]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания конфигурации ip-адреса службы личной ссылки создается **cmdlet New-AzPrivateLinkServiceIpConfig.** Эта конфигурация используется для nat-ing источника входящих трафика из закрытой конечной точки. 

## ПРИМЕРЫ

### Пример 1
```powershell
New-AzPrivateLinkServiceIpConfig -Name $IpConfigurationName -PrivateIpAddress "10.0.0.5" -Primary
```

В этом примере создается конфигурация IP-адреса службы частных ссылок в памяти.

# Test-AzDataLakeAnalyticsAccount

## КРАТКИй обзор
Проверка наличия учетной записи Data Lake Analytics.

## Максимальное

```
Test-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Test-AzDataLakeAnalyticsAccount** проверяет наличие учетной записи Data Lake Analytics.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Проверка наличия учетной записи
```
PS C:\>Test-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Эта команда проверяет, существует ли учетная запись с именем ContosoAdlAccount.

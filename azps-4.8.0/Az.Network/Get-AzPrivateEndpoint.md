# Get-AzPrivateEndpoint

## КРАТКИй обзор
Получение закрытой конечной точки

## Максимальное

### NOEXPAND (по умолчанию)
```
Get-AzPrivateEndpoint [-Name <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Кройте
```
Get-AzPrivateEndpoint -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzPrivateEndpoint** получает одну или несколько закрытых конечных точек.

# Set-AzApiManagementTenantAccess

## SYNOPSIS
Включает или отключает клиентский доступ.

## СИНТАКСИС

```
Set-AzApiManagementTenantAccess -Context <PsApiManagementContext> -Enabled <Boolean> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Set-AzApiManagementTenantAccess** можно включить или отключить доступ к клиенту.

## ПРИМЕРЫ

### Пример 1. Включить клиентский доступ
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementTenantAccess -Context $apimContext -Enabled $True
```

Эта команда обеспечивает доступ клиента в указанном контексте.

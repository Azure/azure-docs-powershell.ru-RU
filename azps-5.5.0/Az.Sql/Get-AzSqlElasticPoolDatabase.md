# Get-AzSqlElasticPoolDatabase

## SYNOPSIS
Получает эластичные базы данных в эластичном пуле и значениях их свойств.

## СИНТАКСИС

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzSqlElasticPoolDatabase** получает эластичные базы данных в эластичном пуле и значениях их свойств.
В базе данных Azure SQL можно указать имя эластичной базы данных, чтобы увидеть значения свойств только для этой базы данных.

## ПРИМЕРЫ

### Пример 1. Получить все базы данных в эластичном пуле
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Эта команда получает все базы данных в эластичном пуле Под названием "ЭластичнаяPool01".

### Пример 2. Получить все базы данных в эластичном пуле с помощью фильтрации
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

Эта команда получает все базы данных в эластичном пуле Под названием "ЭластичнаяPool01", которые начинаются с "База данных".

# Remove-AzureLocalNetworkGateway

## КРАТКИй обзор
Удаление шлюза локальной сети Azure.

## Максимальное

```
Remove-AzureLocalNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureLocalNetworkGateway** удаляет шлюз локальной сети Azure.

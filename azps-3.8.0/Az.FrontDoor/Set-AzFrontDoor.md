# Set-AzFrontDoor

## КРАТКИй обзор

## NОПИСАНИЕ
Командлет **Set-AzFrontDoor** обновляет подсистему балансировки нагрузки передней дверцы. Если входные параметры не предоставлены, будут использоваться старые параметры существующей передней дверцы.

# Set-AzureRmDataLakeStoreTrustedIdProvider

## КРАТКИй обзор
Изменяет указанного доверенного поставщика удостоверений в указанном Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Set-AzureRmDataLakeStoreTrustedIdProvider [-Account] <String> [-Name] <String> [-ProviderEndpoint] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Set-AzureRmDataLakeStoreTrustedIdProvider** изменяет указанный доверенный поставщик удостоверений в указанном Data Lake Store.

## ИЛЛЮСТРИРУЮТ

### Пример 1: обновление доверенной конечной точки поставщика удостоверений
```
PS C:\> Set-AzureRmDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider -ProviderEndpoint "https://sts.windows.net/6b04908c-b91f-40ce-8024-7ee8a4fd6150"
```

Это обновление поставщика endpoing для правила межсетевого экрана "MyProvider" в учетной записи "ContosoADL" на " https://sts.windows.net/6b04908c-b91f-40ce-8024-7ee8a4fd6150 "

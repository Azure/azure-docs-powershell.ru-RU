# Remove-AzApiManagementIdentityProvider

## SYNOPSIS
Удаляет существующую конфигурацию поставщика удостоверений.

## СИНТАКСИС

```
Remove-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Удаляет существующую конфигурацию поставщика удостоверений.

## ПРИМЕРЫ

### Пример 1. Удаление параметров поставщика удостоверений Facebook из службы ApiManagement
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementIdentityProvider -Context $apimContext -Type 'Facebook' -PassThru
```

Удаляет конфигурацию, связанную с конфигурацией поставщика удостоверений Facebook.

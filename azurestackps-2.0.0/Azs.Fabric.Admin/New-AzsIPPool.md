# New-AzsIPPool

## КРАТКИй обзор
Создание пула IP-адресов.
После создания пул IP-адресов не может быть удален.

## Максимальное

### CreateExpanded (по умолчанию)
```
New-AzsIPPool -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String>]
 [-AddressPrefix <String>] [-EndIPAddress <String>]
 [-NumberOfAllocatedIPAddress <Int64>] [-NumberOfIPAddress <Int64>] [-NumberOfIPAddressesInTransition <Int64>]
 [-StartIPAddress <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### Заново
```
New-AzsIPPool -Name <String> -Pool <IIPPool> [-Location <String>] [-ResourceGroupName <String>]
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Создание пула IP-адресов.
После создания пул IP-адресов не может быть удален.

## ИЛЛЮСТРИРУЮТ

### Пример 1:
```powershell
PS C:\> New-AzsIpPool -Name IpPool4 -StartIpAddress ***.***.***.*** -EndIpAddress ***.***.***.*** -AddressPrefix ***.***.***.***/24

```

Создание нового пула IP-адресов.

## Пуск

Свойства сложных параметров для создания описанных ниже параметров создайте хэш-таблицу, содержащую соответствующие свойства. Для получения сведений о хэш-таблицах запустите Get-Help about_Hash_Tables.

ПУЛ <IIPPool> : этот ресурс определяет диапазон IP-адресов, на основе которых будут выделяться адреса для узлов в подсети.
  - `[Location <String>]`: Область, в которой находится ресурс.
  - `[Tag <IResourceTags>]`: Список пар "ключ-значение".
    - `[(Any) <String>]`: Это указывает на то, что в этот объект можно добавить любое свойство.
  - `[AddressPrefix <String>]`: Префикс адреса.
  - `[EndIPAddress <String>]`: Конечный IP-адрес.
  - `[NumberOfAllocatedIPAddresses <Int64?>]`: Количество выделенных в настоящее время IP-адресов.
  - `[NumberOfIPAddresses <Int64?>]`: Общее число IP-адресов.
  - `[NumberOfIPAddressesInTransition <Int64?>]`: Текущее число IP-адресов в переходе.
  - `[StartIPAddress <String>]`: Начальный IP-адрес.

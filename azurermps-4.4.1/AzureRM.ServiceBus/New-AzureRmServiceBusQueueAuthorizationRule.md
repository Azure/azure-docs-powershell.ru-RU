# New-AzureRmServiceBusQueueAuthorizationRule

## КРАТКИй обзор
Создание нового правила авторизации для указанной очереди обслуживания служебной шины.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmServiceBusQueueAuthorizationRule [-ResourceGroup] <String> -Namespace <String> -Queue <String>
 -Name <String> [-Rights] <String[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmServiceBusQueueAuthorizationRule** создает новое правило авторизации для указанной очереди служебной шины.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> New-AzureRmServiceBusQueueAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -QueueName SB-Queue_exampl1 -AuthorizationRuleName SBAuthoRule1 -Rights @("Listen","Send")
```

Создание правила авторизации `SBAuthoRule1` с правами **прослушивания и отправки** для очереди `SB-Queue_exampl1` .

# New-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Создает правило маршрутки запросов для шлюза приложения.

## СИНТАКСИС

```
New-AzApplicationGatewayRewriteRuleSet -Name <String>
 -RewriteRule <System.Collections.Generic.List`1[Microsoft.Azure.Commands.Network.Models.PSApplicationGatewayRewriteRule]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для шлюза приложения Azure создается набор правил переописи для нового **azApplicationGatewayRewriteRuleSet.**

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $ruleset = New-AzApplicationGatewayRewriteRuleSet -Name ruleset1 -RewriteRule $rule
```

Эта команда создает набор правил переописи с именем ruleset1 и сохраняет результат в переменной с $ruleset.

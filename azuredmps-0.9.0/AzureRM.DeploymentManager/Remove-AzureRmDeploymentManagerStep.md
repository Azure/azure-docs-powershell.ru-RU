# Remove-AzureRmDeploymentManagerStep

## КРАТКИй обзор
Удаление шага.

## Максимальное

### Интерактивный (по умолчанию)
```
Remove-AzureRmDeploymentManagerStep [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzureRmDeploymentManagerStep [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzureRmDeploymentManagerStep [-Step] <PSStepResource> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmDeploymentManagerStep** удаляет шаг.
Укажите шаг, указав имя и имя группы ресурсов. Кроме того, вы можете предоставить объект Step или ResourceId.

## ИЛЛЮСТРИРУЮТ
### Пример 1: удаление шага
```powershell
PS C:\> Remove-AzureRmDeploymentManagerStep -ResourceGroupName ContosoResourceGroup -Name ContosoService1WaitStep
```

Эта команда удаляет шаг с именем ContosoService1WaitStep в ContosoResourceGroup.

### Пример 2: удаление шага с помощью идентификатора ресурса
```powershell
PS C:\> Remove-AzureRmDeploymentManagerStep -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/steps/ContosoService1WaitStep"
```

Эта команда удаляет шаг с именем ContosoService1WaitStep в ContosoResourceGroup.

### Пример 3: удаление шага с помощью объекта, возвращаемого New-AzureRmDeploymentManagerStep
```powershell
PS C:\> Remove-AzureRmDeploymentManagerStep -Step $stepObject
```

 Эта команда удаляет шаг, имя и значение ResourceGroup которого совпадают со свойствами Name и ResourceGroupName для $stepObject соответственно.

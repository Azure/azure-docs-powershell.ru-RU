# Add-AzureRmSqlServerKeyVaultKey

## КРАТКИй обзор
Добавляет ключ хранилища ключей на сервер SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Add-AzureRmSqlServerKeyVaultKey [-KeyId] <String> [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Add-AzureRmSqlServerKeyVaultKey добавляет ключ хранилища ключей на указанный SQL Server.
У сервера должны быть разрешения "получить, wrapKey, unwrapKey" для хранилища.

## ИЛЛЮСТРИРУЮТ

### --------------------------Пример 1: Добавление ключа хранилища ключей--------------------------
```
PS C:\> Add-AzureRmSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Эта команда добавляет ключ хранилища ключей с идентификатором " https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 ContosoServer" в SQL Server в группе ресурсов "ContosoResourceGroup".

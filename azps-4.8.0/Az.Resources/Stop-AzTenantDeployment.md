# Stop-AzTenantDeployment

## КРАТКИй обзор
Отмена выполняющегося развертывания в области клиента

## Максимальное

### StopByDeploymentName (по умолчанию)
```
Stop-AzTenantDeployment [-Name] <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzTenantDeployment -Id <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzTenantDeployment -InputObject <PSDeployment> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Stop-AzTenantDeployment** отменяет развертывание, которое началось, но не завершено в текущей области клиента.
Чтобы остановить развертывание, необходимо, чтобы развертывание имело неполное состояние подготовки, такое как подготовка, а не завершенное состояние, например подготовка или сбой.

Чтобы создать развертывание в области клиента, используйте командлет New-AzTenantDeployment.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\>Stop-AzTenantDeployment -Name "deployment01"
```

Эта команда отменяет запущенное развертывание "deployment01" в текущей области клиента.

### Пример 2
```
PS C:\>Get-AzTenantDeployment -Name "deployment01" | Stop-AzTenantDeployment
```

Эта команда получает развертывание "deployment01" в текущей области клиента и отменяет его.

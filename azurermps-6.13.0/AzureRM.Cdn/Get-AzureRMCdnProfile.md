# Get-AzureRmCdnProfile

## КРАТКИй обзор
Получает профиль CDN.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmCdnProfile [-ProfileName <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRMCdnProfile** получает профиль сети доставки содержимого (CDN) для Azure и связанную с ней информацию.

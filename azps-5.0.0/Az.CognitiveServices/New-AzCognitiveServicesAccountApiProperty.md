# New-AzCognitiveServicesAccountApiProperty

## КРАТКИй обзор
Создание нового экземпляра учетной записи самообслуживания ApiProperties

## Максимальное

```
New-AzCognitiveServicesAccountApiProperty [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Создайте новый экземпляр учетной записи службы ApiProperties.
ApiProperties можно использовать при создании новой или обновлении существующей учетной записи.
ApiProperties является обязательным для некоторых типов учетных записей.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $apiProperties = New-AzCognitiveServicesAccountApiProperty
PS C:\> $apiProperties.QnaRuntimeEndpoint = "https://qnamaker.azurewebsites.net"
PS C:\> New-AzCognitiveServicesAccount -ResourceGroupName cognitive-services-resource-group -name qnamaker -Type QnAMaker -SkuName S0 -Locatio WestUS -ApiProperty $apiProperties
```

В приведенном выше примере создается учетная запись QnAMaker с свойством API "QnaRuntimeEndpoint".

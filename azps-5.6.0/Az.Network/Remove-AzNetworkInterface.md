# Remove-AzNetworkInterface

## SYNOPSIS
Удаляет сетевой интерфейс.

## СИНТАКСИС

```
Remove-AzNetworkInterface -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Командлет **Remove-AzNetworkInterface** удаляет сетевой интерфейс Azure.

## ПРИМЕРЫ

### Пример 1. Удаление сетевого интерфейса
```
PS C:\>Remove-AzNetworkInterface -Name "NetworkInterface1" -ResourceGroup "ResourceGroup1"
```

Эта команда удаляет сетевой интерфейс NetworkInterface1 в группе ресурсов ResourceGroup1.
Так как *параметр Force* не используется, пользователю будет предложено подтвердить это действие.

### Пример 2. Удаление сетевого интерфейса
```
PS C:\>Get-AzNetworkInterface -ResourceGroupName "ResourceGroup1" | Remove-AzNetworkInterface -Force
```

Эта команда удаляет все сетевые интерфейсы группы ресурсов ResourceGroup1.
Так как используется параметр *Force,* пользователю не будет предложено подтверждение.

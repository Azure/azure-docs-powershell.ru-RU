# New-AzApiManagementCertificate

## SYNOPSIS
Создание сертификата управления API, который будет использоваться во время проверки подлинности с помощью backend.

## СИНТАКСИС

### LoadFromFile (по умолчанию)
```
New-AzApiManagementCertificate -Context <PsApiManagementContext> [-CertificateId <String>]
 -PfxFilePath <String> -PfxPassword <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Необработанные
```
New-AzApiManagementCertificate -Context <PsApiManagementContext> [-CertificateId <String>] -PfxBytes <Byte[]>
 -PfxPassword <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Новый-AzApiManagementCertificate создает** сертификат управления API Azure.

## ПРИМЕРЫ

### Пример 1. Создание и отправка сертификата
```powershell
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>New-AzApiManagementCertificate -Context $ApiMgmtContext -PfxFilePath "C:\contoso\certificates\apimanagement.pfx" -PfxPassword "1111"
```

Эта команда передает сертификат в управление Api. Этот сертификат можно использовать для совместной проверки подлинности с использованием backend с помощью политик.

### Пример 2

Создание сертификата управления API, который будет использоваться во время проверки подлинности с помощью backend. (autogenerated)


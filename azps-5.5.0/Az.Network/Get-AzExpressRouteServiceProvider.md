# Get-AzExpressRouteServiceProvider

## SYNOPSIS
Список поставщиков услуг ExpressRoute и их атрибутов.

## СИНТАКСИС

```
Get-AzExpressRouteServiceProvider [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения списка поставщиков услуг ExpressRoute и их атрибутов извлекается **cmdlet Get-AzExpressRouteServiceProvider.** Атрибут включает параметры расположения и пропускной способности.

## ПРИМЕРЫ

### Пример 1. Получить список поставщиков услуг с расположениями в "Кремниевой долине"
```
Get-AzExpressRouteServiceProvider |
   Where-Object PeeringLocations -Contains "Silicon Valley" |
   Select-Object Name
```

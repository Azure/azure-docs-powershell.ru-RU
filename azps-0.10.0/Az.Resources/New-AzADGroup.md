# New-AzADGroup

## КРАТКИй обзор
Создание новой группы Active Directory.

## Максимальное

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Создание новой группы Active Directory.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание новой группы рекламных объявлений

```
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Создание новой группы Active Directory с именем "MyGroupDisplayName" и псевдонимом почты " myemail@domain.com ".

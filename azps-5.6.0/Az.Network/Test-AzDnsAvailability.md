# Test-AzDnsAvailability

## SYNOPSIS
Проверяет, доступно ли доменное имя в cloudapp.azure.com зоне.

## СИНТАКСИС

```
Test-AzDnsAvailability -DomainNameLabel <String> -Location <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Проверяет, доступно ли доменное имя в cloudapp.azure.com зоне.

## ПРИМЕРЫ

### Пример 1. Проверьте, contoso.westus.cloudapp.azure.com ли contoso.westus.cloudapp.azure.com можно использовать.
```
Test-AzDnsAvailability -DomainNameLabel contoso -Location westus
```

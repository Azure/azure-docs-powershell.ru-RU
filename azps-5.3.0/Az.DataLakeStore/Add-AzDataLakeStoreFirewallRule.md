# Add-AzDataLakeStoreFirewallRule

## SYNOPSIS
Добавляет правило брандмауэра к указанной учетной записи Data Lake Store.

## СИНТАКСИС

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для указанной учетной записи Data Lake Store будет добавлено правило брандмауэра **Add-AzDataLakeStoreFirewallRule.**

## ПРИМЕРЫ

### Пример 1. Добавление нового правила брандмауэра в учетную запись Data Lake Store
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Будет создаваться новое правило брандмауэра под названием "MyRule" в учетной записи ContosoADL с диапазоном IP-адресов 127.0.0.1–127.0.0.2.

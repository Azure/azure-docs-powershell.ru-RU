# Remove-AzDataBoxEdgeDevice

## SYNOPSIS
Удаляет устройство Edge "Поле данных".

## СИНТАКСИС

### DeleteByNameParameterSet (по умолчанию)
```
Remove-AzDataBoxEdgeDevice [-ResourceGroupName] <String> [-Name] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzDataBoxEdgeDevice -ResourceId <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzDataBoxEdgeDevice -InputObject <PSDataBoxEdgeDevice> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При **наличии cmdlet Remove-AzDataBoxEdgeDevice** конфигурация устройства Edge box данных удаляется.
Обратите внимание, что устройство можно удалить только после того, как вы разместили заказ и не подготовили его к отправке.

Перед использованием этого [cmdlet](https://docs.microsoft.com/en-us/azure/databox-online/data-box-edge-return-device#delete-the-resource) сослаться на документацию об удалении ресурса

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Remove-AzDataBoxEdgeDevice ResourceGroupName resourceGroupName -Name deviceName
```

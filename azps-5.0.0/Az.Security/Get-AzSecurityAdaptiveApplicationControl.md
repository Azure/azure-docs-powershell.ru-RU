# Get-AzSecurityAdaptiveApplicationControl

## КРАТКИй обзор
Получает список групп виртуальных машин и серверов для подписки на элементы управления приложением.

## Максимальное

### SubscriptionScope (по умолчанию)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Адаптивные элементы управления приложениями автоматически рассчитываются центром безопасности Azure, используют этот командлет для получения списка ресурсов адаптивных элементов управления приложениями в области подписки.

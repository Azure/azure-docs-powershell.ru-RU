## SYNOPSIS

## ОПИСАНИЕ
Для создания календарного плана в автоматизации Azure создается календарный план **New-AzAutomationSchedule.**

## ПРИМЕРЫ

### Пример 1. Создание одно расписания в локальное время
```
PS C:\> $TimeZone = ([System.TimeZoneInfo]::Local).Id
PS C:\> New-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -StartTime "23:00" -OneTime -ResourceGroupName "ResourceGroup01" -TimeZone $TimeZone
```

Первая команда получает из системы и сохраняет его в переменной $TimeZone часовой пояс.
Вторая команда создает расписание, которое выполняется один раз к текущей дате в 23:00 в указанном часовом поясе.

### Пример 2. Создание повторяющегося расписания
```
PS C:\> $StartTime = Get-Date "13:00:00"
PS C:\> $EndTime = $StartTime.AddYears(1)
PS C:\> New-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule02" -StartTime $StartTime -ExpiryTime $EndTime -DayInterval 1 -ResourceGroupName "ResourceGroup01"
```

Первая команда создает объект даты с помощью командлета **Get-Date,** а затем сохраняет его в $StartDate переменной.
Укажите время, которое будет по крайней мере пять минут в будущем.
Вторая команда создает объект даты с помощью командлета **Get-Date,** а затем сохраняет его в $EndDate переменной.
Команда определяет будущее время.
Конечная команда создает ежедневный календарный план с именем Schedule02, который начинается с того времени, $StartDate и истекает в то время, $EndDate.

### Пример 3. Создание повторяющегося расписания на неделю
```
PS C:\> $StartTime = (Get-Date "13:00:00").AddDays(1)
PS C:\> [System.DayOfWeek[]]$WeekDays = @([System.DayOfWeek]::Monday..[System.DayOfWeek]::Friday)
PS C:\> New-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule03" -StartTime $StartTime - WeekInterval 1 -DaysOfWeek $WeekDays -ResourceGroupName "ResourceGroup01"
```

Первая команда создает объект даты с помощью командлета **Get-Date,** а затем сохраняет его в $StartDate переменной.
Вторая команда создает массив дней недели, содержащий понедельник, вторник, среду, четверг и пятницу.
Конечная команда создает ежедневное расписание "Расписание03", которое будет еженедельно с понедельника по пятницу в 13:00. Срок действия расписания не истечет.

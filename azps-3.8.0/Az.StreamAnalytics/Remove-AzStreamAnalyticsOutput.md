# Remove-AzStreamAnalyticsOutput

## КРАТКИй обзор
Удаляет вывод из задания Stream Analytics.

## Максимальное

```
Remove-AzStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzStreamAnalyticsOutput** асинхронно удаляет выходные данные из задания Stream Analytics в Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление выхода из задания
```
PS C:\>Remove-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Эта команда удаляет выходные данные в StreamingJob задания.

# New-AzPrivateDnsZone

## SYNOPSIS
Создает новую частную зону DNS.

## СИНТАКСИС

```
New-AzPrivateDnsZone -ResourceGroupName <String> -Name <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания новой зоны частного доменного имени (DNS) в указанной группе ресурсов будет указана новая зона **AzPrivateDnsZone.** Необходимо указать уникальное имя зоны DNS для параметра *Name,* иначе при этом будет возвращена ошибка. После создания зоны используйте New-AzPrivateDnsRecordSet для создания наборов записей в этой зоне.
Параметр Confirm  и переменная $ConfirmPreference Windows PowerShell можно использовать для управления запросом на подтверждение с помощью cmdlet.

## ПРИМЕРЫ

### Пример 1. Создание частной зоны DNS
```
PS C:\>$Zone = New-AzPrivateDnsZone -Name "myzone.com" -ResourceGroupName "MyResourceGroup"


This command creates a new private DNS zone named myzone.com in the specified resource group, and then
stores it in the $Zone variable. $Zone object looks something like this,

Name                          : myzone.com
ResourceId                    : "/subscriptions/xxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/MyResourceGroup/PrivateZones/myzone.com"
ResourceGroupName             : MyResourceGroup
Location                      : 
Etag                          : xxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
Tags                          : {}
NumberOfRecordSets            : 1
MaxNumberOfRecordSets         : 5000
```

# Stop-AzManagementGroupDeployment

## SYNOPSIS
Отмена развертывания в группе управления

## СИНТАКСИС

### StopByDeploymentName (по умолчанию)
```
Stop-AzManagementGroupDeployment [-ManagementGroupId] <String> [-Name] <String> [-PassThru] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzManagementGroupDeployment -Id <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzManagementGroupDeployment -InputObject <PSDeployment> [-PassThru] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Выполнение **команды Stop-AzManagementGroupDeployment** отменяет развертывание, которое началось, но не завершено в группе управления.
Чтобы завершить развертывание, необходимо иметь неполное состояние подготовка, например подготовка, а не завершалось состояние, например "Подготовка" или "Сбой".

Чтобы создать развертывание в группе управления, воспользуйтесь New-AzManagementGroupDeployment управления.

## ПРИМЕРЫ

### Пример 1
```
PS C:\>Stop-AzManagementGroupDeployment -ManagementGroupId "myMG" -Name "deployment01"
```

Эта команда отменяет развертывание "deployment01" в группе управления "myMG".

### Пример 2
```
PS C:\>Get-AzManagementGroupDeployment -ManagementGroupId "myMG" -Name "deployment01" | Stop-AzManagementGroupDeployment
```

Эта команда получает "deployment01" в группе управления MyMG и отменяет ее.

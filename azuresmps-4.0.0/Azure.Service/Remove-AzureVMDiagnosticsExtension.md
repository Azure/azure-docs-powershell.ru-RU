# Remove-AzureVMDiagnosticsExtension

## КРАТКИй обзор
Удаление расширения диагностики Azure из виртуальной машины.

## Максимальное

```
Remove-AzureVMDiagnosticsExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureVMDiagnosticsExtension** удаляет расширение диагностики Microsoft Azure из виртуальной машины.
Выходные данные этого командлета необходимо передать командлету **Update-AzureVM** .

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление расширения диагностики Azure из виртуальной машины
```
PS C:\> Remove-AzureVMDiagnosticsExtension -VM $VM | Update-AzureVM
```

Эта команда удаляет расширение диагностики Azure из виртуальной машины.

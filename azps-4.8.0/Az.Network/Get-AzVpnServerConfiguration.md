# Get-AzVpnServerConfiguration

## КРАТКИй обзор
Возвращает существующий VpnServerConfiguration для подключения к сайту.

## Максимальное

### ListBySubscriptionId (по умолчанию)
```
Get-AzVpnServerConfiguration [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ListByResourceGroupName
```
Get-AzVpnServerConfiguration [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzVpnServerConfiguration** возвращает существующий VpnServerConfiguration для подключения к сайту.

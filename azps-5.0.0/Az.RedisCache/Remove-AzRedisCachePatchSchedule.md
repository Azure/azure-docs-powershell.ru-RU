# Remove-AzRedisCachePatchSchedule

## КРАТКИй обзор
Удаление расписания исправлений.

## Максимальное

```
Remove-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzRedisCachePatchSchedule** Удаляет расписание исправлений из кэша в Azure Redis Cache.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление расписания исправлений
```
PS C:\>Remove-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Эта команда удаляет расписание исправлений из кэша с именем RedisCache06.

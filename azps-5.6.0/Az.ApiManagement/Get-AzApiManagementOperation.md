# Get-AzApiManagementOperation

## SYNOPSIS
Возвращает список или указанную операцию API.

## СИНТАКСИС

### GetAllApiOperations (по умолчанию)
```
Get-AzApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 -OperationId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Get-AzApiManagementOperation** получает список или указанную операцию API.

## ПРИМЕРЫ

### Пример 1. Получить все операции управления API
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOperation -Context $apimContext -ApiId $APIId
```

Эта команда получает все операции управления API.

### Пример 2. Получить операцию управления API по ИД операции
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOperation -Context $apimContext -ApiId $APIId -OperationId "Operation003"
```

Эта команда получает операцию управления API по ИД операции Operation0003.

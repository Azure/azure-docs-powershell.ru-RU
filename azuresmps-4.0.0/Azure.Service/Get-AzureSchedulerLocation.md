# Get-AzureSchedulerLocation

## КРАТКИй обзор
Получает доступные расположения планировщиков.

## Максимальное

```
Get-AzureSchedulerLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
В этом разделе описан командлет в версии 0.8.10 модуля Microsoft Azure PowerShell.
Чтобы получить версию модуля, который вы используете, введите в командной консоли Azure PowerShell `(Get-Module -Name Azure).Version` .

Командлет **Get-AzureSchedulerLocation** получает доступные расположения планировщиков.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение доступных папок планировщика
```
PS C:\> Get-AzureSchedulerLocation
```

Эта команда получает доступные расположения планировщиков.

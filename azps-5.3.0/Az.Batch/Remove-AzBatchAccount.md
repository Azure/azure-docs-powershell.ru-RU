# Remove-AzBatchAccount

## SYNOPSIS
Удаляет пакетную учетную запись.

## СИНТАКСИС

```
Remove-AzBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его использованием удаляется** учетная запись Azure Batch.
При этом будет предложено удалить учетную запись, если не указан параметр *Force.*

## ПРИМЕРЫ

### Пример 1. Удаление учетной записи пакета
```
PS C:\>Remove-AzBatchAccount -AccountName "pfuller"
```

Эта команда удаляет пакетную учетную запись с именем pfuller.
Эта команда запросит подтверждение перед удалением учетной записи.

# Модуль AZS. KeyVault. Админ
## Nописание
Предварительный просмотр выпуска модуля оператора AzureStack KeyVault, который позволяет операторам просматривать KeyVault квоты.

## Командлеты AZS. KeyVault. admin
### [Get-AzsKeyVaultQuota](Get-AzsKeyVaultQuota.md)
Получение списка всех объектов квот для KeyVault в местоположении.

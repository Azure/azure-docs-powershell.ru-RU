# Remove-AzApplicationGatewayPrivateLinkConfiguration

## SYNOPSIS
Удаляет конфигурацию PrivateLink из шлюза приложения.

## СИНТАКСИС

```
Remove-AzApplicationGatewayPrivateLinkConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его** использованием можно удалить конфигурацию PrivateLink из шлюза приложения Azure.

## ПРИМЕРЫ

### Пример 1. Удаление конфигурации PrivateLink шлюза приложения
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01"
```

Первая команда получает шлюз приложения и сохраняет его в переменной $AppGw.
Вторая команда удаляет конфигурацию PrivateLink с именем privateLinkConfig01 из шлюза приложения, $AppGw.

# Update-AzAutomationSourceControl

## КРАТКИй обзор
Обновляет элемент управления источником автоматизации Azure.

## Максимальное

```
Update-AzAutomationSourceControl -Name <String> [-AccessToken <SecureString>] [-FolderPath <String>]
 [-Branch <String>] [-Description <String>] [-AutoSync <Boolean>] [-PublishRunbook <Boolean>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Update-AzAutomationSourceControl изменяет значение поля в системе управления версиями в Azure Automation.

## ИЛЛЮСТРИРУЮТ

### Пример 1
Эти команды устанавливают в поле PublishRunbook значение false для элемента управления источником автоматизации Azure с именем VSTSNative в учетной записи с именем devAccount.


```powershell
Update-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                      -AutomationAccountName "devAccount" `
                                      -Name "VSTSNative" `
                                      -PublishRunbook $false

```

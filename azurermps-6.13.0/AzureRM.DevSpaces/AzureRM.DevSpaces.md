# Модуль AzureRM. DevSpaces
## Nописание
Команды для взаимодействия с контроллером Azure DevSpaces.

## Командлеты AzureRM. DevSpaces
### [Get-AzureRmDevSpacesController](Get-AzureRmDevSpacesController.md)
Получить или просмотреть контроллер DevSpaces Azure.

### [New-AzureRmDevSpacesController](New-AzureRmDevSpacesController.md)
Создайте новый контроллер DevSpaces Azure.

### [Remove-AzureRmDevSpacesController](Remove-AzureRmDevSpacesController.md)
Удалите контроллер DevSpaces.

### [Update-AzureRmDevSpacesController](Update-AzureRmDevSpacesController.md)
Обновите контроллер DevSpaces, чтобы добавить теги.

# Remove-AzIntegrationAccountSchema

## SYNOPSIS
Удаляет схему интеграции учетной записи.

## СИНТАКСИС

```
Remove-AzIntegrationAccountSchema -ResourceGroupName <String> -Name <String> -SchemaName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При **этом из группы** ресурсов удаляется схема учетной записи интеграции.
Укажите имя учетной записи интеграции, имя группы ресурсов и имя схемы.
Этот модуль поддерживает динамические параметры.
Чтобы использовать динамический параметр, введите его в команду.
Чтобы найти имена динамических параметров, введите дефис (-) после имени cmdlet, а затем нажимая клавишу TAB для циклическиго перебора доступных параметров.
Если параметр обязательного шаблона не задан, будет выдан запрос на его значение.

## ПРИМЕРЫ

### Пример 1. Удаление схемы учетной записи интеграции
```
PS C:\>Remove-AzIntegrationAccountSchema -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -SchemaName "IntegrationAccountSchema43"
```

Эта команда удаляет схему интеграции учетной записи IntegrationAccountSchema43.

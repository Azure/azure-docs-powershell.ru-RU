# Get-AzureVMDiagnosticsExtension

## КРАТКИй обзор
Получает параметры расширения диагностики Azure на виртуальной машине.

## Максимальное

```
Get-AzureVMDiagnosticsExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureVMDiagnosticsExtension** получает параметры расширения диагностики Microsoft Azure на виртуальной машине.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение расширения диагностики, примененного к виртуальной машине
```
PS C:\> Get-AzureVMDiagnosticsExtension -VM $VM
```

Эта команда получает расширение диагностики, примененное к указанной виртуальной машине, в соответствии с переменной $VM.

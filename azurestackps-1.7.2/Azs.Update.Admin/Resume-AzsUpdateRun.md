# Resume-AzsUpdateRun

## КРАТКИй обзор
Возобновляет Запущенный ранее прогон обновления, который завершился сбоем.

## Максимальное

### UpdateRuns_Rerun (по умолчанию)
```
Resume-AzsUpdateRun -Name <String> [-Location <String>] [-ResourceGroupName <String>] -UpdateName <String>
 [-AsJob] [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Resume-AzsUpdateRun [-AsJob] -ResourceId <String> [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Возобновляет Запущенный ранее прогон обновления, который завершился сбоем. Возобновление прогонов обновления возобновится на момент последнего сбоя.

## ИЛЛЮСТРИРУЮТ

### --------------------------ПРИМЕР 1--------------------------
```
Get-AzsUpdateRun -Name 5173e9f4-3040-494f-b7a7-738a6331d55c -UpdateName Microsoft1.0.180305.1 | Resume-AzsUpdateRun
```

Возобновляет Запущенный ранее прогон обновления, который завершился сбоем.

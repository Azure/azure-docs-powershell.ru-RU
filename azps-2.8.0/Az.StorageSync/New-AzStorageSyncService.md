# New-AzStorageSyncService

## КРАТКИй обзор
Эта команда создает новую службу синхронизации хранилища в группе ресурсов.

## Максимальное

```
New-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Служба синхронизации хранилища — это ресурс верхнего уровня для синхронизации файлов Azure. Эта команда создает новую службу синхронизации хранилища в группе ресурсов. Для различения отдельных групп серверов в организации рекомендуется создать как можно меньше служб синхронизации хранилища. Служба синхронизации хранилища включает группы синхронизации, а также работает как целевой объект для регистрации серверов. Сервер может быть зарегистрирован только в одной службе синхронизации хранилища. Если нужно, чтобы серверы использовали один и тот же набор файлов, зарегистрируйте их в одной и той же службе синхронизации хранилища.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzStorageSyncService -ResourceGroupName "myResourceGroup" -Location "myLocation" -StorageSyncServiceName "myStorageSyncServiceName"
```

Эта команда создаст службу синхронизации хранилища.

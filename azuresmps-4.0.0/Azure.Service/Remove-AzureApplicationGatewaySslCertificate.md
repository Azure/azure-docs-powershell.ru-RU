# Remove-AzureApplicationGatewaySslCertificate

## КРАТКИй обзор
Удаление SSL-сертификата из шлюза приложений.

## Максимальное

```
Remove-AzureApplicationGatewaySslCertificate -Name <String> -CertificateName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureApplicationGatewaySslCertificate** УДАЛЯЕТ сертификат SSL (Secure Sockets Layer) из шлюза приложений.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление SSL-сертификата
```
PS C:\> Remove-AzureApplicationGatewaySslCertificate -Name "ApplicationGateway08" -CertificateName "SslCertificate13"
```

Эта команда удаляет SSL-сертификат с именем SslCertificate13 с шлюза приложений с именем ApplicationGateway08.

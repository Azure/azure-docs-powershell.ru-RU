# Get-AzureRmStreamAnalyticsInput

## КРАТКИй обзор
Возвращает входные данные для задания Stream Analytics.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmStreamAnalyticsInput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmStreamAnalyticsInput** перечисляет все входные данные, определенные в задании Stream Analytics, или получает сведения о конкретных входных данных.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение сведений о входных данных, определенных для задания
```
PS C:\>Get-AzureRmStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

Эта команда возвращает сведения обо всех входных данных, определенных для задания StreamingJob.

### Пример 2: получение сведений о конкретном вводе, определенном для задания
```
PS C:\>Get-AzureRmStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Эта команда возвращает сведения о входных данных с именем EntryStream, определенных в задании StreamingJob.

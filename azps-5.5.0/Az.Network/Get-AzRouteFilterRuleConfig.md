# Get-AzRouteFilterRuleConfig

## SYNOPSIS
Получает правило фильтрации маршрутов в фильтре маршрутов.

## СИНТАКСИС

```
Get-AzRouteFilterRuleConfig [-Name <String>] -RouteFilter <PSRouteFilter>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения правила фильтрации маршрутов или списка правил фильтрации маршрутов в фильтре маршрутов возвращается cmdlet **Get-AzRouteFilterRuleConfig.**

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "MyRouteFilter" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf
```

Первая команда получает фильтр маршрутов MyRouteFilter, а затем сохраняет его в переменной $rf.
Вторая команда получает правило фильтра маршрутов с именем "Правило01", связанное с этим фильтром маршрутов.
Третья команда получает список правил фильтрации маршрутов, связанных с этим фильтром.

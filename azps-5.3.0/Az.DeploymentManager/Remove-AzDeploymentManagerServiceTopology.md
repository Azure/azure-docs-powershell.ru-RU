# Remove-AzDeploymentManagerServiceTopology

## SYNOPSIS
Удаляет топологию службы. Перед удалением топологии службы необходимо удалить все службы, созданные в топологии службы.

## СИНТАКСИС

### Интерактивный (по умолчанию)
```
Remove-AzDeploymentManagerServiceTopology [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerServiceTopology [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerServiceTopology [-InputObject] <PSServiceTopologyResource> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **удаления топологии службы** будет удалена топология службы.

Укажите топологию службы по имени и имени группы ресурсов. Кроме того, у вас может быть объект ServiceTopology или ResourceId.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Remove-AzDeploymentManagerServiceTopology -ResourceGroupName ContosoResourceGroup -Name ContosoServiceTopology
```

Эта команда удаляет топологию службы ContosoServiceTopology в contosoResourceGroup.

### Пример 2. Удаление топологии службы с помощью идентификатора ресурса.
```powershell
PS C:\> Remove-AzDeploymentManagerServiceTopology -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/serviceTopologies/ContosoServiceTopology"
```

Эта команда удаляет топологию службы ContosoServiceTopology в contosoResourceGroup.

### Пример 3. Удаление топологии службы с помощью объекта топологии службы.
```powershell
PS C:\> Remove-AzDeploymentManagerService -InputObject $serviceTopologyObject
```

Эта команда удаляет топологию службы, имя и группа ресурсов которой соответствуют свойствам Name и ResourceGroupName соответствующего $serviceTopologyObject службы.

# Get-AzSqlElasticPool

## КРАТКИй обзор
Возвращает эластичные пулы и значения их свойств в базе данных Azure SQL.

## Максимальное

```
Get-AzSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlElasticPool** получает эластичные пулы и значения их свойств.
Укажите имя существующего эластичного пула, чтобы просмотреть значения свойств только для этого пула.

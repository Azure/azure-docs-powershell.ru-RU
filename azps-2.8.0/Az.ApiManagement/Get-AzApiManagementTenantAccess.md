# Get-AzApiManagementTenantAccess

## КРАТКИй обзор
Получает конфигурацию доступа для клиента.

## Максимальное

```
Get-AzApiManagementTenantAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzApiManagementTenantAccess** получает конфигурацию доступа клиента для клиента.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение конфигурации доступа клиента
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccess -Context $apimContext
```

Эта команда получает конфигурацию доступа клиента для указанного контекста.

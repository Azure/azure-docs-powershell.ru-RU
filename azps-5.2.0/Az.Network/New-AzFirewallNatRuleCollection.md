# New-AzFirewallNatRuleCollection

## SYNOPSIS
Создает набор правил NAT брандмауэра.

## СИНТАКСИС

```
New-AzFirewallNatRuleCollection -Name <String> -Priority <UInt32> -Rule <PSAzureFirewallNatRule[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet New-AzFirewallNatRuleCollection** создается коллекция правил NAT брандмауэра.

## ПРИМЕРЫ

### Пример 1. Создание коллекции с одним правилом
```powershell
$rule1 = New-AzFirewallNatRule -Name "natRule" -Protocol "TCP" -SourceAddress "*" -DestinationAddress "10.0.0.1" -DestinationPort "80" -TranslatedAddress "10.0.0.2" -TranslatedPort "8080"
New-AzFirewallNatRuleCollection -Name "MyNatRuleCollection" -Priority 1000 -Rule $rule1
```

В этом примере создается коллекция с одним правилом. Весь трафик, который соответствует условиям, зазначимых $rule 1, будет переведен на переведенный адрес и порт.

### Пример 2. Добавление правила в коллекцию правил
```powershell
$rule1 = New-AzFirewallNatRule -Name R1 -Protocol "UDP","TCP" -SourceAddress "*" -DestinationAddress "10.0.0.1" -DestinationPort "80" -TranslatedAddress "10.0.0.2" -TranslatedPort "8080"
$ruleCollection = New-AzFirewallNatRuleCollection -Name "MyNatRuleCollection" -Priority 100 -Rule $rule1

$rule2 = New-AzFirewallNatRule -Name R2 -Protocol "TCP" -SourceAddress "*" -DestinationAddress "10.0.0.1" -DestinationPort "443" -TranslatedAddress "10.0.0.2" -TranslatedPort "8443"
$ruleCollection.AddRule($rule2)
```

В этом примере создается новое коллекцию правил NAT с одним правилом, а затем добавляется второе правило к коллекции правил с использованием метода AddRule для объекта коллекции правил. Каждое имя правила в данном наборе правил должно иметь уникальное имя без учета case.

### Пример 3. Получите правило из коллекции правил
```powershell
$rule1 = New-AzFirewallNatRule -Name R1 -Protocol "TCP" -SourceAddress "10.0.0.0/24" -DestinationAddress "10.0.1.0/24" -DestinationPort "443" -TranslatedAddress "10.0.0.2" -TranslatedPort "8443"
$ruleCollection = New-AzFirewallNatRuleCollection -Name "MyNatRuleCollection" -Priority 100 -Rule $rule1

$rule=$ruleCollection.GetRuleByName("r1")
```

В этом примере создается коллекция правил NAT с одним правилом, которое затем получает правило по имени, методу вызова GetRuleByName для объекта коллекции правил. Имя правила для метода GetRuleByName нечувствительно к делу.

### Пример 4. Удаление правила из коллекции правил
```powershell
$rule1 = New-AzFirewallNatRule -Name R1 -Protocol "UDP","TCP" -SourceAddress "*" -DestinationAddress "10.0.0.1" -DestinationPort "80" -TranslatedAddress "10.0.0.2" -TranslatedPort "8080"
$rule2 = New-AzFirewallNatRule -Name R2 -Protocol "TCP" -SourceAddress "*" -DestinationAddress "10.0.0.1" -DestinationPort "443" -TranslatedAddress "10.0.0.2" -TranslatedPort "8443"
$ruleCollection = New-AzFirewallNatRuleCollection -Name "MyNatRuleCollection" -Priority 100 -Rule $rule1, $rule2
$ruleCollection.RemoveRuleByName("r1")
```

В этом примере создается новое коллекцию правил NAT с двумя правилами, а затем удаляется первое правило из коллекции правил путем вызова метода RemoveRuleByName для объекта коллекции правил. Имя правила для метода RemoveRuleByName нечувствительно к делу.

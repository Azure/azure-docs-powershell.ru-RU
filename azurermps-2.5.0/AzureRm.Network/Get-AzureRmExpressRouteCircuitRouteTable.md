# Get-AzureRmExpressRouteCircuitRouteTable

## КРАТКИй обзор
Возвращает таблицу маршрутов из канала ExpressRoute.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmExpressRouteCircuitRouteTable -ResourceGroupName <String> -ExpressRouteCircuitName <String>
 [-PeeringType <String>] -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmExpressRouteCircuitRouteTable** извлекает детальную таблицу маршрутов канала ExpressRoute. В таблице маршрутов отобразятся все маршруты или их можно отфильтровать, чтобы показать маршруты для определенного типа пиринга. Вы можете использовать таблицу маршрутов для проверки конфигурации и подключения к одноранговым данным.

## ИЛЛЮСТРИРУЮТ

### Пример 1: отображение таблицы маршрутов для основного пути
```
Get-AzureRmExpressRouteCircuitRouteTable -ResourceGroupName $RG -ExpressRouteCircuitName $CircuitName -DevicePath 'Primary'
```

# Remove-AzureVMSqlServerExtension

## КРАТКИй обзор
Удаляет расширение SQL Server виртуальной машины Azure из объекта виртуальной машины.

## Максимальное

```
Remove-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureVMSqlServerExtension** УДАЛЯЕТ расширение SQL Server виртуальной машины Azure из объекта виртуальной машины.

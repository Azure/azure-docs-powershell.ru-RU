# Stop-AzureRmSqlSyncGroupSync

## КРАТКИй обзор
Останавливает синхронизацию группы синхронизации.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Stop-AzureRmSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Stop-AzureRmSqlSyncGroupSync** останавливает синхронизацию группы синхронизации.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Stop-AzureRmSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Эта команда останавливает синхронизацию, которая выполняется для группы синхронизации mysg.

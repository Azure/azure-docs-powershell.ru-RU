# Remove-AzApplicationGatewayPrivateLinkConfiguration

## КРАТКИй обзор
Удаляет конфигурацию privateLink из шлюза приложений.

## Максимальное

```
Remove-AzApplicationGatewayPrivateLinkConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApplicationGatewayPrivateLinkConfiguration** удаляет конфигурацию privateLink из шлюза приложений Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление конфигурации PrivateLink шлюза приложений
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01"
```

Первая команда получает шлюз приложения и сохраняет его в переменной $AppGw.
Вторая команда удаляет конфигурацию privateLink с именем privateLinkConfig01 из шлюза приложения, который хранится в $AppGw.

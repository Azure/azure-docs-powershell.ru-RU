# Get-AzSqlDatabaseTransparentDataEncryptionActivity

## SYNOPSIS
Возвращает ход сканирования базы данных по TDE.

## СИНТАКСИС

```
Get-AzSqlDatabaseTransparentDataEncryptionActivity [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для выполнения проверки прозрачного шифрования данных (TDE) в базе данных Azure SQL используется **cmdlet Get-AzSqlDataTransparentDataEncryptionActivity.**
Если диапазон шифрования не используется, этот cmdlet возвращает пустой список.
Дополнительные сведения см. в теме "Прозрачное шифрование данных с помощью SQL Azure "База данных" https://msdn.microsoft.com/library/dn948096 https://msdn.microsoft.com/library/dn948096) (Microsoft Developer Network библиотеки).
Этот cmdlet также поддерживается службой SQL Server Stretch Database в Azure.

## ПРИМЕРЫ

### Пример 1. Получить действие TDE для базы данных
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryptionActivity -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName : resourcegroup01
ServerName        : server01
DatabaseName      : database01
Status            : Encrypting
PercentComplete   : 3.662109
```

Эта команда возвращает действие TDE для базы данных database01 на сервере с именем server01.

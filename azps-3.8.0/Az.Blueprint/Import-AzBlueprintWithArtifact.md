# Import-AzBlueprintWithArtifact

## КРАТКИй обзор
Импортируйте файл в формате JSON в объект чертежа и сохраните его в указанной подписке или группе управления.

## Максимальное

```
Import-AzBlueprintWithArtifact -Name <String> [-SubscriptionId <String>] [-ManagementGroupId <String>]
 -InputPath <String> [-Force] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Импортируйте определение чертежа с его артефактами. 

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Import-AzBlueprintWithArtifact -Name MySimpleBlueprint -SubscriptionId 00000000-1111-0000-1111-000000000000 -InputPath  C:\Blueprints\SimpleBlueprint
```

Импортируйте определение чертежа с артефактами и сохраните его в рамках подписки.

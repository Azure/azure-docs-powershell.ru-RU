# Get-AzApiManagementNetworkStatus

## SYNOPSIS
Возвращает состояние подключения к внешним ресурсам, от которых зависит служба управления API в облачной службе. Это также возвращает DNS-серверы, видимые облачной службе.

## СИНТАКСИС

### ByInputObject (по умолчанию)
```
Get-AzApiManagementNetworkStatus -ApiManagementObject <PsApiManagement> [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExpandedParameter
```
Get-AzApiManagementNetworkStatus -ResourceGroupName <String> -Name <String> [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Состояние сети службы управления Api

# Get-AzFrontDoorRulesEngine

## КРАТКИй обзор
Получение конфигураций обработчика правил.

## Максимальное

```
Get-AzFrontDoorRulesEngine -ResourceGroupName <String> -FrontDoorName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzFrontDoorRulesEngine** возвращает определенную конфигурацию обработчика правил или получает все конфигурации обработчика правил, связанные с передней дверцей. 

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name rulesengine3

Name         RulesEngineRules
----         ----------------
rulesEngine3 {rules1}
```

Получить определенную конфигурацию обработчика правил.

### Пример 2

```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName

Name         RulesEngineRules
----         ----------------
rulesEngine1 {Rule1}
rulesEngine2 {Rule1}
rulesEngine3 {rules1}
```

Получение всех конфигураций обработчика правил на передней дверце.

### Пример 3

# Get-AzDataMigrationService

## SYNOPSIS
Извлекает свойства, связанные с экземпляром службы миграции баз данных Azure. 

## СИНТАКСИС

### ResourceGroupSet (по умолчанию)
```
Get-AzDataMigrationService [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzDataMigrationService [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ServiceNameGroupSet
```
Get-AzDataMigrationService [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С Get-AzDataMigrationService- именем группы ресурсов Azure в качестве входных параметров извлекает свойства, связанные с экземпляром службы миграции баз данных Azure. 

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Get-AzDataMigrationService -ResourceGroupName testResourceGroup -Name testService
```

В примере выше извлекались свойства экземпляра службы миграции баз данных Azure, который называется testService. 

### Пример 2
```
PS C:\> Get-AzDataMigrationService -ResourceGroupName testResourceGroup
```

В примере выше извлекаются службы миграции баз данных Azure в группе ресурсов testResourceGroup.

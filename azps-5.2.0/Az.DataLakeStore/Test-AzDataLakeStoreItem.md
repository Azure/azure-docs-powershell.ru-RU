# Test-AzDataLakeStoreItem

## SYNOPSIS
Проверяет, существует ли файл или папка в Data Lake Store.

## СИНТАКСИС

```
Test-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance> [[-PathType] <PathType>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Test-AzDataLakeStoreItem** проверяет наличие файлов или папок в магазине Data Lake Store.

## ПРИМЕРЫ

### Пример 1. Проверка файла
```powershell
PS C:\>Test-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/MyFiles/Test.csv"
```

Эта команда проверяет, существует ли Test.csv в учетной записи ContosoADL.

### Пример 2

Проверяет, существует ли файл или папка в Data Lake Store. (autogenerated)

<!-- Aladdin Generated Example -->
```powershell
Test-AzDataLakeStoreItem -Account 'ContosoADL' -Path '/MyFiles/Test.csv' -PathType Any
```

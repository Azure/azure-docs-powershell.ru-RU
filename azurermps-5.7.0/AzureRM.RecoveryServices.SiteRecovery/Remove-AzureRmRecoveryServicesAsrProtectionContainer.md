# Remove-AzureRmRecoveryServicesAsrProtectionContainer

## КРАТКИй обзор
Удаляет указанный контейнер защиты из его структуры.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmRecoveryServicesAsrProtectionContainer -InputObject <ASRProtectionContainer>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzureRmRecoveryServicesAsrProtectionContainer удаляет указанный контейнер защиты Azure Site Recovery.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Get-AzureRmRecoveryServicesAsrProtectionContainer -Name xxxxx  -Fabric $fabric
PS C:\> Remove-AzureRmRecoveryServicesAsrProtectionContainer -InputObject $protectionContainer
```

Запускает удаление указанного контейнера защиты и возвращает задание ASR, используемое для отслеживания операции удаления.

# Модуль Az.DataLakeStore
## Описание
В разделах этой статьи хранятся командылеты Azure PowerShell для Azure Data Lake Store в платформе Azure Resource Manager (ARM). Командлеты существуют в области имен Microsoft.Azure.Commands.DataLakeStore. Эти cmdlets работают только с учетными записьми Azure Data Lake Storage Gen1.

## Az.DataLakeStore Cmdlets
### [Add-AzDataLakeStoreFirewallRule](Add-AzDataLakeStoreFirewallRule.md)
Добавляет правило брандмауэра к указанной учетной записи Data Lake Store.

### [Add-AzDataLakeStoreItemContent](Add-AzDataLakeStoreItemContent.md)
Добавляет содержимое к элементу в магазине Data Lake Store.

### [Add-AzDataLakeStoreTrustedIdProvider](Add-AzDataLakeStoreTrustedIdProvider.md)
Добавляет надежного поставщика удостоверений в указанную учетную запись Data Lake Store.

### [Add-AzDataLakeStoreVirtualNetworkRule](Add-AzDataLakeStoreVirtualNetworkRule.md)
Добавляет виртуальное правило сети в указанную учетную запись Data Lake Store.

### [Enable-AzDataLakeStoreKeyVault](Enable-AzDataLakeStoreKeyVault.md)
Попытка включить управляемый пользователем key Vault для шифрования указанной учетной записи Data Lake Store.

### [Export-AzDataLakeStoreChildItemProperty](Export-AzDataLakeStoreChildItemProperty.md)
Экспорт свойств (использование диска и ACL) для всего дерева из указанного пути на выходной путь.

### [Export-AzDataLakeStoreItem](Export-AzDataLakeStoreItem.md)
Скачивает файл из Data Lake Store.

### [Get-AzDataLakeStoreAccount](Get-AzDataLakeStoreAccount.md)
Сведения об учетной записи Data Lake Store.

### [Get-AzDataLakeStoreChildItem](Get-AzDataLakeStoreChildItem.md)
Получает список элементов в папке в Data Lake Store.

### [Get-AzDataLakeStoreChildItemSummary](Get-AzDataLakeStoreChildItemSummary.md)
Возвращает сводку общего размера, файлов и каталогов, содержащихся в указанном пути.

### [Get-AzDataLakeStoreFirewallRule](Get-AzDataLakeStoreFirewallRule.md)
Получает указанные правила брандмауэра в указанном магазине data Lake Store.
Если правило брандмауэра не задано, будут перечислены все правила брандмауэра для учетной записи.

### [Get-AzDataLakeStoreItem](Get-AzDataLakeStoreItem.md)
Получает сведения о файле или папке в Data Lake Store.

### [Get-AzDataLakeStoreItemAclEntry](Get-AzDataLakeStoreItemAclEntry.md)
Возвращает запись в ACL файла или папки в магазине Data Lake Store.

### [Get-AzDataLakeStoreItemContent](Get-AzDataLakeStoreItemContent.md)
Получает содержимое файла в магазине Data Lake Store.

### [Get-AzDataLakeStoreItemOwner](Get-AzDataLakeStoreItemOwner.md)
Получает владельца файла или папки в Data Lake Store.

### [Get-AzDataLakeStoreItemPermission](Get-AzDataLakeStoreItemPermission.md)
Получает разрешение восьмеричего разрешения для файла или папки в Data Lake Store.

### [Get-AzDataLakeStoreTrustedIdProvider](Get-AzDataLakeStoreTrustedIdProvider.md)
Получает указанного надежного поставщика удостоверений в указанном Data Lake Store.
Если не указан поставщик, перечислены все поставщики для учетной записи.

### [Get-AzDataLakeStoreVirtualNetworkRule](Get-AzDataLakeStoreVirtualNetworkRule.md)
Получает указанные правила виртуальной сети в указанном магазине для озера Data Lake Store.
Если виртуальное правило сети не задано, будут перечислены все правила виртуальной сети для учетной записи.

### [Import-AzDataLakeStoreItem](Import-AzDataLakeStoreItem.md)
Отправка локального файла или каталога в Data Lake Store.

### [Join-AzDataLakeStoreItem](Join-AzDataLakeStoreItem.md)
Соединяет один или несколько файлов, чтобы создать один файл в Data Lake Store.

### [Move-AzDataLakeStoreItem](Move-AzDataLakeStoreItem.md)
Перемещает или переименовыв файл или папку в Data Lake Store.

### [New-AzDataLakeStoreAccount](New-AzDataLakeStoreAccount.md)
Создает новую учетную запись Data Lake Store.

### [New-AzDataLakeStoreItem](New-AzDataLakeStoreItem.md)
Создает файл или папку в магазине Data Lake Store.

### [Remove-AzDataLakeStoreAccount](Remove-AzDataLakeStoreAccount.md)
Удаление учетной записи Data Lake Store навсегда.

### [Remove-AzDataLakeStoreFirewallRule](Remove-AzDataLakeStoreFirewallRule.md)
Удаляет указанное правило брандмауэра для указанного магазина Data Lake Store.

### [Remove-AzDataLakeStoreItem](Remove-AzDataLakeStoreItem.md)
Удаляет файл или папку из магазина Data Lake Store.

### [Remove-AzDataLakeStoreItemAcl](Remove-AzDataLakeStoreItemAcl.md)
Очищает ACL файла или папки в Data Lake Store.

### [Remove-AzDataLakeStoreItemAclEntry](Remove-AzDataLakeStoreItemAclEntry.md)
Удаляет запись из ACL файла или папки в Data Lake Store.

### [Remove-AzDataLakeStoreTrustedIdProvider](Remove-AzDataLakeStoreTrustedIdProvider.md)
Удаление указанного надежного поставщика удостоверений из указанного магазина Data Lake Store.

### [Remove-AzDataLakeStoreVirtualNetworkRule](Remove-AzDataLakeStoreVirtualNetworkRule.md)
Удаляет указанное правило виртуальной сети из указанного магазина Data Lake Store.

### [Set-AzDataLakeStoreAccount](Set-AzDataLakeStoreAccount.md)
Изменяет учетную запись Data Lake Store.

### [Set-AzDataLakeStoreFirewallRule](Set-AzDataLakeStoreFirewallRule.md)
Изменяет указанное правило брандмауэра в указанном хранилище данных.

### [Set-AzDataLakeStoreItemAcl](Set-AzDataLakeStoreItemAcl.md)
Изменяет ACL файла или папки в магазине Data Lake Store.

### [Set-AzDataLakeStoreItemAclEntry](Set-AzDataLakeStoreItemAclEntry.md)
Изменяет запись в ACL файла или папки в Data Lake Store.

### [Set-AzDataLakeStoreItemExpiry](Set-AzDataLakeStoreItemExpiry.md)
Задает или удаляет срок действия файла в учетной записи Azure Data Lake Store.

### [Set-AzDataLakeStoreItemOwner](Set-AzDataLakeStoreItemOwner.md)
Изменяет владельца файла или папки в магазине Data Lake Store.

### [Set-AzDataLakeStoreItemPermission](Set-AzDataLakeStoreItemPermission.md)
Изменяет разрешение восьмеричего разрешения для файла или папки в Data Lake Store.

### [Set-AzDataLakeStoreTrustedIdProvider](Set-AzDataLakeStoreTrustedIdProvider.md)
Изменяет указанного надежного поставщика удостоверений в указанном Data Lake Store.

### [Set-AzDataLakeStoreVirtualNetworkRule](Set-AzDataLakeStoreVirtualNetworkRule.md)
Изменяет указанное правило виртуальной сети в указанном хранилище данных.

### [Test-AzDataLakeStoreAccount](Test-AzDataLakeStoreAccount.md)
Проверяет наличие учетной записи Data Lake Store.

### [Test-AzDataLakeStoreItem](Test-AzDataLakeStoreItem.md)
Проверяет, существует ли файл или папка в Data Lake Store.

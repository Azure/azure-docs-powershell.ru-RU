# New-AzureRmBatchApplication

## КРАТКИй обзор
Добавляет приложение в указанную учетную запись пакета.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationId] <String>
 [[-AllowUpdates] <Boolean>] [[-DisplayName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmBatchApplication** добавляет приложение в указанную учетную запись пакета Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Добавление пустого приложения в пакетную учетную запись
```
PS C:\>New-AzureRmBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "Litware" -AllowUpdates $True -DisplayName "Litware Advanced Reticulator"
```

Эта команда создает приложение беседа Litware в учетной записи ContosoBatch.
Приложение изначально не включает пакеты.

# New-AzureRmUserAssignedIdentity

## КРАТКИй обзор
Создание нового пользователя, которому назначено удостоверение, или обновление существующего удостоверения, назначенного пользователю.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
С помощью командлета **New-AzureRmUserAssignedIdentity** создается новый пользователь, которому назначено удостоверение. При использовании с уже существующим удостоверением оно обновило удостоверение.
Чтобы добавить теги диспетчера ресурсов Azure в удостоверение, используйте командлет Set-AzureRmResource.

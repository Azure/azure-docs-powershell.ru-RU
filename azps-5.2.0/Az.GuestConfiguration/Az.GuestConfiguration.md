# Модуль Az.GuestConfiguration
## Описание
Модуль с командлетами, позволяющими пользователям понять состояние соответствия требованиям своих виртуальных машин в Azure на основе результатов политики Azure.

## Az.GuestConfiguration Cmdlets
### [Get-AzVMGuestPolicyStatus](Get-AzVMGuestPolicyStatus.md)
Получает статусы политики настройки гостей (подробный) для инициативы типа "Гостевая конфигурация", которая назначена VM.
Инициатива — это политика типа определения "Инициатива".

### [Get-AzVMGuestPolicyStatusHistory](Get-AzVMGuestPolicyStatusHistory.md)
Возвращает историю состояния соответствия политики гостевой конфигурации инициативе типа "Гостевая конфигурация", которая назначена VM.
Инициатива — это политика типа определения "Инициатива".

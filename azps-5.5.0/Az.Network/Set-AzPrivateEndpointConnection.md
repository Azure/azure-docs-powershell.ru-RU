# Set-AzPrivateEndpointConnection

## SYNOPSIS
Обновляет состояние подключения частной конечной точки в службе частных ссылок.

## СИНТАКСИС

### ByResourceId (по умолчанию)
```
Set-AzPrivateEndpointConnection -ResourceId <String>
 -PrivateLinkServiceConnectionState <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResource
```
Set-AzPrivateEndpointConnection -Name <String> -ServiceName <String> -ResourceGroupName <String> [-PrivateLinkResourceType <String>]
 -PrivateLinkServiceConnectionState <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Set-AzPrivateEndpointConnection** обновляет состояние подключения частной конечной точки в частной службе связывания

## ПРИМЕРЫ

### Пример 1
```
Set-AzPrivateEndpointConnection -Name TestPrivateEndpointConnection -ResourceGroupName TestResourceGroup -ServiceName TestPrivateLinkService -PrivateLinkServiceConnectionState "Approved"
```

В этом примере состояние подключения частной конечной точки обновляется на "Утверждено".

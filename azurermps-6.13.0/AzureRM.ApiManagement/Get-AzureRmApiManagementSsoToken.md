# Get-AzureRmApiManagementSsoToken

## КРАТКИй обзор
Возвращает ссылку с маркером единого входа на развернутый портал управления в службе управления API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmApiManagementSsoToken -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmApiManagementSsoToken** возвращает ссылку (URL-адрес), содержащую маркер единого входа (SSO), на развернутый портал управления для службы управления API.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение маркера единого входа для службы управления API
```
PS C:\>Get-AzureRmApiManagementSsoToken -ResourceGroupName "Contoso" -Name "ContosoApi"
```

Эта команда получает маркер SSO службы управления API.

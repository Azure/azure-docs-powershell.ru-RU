# Stop-AzVM

## SYNOPSIS
Остановка виртуальной машины Azure.

## СИНТАКСИС

### ResourceGroupNameParameterSetName (по умолчанию)
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-ResourceGroupName] <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Stop-AzVM [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-Id] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для остановки виртуальной машины Azure будет отсючена **cmdlet Stop-AzVM.**

## ПРИМЕРЫ

### Пример 1. Остановка виртуальной машины
```
PS C:\> Stop-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Эта команда останавливает виртуальную машину с именем VirtualMachine07 в ResourceGroup11.

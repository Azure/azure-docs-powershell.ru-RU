# Get-AzureVirtualNetworkGatewayKey

## КРАТКИй обзор
Получает ключ для шлюза виртуальной сети Azure.

## Максимальное

```
Get-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureVirtualNetworkGatewayKey** получает ключ для шлюза виртуальной сети Azure.

# Invoke-AzIotHubDeploymentMetricsQuery

## SYNOPSIS
Вызов запроса метрики развертывания IoT Edge.

## СИНТАКСИС

### Набор ресурсов (по умолчанию)
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 -MetricName <String> [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-InputObject] <PSIotHub> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceId] <String> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Оценка целевой настраиваемой или системной метрик, определенной в развертывании IoT Edge.
Существуют предопределяемые системные показатели, которые рассчитываются с помощью IOT-концентратора и не могут быть настроены.
- Targeted (Целевой) — это устройства IoT Edge, которые соответствуют условию целевого развертывания.
- "Применено" показывает целевые устройства IoT Edge, которые не являются целевыми при другом развертывании с более высоким приоритетом.
- "Отчет об успешном развертывании" — это устройства IoT Edge, которые сообщили об успешном развертывании модулей.
- "Сбой отчетов" — это устройства IoT Edge, которые сообщили о том, что один или несколько модулей не были развернуты. 
  Чтобы изучить ошибку, подключите ее удаленно к этим устройствам и изучите файлы журнала.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Invoke-AzIotHubDeploymentMetricsQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "warningLimit"
```

Оцените настраиваемую метрику warningLimit.

### Пример 2
```powershell
PS C:\> Invoke-AzIotHubDeployMetric -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "Reporting Success" -MetricType "system"
```

Оценка успешности системы.

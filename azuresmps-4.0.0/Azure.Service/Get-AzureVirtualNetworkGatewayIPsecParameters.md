# Get-AzureVirtualNetworkGatewayIPsecParameters

## КРАТКИй обзор
Получает параметры IPsec для шлюза виртуальной сети Azure.

## Максимальное

```
Get-AzureVirtualNetworkGatewayIPsecParameters -GatewayId <String> -ConnectedEntityId <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureVirtualNetworkGatewayIPsecParameters** получает параметры IPsec для шлюза виртуальной сети Azure.

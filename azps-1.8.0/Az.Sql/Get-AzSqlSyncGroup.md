# Get-AzSqlSyncGroup

## КРАТКИй обзор
Возвращает сведения о группах синхронизации базы данных Azure SQL.

## Максимальное

```
Get-AzSqlSyncGroup [[-Name] <String>] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSqlSyncGroup** возвращает сведения о одной или нескольких группах синхронизации базы данных Azure SQL.
Укажите имя группы синхронизации, чтобы просмотреть сведения только для этой группы синхронизации.

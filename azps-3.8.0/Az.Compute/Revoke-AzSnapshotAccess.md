# Revoke-AzSnapshotAccess

## КРАТКИй обзор
Отменяет доступ к снимку.

## Максимальное

```
Revoke-AzSnapshotAccess [-ResourceGroupName] <String> [-SnapshotName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **REVOKE-AzSnapshotAccess** отменяет доступ к моментальному снимку.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Revoke-AzSnapshotAccess -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01'
```

Отмена доступа к моментальному снимку с именем "Snapshot01" в группе ресурсов с именем "ResourceGroup01"

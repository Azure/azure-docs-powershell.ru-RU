# Get-AzVmssSku

## SYNOPSIS
Возвращает доступные skus для VMSS.

## СИНТАКСИС

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **набора виртуальных** машин (VMSS) доступны коды SKKU, доступные для этого набора.

## ПРИМЕРЫ

### Пример 1. Получить все доступные skUs из VMSS
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Эта команда получает все доступные skUs из VMSS ContosoVMSS, которые принадлежат группе ресурсов ContosoGroup.

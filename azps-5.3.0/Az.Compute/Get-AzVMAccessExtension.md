# Get-AzVMAccessExtension

## SYNOPSIS
Сведения о расширении VMAccess.

## СИНТАКСИС

```
Get-AzVMAccessExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzVMAccessExtension** получает сведения о расширении виртуальной машины (VMAccess).

## ПРИМЕРЫ

### Пример 1. Получить расширение VMAccess
```
PS C:\> $VMAccessExtension = Get-AzVMAccessExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -Name "ContosoTest"
```

Эта команда получает расширение VMAccess ContosoTest для виртуальной машины с именем VirtualMachine07.

### Пример 2. Просмотр представления экземпляра расширения VMAccess
```
PS C:\> $VMAccessExtension = Get-AzVMAccessExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -Name "ContosoTest" -Status
```

Эта команда возвращает представление экземпляра расширения VMAccess ContosoTest для виртуальной машины с именем VirtualMachine07.

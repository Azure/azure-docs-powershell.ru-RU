# Get-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Возвращает указанные правила виртуальной сети в указанном Магазине для озера Данных.
Если виртуальное правило сети не задано, будут перечислены все правила виртуальной сети для учетной записи.

## СИНТАКСИС

```
Get-AzDataLakeStoreVirtualNetworkRule [-Account] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Get-AzDataLakeStoreVirtualNetworkRule возвращает указанные правила виртуальной сети в указанном магазине.
Если виртуальное правило сети не задано, будут перечислены все правила виртуальной сети для учетной записи.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"

ResourceGroupName                :
AccountName                      :
VirtualNetworkRuleName           : myVNET
VirtualNetworkSubnetId           : /subscriptions/<subscriptionId>/resourceGroups/<resourceGroup>/providers/Microsoft.Network/virtualNetworks/myVNET/subnets/testId
IgnoreMissingVnetServiceEndpoint :
State                            :
```

Возвращает правило виртуальной сети с именем "myVNET" из учетной записи "dls"

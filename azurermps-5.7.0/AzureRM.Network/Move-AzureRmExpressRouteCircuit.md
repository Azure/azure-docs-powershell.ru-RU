# Move-AzureRmExpressRouteCircuit

## КРАТКИй обзор
Перемещает цепь ExpressRoute из классической модели развертывания в модель развертывания диспетчера ресурсов.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Move-AzureRmExpressRouteCircuit -Name <String> -ResourceGroupName <String> -Location <String>
 -ServiceKey <String> [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Move-AzureRmExpressRouteCircuit** перемещает канал ExpressRoute из классической модели развертывания в модель развертывания диспетчера ресурсов. После перемещения канал ExpressRoute работает так же, как и любой другой канал ExpressRoute, созданный в модели развертывания диспетчера ресурсов. Ссылки на каналы, виртуальные сети и шлюзы VPN не перемещаются по этой операции. Эти ресурсы необходимо перенастроить после перемещения.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Перемещение канала ExpressRoute в модель развертывания диспетчера ресурсов
```
Move-AzureRmExpressRouteCircuit -Name $CircuitName -ResourceGroupName $RG -Location $Location -ServiceKey $ServiceKey
```

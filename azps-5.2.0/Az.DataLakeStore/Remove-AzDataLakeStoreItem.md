# Remove-AzDataLakeStoreItem

## SYNOPSIS
Удаляет файл или папку из магазина Data Lake Store.

## СИНТАКСИС

```
Remove-AzDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]> [-Recurse] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его учетом** можно удалить файл или папку из магазина Data Lake Store.

## ПРИМЕРЫ

### Пример 1. Удаление нескольких элементов
```
PS C:\>Remove-AzDataLakeStoreItem -AccountName "ContosoADL" -Paths "/File01.txt","/MyFiles/File.csv"
```

Эта команда удаляет файлы, File01.txt и File.csv из магазина Data Lake Store.

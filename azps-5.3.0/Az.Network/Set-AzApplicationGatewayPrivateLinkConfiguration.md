# Set-AzApplicationGatewayPrivateLinkConfiguration

## SYNOPSIS
Изменяет конфигурацию PrivateLink для шлюза приложения.

## СИНТАКСИС

```
Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 -IpConfiguration <PSApplicationGatewayPrivateLinkIpConfiguration[]> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Set-AzApplicationGatewayPrivateLinkConfiguration** изменяет конфигурацию PrivateLink для шлюза приложения Azure.

## ПРИМЕРЫ

### Пример 1. Настройка конфигурации PrivateLink
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01" -IpConfiguration $privateLinkIpConfiguration01
```

Первая команда получает шлюз приложения ApplicationGateway01, который принадлежит к группе ресурсов ResourceGroup01, и сохраняет его в переменной $AppGw ресурса.
Вторая команда задает конфигурацию PrivateLink с именем privateLinkConfig01, чтобы использовать конфигурацию IP-адреса, храняную в $privateLinkIpConfiguration 01

# Модуль AzureRM. IotCentral
## Nописание
Предоставляет управление CRUD для приложений центра Интернета вещей.

## Командлеты AzureRM. IotCentral
### [Get-AzureRmIotCentralApp](Get-AzureRmIotCentralApp.md)
Возвращает свойства для одного или нескольких приложений Интернета вещей.

### [New-AzureRmIotCentralApp](New-AzureRmIotCentralApp.md)
Создание центра администрирования Интернет вещей.

### [Remove-AzureRmIotCentralApp](Remove-AzureRmIotCentralApp.md)
Удаляет центральное приложение IoT.

### [Set-AzureRmIotCentralApp](Set-AzureRmIotCentralApp.md)
Обновляет метаданные приложения центра администрирования Интернет вещей.

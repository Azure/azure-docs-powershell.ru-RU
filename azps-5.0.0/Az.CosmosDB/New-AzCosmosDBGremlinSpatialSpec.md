# New-AzCosmosDBGremlinSpatialSpec

## КРАТКИй обзор
Создает новый объект типа PSSpatialSpec. Она может быть передана в качестве значения параметра Set-AzCosmosDBGremlinIndexingPolicy.

## Максимальное

```
New-AzCosmosDBGremlinSpatialSpec -Path <String> -Type <String[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Создает объект, соответствующий SpatialSpec API Gremlin.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzCosmosDBGremlinSpatialSpec -Path "/abc" -Type String
Path Types
---- -----
/abc {String}
```

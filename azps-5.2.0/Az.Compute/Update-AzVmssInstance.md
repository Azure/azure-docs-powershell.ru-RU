# Update-AzVmssInstance

## SYNOPSIS
Запускает обновление экземпляра VMSS вручную.

## СИНТАКСИС

```
Update-AzVmssInstance [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-InstanceId] <String[]>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С Update-AzVmssInstance запускается ручное обновление указанного экземпляра набора масштаба виртуальной машины (VMSS).
Это используется, если политика обновления для набора масштаба VMSS настроена на ручную.

## ПРИМЕРЫ

### Пример 1. Запуск обновления экземпляра VMSS
```
PS C:\> Update-AzVmssInstance -ResourceGroupName "Group011" -VMScaleSetName "VMScaleSet001" -InstanceId "0"
```

Эта команда запускает обновление VMSS VMScaleSet001 с ИД экземпляра 0.

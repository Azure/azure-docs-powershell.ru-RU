# Export-AzMlWebService

## SYNOPSIS
Экспорт объекта определения веб-службы в виде строки в формате JSON.

## СИНТАКСИС

### ExportToFile
```
Export-AzMlWebService -WebService <WebService> -OutputFile <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ExportToJSON
```
Export-AzMlWebService -WebService <WebService> [-ToJsonString] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Экспорт объекта определения для указанной веб-службы в виде строки в формате JSON.
Вы можете сразу же вернуть строку или сохранить ее в файл.

## ПРИМЕРЫ

### Пример 1. Экспорт в качестве строки
```
Export-AzMlWebService -WebService $svc -ToJsonString
```

### Пример 2. Экспорт в файл
```
Export-AzMlWebService -WebService $svc -OutputFile "C:\mlservice.json"
```

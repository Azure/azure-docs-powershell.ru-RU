# Restart-AzsInfrastructureRole

## КРАТКИй обзор
Перезапускает запрошенную роль инфраструктуры.

## Максимальное

### Перезагрузка (по умолчанию)
```
Restart-AzsInfrastructureRole -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-AsJob]
 [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Restart-AzsInfrastructureRole -ResourceId <String> [-AsJob] [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Перезапускает запрошенную роль инфраструктуры.

## ИЛЛЮСТРИРУЮТ

### --------------------------ПРИМЕР 1--------------------------
```
Restart-AzsInfrastructureRole -Name "Compute Controller"
```

Перезапустите роль инфраструктуры, которая завершилась сбоем.

# Get-AzureRmAutomationCredential

## КРАТКИй обзор
Получает учетные данные автоматизации.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### ByAll (по умолчанию)
```
Get-AzureRmAutomationCredential [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzureRmAutomationCredential [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmAutomationCredential** получает один или несколько учетных данных службы автоматизации Azure.
По умолчанию возвращаются все учетные данные.
Укажите имя учетных данных для получения определенных учетных данных.
В целях безопасности этот командлет не возвращает пароли учетных данных.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение всех учетных данных
```
PS C:\>Get-AzureRmAutomationCredential -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17"
```

Эта команда получает метаданные для всех учетных данных в учетной записи автоматизации с именем Contoso17.

### Пример 2: получение учетных данных
```
PS C:\>Get-AzureRmAutomationCredential -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17" -Name "ContosoCredential"
```

Эта команда получает метаданные для учетных данных с именем ContosoCredential в учетной записи автоматизации с именем Contoso17.

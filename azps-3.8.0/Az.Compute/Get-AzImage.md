# Get-AzImage

## КРАТКИй обзор
Получает свойства изображения.

## Максимальное

```
Get-AzImage [[-ResourceGroupName] <String>] [[-ImageName] <String>] [[-Expand] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzImage** получает свойства изображения.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Get-AzImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01'

                       iders/Microsoft.Compute/images/Image01

```

Эта команда получает свойства изображения с именем "image01" в группе ресурсов "ResourceGroup01".

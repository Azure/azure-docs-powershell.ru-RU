# Start-AzureWebsiteJob

## КРАТКИй обзор
Запуск веб-задания для веб-сайта.

## Максимальное

```
Start-AzureWebsiteJob -JobName <String> -JobType <WebJobType> [-PassThru] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Start-AzureWebsiteJob** запускает веб-задание для веб-сайта.

## ИЛЛЮСТРИРУЮТ

### Пример 1: запуск веб-задания для веб-сайта
```
PS C:\> Start-AzureWebsiteJob -Name MyWebsite -JobName MyWebJob -JobType Continuous
```

Запуск веб-задания с именем MyWebJob для MyWebSite.

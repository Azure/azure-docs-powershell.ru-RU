# Get-AzHpcCacheUsageModel

## КРАТКИй обзор
Получает все целевые объекты хранилища usageModels для NFS.

## Максимальное

```
Get-AzHpcCacheUsageModel [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzHpcCacheUsageModel** возвращает список моделей использования для целевого хранилища NFS.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzHpcCacheUsageModel
```

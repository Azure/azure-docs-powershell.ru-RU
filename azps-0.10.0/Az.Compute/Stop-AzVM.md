# Stop-AzVM

## КРАТКИй обзор
Остановка виртуальной машины Azure.

## Максимальное

### ResourceGroupNameParameterSetName (по умолчанию)
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-Id] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Stop-AzVM** останавливает виртуальную машину Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: остановка виртуальной машины
```
PS C:\> Stop-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Эта команда останавливает виртуальную машину с именем VirtualMachine07 в ResourceGroup11.

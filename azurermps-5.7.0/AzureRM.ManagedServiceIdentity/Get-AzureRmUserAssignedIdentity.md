# Get-AzureRmUserAssignedIdentity

## КРАТКИй обзор
Возвращает пользователей, которым назначены идентификаторы и удостоверения.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### SuscriptionParameterSet (по умолчанию)
```
Get-AzureRmUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzureRmUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
**Get-AzureRmUserAssignedIdentity** получает идентификаторы существующих пользователей, которым они назначены.

# Get-AzIotHubValidSku

## SYNOPSIS
Получает все допустимые skus, к которые может перейти этот IotHub.

## СИНТАКСИС

```
Get-AzIotHubValidSku [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Получает все действительные skus, к которые может перейти этот IotHub.
IotHub не может перейти между бесплатными и платными skus и наоборот. Для этого вам придется удалить и заново создать iothub.

## ПРИМЕРЫ

### Пример 1. Получить действительные skus
```
PS C:\> Get-AzIotHubValidSku -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Получает список всех skus для IotHub с именем myiothub.

## NОПИСАНИЕ
Командлет **New-AzureRmResource** создает ресурс Azure, например веб-сайт, сервер базы данных Azure SQL или базу данных Azure SQL, в группе ресурсов.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Создание ресурса
```
PS> New-AzureRmResource -Location "West US" -Properties @{test="test"} -ResourceName TestSite06 -ResourceType microsoft.web/sites -ResourceGroupName ResourceGroup11 -Force
```

Эта команда создает ресурс, который является веб-сайтом в ResourceGroup11.

### Пример 2: Создание ресурса с помощью splatting
```
PS> $prop = @{
    Location          = "West US" 
    Properties        = @{test = "test"} 
    ResourceName      = "TestSite06" 
    ResourceType      = "microsoft.web/sites" 
    ResourceGroupName = "ResourceGroup11" 
    Force             = $true
}

PS> New-AzureRmResource @prop
```

Эта команда создает ресурс, который является веб-сайтом в ResourceGroup11.

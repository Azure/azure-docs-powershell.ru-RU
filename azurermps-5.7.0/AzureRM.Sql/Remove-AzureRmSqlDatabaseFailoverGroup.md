# Remove-AzureRmSqlDatabaseFailoverGroup

## КРАТКИй обзор
Удаляет группу отработки отказа базы данных Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Эта команда удаляет группу отработки отказа с указанным именем, оставляя все базы данных и связи репликации неизменными. Конечная точка прослушивателя будет отменена из DNS.

Для выполнения команды следует использовать основной сервер группы отработки отказа.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Remove-AzureRmSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Удаление группы отработки отказа.

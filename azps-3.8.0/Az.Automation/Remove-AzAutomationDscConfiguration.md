# Remove-AzAutomationDscConfiguration

## КРАТКИй обзор
Удаляет конфигурации DSC из автоматизации.

## Максимальное

```
Remove-AzAutomationDscConfiguration [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzAutomationDscConfiguration** удаляет из Azure Automation параметры необходимой конфигурации состояния (DSC).

# Revoke-AzStorageAccountUserDelegationKeys

## SYNOPSIS
Ото имя всех ключей делегирования пользователей для учетной записи хранения.

## СИНТАКСИС

### AccountName (по умолчанию)
```
Revoke-AzStorageAccountUserDelegationKeys [-ResourceGroupName] <String> [-StorageAccountName] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountObject
```
Revoke-AzStorageAccountUserDelegationKeys -InputObject <PSStorageAccount> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountResourceId
```
Revoke-AzStorageAccountUserDelegationKeys [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Revoke-AzStorageAccountUserDelegationKeys** отменяет все ключи делегирования пользователей учетной записи хранения, поэтому все маркеры SAS удостоверений учетной записи хранения также будут отозваны.

## ПРИМЕРЫ

### Пример 1. Ото имя всех ключей делегирования пользователей учетной записи хранения
```powershell
PS C:\>Revoke-AzStorageAccountUserDelegationKeys -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount"
```

В этом примере все ключи делегирования пользователей учетной записи хранения отозваны, поэтому все маркеры SAS удостоверений, созданные на их примере, также будут отозваны.

# Remove-AzRecoveryServicesAsrPolicy

## SYNOPSIS
Удаляет указанную политику репликации ASR из хранилища служб восстановления.

## СИНТАКСИС

```
Remove-AzRecoveryServicesAsrPolicy -InputObject <ASRPolicy> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для удаления указанной политики репликации из хранилища служб восстановления был удален cmdlet **Remove-AzRecoveryServicesAsrPolicy.**

## ПРИМЕРЫ

### Пример 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrPolicy -Policy $Policy
```

Начало удаления указанной политики репликации и возврат задания ASR, используемого для отслеживания операции.

# Get-AzWebAppAccessRestrictionConfig

## КРАТКИй обзор
Получает доступ к конфигурации restiction для веб-приложения Azure.

## Максимальное

```
Get-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String> [[-SlotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzWebAppAccessRestrictionConfig** получает конфигурацию ограничения доступа к веб-приложению Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение конфигурации ограничения доступа для веб-приложения из группы ресурсов
```
PS C:\>Get-AzWebAppAccessRestrictionConfig -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Эта команда получает конфигурацию ограничения доступа для веб-приложения с именем ContosoSite, которое принадлежит к группе ресурсов Default-Web-WestUS.

# Remove-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Удаляет сертификат проверки подлинности из шлюза приложения.

## СИНТАКСИС

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Remove-AzApplicationGatewayAuthenticationCertificate** удаляет сертификат проверки подлинности из шлюза приложения Azure.

## ПРИМЕРЫ

### Пример 1. Удаление сертификата проверки подлинности из шлюза приложения
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Первая команда получает шлюз приложения appGwName и сохраняет результат в переменной $appgw.
Вторая команда удаляет сертификат проверки подлинности с именем cert01 из шлюза приложения.
Третья команда обновляет шлюз приложения.

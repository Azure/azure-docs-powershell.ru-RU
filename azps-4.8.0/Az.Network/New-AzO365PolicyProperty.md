# New-AzO365PolicyProperty

## КРАТКИй обзор
Создание объекта политики передачи трафика в Office 365.

## Максимальное

```
New-AzO365PolicyProperty [-Allow] [-Optimize] [-Default] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Создание политики Office 365 для использования с командлетами New-AzVpnSite и Update-AzVpnSite.
## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $policy = New-AzO365PolicyProperty -Allow -Optimize
```

Создание политики трафика Office 365 с разрешенными помещениями для разрешения и оптимизации категории трафика.

# Get-AzCognitiveServicesAccountNetworkRuleSet

## КРАТКИй обзор
Получение свойства NetworkRule учетной записи службы для персонализированных служб

## Максимальное

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzCognitiveServicesAccountNetworkRuleSet** получает свойство NetworkRule учетной записи службы для персонализированных служб.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение свойства NetworkRule для указанной учетной записи самообслуживания
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

Эта команда получает свойство NetworkRule для указанной учетной записи службы.

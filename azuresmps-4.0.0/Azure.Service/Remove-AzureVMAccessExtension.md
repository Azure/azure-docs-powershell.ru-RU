# Remove-AzureVMAccessExtension

## КРАТКИй обзор
Удаляет расширение VMAccess, примененное на виртуальной машине.

## Максимальное

```
Remove-AzureVMAccessExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureVMAccessExtension** удаляет расширение VMAccess, примененное на виртуальной машине.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление расширения VMAccess из виртуальной машины
```
PS C:\> Remove-AzureVMAccessExtension -VM $VM;
```

Эта команда удаляет расширение VMAccess из виртуальной машины.

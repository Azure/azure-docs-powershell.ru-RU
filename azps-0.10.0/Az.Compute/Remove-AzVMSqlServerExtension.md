# Remove-AzVMSqlServerExtension

## КРАТКИй обзор
Удаляет расширение SQL Server из виртуальной машины.

## Максимальное

```
Remove-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzVMSqlServerExtension** удаляет расширение сервера AzureSQL из виртуальной машины.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление расширения SQL Server
```
PS C:\> Remove-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" -Name "SqlIaaSAgent"
```

Эта команда удаляет расширение SQL Server из виртуальной машины с именем ContosoVM22.

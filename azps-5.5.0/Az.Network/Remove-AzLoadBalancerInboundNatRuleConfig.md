# Remove-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Удаляет из балансиров нагрузки конфигурацию входящие правила NAT.

## СИНТАКСИС

```
Remove-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для балансиров нагрузки Azure конфигурация правила перевода сетевых адресов (NAT) **remove-AzLoadBalancerInboundNatRuleConfig.**

## ПРИМЕРЫ

### 1. Удаление входящий правила NAT из балансиров нагрузки Azure
```
$loadbalancer = Get-AzLoadBalancer -Name mylb -ResourceGroupName myrg

 Remove-AzLoadBalancerInboundNatRuleConfig -Name "myinboundnatrule" -LoadBalancer $loadbalancer
```

Первая команда загружает уже существующий баланс нагрузки "mylb" и сохраняет его в переменной $load балансе. Вторая команда удаляет правило NAT, связанное с этим балансировом нагрузки.

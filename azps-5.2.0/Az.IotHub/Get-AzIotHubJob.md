# Get-AzIotHubJob

## SYNOPSIS
Получает сведения о заданиях IotHub.

## СИНТАКСИС

```
Get-AzIotHubJob [-ResourceGroupName] <String> [-Name] <String> [[-JobId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Получает сведения о заданиях IotHub.
Задание IotHub создается при инициализации операции импорта или экспорта с New-AzIotHubExportDevices или New-AzIotHubImportDevices экспорта.
Вы можете перечислить все задания или отфильтровать их по идентификатору.

## ПРИМЕРЫ

### Пример 1. Список всех заданий
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Все задания для IotHub с именем "myiothub"

### Пример 2. Получить задание
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub" -JobId 3630fc31-4caa-43e8-a232-ea0577221cb2
```

Сведения о работе с идентификатором 3630fc31-4caa-43e8-a232-ea0577221cb2" для IotHub с именем "myiothub"

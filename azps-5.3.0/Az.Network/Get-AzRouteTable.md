# Get-AzRouteTable

## SYNOPSIS
Получает таблицы маршрутов.

## СИНТАКСИС

### NoExpand (по умолчанию)
```
Get-AzRouteTable [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Развернуть
```
Get-AzRouteTable -ResourceGroupName <String> -Name <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения таблиц маршрутов Azure можно получить таблицы **маршрутов Get-AzRouteTable.**
Вы можете получить одну таблицу маршрутов или все таблицы маршрутов в группе ресурсов или в подписке.

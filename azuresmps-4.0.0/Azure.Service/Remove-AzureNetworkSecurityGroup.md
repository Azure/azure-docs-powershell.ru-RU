# Remove-AzureNetworkSecurityGroup

## КРАТКИй обзор
Удаление группы безопасности сети.

## Максимальное

```
Remove-AzureNetworkSecurityGroup -Name <String> [-Force] [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureNetworkSecurityGroup** удаляет группу безопасности сети Azure из подписки.

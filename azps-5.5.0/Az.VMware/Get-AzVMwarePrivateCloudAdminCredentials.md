# Get-AzVMwarePrivateCloudAdminCredentials

## SYNOPSIS
Укаймь учетных данных администратора для закрытого облака

## СИНТАКСИС

```
Get-AzVMwarePrivateCloudAdminCredentials -PrivateCloudName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## ОПИСАНИЕ
Укаймь учетных данных администратора для закрытого облака

## ПРИМЕРЫ

### Пример 1. Получите учетные данные администратора закрытого облака
```powershell
PS C:\> Get-AzVMwarePrivateCloudAdminCredentials -PrivateCloudName azps-test-cloud -ResourceGroupName azps-test-group

NsxtPassword NsxtUsername VcenterPassword VcenterUsername
------------ ------------ --------------- ---------------
************ admin        ************    cloudadmin@vsphere.local
```

Получить учетные данные администратора частного облака

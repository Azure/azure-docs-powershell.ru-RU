# Get-AzSqlCapability

## SYNOPSIS
Возвращает SQL базы данных для текущей подписки.

## СИНТАКСИС

### FilterResults (по умолчанию)
```
Get-AzSqlCapability [-LocationName] <String> [-ServerVersionName <String>] [-EditionName <String>]
 [-ServiceObjectiveName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### DefaultResults
```
Get-AzSqlCapability [-LocationName] <String> [-Defaults] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **его использованием** можно получить возможности azure SQL Database, доступные в текущей подписке для региона.
Если указать *параметры ServerVersionName,* *EditionName* или *ServiceObjectiveName, этот cmdlet* возвратит указанные значения и их предшественники.

## ПРИМЕРЫ

### Пример 1. Получить возможности для текущей подписки для региона
```
PS C:\>Get-AzSqlCapability -LocationName "Central US"
Location                : Central US
Status                  : Available
SupportedServerVersions : {12.0, 2.0}
```

Эта команда возвращает возможности экземпляров SQL базы данных в текущей подписке для центрального региона США.

### Пример 2. Получить возможности по умолчанию для текущей подписки для региона
```
PS C:\>Get-AzSqlCapability -LocationName "Central US" -Defaults
Location        : Central US
Status          : Available
ExpandedDetails : Version: 2.0 (Default) -> Edition: Standard (Default) -> Service Objective: S0 (Default)
```

Эта команда возвращает возможности по умолчанию для SQL баз данных в текущей подписке в центральной части США.

### Пример 3. Подробные сведения о цели обслуживания
```
PS C:\>Get-AzSqlCapability -LocationName "Central US" -ServiceObjectiveName "S1"
Location        : Central US
Status          : Available
ExpandedDetails : Version: 12.0 (Available) -> Edition: Standard (Default) -> Service Objective: S1 (Available) 
                  Version: 2.0 (Default) -> Edition: Standard (Default) -> Service Objective: S1 (Available)
```

Эта команда по умолчанию может SQL базы данных для указанной цели обслуживания по текущей подписке.

# Set-AzDtlAutoShutdownPolicy

## SYNOPSIS
Задает политику автоматического закрытия лабораторной работы DevTest Labs.

## СИНТАКСИС

### Включить (по умолчанию)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### "Отключить"
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Set-AzDtlAutoShutdownPolicy** задает политику автоматического закрытия лаборатории, которая автоматически закрывает все виртуальные машины в заданное время суток.
Для этого используется заданная группа ресурсов и название лабораторных работы.

## ПРИМЕРЫ

### Пример 1

Задает политику автоматического завершения работы лабораторных лаборатории DevTest Labs. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoShutdownPolicy -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```

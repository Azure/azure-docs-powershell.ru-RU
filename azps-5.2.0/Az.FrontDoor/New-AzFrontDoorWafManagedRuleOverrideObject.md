# New-AzFrontDoorWafManagedRuleOverrideObject

## SYNOPSIS
Создание объекта переопределения управляемых правил

## СИНТАКСИС

```
New-AzFrontDoorWafManagedRuleOverrideObject -RuleId <String> [-Action <String>] [-Disabled]
 [-Exclusion <PSManagedRuleExclusion[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Создайте объект PSAzureManagedRuleOverride для управляемой группы правил WAF, переопределяйте создание объектов.

## ПРИМЕРЫ

### Пример 1
Создайте объект переопределения управляемого правила для правила 942250 (в группе SQLI).

```powershell
PS C:\> New-AzFrontDoorWafManagedRuleOverrideObject -RuleId "942250" -Action Log

RuleId EnabledState Action
------ ------------ ------
942250      Enabled    Log
```

# Remove-AzureRmDdosProtectionPlan

## КРАТКИй обзор
Удаление плана защиты Distributed.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzureRmDdosProtectionPlan удаляет план защиты Distributed.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление пустого плана защиты Distributed
```
D:\> Remove-AzureRmDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

В этом случае мы удалим план защиты Distributed, как указано.

### Пример 2: Удаление плана защиты Distributed, связанного с виртуальной сетью

Планы защиты Distributed невозможно удалить, если они связаны с виртуальной сетью. Для начала необходимо разосоединить оба объекта. Здесь мы получаем самую обновленную версию виртуальной сети, связанную с планом, и установили для свойства **DdosProtectionPlan** пустое значение и флаг **EnableDdosProtection** (этот флаг не может быть истинным, но не планом).
Затем мы сохраняем новое состояние, перетаскивая локальную переменную на **Set-AzureRmVirtualNetwork**. На этом этапе план больше не связан с виртуальной сетью.
Если это последнее значение, связанное с планом, мы можем удалить план защиты Distributed с помощью команды Remove-AzureRmDdosProtectionPlan.

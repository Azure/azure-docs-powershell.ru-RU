# Move-AzExpressRouteCircuit

## SYNOPSIS
Перемещение схемы ExpressRoute из классической модели развертывания в модель развертывания Диспетчера ресурсов.

## СИНТАКСИС

```
Move-AzExpressRouteCircuit -Name <String> -ResourceGroupName <String> -Location <String> -ServiceKey <String>
 [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для **перемещения-AzExpressRouteCircuit** канал ExpressRoute из классической модели развертывания в модель развертывания Диспетчера ресурсов. После перемещения канал ExpressRoute работает и работает так же, как любой другой канал ExpressRoute, созданный в модели развертывания Диспетчера ресурсов. В ходе этой операции не перемещаются сетевые, виртуальные сети и VPN-шлюзы. После перемещения эти ресурсы необходимо перенастроить.

## ПРИМЕРЫ

### Пример 1. Перемещение схемы ExpressRoute в модель развертывания Диспетчера ресурсов
```
Move-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $RG -Location $Location -ServiceKey $ServiceKey
```

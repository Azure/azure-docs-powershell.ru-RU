# Enable-AzureRmSchedulerJobCollection

## КРАТКИй обзор
Включает коллекцию заданий.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Enable-AzureRmSchedulerJobCollection -ResourceGroupName <String> -JobCollectionName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Enable-AzureRmSchedulerJobCollection** включает коллекцию заданий в планировщике Azure.

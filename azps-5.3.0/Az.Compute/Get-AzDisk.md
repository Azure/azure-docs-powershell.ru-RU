# Get-AzDisk

## SYNOPSIS
Свойства управляемого диска.

## СИНТАКСИС

```
Get-AzDisk [[-ResourceGroupName] <String>] [[-DiskName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для управляемого диска свойства управляемого диска получает **cmdlet Get-AzDisk.**

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Get-AzDisk -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'

                     te/disks/Disk01
```

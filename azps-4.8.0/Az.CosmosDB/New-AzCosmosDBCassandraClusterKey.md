# New-AzCosmosDBCassandraClusterKey

## КРАТКИй обзор
Создает новый ключ кластера CosmosDB Cassandra.

## Максимальное

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
**New-AzCosmosDBCassandraClusterKey** создает новый ключ кластера CosmosDB Cassandra.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```

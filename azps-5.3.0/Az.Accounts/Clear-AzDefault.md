# Clear-AzDefault

## SYNOPSIS
Очищает значения по умолчанию, установленные пользователем в текущем контексте.

## СИНТАКСИС

```
Clear-AzDefault [-ResourceGroup] [-PassThru] [-Force] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот Clear-AzDefault удаляет значения по умолчанию, установленные пользователем, в зависимости от параметров переключения, заданных пользователем.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Clear-AzDefault
```

Эта команда удаляет все значения по умолчанию, установленные пользователем в текущем контексте.

### Пример 2
```powershell
PS C:\> Clear-AzDefault -ResourceGroup
```

Эта команда удаляет группу ресурсов по умолчанию, заданной пользователем в текущем контексте.

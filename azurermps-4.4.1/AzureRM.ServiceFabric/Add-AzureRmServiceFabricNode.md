# Add-AzureRmServiceFabricNode

## КРАТКИй обзор
Добавьте узлы в конкретный тип узла в кластере.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Add-AzureRmServiceFabricNode [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 -NumberOfNodesToAdd <Int32> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Используйте **Add-AzureRmServiceFabricNode** , чтобы добавить узлы в конкретный тип узла. Вам нужно просто указать количество узлов, которые нужно добавить в тип узла.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS c:> Add-AzureRmServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Эта команда добавит 2 узла в тип "N1".

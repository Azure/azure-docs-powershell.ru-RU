# Модуль Az.ContainerInstance
## Описание
В этой теме вы можете найти разделы справки по экземплярам контейнеров Azure.

## Az.ContainerInstance Cmdlets
### [Get-AzContainerGroup](Get-AzContainerGroup.md)
Возвращает группы контейнеров.

### [Get-AzContainerInstanceLog](Get-AzContainerInstanceLog.md)
Получить журналы экземпляра контейнера в группе контейнеров.

### [New-AzContainerGroup](New-AzContainerGroup.md)
Создает группу контейнеров.

### [Remove-AzContainerGroup](Remove-AzContainerGroup.md)
Удаляет группу контейнеров.

# Disable-AzureRmHDInsightOperationsManagementSuite

## КРАТКИй обзор
Отключает набор Operations Management Suite (OMS) в HDInsight кластере, и необходимые журналы перестают передаваться в рабочую область OMS, указанную во время включения.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Disable-AzureRmHDInsightOperationsManagementSuite [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Disable-AzureRmHDInsightOperationsManagementSuite** отключает набор Operations Management Suite (OMS) в кластере Azure HDInsight.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Disable-AzureRmHDInsightOMS -Name testcluster

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

Набор Operations Management Suite (OMS) будет отключен в кластере HDInsight, а связанные журналы будут остановлены в рабочую область OMS.

### Пример 2
```
PS C:\> Disable-AzureRmHDInsightOMS -Name testcluster -ResourceGroupName testrg

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

Набор Operations Management Suite (OMS) будет отключен в кластере HDInsight, а связанные журналы будут остановлены в рабочую область OMS.

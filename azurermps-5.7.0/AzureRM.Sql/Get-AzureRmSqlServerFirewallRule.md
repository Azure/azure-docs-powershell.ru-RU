# Get-AzureRmSqlServerFirewallRule

## КРАТКИй обзор
Возвращает правила брандмауэра для сервера базы данных SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlServerFirewallRule [[-FirewallRuleName] <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSqlServerFirewallRule** получает правила брандмауэра для сервера базы данных SQL Azure.
Если вы задаете имя правила брандмауэра, этот командлет получает сведения о конкретном правиле брандмауэра.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение всех правил для сервера
```
PS C:\>Get-AzureRmSqlServerFirewallRule -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
ResourceGroupName : ResourceGroup01
ServerName        : server01
StartIpAddress    : 0.0.0.0
EndIpAddress      : 0.0.0.0
FirewallRuleName  : AllowAllWindowsAzureIps

ResourceGroupName : ResourceGroup01
ServerName        : Server01
StartIpAddress    : 1.2.3.4
EndIpAddress      : 4.3.2.1
FirewallRuleName  : Rule01
```

Эта команда получает все правила брандмауэра для сервера с именем Server01.

# Suspend-AzureRmSqlDatabase

## КРАТКИй обзор
Приостанавливает базу данных хранилища данных SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Suspend-AzureRmSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Suspend-AzureRmSqlDatabase** приостанавливает базу данных хранилища данных Azure SQL.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Приостановка базы данных хранилища данных SQL Azure
```
PS C:\>Suspend-AzureRmSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда приостанавливает активную базу данных хранилища данных SQL Azure.

## Пуск
* Командлет **Suspend-AzureRmSqlDatabase** работает только в базах данных хранилища данных Azure SQL. Эта операция не поддерживается в основных, стандартных и расширенных выпусках базы данных Azure SQL.

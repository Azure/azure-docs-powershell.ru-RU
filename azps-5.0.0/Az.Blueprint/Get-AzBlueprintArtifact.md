# Get-AzBlueprintArtifact

## КРАТКИй обзор
Получение артефактов из определения чертежа.

## Максимальное

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Получение артефактов из определения чертежа. Если версия определения схемы не задана, извлекается черновая версия. В случае отсутствия черновой версии возвращается последнее определение опубликованного чертежа.

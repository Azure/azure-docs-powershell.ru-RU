# Remove-AzureRmApiManagement

## КРАТКИй обзор
Удаляет службу управления API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmApiManagement -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmApiManagement** Удаляет службу управления API Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление службы управления API
```
PS C:\>Remove-AzureRmApiManagement -ResourceGroupName "ContosoGroup02" -Name "ContosoApi"
```

Эта команда удаляет службу управления API с именем ContosoApi.

# Get-AzHpcCacheSku

## КРАТКИй обзор
Получает все номера SKU, доступные в подписке.

## Максимальное

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzHpcCacheSku** возвращает список SKU, доступных в подписке.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzHpcCacheSku
```

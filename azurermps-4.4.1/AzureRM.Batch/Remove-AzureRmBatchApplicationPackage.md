# Remove-AzureRmBatchApplicationPackage

## КРАТКИй обзор
Удаляет запись пакета приложения и двоичный файл.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String>
 [-ApplicationId] <String> [-ApplicationVersion] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmBatchApplicationPackage** удаляет запись пакета приложения и двоичный файл из пакетной учетной записи Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление пакета приложения из пакетной учетной записи
```
PS C:\>Remove-AzureRmBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "litware" -ApplicationVersion "1.0"
```

Эта команда удаляет версию 1,0 приложения беседа Litware из учетной записи ContosoBatchGroup.
Команда удаляет запись пакета и большой двоичный объект, который содержит двоичный файл пакета.

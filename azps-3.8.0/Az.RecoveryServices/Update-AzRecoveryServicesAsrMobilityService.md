# Update-AzRecoveryServicesAsrMobilityService

## КРАТКИй обзор
Отправка обновлений агента службы Mobility Service на защищенные компьютеры.

## Максимальное

```
Update-AzRecoveryServicesAsrMobilityService [-Account <ASRRunAsAccount>]
 -ReplicationProtectedItem <ASRReplicationProtectedItem> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Update-AzRecoveryServicesAsrMobilityService** пытается отправить обновления агента службы Mobility Service на защищенные компьютеры (если доступно обновление).

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Update-AzRecoveryServicesAsrMobilityService -ReplicationProtectedItem $rpi -Account $fabric.fabricSpecificDetails.RunAsAccounts[0]
```

Задание для отслеживания агента обслуживания служб Mobility Service, защищенного с репликацией обновлений.

# AZ. DataBox Module
## Nописание
Модуль для создания, отмены, удаления и получения заданий Databox

## Командлеты AZ. DataBox
### [Get-AzDataBoxCredential](Get-AzDataBoxCredential.md)
Получает учетные данные databox для определенного задания.

### [Get-AzDataBoxJob](Get-AzDataBoxJob.md)
Получение сведений о заданиях Databox

### [New-AzDataBoxJob](New-AzDataBoxJob.md)
Создает новое задание databox с использованием указанных параметров.

### [Remove-AzDataBoxJob](Remove-AzDataBoxJob.md)
Удаляет задание databox

### [Остановить-AzDataBoxJob](Stop-AzDataBoxJob.md)
Отменяет текущее задание databox, если задание находится в состоянии отмены.

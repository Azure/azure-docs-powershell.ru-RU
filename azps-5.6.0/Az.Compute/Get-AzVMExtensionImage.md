# Get-AzVMExtensionImage

## SYNOPSIS
Получает все версии для расширения Azure.

## СИНТАКСИС

```
Get-AzVMExtensionImage -Location <String> -PublisherName <String> -Type <String> [-FilterExpression <String>]
 [-Version <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для расширения Azure можно получить все версии для cmdlet **Get-AzVMExtensionImage.**

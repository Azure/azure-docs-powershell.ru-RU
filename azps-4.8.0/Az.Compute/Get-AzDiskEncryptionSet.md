# Get-AzDiskEncryptionSet

## КРАТКИй обзор
Получение или перечисление наборов шифрования дисков.

## Максимальное

```
Get-AzDiskEncryptionSet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Получение или перечисление наборов шифрования дисков.

## ИЛЛЮСТРИРУЮТ

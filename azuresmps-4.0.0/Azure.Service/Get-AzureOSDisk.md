# Get-AzureOSDisk

## КРАТКИй обзор
Получает диск операционной системы виртуальной машины Azure.

## Максимальное

```
Get-AzureOSDisk -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureOSDisk** получает диск операционной системы виртуальной машины Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение диска операционной системы
```
PS C:\> Get-AzureVM -ServiceName "ContosoService" -Name "VirtualMachine02" | Get-AzureOSDisk
```

Эта команда получает виртуальную машину с именем VirtualMachine02 в службе с именем ContosoService с помощью командлета **Get-AzureVM** .
Команда передает виртуальную машину в текущий командлет с помощью оператора конвейера.
Текущий командлет получает диск операционной системы этой виртуальной машины.

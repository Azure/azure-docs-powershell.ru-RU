# Get-AzureRmSqlServerRecommendedAction

## КРАТКИй обзор
Получает одно или несколько рекомендованных действий для советника SQL Server Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlServerRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -AdvisorName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSqlServerRecommendedAction** получает одно или несколько рекомендованных действий для консультанта SQL Server Azure.

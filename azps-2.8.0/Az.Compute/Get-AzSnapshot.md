# Get-AzSnapshot

## КРАТКИй обзор
Получает свойства снимка.

## Максимальное

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzSnapshot** получает свойства снимка.

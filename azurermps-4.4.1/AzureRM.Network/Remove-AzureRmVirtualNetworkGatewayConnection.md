# Remove-AzureRmVirtualNetworkGatewayConnection

## КРАТКИй обзор
Удаление подключения к шлюзу виртуальной сети

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmVirtualNetworkGatewayConnection -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Подключение к шлюзу виртуальной сети является объектом, представляющим туннель IPsec ("сайт-сайт" или "Виртуальная сеть-сеть"), подключенного к шлюзу виртуальной сети в Azure.

Командлет **Remove-AzureRmVirtualNetworkGatewayConnection** удаляет объект подключения на основе имени и имени группы ресурсов.

## ИЛЛЮСТРИРУЮТ

### 1: Удаление подключения к шлюзу виртуальной сети
```
Remove-AzureRmVirtualNetworkGatewayConnection -Name myTunnel -ResourceGroupName myRG
```

Удаляет объект подключения к шлюзу виртуальной сети с именем "myTunnel" в группе ресурсов "myRG"

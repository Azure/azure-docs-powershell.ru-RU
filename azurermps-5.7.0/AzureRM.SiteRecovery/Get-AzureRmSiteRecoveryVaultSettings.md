# Get-AzureRmSiteRecoveryVaultSettings

## КРАТКИй обзор
Получает сведения о параметрах для текущего хранилища на сайте восстановления сайта.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSiteRecoveryVaultSettings [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSiteRecoveryVaultSettings** получает сведения о параметрах, связанных с текущим хранилищем Azure Site Recovery.

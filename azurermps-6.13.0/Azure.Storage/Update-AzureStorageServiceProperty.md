# Update-AzureStorageServiceProperty

## КРАТКИй обзор
Изменяет свойства службы хранилища Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Update-AzureStorageServiceProperty [-ServiceType] <StorageServiceType> [-DefaultServiceVersion <String>]
 [-PassThru] [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Update-AzureStorageServiceProperty** изменяет свойства службы хранилища Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Настройка службы BLOB-объектов DefaultServiceVersion в 2017-04-17
```
C:\PS>Update-AzureStorageServiceProperty -ServiceType Blob -DefaultServiceVersion 2017-04-17
```

Эта команда задает DefaultServiceVersion для службы больших двоичных объектов в 2017-04-17

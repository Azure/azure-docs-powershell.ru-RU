# Remove-AzureRmDataLakeStoreAccount

## КРАТКИй обзор
Удаление учетной записи Data Lake Store без возможности восстановления.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmDataLakeStoreAccount** удаляет учетную запись Data Lake Store без возможности восстановления.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление учетной записи Data Lake Store
```
PS C:\>Remove-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

Эта команда удаляет учетную запись с именем ContosoADL из Data Lake Store.

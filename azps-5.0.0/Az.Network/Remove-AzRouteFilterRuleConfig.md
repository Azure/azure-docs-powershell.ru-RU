# Remove-AzRouteFilterRuleConfig

## КРАТКИй обзор
Удаляет правило фильтра маршрутов из фильтра маршрутов.

## Максимальное

```
Remove-AzRouteFilterRuleConfig -Name <String> -RouteFilter <PSRouteFilter> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzRouteFilterRuleConfig** удаляет правило фильтра маршрутов из фильтра маршрутов.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
```

Первая команда получает фильтр маршрутов с именем RouteFilter01, который принадлежит группе ресурсов с именем ResourceGroup01, и сохраняет ее в переменной $rf.
Вторая команда удаляет правило фильтра маршрутов с именем Rule01 из фильтра маршрутов, хранящегося в $rf.

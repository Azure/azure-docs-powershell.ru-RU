# Get-AzAutomationDscCompilationJobOutput

## КРАТКИй обзор
Получает потоки ведения журнала для задания компиляции DSC Automation.

## Максимальное

```
Get-AzAutomationDscCompilationJobOutput [-Id] <Guid> [-Stream <CompilationJobStreamType>]
 [-StartTime <DateTimeOffset>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzAutomationDscCompilationJobOutput** получает записи потока для задания компиляции с требуемой конфигурацией AP (DSC) в службе автоматизации Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение журналов для задания компиляции DSC
```
PS C:\>$Jobs = Get-AzAutomationDscCompilationJob -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17"
PS C:\> $Jobs[0] | Get-AzAutomationDscCompilationJobOutput -Stream "Any"
```

Первая команда получает задания компиляции в учетной записи автоматизации с именем Contoso17 с помощью командлета Get-AzAutomationDscCompilationJob.
Команда хранит эти объекты в переменной $Jobs.
Вторая команда возвращает выходные данные задания компиляции для любого потока для первого элемента массива $Jobs.

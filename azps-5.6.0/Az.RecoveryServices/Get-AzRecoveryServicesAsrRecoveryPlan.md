# Get-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Получает план восстановления или все планы восстановления в хранилище служб восстановления

## СИНТАКСИС

### По умолчанию (по умолчанию)
```
Get-AzRecoveryServicesAsrRecoveryPlan [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzRecoveryServicesAsrRecoveryPlan -Name <String> [[-Path] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFriendlyName
```
Get-AzRecoveryServicesAsrRecoveryPlan -FriendlyName <String> [[-Path] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения подробных сведений о указанном плане восстановления или всех планах восстановления в хранилище служб восстановления возвращается cmdlet **Get-AzRecoveryServicesAsrRecoveryPlan.**

## ПРИМЕРЫ

### Пример 1
```
PS C:\> $RP = Get-AzRecoveryServicesAsrRecoveryPlan -Name $RPName
```

Возвращает план восстановления с указанным именем.

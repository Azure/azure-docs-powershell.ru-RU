# Remove-AzAutomationVariable

## SYNOPSIS
Удаляет переменную автоматизации.

## СИНТАКСИС

```
Remove-AzAutomationVariable [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого из автоматизации** Azure удаляется переменная.

## ПРИМЕРЫ

### Пример 1. Удаление переменной
```
PS C:\>Remove-AzAutomationVariable -AutomationAccountName "Contoso17" -Name "StringVariable22" -Force -ResourceGroupName "ResourceGroup01"
```

Эта команда удаляет переменную с именем StringVariable22 в учетной записи автоматизации Contoso17.
Эта команда определяет параметр *Force.*
Поэтому запрос на подтверждение не будет.

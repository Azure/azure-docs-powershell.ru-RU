# Get-AzureRmNetworkUsage

## КРАТКИй обзор
Список использования сети для подписки

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Get-AzureRmNetworkUsage получает ограничения и текущее использование сетевых ресурсов.

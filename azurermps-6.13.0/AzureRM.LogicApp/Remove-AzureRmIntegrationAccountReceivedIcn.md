# Remove-AzureRmIntegrationAccountReceivedIcn

## КРАТКИй обзор
Этот командлет удаляет определенный полученный номер, указанный в параметрах управления обменом, для каждого соглашения и номера элемента управления.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmIntegrationAccountReceivedIcn -ResourceGroupName <String> -Name <String> -AgreementName <String>
 -ControlNumberValue <String> [-AgreementType <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Этот командлет используется в сценариях аварийного восстановления для удаления полученного номера управления обменом из учетной записи интеграции, чтобы соединитель B2B мог обработать сообщение повторно, если включено обнаружение дубликатов номеров.
В редких случаях полученный номер управления обменом может быть зарезервирован вскоре до аварии и до тех пор, пока соединитель B2B не отклоняет обмен как ошибочный.
В таких случаях может потребоваться, чтобы веб-сайт восстановления повторно мог обработать тот же обмен после его устранения.
Укажите параметр "-AgreementType", чтобы указать, нужно ли возвращаться возвращаемые числа X12 или EDIFACT

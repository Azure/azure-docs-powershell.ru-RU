# AZ. DataLakeStore Module
## Nописание
В этом разделе приведены командлеты PowerShell для Azure Data Lake Store в платформе диспетчера ресурсов Azure (ARM). Командлеты существуют в пространстве имен Microsoft. Azure. Commands. DataLakeStore. Эти командлеты работают только с учетными записями Azure Data Lake Storage Gen1.

## Командлеты AZ. DataLakeStore
### [Add-AzDataLakeStoreFirewallRule](Add-AzDataLakeStoreFirewallRule.md)
Добавляет правило брандмауэра в указанную учетную запись Data Lake Store.

### [Add-AzDataLakeStoreItemContent](Add-AzDataLakeStoreItemContent.md)
Добавляет содержимое в элемент в Data Lake Store.

### [Add-AzDataLakeStoreTrustedIdProvider](Add-AzDataLakeStoreTrustedIdProvider.md)
Добавляет доверенный поставщик удостоверений в указанную учетную запись Data Lake Store.

### [Add-AzDataLakeStoreVirtualNetworkRule](Add-AzDataLakeStoreVirtualNetworkRule.md)
Добавляет правило виртуальной сети к указанной учетной записи Data Lake Store.

### [Enable-AzDataLakeStoreKeyVault](Enable-AzDataLakeStoreKeyVault.md)
Пытается включить шифрование для указанной учетной записи Data Lake Store с помощью пользовательского хранилища ключей, управляемого пользователем.

### [Export-AzDataLakeStoreChildItemProperty](Export-AzDataLakeStoreChildItemProperty.md)
Экспорт свойств (использование дискового пространства и ACL) для всего дерева по заданному пути на выходном пути

### [Export-AzDataLakeStoreItem](Export-AzDataLakeStoreItem.md)
Загрузка файла из Data Lake Store.

### [Get-AzDataLakeStoreAccount](Get-AzDataLakeStoreAccount.md)
Получение сведений об учетной записи для Data Lake Store.

### [Get-AzDataLakeStoreChildItem](Get-AzDataLakeStoreChildItem.md)
Получает список элементов в папке для Data Lake Store.

### [Get-AzDataLakeStoreChildItemSummary](Get-AzDataLakeStoreChildItemSummary.md)
Возвращает сводку общего размера, файлов и каталогов, содержащихся в указанном пути.

### [Get-AzDataLakeStoreFirewallRule](Get-AzDataLakeStoreFirewallRule.md)
Получает указанные правила брандмауэра в заданном Data Lake Store.
Если не указано правило брандмауэра, выводится список всех правил брандмауэра для учетной записи.

### [Get-AzDataLakeStoreItem](Get-AzDataLakeStoreItem.md)
Получает сведения о файле или папке в Data Lake Store.

### [Get-AzDataLakeStoreItemAclEntry](Get-AzDataLakeStoreItemAclEntry.md)
Возвращает запись в ACL для файла или папки в Data Lake Store.

### [Get-AzDataLakeStoreItemContent](Get-AzDataLakeStoreItemContent.md)
Получает содержимое файла в Data Lake Store.

### [Get-AzDataLakeStoreItemOwner](Get-AzDataLakeStoreItemOwner.md)
Получает владельца файла или папки в Data Lake Store.

### [Get-AzDataLakeStoreItemPermission](Get-AzDataLakeStoreItemPermission.md)
Получает восьмеричные разрешения для файла или папки в Data Lake Store.

### [Get-AzDataLakeStoreTrustedIdProvider](Get-AzDataLakeStoreTrustedIdProvider.md)
Получает указанного доверенного поставщика удостоверений в указанном Data Lake Store.
Если поставщик не указан, выводит список всех поставщиков для учетной записи.

### [Get-AzDataLakeStoreVirtualNetworkRule](Get-AzDataLakeStoreVirtualNetworkRule.md)
Получает указанные правила виртуальной сети в указанном Data Lake Store.
Если правило для виртуальной сети не задано, выводится список всех правил для учетной записи в сети.

### [Import-AzDataLakeStoreItem](Import-AzDataLakeStoreItem.md)
Передача локального файла или каталога в Data Lake Store.

### [Присоединиться к AzDataLakeStoreItem](Join-AzDataLakeStoreItem.md)
Присоединяется к одному или нескольким файлам для создания одного файла в Data Lake Store.

### [Move-AzDataLakeStoreItem](Move-AzDataLakeStoreItem.md)
Перемещение или переименование файла или папки в Data Lake Store.

### [New-AzDataLakeStoreAccount](New-AzDataLakeStoreAccount.md)
Создание новой учетной записи для Data Lake Store.

### [New-AzDataLakeStoreItem](New-AzDataLakeStoreItem.md)
Создание нового файла или папки в Data Lake Store.

### [Remove-AzDataLakeStoreAccount](Remove-AzDataLakeStoreAccount.md)
Удаление учетной записи Data Lake Store без возможности восстановления.

### [Remove-AzDataLakeStoreFirewallRule](Remove-AzDataLakeStoreFirewallRule.md)
Удаляет указанное правило брандмауэра в указанном Data Lake Store.

### [Remove-AzDataLakeStoreItem](Remove-AzDataLakeStoreItem.md)
Удаление файла или папки в Data Lake Store.

### [Remove-AzDataLakeStoreItemAcl](Remove-AzDataLakeStoreItemAcl.md)
Очистка списка ACL для файла или папки в Data Lake Store.

### [Remove-AzDataLakeStoreItemAclEntry](Remove-AzDataLakeStoreItemAclEntry.md)
Удаляет запись из ACL файла или папки в Data Lake Store.

### [Remove-AzDataLakeStoreTrustedIdProvider](Remove-AzDataLakeStoreTrustedIdProvider.md)
Удаляет указанного доверенного поставщика удостоверений в указанном Data Lake Store.

### [Remove-AzDataLakeStoreVirtualNetworkRule](Remove-AzDataLakeStoreVirtualNetworkRule.md)
Удаляет указанное правило виртуальной сети в указанном Data Lake Store.

### [Set-AzDataLakeStoreAccount](Set-AzDataLakeStoreAccount.md)
Изменение учетной записи Data Lake Store.

### [Set-AzDataLakeStoreFirewallRule](Set-AzDataLakeStoreFirewallRule.md)
Изменяет указанное правило брандмауэра в указанном Data Lake Store.

### [Set-AzDataLakeStoreItemAcl](Set-AzDataLakeStoreItemAcl.md)
Изменение ACL для файла или папки в Data Lake Store.

### [Set-AzDataLakeStoreItemAclEntry](Set-AzDataLakeStoreItemAclEntry.md)
Изменение записи в списке ACL для файла или папки в Data Lake Store.

### [Set-AzDataLakeStoreItemExpiry](Set-AzDataLakeStoreItemExpiry.md)
Задает или удаляет время окончания срока действия файла в учетной записи Azure Data Lake Store.

### [Set-AzDataLakeStoreItemOwner](Set-AzDataLakeStoreItemOwner.md)
Изменение владельца файла или папки в Data Lake Store.

### [Set-AzDataLakeStoreItemPermission](Set-AzDataLakeStoreItemPermission.md)
Изменение восьмеричного разрешения для файла или папки в Data Lake Store.

### [Set-AzDataLakeStoreTrustedIdProvider](Set-AzDataLakeStoreTrustedIdProvider.md)
Изменяет указанного доверенного поставщика удостоверений в указанном Data Lake Store.

### [Set-AzDataLakeStoreVirtualNetworkRule](Set-AzDataLakeStoreVirtualNetworkRule.md)
Изменяет указанное правило виртуальной сети в указанном Data Lake Store.

### [Test-AzDataLakeStoreAccount](Test-AzDataLakeStoreAccount.md)
Проверка наличия учетной записи Data Lake Store.

### [Test-AzDataLakeStoreItem](Test-AzDataLakeStoreItem.md)
Проверка существования файла или папки в Data Lake Store.

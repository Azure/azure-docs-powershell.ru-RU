# Get-AzApiManagementTenantGitAccess

## SYNOPSIS
Получает конфигурацию доступа GIT для клиента.

## СИНТАКСИС

```
Get-AzApiManagementTenantGitAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для клиента конфигурацию доступа Git получает cmdlet **Get-AzApiManagementTenantGitAccess.**
Ключи не включаются в сведения о результатах. Чтобы получить секрет клиента, используйте **Get-AzApiManagementTenantGitAccessSecret.**

## ПРИМЕРЫ

### Пример 1. Настройка доступа к клиенту
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantGitAccess -Context $apimContext
```

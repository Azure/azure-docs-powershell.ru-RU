# Import-AzBlueprintWithArtifact

## SYNOPSIS
Импорт файла чертежа в формате JSON в объект чертежа и сохранение его в указанной подписке или группе управления.

## СИНТАКСИС

```
Import-AzBlueprintWithArtifact -Name <String> [-SubscriptionId <String>] [-ManagementGroupId <String>]
 -InputPath <String> [-IncludeSubFolders] [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Импорт определения чертежа с его артефактами. 

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Import-AzBlueprintWithArtifact -Name MySimpleBlueprint -SubscriptionId 00000000-1111-0000-1111-000000000000 -InputPath  C:\Blueprints\SimpleBlueprint
```

Импорт определения чертежа с его артефактами и сохранение в рамках подписки.

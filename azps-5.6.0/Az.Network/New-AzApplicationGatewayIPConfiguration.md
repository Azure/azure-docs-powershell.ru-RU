# New-AzApplicationGatewayIPConfiguration

## SYNOPSIS
Создает конфигурацию IP-адреса для шлюза приложения.

## СИНТАКСИС

### SetByResourceId
```
New-AzApplicationGatewayIPConfiguration -Name <String> [-SubnetId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResource
```
New-AzApplicationGatewayIPConfiguration -Name <String> [-Subnet <PSSubnet>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для шлюза приложения создается конфигурация **IP-адреса.**
Конфигурация IP-адреса содержит подсеть, в которой развернут шлюз приложения.

## ПРИМЕРЫ

### Пример 1. Создание конфигурации IP-адреса для шлюза приложения.
```
PS C:\>$VNet = Get-AzVirtualNetwork -Name "VNet01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Subnet = Get-AzVirtualNetworkSubnetConfig -Name "Subnet01" -VirtualNetwork $VNet 
PS C:\ $GatewayIpConfig = New-AzApplicationGatewayIPConfiguration -Name "AppGwSubnet01" -Subnet $Subnet
```

Первая команда получает виртуальную сеть VNet01, которая принадлежит к группе ресурсов ResourceGroup01.
Вторая команда получает конфигурацию подсети для подсети, к которой относится виртуальная сеть в предыдущей команде, и сохраняет ее в переменной $Subnet сети.
Третья команда создает конфигурацию IP-адреса с помощью $Subnet.

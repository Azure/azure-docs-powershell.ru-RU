# Resume-AzureRmSqlDatabase

## КРАТКИй обзор
Возобновляет базу данных хранилища данных SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Resume-AzureRmSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Resume-AzureRmSqlDatabase** возобновляет базу данных хранилища данных SQL Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: возобновление базы данных хранилища данных SQL Azure
```
PS C:\>Resume-AzureRmSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда Возобновляет приостановленную базу данных хранилища данных SQL Azure.

## Пуск
* Командлет **Resume-AzureRmSqlDatabase** работает только в базах данных хранилища данных Azure SQL. Эта операция не поддерживается в основных, стандартных и расширенных выпусках базы данных Azure SQL.

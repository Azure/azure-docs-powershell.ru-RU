# Remove-AzureRmSqlSyncAgent

## КРАТКИй обзор
Удаляет агент Azure SQL Sync.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzureRmSqlSyncAgent** удаляет агент Azure SQL Sync Agent.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление агента синхронизации
```
PS C:\>Remove-AzureRmSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Эта команда удаляет агент Azure SQL Sync Agent с именем syncAgent01.

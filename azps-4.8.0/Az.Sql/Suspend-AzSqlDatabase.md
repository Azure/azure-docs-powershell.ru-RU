# Suspend-AzSqlDatabase

## КРАТКИй обзор
Приостанавливает базу данных хранилища данных SQL.

## Максимальное

```
Suspend-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Suspend-AzSqlDatabase** приостанавливает базу данных хранилища данных Azure SQL.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Приостановка базы данных хранилища данных SQL Azure
```
PS C:\>Suspend-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда приостанавливает активную базу данных хранилища данных SQL Azure.

## Пуск
* Командлет **Suspend-AzSqlDatabase** работает только в базах данных хранилища данных Azure SQL. Эта операция не поддерживается в основных, стандартных и расширенных выпусках базы данных Azure SQL.

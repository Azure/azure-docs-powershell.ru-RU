# Remove-AzVMAEMExtension

## КРАТКИй обзор
Удаляет расширение AEM из виртуальной машины.

## Максимальное

```
Remove-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [[-OSType] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzVMAEMExtension** удаляет расширение службы улучшенного мониторинга Azure (AEM) с виртуальной машины.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление расширения AEM
```
PS C:\> Remove-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Эта команда удаляет расширение AEM для виртуальной машины с именем Contoso-Server.

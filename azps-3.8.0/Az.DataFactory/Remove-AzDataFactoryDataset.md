# Remove-AzDataFactoryDataset

## КРАТКИй обзор
Удаляет набор данных из фабрики данных Azure.

## Максимальное

### ByFactoryName (по умолчанию)
```
Remove-AzDataFactoryDataset [-Force] [-DataFactoryName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryDataset [-Force] [-DataFactory] <PSDataFactory> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzDataFactoryDataset** удаляет набор данных из фабрики данных Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление набора данных
```
PS C:\>Remove-AzDataFactoryDataset -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -Name "DAWikiAggregatedData"
Confirm
Are you sure you want to remove dataset 'DAWikiAggregatedData' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
True
```

Эта команда удаляет набор данных с именем DAWikiAggregatedData из фабрики данных с именем WikiADF.
Команда возвращает значение $True.

# Get-AzBgpServiceCommunity

## SYNOPSIS
Содержит список всех служб и регионов, сообществ BGP и связанных префиксов.

## СИНТАКСИС

```
Get-AzBgpServiceCommunity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Этот cmdlet содержит список всех служб и регионов, сообществ BGP и связанных префиксов.

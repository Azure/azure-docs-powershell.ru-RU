# Remove-AzServiceFabricNodeType

## КРАТКИй обзор
Удаление типа полного узла из кластера.

## Максимальное

```
Remove-AzServiceFabricNodeType [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
С помощью **Remove-AzServiceFabricNodeType** можно удалить все узлы из определенного типа узла и типа узла из кластера. Эту команду нельзя использовать для удаления типа основного узла.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS c:> Remove-AzServiceFabricNodeType -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1'
```

Эта команда удалит из кластера NodeType "NT1".

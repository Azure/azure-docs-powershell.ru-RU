# Remove-AzSqlServerThreatDetectionPolicy

## КРАТКИй обзор
Удаление политики обнаружения угроз с сервера.

## Максимальное

```
Remove-AzSqlServerThreatDetectionPolicy [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzSqlServerThreatDetectionPolicy удаляет политику обнаружения угроз с сервера Azure SQL Server.
Чтобы использовать этот командлет, укажите параметры ResourceGroupName и ServerName для идентификации сервера, с которого этот командлет удаляет политику.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление политики обнаружения угроз для базы данных
```
PS C:\> Remove-AzSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Эта команда удаляет политику обнаружения угроз с сервера с именем Server01.

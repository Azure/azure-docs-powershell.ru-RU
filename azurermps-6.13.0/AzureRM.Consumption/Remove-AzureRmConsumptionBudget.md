# Remove-AzureRmConsumptionBudget

## КРАТКИй обзор
Удаление бюджета либо в подписке, либо в группе ресурсов.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

### Подписка (по умолчанию)
```
Remove-AzureRmConsumptionBudget [-DefaultProfile <IAzureContextContainer>] -Name <String>
 [-ResourceGroupName <String>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Трубодержателей
```
Remove-AzureRmConsumptionBudget [-DefaultProfile <IAzureContextContainer>] -InputObject <PSBudget> [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет Remove-AzureRmConsumptionBudget удаляет бюджет либо в подписке, либо в группе ресурсов.

## ИЛЛЮСТРИРУЮТ

### Пример 1: удаление бюджета с именем бюджета на уровне подписки
```powershell
PS C:\> Remove-AzureRmConsumptionBudget -Name PSBudget -PassThru
True
```

### Пример 2: удаление бюджета с именем бюджета на уровне группы ресурсов
```powershell
PS C:\> Remove-AzureRmConsumptionBudget -ResourceGroupName RGBudgets -Name PSBudgetRG -PassThru
True
```

### Пример 3: удаление бюджета с помощью трубопроводов на уровне подписки
```powershell
PS C:\> Get-AzureRmConsumptionBudget -Name PSBudget | Remove-AzureRmConsumptionBudget -PassThru
True
```

# Test-AzureRmDataLakeStoreAccount

## КРАТКИй обзор
Проверка наличия учетной записи Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Test-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Test-AzureRmDataLakeStoreAccount** проверяет наличие учетной записи Data Lake Store.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Проверка учетной записи
```
PS C:\>Test-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

Эта команда проверяет, существует ли учетная запись с именем ContosoADL.

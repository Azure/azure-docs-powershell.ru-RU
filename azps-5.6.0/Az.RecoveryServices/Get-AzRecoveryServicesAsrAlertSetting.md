# Get-AzRecoveryServicesAsrAlertSetting

## SYNOPSIS
Настраивает параметры уведомлений о восстановлении сайта Azure для хранилища.

## СИНТАКСИС

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **параметра Get-AzRecoveryServicesAsrAlertSetting** настраиваются параметры уведомлений восстановления сайта Azure для хранилища.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Настройка оповещений и уведомлений для восстановления сайта Azure.

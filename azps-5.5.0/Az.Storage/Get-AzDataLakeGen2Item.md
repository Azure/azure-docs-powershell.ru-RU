# Get-AzDataLakeGen2Item

## SYNOPSIS
Получает сведения о файле или каталоге в файловойсистеме.

## СИНТАКСИС

```
Get-AzDataLakeGen2Item [-FileSystem] <String> [-Path <String>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для получения сведений о файле или каталоге в Filesystem учетной записи хранилища Azure можно получить сведения о файле или каталоге **Get-AzDataLakeGen2Item.**
Этот cmdlet работает только в том случае, если для учетной записи хранения включено иерархическое пространство имен. Учетную запись такого типа можно создать с помощью cmdlet "New-AzStorageAccount" с помощью функции "-EnableHierarchicalNamespace $true".

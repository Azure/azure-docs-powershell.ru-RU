# Set-AzDtlAutoStartPolicy

## SYNOPSIS
Задает политику автоматического запуска лабораторных анализов DevTest Labs.

## СИНТАКСИС

### Включить (по умолчанию)
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### "Отключить"
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Для **этого используется** поле Auto Start policy для лабораторных виртуальных машин, которое можно запланизировать на автоматический запуск.
Для этого используется заданная группа ресурсов и название лабораторных работы.

## ПРИМЕРЫ

### Пример 1

Задает политику автоматического запуска лабораторных анализов DevTest Labs. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoStartPolicy -Days Sunday -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```

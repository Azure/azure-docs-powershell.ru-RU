# Get-AzSynapseNotebook

## SYNOPSIS
Получает сведения о записных книжках в рабочей области.

## СИНТАКСИС

### GetByName (по умолчанию)
```
Get-AzSynapseNotebook -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseNotebook -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Чтобы получить сведения о записных книжках в рабочей области, можно использовать **cmdletNotebook Get-AzSynapseNotebook.** Если указать имя записной книжки, он получит сведения о записной книжке. Если имя не указано, он получает сведения обо всех записных книжках в рабочей области.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzSynapseNotebook -WorkspaceName ContosoWorkspace | Format-Table

WorkspaceName    Properties                                         Name
-------------    ----------                                         --
ContosoWorkspace Microsoft.Azure.Commands.Synapse.Models.PSNotebook ContosoNotebook1
ContosoWorkspace Microsoft.Azure.Commands.Synapse.Models.PSNotebook ContosoNotebook2
```

Возвращает список всех записных книг в рабочей области ContosoWorkspace.

### Пример 2
```powershell
PS C:\> Get-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook
```

Получает одну записную книжку с названием ContosoNotebook в рабочей области ContosoWorkspace.

### Пример 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseNotebook -Name ContosoNotebook
```

Получает одну записную книжку с названием ContosoNotebook в рабочей области ContosoWorkspace через канал.

## ОПИСАНИЕ
Изменяет спецификацию Templace. Если спецификация шаблона с указанным именем или определенной версией еще не существует, она будет создана.

При изменении содержимого шаблона шаблона ARM спецификации шаблона его содержимое может быть либо из необработанных строк JSON (с помощью набора параметров **UpdateVersionByNameFromJsonParameterSet),** либо из указанного файла JSON (с помощью **параметра UpdateVersionByNameFromJsonFileParameterSet).**

## ПРИМЕРЫ

### Пример 1.
```powershell
PS C:\> $templateJson = @"
{
    "$schema": "http://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
    "contentVersion": "1.0.0.0",
    "parameters": {},
    "resources": []
}
"@
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v1.0' -Location 'West US' -TemplateJson $templateJson
```

Изменяет версию "v1.0" спецификации шаблона с именем "myTemplateSpec". В указанной версии $templateJson как содержимое шаблона ARM версии. Если корневой спецификации шаблона или версии еще не существует, они будут созданы.


**Примечания.** 

* Шаблон ARM в этом примере не имеет никакого никакого ресурса, так как он не содержит фактических ресурсов.
* Расположение является требоваться только в том случае, если шаблон спецификации еще не существует

### Пример 2.
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v2.0' -Location 'West US' -TemplateFile 'myTemplateContent.json'
```

Изменяет версию "v2.0" спецификации шаблона с именем "myTemplateSpec". В качестве содержимого шаблона для указанной версии будет указано содержимое локального myTemplateContent.js"ARM". Если корневой спецификации шаблона или версии еще не существует, они будут созданы.

**Примечание.** Расположение является требоваться только в том случае, если шаблон спецификации еще не существует

### Пример 3.
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec'  -Location 'West US' -Description 'My updated Template Spec'
```

Изменяет описание шаблона "MyTemplateSpec" в группе ресурсов myRG. Если спецификация шаблона еще не существует, она будет создана.

**Примечание.** Расположение является требоваться только в том случае, если шаблон спецификации еще не существует

# Get-AzureRmSqlSyncAgent

## КРАТКИй обзор
Возвращает сведения об агентах синхронизации Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureRmSqlSyncAgent** возвращает сведения об одном или нескольких агентах синхронизации Azure SQL.
Укажите имя агента синхронизации, чтобы просмотреть сведения только для этого агента синхронизации.

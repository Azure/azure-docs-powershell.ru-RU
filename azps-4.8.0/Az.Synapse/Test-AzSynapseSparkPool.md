# Test-AzSynapseSparkPool

## КРАТКИй обзор
Проверка существования пула Synapse Analytics Spark.

## Максимальное

### TestByNameParameterSet (по умолчанию)
```
Test-AzSynapseSparkPool [-ResourceGroupName <String>] -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByParentObjectParameterSet
```
Test-AzSynapseSparkPool -Name <String> -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Test-AzSynapseSparkPool** проверяет наличие пула Synapse Analytics Spark.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Test-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
```

Эта команда проверяет существование указанного пула Spark.

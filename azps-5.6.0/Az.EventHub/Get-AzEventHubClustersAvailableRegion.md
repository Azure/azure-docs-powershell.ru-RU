# Get-AzEventHubClustersAvailableRegion

## SYNOPSIS
Подробные сведения о кластере Естройка или списке кластеров в данной группе ресурсов.

## СИНТАКСИС

```
Get-AzEventHubClustersAvailableRegion [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Список Get-AzEventHubClustersAvailableRegion, в которых можно создать выделенные области.

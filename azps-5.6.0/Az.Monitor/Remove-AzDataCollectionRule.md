# Remove-AzDataCollectionRule

## SYNOPSIS
Удаление правила сбора данных.

## СИНТАКСИС

### ByName (по умолчанию)
```
Remove-AzDataCollectionRule
   -ResourceGroupName <string> 
   -RuleName <string> 
   [-PassThru]
   [-DefaultProfile <IAzureContextContainer>]
   [-WhatIf]
   [-Confirm]
   [<CommonParameters>]
```

### ByInputObject
```
Remove-AzDataCollectionRule
   -InputObject <PSDataCollectionRuleResource>
   [-PassThru]
   [-DefaultProfile <IAzureContextContainer>]
   [-WhatIf]
   [-Confirm]
   [<CommonParameters>]
```

### ByResourceId
```
Remove-AzDataCollectionRule
   -RuleId <string>
   [-PassThru]
   [-DefaultProfile <IAzureContextContainer>]
   [-WhatIf]
   [-Confirm]
   [<CommonParameters>]
```

## ОПИСАНИЕ
Чтобы удалить правило сбора данных, удалите правило сбора данных **Скайп-AzDataCollectionRule.**

Правила сбора данных (DCR) определяют данные, которые в нее в нее вслали, и определяли, куда они должны быть отправлены или сохранены. Вот полная статья [с обзором DCR.](https://docs.microsoft.com/azure/azure-monitor/platform/data-collection-rule-overview)

## ПРИМЕРЫ

### Пример 1. Удаление правила сбора данных с параметрами имени и группы ресурсов
```
PS C:\>Remove-AzDataCollectionRule -ResourceGroupName "testgroup" -RuleName "testDcr"             
```

### Пример 2. Удаление правила сбора данных с названием и возвращаемой группой ресурсов
```
PS C:\>Remove-AzDataCollectionRule -ResourceGroupName "testgroup" -RuleName "testDcr" -PassThru

True
```

### Пример 3. Удаление правила сбора данных из InputObject
```
PS C:\>Get-AzDataCollectionRule -ResourceGroupName "testdcr" -RuleName "dcrFromPipe95" | Remove-AzDataCollectionRule
```

### Пример 4. Удаление правила сбора данных из ид ресурса
```
PS C:\>Remove-AzDataCollectionRule -RuleId "/subscriptions/{subId}/resourceGroups/testdcr/providers/Microsoft.Insights/dataCollectionRules/{dcrName}"
```

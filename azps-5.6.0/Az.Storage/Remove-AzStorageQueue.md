# Remove-AzStorageQueue

## SYNOPSIS
Удаляет очередь хранилища.

## СИНТАКСИС

```
Remove-AzStorageQueue [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для удаления очереди хранилища **удаляется cmdlet Remove-AzStorageQueue.**

## ПРИМЕРЫ

### Пример 1. Удаление очереди хранилища по имени
```
PS C:\>Remove-AzStorageQueue "ContosoQueue01"
```

Эта команда удаляет очередь с именем ContosoQueue01.

### Пример 2. Удаление нескольких очередей хранилища
```
PS C:\>Get-AzStorageQueue "Contoso*" | Remove-AzStorageQueue
```

Эта команда удаляет все очереди с именами, имена которых начинаются с Contoso.

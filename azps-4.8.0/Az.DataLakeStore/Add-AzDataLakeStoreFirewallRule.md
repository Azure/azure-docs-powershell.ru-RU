# Add-AzDataLakeStoreFirewallRule

## КРАТКИй обзор
Добавляет правило брандмауэра в указанную учетную запись Data Lake Store.

## Максимальное

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Add-AzDataLakeStoreFirewallRule** добавляет правило межсетевого экрана для указанной учетной записи Data Lake Store.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Добавление нового правила межсетевого экрана для учетной записи Data Lake Store
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

В результате создается новое правило брандмауэра с именем "MyRule" в учетной записи "ContosoADL" с IP-диапазоном 127.0.0.1-127.0.0.2

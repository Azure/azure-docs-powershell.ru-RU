# Add-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Добавляет пул конечных адресов в шлюз приложения.

## СИНТАКСИС

```
Add-AzApplicationGatewayBackendAddressPool -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-BackendIPAddresses <String[]>] [-BackendFqdns <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Add-AzApplicationGatewayBackendAddressPool** можно добавить пул адресов на конечный адрес в шлюз приложения.
Для этого используется IP-адрес, полное доменное имя (FQDN) или ИД конфигурации IP-адреса.

## ПРИМЕРЫ

### Пример 1. Добавление серверного пула адресов с использованием серверного FQDN
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "Pool02" -BackendFqdns "contoso1.com", " contoso1.com"
```

Первая команда получает шлюз приложения ApplicationGateway01 в группе ресурсов ResourceGroup01 и сохраняет его в $AppGw переменной. Вторая команда добавляет пул конечных адресов шлюза приложения, который хранится в $AppGw с помощью FQDNs.

### Пример 2. Добавление пула серверных адресов с помощью IP-адресов сервера серверов
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayBackendAddressPool -ApplicationGateway $ AppGw -Name "Pool02" -BackendIPAddresses "10.10.10.10", "10.10.10.11"
```

Первая команда получает шлюз приложения ApplicationGateway01 в группе ресурсов ResourceGroup01 и сохраняет его в $AppGw переменной. Вторая команда добавляет пул конечных адресов шлюза приложения, который хранится в $AppGw с помощью IP-адресов.

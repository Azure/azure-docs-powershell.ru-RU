# New-AzureRmStreamAnalyticsTransformation

## КРАТКИй обзор
Создает или обновляет преобразование в задании.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
New-AzureRmStreamAnalyticsTransformation [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **New-AzureRmStreamAnalyticsTransformation** создает преобразование в рамках задания Stream Analytics или обновляет существующее преобразование.
Имя преобразования можно задать в поле. Файл JSON или в командной строке.
Если заданы оба флажка, имя в командной строке должно совпадать с именем в файле.

Если вы задаете уже существующее и не указали параметр Force, командлет запрашивает необходимость замены существующего преобразования.

Если указать параметр *Force* и задать существующее имя преобразования, преобразование будет заменено без подтверждения.

## ИЛЛЮСТРИРУЮТ

### Пример 1: создание или замена преобразования в задании
```
PS C:\>New-AzureRmStreamAnalyticsTransformation -ResourceGroupName "StreamAnalytics-Default-West-US" -File "C:\Transformation.json" -JobName "StreamingJob" -Name "StreamingJobTransform"
```

Эта команда создает преобразование с именем StreamingJobTransform в задании, именуемом StreamingJob.
Если существующее преобразование уже определено с этим именем, командлет задаст вопрос, нужно ли его заменить.

### Пример 2: замена преобразования в задании
```
PS C:\>New-AzureRmStreamAnalyticsTransformation -ResourceGroupName "StreamAnalytics-Default-West-US" -File "C:\Transformation.json" -JobName "StreamingJob" -Name "StreamingJobTransform" -Force
```

Эта команда заменяет определение StreamingJobTransform в задании StreamingJob без подтверждения.

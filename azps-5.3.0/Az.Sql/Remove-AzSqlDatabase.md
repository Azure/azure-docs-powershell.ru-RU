# Remove-AzSqlDatabase

## SYNOPSIS
Удаляет базу данных Azure SQL.

## СИНТАКСИС

```
Remove-AzSqlDatabase [-DatabaseName] <String> [-Force] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При этом удаляется база данных Azure SQL **AzSqlData.**
Этот cmdlet также поддерживается службой SQL Server Stretch Database в Azure.

## ПРИМЕРЫ

### Пример 1. Удаление базы данных с сервера Azure SQL
```
PS C:\>Remove-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Эта команда удаляет базу данных с именем Database01 с сервера Server01.

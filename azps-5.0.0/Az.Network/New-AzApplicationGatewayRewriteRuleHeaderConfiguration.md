# New-AzApplicationGatewayRewriteRuleHeaderConfiguration

## КРАТКИй обзор
Создает конфигурацию заголовка правила перезаписи для шлюза приложения.

## Максимальное

```
New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName <String> [-HeaderValue <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **AzApplicationGatewayRewriteRuleHeaderConfiguration** создает набор действий правила перезаписи для шлюза приложений Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> $hc = New-AzApplicationGatewayRewriteRuleHeaderConfiguration -HeaderName abc -HeaderValue def
```

Эта команда создает конфигурацию заголовка правила перезаписи и сохраняет результат в переменной с именем $hc.

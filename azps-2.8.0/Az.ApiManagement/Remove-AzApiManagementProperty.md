# Remove-AzApiManagementProperty

## КРАТКИй обзор
Удаляет свойство управления API.

## Максимальное

```
Remove-AzApiManagementProperty -Context <PsApiManagementContext> -PropertyId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Remove-AzApiManagementProperty** удаляет **свойство** управления API Azure.

## ИЛЛЮСТРИРУЮТ

### Пример 1: Удаление свойства
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementProperty -Context $apimContext -PropertyId "Property11" -PassThru
```

Эта команда удаляет свойство с ИДЕНТИФИКАТОРом Property11.

# Get-AzApplicationGatewaySslPredefinedPolicy

## SYNOPSIS
Получает предопределять политики SSL, предоставляемые шлюзом приложения.

## СИНТАКСИС

```
Get-AzApplicationGatewaySslPredefinedPolicy [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzApplicationGatewaySslPredefinedPolicy** можно получить предопределять политики SSL, предоставляемые шлюзом приложения.

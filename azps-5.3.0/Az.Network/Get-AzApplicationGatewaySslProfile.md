# Get-AzApplicationGatewaySslProfile

## SYNOPSIS
Возвращает SSL-профиль шлюза приложения.

## СИНТАКСИС

```
Get-AzApplicationGatewaySslProfile [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Get-AzApplicationGatewaySslProfile** получает профиль SSL шлюза приложения.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $profile  = Get-AzApplicationGatewaySslProfile -Name "SslProfile01" -ApplicationGateway $AppGw
```

Первая команда получает шлюз приложения ApplicationGateway01 в группе ресурсов ResourceGroup01 и сохраняет его в $AppGw переменной. Вторая команда получает профиль SSL SslProfile01 для $AppGw и сохраняет его в $profile переменной.

# Get-AzApiManagementGateway

## КРАТКИй обзор
Возвращает все или конкретные шлюзы управления API.

## Максимальное

### GetAllGateways (по умолчанию)
```
Get-AzApiManagementGateway -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByGatewayId
```
Get-AzApiManagementGateway -Context <PsApiManagementContext> -GatewayId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzApiManagementGateway** получает все или определенные шлюзы управления API.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение всех шлюзов
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGateway -Context $apimContext
```

Эта команда получает все шлюзы.

### Пример 2: получение шлюза с помощью идентификатора
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGateway -Context $apimContext -GatewayId "0123456789"
```

Эта команда возвращает шлюз 0123456789.

# Get-AzServiceEndpointPolicyDefinition

## КРАТКИй обзор
Получает определение политики конечной точки службы.

## Максимальное

```
Get-AzServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzServiceEndpointPolicyDefinition** получает определение политики конечной точки службы.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
$policydef= Get-AzServiceEndpointPolicyDefinition -Name "ServiceEndpointPolicyDefinition1" -ServiceEndpointPolicy $Policy
```

Эта команда получает определение политики конечной точки службы с именем ServiceEndpointPolicyDefinition1 в ServiceEndpointPolicy $Policy сохраняет его в переменной $policydef.

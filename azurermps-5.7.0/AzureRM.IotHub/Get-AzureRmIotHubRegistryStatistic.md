# Get-AzureRmIotHubRegistryStatistic

## КРАТКИй обзор
Возвращает RegistryStatistics для IotHub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Get-AzureRmIotHubRegistryStatistic [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Возвращает RegistryStatistics для IotHub.
Это предоставляет сведения о количестве включенных и отключенных устройств в IotHub.

## ИЛЛЮСТРИРУЮТ

### Пример 1. Скачайте RegistryStatistics
```
PS C:\> Get-AzureRmIotHubRegistryStatistic -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Возвращает RegistryStatictics для IotHub с именем "myiothub"

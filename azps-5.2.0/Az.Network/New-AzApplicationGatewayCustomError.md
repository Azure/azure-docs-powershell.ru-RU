# New-AzApplicationGatewayCustomError

## SYNOPSIS
Создает настраиваемую ошибку с кодом состояния http и URL-адресом страницы настраиваемой ошибки 

## СИНТАКСИС

```
New-AzApplicationGatewayCustomError -StatusCode <String> -CustomErrorPageUrl <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для создания пользовательской ошибки будет создаваться новый **cmdlet New-AzApplicationGatewayCustomError.**

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $customError403Url = "https://mycustomerrorpages.blob.core.windows.net/errorpages/403-another.htm"
PS C:\> $ce = New-AzApplicationGatewayCustomError -StatusCode HttpStatus403 -CustomErrorPageUrl $customError403Url
```

Эта команда создает настраиваемую ошибку http-кода состояния 403.

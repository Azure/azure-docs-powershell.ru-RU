# Get-AzUserAssignedIdentity

## КРАТКИй обзор
Возвращает пользователей, которым назначены идентификаторы и удостоверения.

## Максимальное

### SuscriptionParameterSet (по умолчанию)
```
Get-AzUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
**Get-AzUserAssignedIdentity** получает идентификаторы существующих пользователей, которым они назначены.

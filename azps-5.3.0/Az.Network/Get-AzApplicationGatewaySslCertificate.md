# Get-AzApplicationGatewaySslCertificate

## SYNOPSIS
Получает SSL-сертификат для шлюза приложения.

## СИНТАКСИС

```
Get-AzApplicationGatewaySslCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Get-AzApplicationGatewaySslCertificate** получает SSL-сертификат для шлюза приложения.

## ПРИМЕРЫ

### Пример 1. Получить определенный SSL-сертификат
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Cert = Get-AzApplicationGatewaySslCertificate -Name "Cert01" -ApplicationGateway $AppGW
```

Первая команда получает шлюз приложения ApplicationGateway01 и сохраняет результат в переменной с $AppGW.
Вторая команда получает SSL-сертификат с именем Cert01 из шлюза приложения, храняного в переменной $AppGW.
Команда хранит сертификат в переменной с именем $Cert.

### Пример 2. Получить список SSL-сертификатов
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Certs = Get-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW
```

Первая команда получает шлюз приложения ApplicationGateway01 и сохраняет результат в переменной с $AppGW.
Вторая команда получает список SSL-сертификатов из шлюза приложения, храняного в переменной с $AppGW.
Затем команда сохраняет результаты в переменной с именем $Certs.

# Get-AzSynapseIntegrationRuntime

## SYNOPSIS
Сведения о ресурсах интеграции и времени работы.

## СИНТАКСИС

### GetByNameParameterSet (по умолчанию)
```
Get-AzSynapseIntegrationRuntime [-ResourceGroupName <String>] -WorkspaceName <String> [-Name <String>]
 [-Status] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseIntegrationRuntime [-Name <String>] -WorkspaceObject <PSSynapseWorkspace> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseIntegrationRuntime -ResourceId <String> [-Status] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByInputObjectParameterSet
```
Get-AzSynapseIntegrationRuntime -InputObject <PSIntegrationRuntime> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Get-AzSynapseIntegrationRuntime** получает сведения о времени интеграции в рабочей области.
Если указать имя времени запуска интеграции, этот cmdlet получит сведения об этом времени запуска интеграции.
Если имя не указано, этот cmdlet получает сведения обо всех процессах интеграции в рабочей области.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> Get-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace
```

Перечислите все время интеграции в рабочей области ContosoWorkspace.

### Пример 2
```powershell
PS C:\> Get-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir'
```

Эта команда отображает сведения об интегрированной рабочей области с именем Test-selfhost-ir в рабочей области ContosoWorkspace.

### Пример 3
```powershell
PS C:\> Get-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' -Status
```

Эта команда отображает подробные состояния интеграции с именем test-selfhost-ir в рабочей области ContosoWorkspace.

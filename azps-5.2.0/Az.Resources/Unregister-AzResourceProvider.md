# Unregister-AzResourceProvider

## SYNOPSIS
Отрегистрет поставщика ресурсов.

## СИНТАКСИС

```
Unregister-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Из-за cmdister-AzResourceProvider** поставщик ресурсов Azure будет отсутствован.

## ПРИМЕРЫ

### Пример 1. Unregister resource provider with ProviderNamespace

```powershell
PS C:\>Unregister-AzResourceProvider -ProviderNamespace "Microsoft.support"
```

Эта команда отвязкает поставщика ресурсов "Microsoft.support".

# Remove-AzVMAccessExtension

## SYNOPSIS
Удаляет расширение VMAccess с виртуальной машины.

## СИНТАКСИС

```
Remove-AzVMAccessExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
При **этом с** виртуальной машины удаляется расширение виртуальной машины (VMAccess).

## ПРИМЕРЫ

### Пример 1

Удаляет расширение VMAccess с виртуальной машины. (autogenerated)

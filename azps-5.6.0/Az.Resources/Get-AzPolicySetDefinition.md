# Get-AzPolicySetDefinition

## SYNOPSIS
Получает определения набора политик.

## СИНТАКСИС

### NameParameterSet (по умолчанию)
```
Get-AzPolicySetDefinition [-Name <String>] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ManagementGroupNameParameterSet
```
Get-AzPolicySetDefinition [-Name <String>] -ManagementGroupName <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionIdParameterSet
```
Get-AzPolicySetDefinition [-Name <String>] -SubscriptionId <Guid> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdParameterSet
```
Get-AzPolicySetDefinition -Id <String> [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### BuiltinFilterParameterSet
```
Get-AzPolicySetDefinition [-ManagementGroupName <String>] [-SubscriptionId <Guid>] [-Builtin]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### CustomFilterParameterSet
```
Get-AzPolicySetDefinition [-ManagementGroupName <String>] [-SubscriptionId <Guid>] [-Custom]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzPolicySetDefinition** можно получить набор определений набора политик или определение, задаваемого по имени или коду.

## ПРИМЕРЫ

### Пример 1. Получить все определения набора политик
```
PS C:\> Get-AzPolicySetDefinition
```

Эта команда получает все определения набора политик.

### Пример 2. Определение набора политик из текущей подписки по имени
```
PS C:\> Get-AzPolicySetDefinition -Name 'VMPolicySetDefinition'
```

Эта команда получает определение набора политик VMPolicySetDefinition из текущей подписки по умолчанию.

### Пример 3. Определение набора политик из подписки по имени
```
PS C:\> Get-AzPolicySetDefinition -Name 'VMPolicySetDefinition' -subscriptionId '3bf44b72-c631-427a-b8c8-53e2595398ca'
```

Эта команда получает определение политики VMPolicySetDefinition из подписки с ИД 3bf44b72-c631-427a-b8c8-53e2595398ca.

### Пример 4. Получить все определения настраиваемого набора политик из группы управления
```
PS C:\> Get-AzPolicySetDefinition -ManagementGroupName 'Dept42' -Custom
```

Эта команда получает все определения настраиваемого набора политик из группы управления "Отдел42".

### Пример 5. Получать определения набора политик из данной категории
```
PS C:\> Get-AzPolicySetDefinition | where-object {$_.Properties.metadata.category -eq "Virtual Machine"}
```

Эта команда получает все определения набора политик в категории "Виртуальная машинка".

# Enable-AzHDInsightMonitoring

## SYNOPSIS
Включает мониторинг в кластере HDInsight, а соответствующие журналы отправляются в рабочее пространство мониторинга, заданное во время его работы.

## СИНТАКСИС

```
Enable-AzHDInsightMonitoring [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Enable-AzHDInsightMonitoring** можно вести мониторинг в кластере Azure HDInsight.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

Мониторинг будет включен на кластере HDInsight, а соответствующие журналы будут отправлены в рабочее пространство мониторинга с id 1d364e89-bb71-4503-aa3d-a23535aea7bd

### Пример 2
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

Мониторинг будет включен в кластере HDInsight, а соответствующие журналы будут отправлены в рабочее пространство мониторинга с id 1d364e89-bb71-4503-aa3d-a23535aea7bd

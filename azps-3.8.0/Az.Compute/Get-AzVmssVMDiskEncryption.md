# Get-AzVmssVMDiskEncryption

## КРАТКИй обзор
Показывает состояние шифрования диска для виртуальных машин в наборе масштабирования виртуальных машин.

## Максимальное

```
Get-AzVmssVMDiskEncryption [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String>]
 [-ExtensionName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Показывает состояние шифрования диска для набора шкал виртуальных машин.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Get-AzVmssVMDiskEncryption -ResourceGroupName "Group001" -VMScaleSetName "VMSS001" -InstanceId "1"
```

Показывает состояние шифрования диска для экземпляра VM 1 в наборе масштабирования ВИРТУАЛЬНЫХ машин с именем VMSS001, который принадлежит группе ресурсов с именем Group001.

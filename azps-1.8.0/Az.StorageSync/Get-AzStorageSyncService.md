# Get-AzStorageSyncService

## КРАТКИй обзор
Эта команда выводит список всех служб синхронизации хранилища в заданной области подписки или группы ресурсов.

## Максимальное

### ParentStringParameterSet (по умолчанию)
```
Get-AzStorageSyncService [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### StringParameterSet
```
Get-AzStorageSyncService [-ResourceGroupName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Эта команда выводит список всех служб синхронизации хранилища в заданной области подписки или группы ресурсов. Он также может использоваться для перечисления атрибутов каждой службы синхронизации хранилища.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```powershell
PS C:\> Get-AzStorageSyncService -ResourceGroupName "myResourceGroup"
```

Эта команда выводит список всех ресурсов службы синхронизации хранилища в заданной группе ресурсов. Он также может использоваться для перечисления атрибутов каждой службы синхронизации хранилища. Укажите-StorageSyncServiceName, чтобы вернуть определенную единицу.

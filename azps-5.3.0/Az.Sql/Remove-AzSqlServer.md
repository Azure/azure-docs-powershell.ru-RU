# Remove-AzSqlServer

## SYNOPSIS
Удаляет сервер базы данных Azure SQL.

## СИНТАКСИС

```
Remove-AzSqlServer [-ServerName] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
С его использованием удаляется сервер базы данных Azure SQL **AzSqlServer.**
Операция удаления является асинхронной и может занять некоторое время, поэтому прежде чем выполнять дополнительные операции, которые зависят от удаления сервера, убедитесь, что операция завершена.
Например, вам нужно создать сервер с таким же именем.

## ПРИМЕРЫ

### Пример 1. Удаление сервера
```
PS C:\>Remove-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Эта команда удаляет сервер базы данных Azure SQL с именем Server01.

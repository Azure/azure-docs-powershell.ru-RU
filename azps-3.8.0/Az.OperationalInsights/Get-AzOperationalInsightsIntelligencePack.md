# Get-AzOperationalInsightsIntelligencePack

## КРАТКИй обзор
Получает доступные пакеты поддержки интеллектуального анализа.

## Максимальное

```
Get-AzOperationalInsightsIntelligencePack [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzOperationalInsightsIntelligencePack** получает доступные пакеты поддержки интеллектуального анализа.

## ИЛЛЮСТРИРУЮТ

### Пример 1: получение пакетов для аналитики
```
PS C:\>Get-AzOperationalInsightsStorageInsight -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace"
```

Эта команда получает доступные пакеты поддержки интеллектуального анализа.

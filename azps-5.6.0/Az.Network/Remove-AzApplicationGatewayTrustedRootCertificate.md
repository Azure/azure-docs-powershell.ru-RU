# Remove-AzApplicationGatewayTrustedRootCertificate

## SYNOPSIS
Удаляет доверенный корневой сертификат из шлюза приложения.

## СИНТАКСИС

```
Remove-AzApplicationGatewayTrustedRootCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Remove-AzApplicationGateWayTrustedRootCertificate** удаляет доверенный корневой сертификат из существующего шлюза приложения.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name "myRootCA"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Первая команда получает шлюз приложения и сохраняет его в переменной $gw.
Вторая команда удаляет доверенный корневой сертификат myRootCA из шлюза приложения, $gw.
Третья команда обновляет шлюз приложения в Azure.

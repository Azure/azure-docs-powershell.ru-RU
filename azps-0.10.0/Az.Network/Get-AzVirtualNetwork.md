# Get-AzVirtualNetwork

## КРАТКИй обзор
Возвращает виртуальную сеть в группе ресурсов.

## Максимальное

### NOEXPAND
```
Get-AzVirtualNetwork [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Кройте
```
Get-AzVirtualNetwork -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzVirtualNetwork** получает одну или несколько виртуальных сетей n группы ресурсов.

## ИЛЛЮСТРИРУЮТ

### 1: получение виртуальной сети
```
Get-AzVirtualNetwork -Name MyVirtualNetwork -ResourceGroupName TestResourceGroup
```

Эта команда возвращает виртуальную сеть с именем MyVirtualNetwork в группе ресурсов TestResourceGroup

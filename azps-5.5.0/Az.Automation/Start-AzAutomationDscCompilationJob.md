# Start-AzAutomationDscCompilationJob

## SYNOPSIS
Компилирует конфигурацию DSC в автоматизации.

## СИНТАКСИС

```
Start-AzAutomationDscCompilationJob [-ConfigurationName] <String> [-Parameters <IDictionary>]
 [-ConfigurationData <IDictionary>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-IncrementNodeConfigurationBuild] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ОПИСАНИЕ
Cmdlet **Start-AzAutomationDscCompilationJob** компилирует конфигурацию APS Desired State Configuration (DSC) в автоматизации Azure.

## ПРИМЕРЫ

### Пример 1. Компиляция конфигурации azure DSC в автоматизации
```
PS C:\>$Params = @{"StringParam"="Hello World";"IntegerParam"=32}
PS C:\> Start-AzAutomationDscCompilationJob -ConfigurationName "Config01" -Parameters $Params -ResourceGroupName "ResourceGroup01"
```

Первая команда создает словарь параметров и сохраняет их в переменной $Params параметрами.
Вторая команда скомпилирует конфигурацию DSC с именем Config01.
Она содержит значения в параметрах конфигурации $Params DSC.

### Пример 2. Компилировать конфигурацию Azure DSC в автоматизации с помощью новой версии сборки конфигурации узла.
```
PS C:\>$Params = @{"StringParam"="Hello World";"IntegerParam"=32}
PS C:\> Start-AzAutomationDscCompilationJob -ConfigurationName "Config01" -Parameters $Params -ResourceGroupName "ResourceGroup01" -IncrementNodeConfigurationBuild
```

Как и в первом примере, первая команда создает словарь параметров и сохраняет их в переменной $Params параметрами.
Вторая команда скомпилирует конфигурацию DSC с именем Config01.
Она содержит значения в параметрах конфигурации $Params DSC.
Она не переопределяет существующую конфигурацию узла путем создания новой конфигурации узла с именем Config01[<2 <NodeName>>]. Номер версии будет приращен с приращением в зависимости от существующего номера версии.

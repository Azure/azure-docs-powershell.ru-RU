# Get-AzServiceBusTopic

## SYNOPSIS
Возвращает описание указанного раздела "Автобусы обслуживания".

## СИНТАКСИС

```
Get-AzServiceBusTopic [-ResourceGroupName <String>] [-Namespace <String>] [-Name <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для **указанного пространства имен в автобусе** с обслуживанием возвращается описание темы с его описанием.


Возвращает описание указанного раздела для указанного пространства имен автобусов обслуживания.

### Пример 2
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Возвращает список разделов для данного пространства имен в автобусе обслуживания. По умолчанию возвращается 100 тем, если возвращается более 100 тем, используйте параметр -MaxCount.

### Пример 3
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Возвращает список из первых 150 разделов, задав для данного пространства имен автобусов обслуживания.

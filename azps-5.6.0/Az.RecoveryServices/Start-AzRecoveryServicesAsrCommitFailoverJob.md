# Start-AzRecoveryServicesAsrCommitFailoverJob

## SYNOPSIS
Запускает отбой для объекта восстановления сайта.

## СИНТАКСИС

### ByRPIObject (по умолчанию)
```
Start-AzRecoveryServicesAsrCommitFailoverJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByRPObject
```
Start-AzRecoveryServicesAsrCommitFailoverJob -RecoveryPlan <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
Для объекта восстановления сайта Azure после отбойной операции запускается процесс сбойной процедуры **Start-AzRecoveryServicesAsrCommitFailoverJob.**

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> $currentJob = Start-AzRecoveryServicesAsrCommitFailoverJob -RecoveryPlan $RP
```

Запускает отбой для указанного плана восстановления и возвращает задание ASR, используемую для отслеживания операции.

### Пример 2

Запускает отбой для объекта восстановления сайта. (autogenerated)


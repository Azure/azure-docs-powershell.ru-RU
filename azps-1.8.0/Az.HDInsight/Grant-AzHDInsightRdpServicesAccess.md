# Grant-AzHDInsightRdpServicesAccess

## КРАТКИй обзор
Предоставление доступа к кластеру Windows для подключения к удаленному рабочему столу.

## Максимальное

```
Grant-AzHDInsightRdpServicesAccess [-ClusterName] <String> [-RdpCredential] <PSCredential>
 [-RdpAccessExpiry] <DateTime> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Grant-AzHDInsightRdpServicesAccess** включает протокол удаленного рабочего стола (RDP) для доступа к кластеру Azure HDInsight на базе Windows.

## ИЛЛЮСТРИРУЮТ

### Пример 1: предоставление доступа к удаленному рабочему столу в кластере HDInsight Azure
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzHDInsightRdpServicesAccess `
            -ClusterName $clusterName `
            -RdpCredential $newRdpCreds `
            -RdpAccessExpiry $newRdpExpiry
```

Эта команда предоставляет доступ к кластеру через RDP с именем-Hadoop-001.

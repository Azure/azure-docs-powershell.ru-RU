# Get-AzIntegrationAccountMap

## SYNOPSIS
Возвращает интеграцию с картой учетной записи.

## СИНТАКСИС

```
Get-AzIntegrationAccountMap [-ResourceGroupName <String>] [-Name <String>] [-MapName <String>]
 [-MapType <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для интеграции с группой ресурсов возвращается карта учетной записи **Get-AzIntegrationAccountMap.**
Укажите имя учетной записи интеграции, имя группы ресурсов и имя карты.
Этот модуль поддерживает динамические параметры.
Чтобы использовать динамический параметр, введите его в команду.
Чтобы найти имена динамических параметров, введите дефис (-) после имени cmdlet, а затем нажимая клавишу TAB для циклическиго перебора доступных параметров.
Если не задан параметр обязательного шаблона, будет выдан запрос на его значение.

# Get-AzLoadBalancerInboundNatPoolConfig

## SYNOPSIS
Получает одну или несколько конфигураций пула NAT для входящий остаток на балансе нагрузки.

## СИНТАКСИС

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
Для этого вы можете получить одну или несколько конфигураций пула NAT для входящий пула **Get-AzLoadBalancerInboundNatPoolConfig.**

## ПРИМЕРЫ

### 1. Получить
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```

# Revoke-AzSnapshotAccess

## SYNOPSIS
Отзыв доступа к моментального снимку.

## СИНТАКСИС

```
Revoke-AzSnapshotAccess [-ResourceGroupName] <String> [-SnapshotName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Revoke-AzSnapshotAccess** отзывает доступ к снимку.

## ПРИМЕРЫ

### Пример 1
```
PS C:\> Revoke-AzSnapshotAccess -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01'
```

Отощут доступ к моментального снимку с именем "Моментальный снимок01" в группе ресурсов "Группа ресурсов01".

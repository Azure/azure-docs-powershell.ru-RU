# Get-AzCognitiveServicesAccountSku

## SYNOPSIS
Возвращает доступные skus для учетной записи.

## СИНТАКСИС

```
Get-AzCognitiveServicesAccountSku [-Type <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
С **помощью cmdlet Get-AzCognitiveServicesAccountSku** можно получить доступную коды skKu для учетной записи Службы когнитивных функций.
SKU — это план уровня для учетной записи.
В нем определяются цена, ограничение и ставка для счета.
F0 SKU — это бесплатный уровень.
Платные уровни: S0, S1, S2 и другие.

## ПРИМЕРЫ

### Пример 1
```powershell
PS C:\> (Get-AzCognitiveServicesAccountSku -Type 'TextAnalytics' -Location "westus").Value | Select-Object -E
xpandProperty Sku;

Name     Tier
----     ----
F0       Free
S0   Standard
```

# Remove-AzureRmEventHubNamespaceAuthorizationRule

## КРАТКИй обзор
Удаляет указанное правило авторизации на заданное пространство имен концентраторов событий.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Максимальное

```
Remove-AzureRmEventHubNamespaceAuthorizationRule [-ResourceGroupName] <String> [-NamespaceName] <String>
 [-AuthorizationRuleName] <String> [-WhatIf] [-Confirm]
```

## NОПИСАНИЕ
Командлет Remove-AzureRmEventHubNamespaceAuthorizationRule удаляет и удаляет указанное правило авторизации на заданное пространство имен концентраторов событий.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\> Remove-AzureRmEventHubNamespaceAuthorizationRule -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName -AuthorizationRuleName MyAuthRuleName
```

Удаляет правило авторизации \` MyAuthRuleName \` из пространства имен \` MyNamespaceName \` .

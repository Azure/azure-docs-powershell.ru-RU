# Get-AzureVirtualNetworkGatewayDiagnostics

## КРАТКИй обзор
Получает результаты диагностики шлюза виртуальной сети Azure.

## Максимальное

```
Get-AzureVirtualNetworkGatewayDiagnostics -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Get-AzureVirtualNetworkGatewayDiagnostics** получает результаты диагностики шлюза виртуальной сети Azure.

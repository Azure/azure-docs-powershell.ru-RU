# Stop-AzManagementGroupDeployment

## КРАТКИй обзор
Отмена выполняющегося развертывания в группе "Управление"

## Максимальное

### StopByDeploymentName (по умолчанию)
```
Stop-AzManagementGroupDeployment [-ManagementGroupId] <String> [-Name] <String> [-PassThru]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzManagementGroupDeployment -Id <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzManagementGroupDeployment -InputObject <PSDeployment> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## NОПИСАНИЕ
Командлет **Stop-AzManagementGroupDeployment** отменяет развертывание, которое было начато, но не завершено в группе управления.
Чтобы остановить развертывание, необходимо, чтобы развертывание имело неполное состояние подготовки, такое как подготовка, а не завершенное состояние, например подготовка или сбой.

Чтобы создать развертывание в группе управления, используйте командлет New-AzManagementGroupDeployment.

## ИЛЛЮСТРИРУЮТ

### Пример 1
```
PS C:\>Stop-AzManagementGroupDeployment -ManagementGroupId "myMG" -Name "deployment01"
```

Эта команда отменяет запущенное развертывание "deployment01" в группе управления "myMG".

### Пример 2
```
PS C:\>Get-AzManagementGroupDeployment -ManagementGroupId "myMG" -Name "deployment01" | Stop-AzManagementGroupDeployment
```

Эта команда получает развертывание "deployment01" в группе управления "myMG" и отменяет ее.

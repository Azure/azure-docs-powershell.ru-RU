# Get-AzLogicAppUpgradedDefinition

## SYNOPSIS
Возвращает обновленное определение для приложения логики.

## СИНТАКСИС

```
Get-AzLogicAppUpgradedDefinition -ResourceGroupName <String> -Name <String> -TargetSchemaVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## ОПИСАНИЕ
**Cmdlet Get-AzLogicAppUpgradedDefinition** получает обновленное определение для версии схемы и приложения логики из группы ресурсов.
Этот cmdlet возвращает объект, который представляет определение обновленного приложения логики.
Укажите имя группы ресурсов, имя приложения логики и версию целевой схемы.
Этот модуль поддерживает динамические параметры.
Чтобы использовать динамический параметр, введите его в команду.
Чтобы найти имена динамических параметров, введите дефис (-) после имени cmdlet, а затем нажимая клавишу TAB для циклическиго перебора доступных параметров.
Если параметр обязательного шаблона не задан, будет выдан запрос на его значение.
